    Subsection 3.5.2 The Cancellation Theorem

    In the last section, we saw that some limits at points outside the domain of a function become infinite when they have the form \(\frac{1}{0}\text{.}\) However, we also saw that some functions can have limits of the form \(\mathbf{\frac{0}{0}}\) at points that are not in their domains. In this section, we learn one way to calculate such limits: algebraic manipulation and cancellation. As we shall see, such limits are sometimes infinite and are sometimes finite.
    ðŸ”—

    Example 3.5.1.

    For example, consider the limit
    \begin{equation*} \lim_{x\to 1}\dfrac{x^2-1}{x-1}\text{.} \end{equation*}
    At \(x=1\text{,}\) the numerator is \((1)^2-1 = 0\) and the denominator is \((1)-1=0\text{,}\) and therefore the limit evaluates to \(\frac{0}{0}\) if we try to plug in \(x=1\text{:.}\)
    \begin{equation*} \lim_{x\to 1}\dfrac{x^2-1}{x-1} = \frac{(1)^2-1}{(1)-1}=\frac{0}{0} \end{equation*}
    ðŸ”—
    As a function, \(\frac{0}{0}\) is undefined. But remember that for limits, we wish to know the behavior near such points, so are not satisfied until we know precisely what happens.
    ðŸ”—
    Fortunately for us in this example, we can determine the limit by factoring the numerator and cancelling a common factor of \(x-1\) in the numerator and denominator:
    \begin{equation*} \frac{x^2-1}{x-1} =\frac{(x-1)(x+1)}{x-1} = \begin{cases}x+1,\amp \text{ if } x\neq 1\\ \text{ undefined } \amp \text{ if } x=1 \end{cases} \end{equation*}
    ðŸ”—
    Remember that limits describe the behavior of a function near the \(x\)-value of the limit. So if we look near \(x=1\text{,}\) we can cancel the \(x-1\) and evaluate the limit of the continuous function \(x+1\text{:}\)
    \begin{equation*} \lim_{x\to 1}\frac{x^2-1}{x-1} = \lim_{x\to 1}\frac{(x-1)(x+1)}{x-1} = \lim_{x\to 1}(x+1) =1+1 = 2\text{.} \end{equation*}
    ðŸ”—
    The cancellation of the common factor \(x-1\) is valid within the limit computation because \(\frac{(x-1)(x+1)}{x-1}\) and \(x+1\) are equal to each other anytime \(x\neq 1\text{,}\) and when we take the limit, we are not concerned with what happens at the point \(x=1\text{,}\) only nearby.
    ðŸ”—
    ðŸ”—
    In general, by definition, limits as \(x\to c\) never have anything to do with what happens at \(x=c\text{,}\) only near \(x=c\text{,}\) which proves the following theorem:
    ðŸ”—

    Theorem 3.5.2. The Cancellation Theorem for Limits.

    If \(\ds\lim_{x\to c}g(x)\) exists, and \(f\) is a function that is equal to \(g\) for all \(x\) sufficiently close to \(c\) except possibly at \(c\) itself, then \(\ds\lim_{x\to c}f(x) = \ds\lim_{x\to c}g(x)\text{.}\)
    ðŸ”—
    ðŸ”—
    \fbox{What this theorem is saying is that we can simplify via cancellation within a limit, and we can use the value of the limit of the simplified function for the value of the original function.
    ðŸ”—
    Said slightly differently, if we have a function \(f(x)\) that is discontinuous at \(x=c\text{,}\) but we can algebraically simplify \(f(x)\) into a function \(g(x)\) that is continuous at \(x=c\) (or at least that has a limit at \(x=c\) that we know), then we can use the value of the limit of \(g(x)\) instead. }}
    ðŸ”—

    Example 3.5.3.

    Letâ€™s evaluate \(\ds\lim_{x\to 1}\dfrac{2x^2+x-3}{x-1}\text{.}\) At first, we get
    \begin{equation*} \lim_{x\to 1}\frac{2x^2+x-3}{x-1} = \frac{2(1)^2+1-3}{1-1} = \frac{0}{0}\text{,} \end{equation*}
    which is not helpful. However, we can factor the numerator and cancel:
    \begin{equation*} \frac{2x^2+x-3}{x-1} = \frac{(2x+3)(x-1)}{x-1} = \begin{cases}2x+3,\amp \text{ if } x\neq1\\ \text{ undefined } ,\amp \text{ if } x=1. \end{cases} \end{equation*}
    ðŸ”—
    By TheoremÂ 3.5.2, we have
    \begin{equation*} \lim_{x\to 1}\frac{2x^2+x-3}{x-1} = \lim_{x\to 1}(2x+3) = 2(1)+3 = 5 \end{equation*}
    by the continuity of \(2x+3\) at \(x=1\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 3.5.4.

    Recall \(h(x) = \dfrac{x-3}{x^3 - 3 x^2 - 4 x + 12}\) from Example ExampleÂ 3.4.45. There, we factored
    \begin{equation*} h(x) = \frac{x-3}{ (x-3)(x-2)(x+2) } \end{equation*}
    and noted that
    \begin{equation*} \lim_{x\to 3} h(x) = \frac{0}{0} \end{equation*}
    ðŸ”—
    Now, we see that we can simply cancel the common factor of \(x-3\) and evaluate:
    \begin{equation*} \lim_{x\to 3} \frac{x-3}{ (x-3)(x-2)(x+2) } = \lim_{x\to 3} \frac{1}{(x-2)(x+2)} = \frac{1}{(3-2)(3+2)} = \frac{1}{5}\text{.} \end{equation*}
    Thus our guess from the table in Example ExampleÂ 3.4.45 that the limit approaches \(\frac{1}{5} = 0.2\) was correct! There is NOT a vertical asymptote at \(x=3\text{.}\) Instead, there is a hole, a.k.a. removable discontinuity, because the original expression is undefined but the limit approaches a finite number.

    Example 3.5.6. Instantaneous Velocity as a Limit of Average Velocity.

    Recall that in Example ExampleÂ 3.1.2 we approximated the instantaneous velocity of a ball with position
    \begin{equation*} s(t) = 64-16(t-1)^2 \end{equation*}
    at the time \(t=1\) using smaller and smaller intervals that approached \(t=1\) from the left:
    \begin{equation*} [0.5,1], [0.9,1], [0.99,1], \ldots \end{equation*}
    and from the right:
    \begin{equation*} [1,1.5], [1,1.1], [1,1.01], \ldots \end{equation*}
    ðŸ”—
    We can use the notation \([1,1+h]\) to represent each interval if we agree to reverse the order of the interval for negative values of \(h\text{.}\) For example, when \(h=-0.5\text{,}\) the interval is
    \begin{equation*} [1,1+(-0.5)] = [1,0.5] = [0.5,1] \end{equation*}
    and when \(h=-0.1\text{,}\) the interval is
    \begin{equation*} [1,1+(-0.1)] = [1,0.9] = [0.9,1]\text{.} \end{equation*}
    So when we consider intervals to the left of \(t=1\text{,}\) we are taking the limit as \(h\to 0^-\text{.}\) When we consider intervals to the right of \(t=1\text{,}\) we are taking limits as \(h\to 0^+\text{.}\) This idea leads to limit calculations that we will study in great depth at the beginning of Chapter 3. For the moment, note that we can find a general formula for the average velocity of \(s(t)\) on intervals \([1,1+h]\text{:}\)
    \begin{equation*} \begin{aligned}AV_{[1,1+h]} \amp = \frac{s(1+h)-s(1)}{(1+h)-1}\\ \amp = \frac{(64-16(1+h-1)^2) - (64-16(1-1)^2)}{(1+h)-1}\\ \amp = \frac{64-16h^2 - 64}{h}\\ \amp = \frac{-16h^2}{h} \end{aligned} \end{equation*}
    ðŸ”—
    If we take \(h\to 0\text{,}\) meaning we let the interval shrink down to the point \(t=1\text{,}\) we get a limit of the form \(\frac{0}{0}\text{,}\) but as weâ€™ve seen, we can cancel in order to evaluate the limit:
    \begin{equation*} \lim_{h\to 0}AV_{[1,1+h]} = \lim_{h\to 0} \frac{-16h^2}{h} = \lim_{h\to 0}{-16h} = 0\text{.} \end{equation*}
    ðŸ”—
    Thus the limit of the average velocities (the instantaneous velocity) at \(t=1\) appears to indeed be 0, meaning that the ball is momentarily motionless in the air at \(t=1\) before beginning its descent back to the ground.

    Subsection 3.5.3 Indeterminate Limit Forms

    You have now been introduced to two different limit forms: \(\frac{1}{0}\) and \(\frac{0}{0}\text{.}\) Limits of the form \(\frac{1}{0}\) are always infinite, though we donâ€™t always know whether they are positive or negative infinity (or one of each from the right and left). Limits of the form \(\frac{0}{0}\) are what is called "indeterminate." If a limit is indeterminate, then we cannot initially say whether or not it exists â€” or if it exists, what real number it is equal to. Many indeterminate limits can be resolved with algebra such as cancellation or factoring.
    ðŸ”—

    Example 3.5.8.

    For example, the four limits that follow are all initially of the indeterminate form \(\frac{0}{0}\text{.}\) After some algebra, we see that three limits exist (and describe holes) and one limit becomes infinite (and describes a vertical asymptote). The three that exist are each equal to different real numbers.
    1. Consider \(f(x) = \frac{x^2-2x+1}{x-1}\) at \(x=1\text{.}\) Initially,
      \begin{equation*} \ds\lim_{x\to 1}\frac{x^2-2x+1}{x-1} = \frac{1-2+1}{1-1} = \frac{0}{0} \end{equation*}
      However, we can factor the numerator: \(x^2-2x+1 = (x-1)(x-1) = (x-1)^2\) and then evaluate the limit by cancelling:
      \begin{equation*} \lim_{x\to 1}\frac{(x-1)^2}{x-1} = \lim_{x\to 1}\frac{x-1}{1} = \frac{0}{1} = 0 \end{equation*}
      The function \(f(x)\) has a hole at the point \((1,0)\text{.}\)
      ðŸ”—
      ðŸ”—
    2. Now consider \(g(x) = \frac{x-1}{(x-1)^3}\) at \(x=1\text{.}\) Again, technically originally the limit has the form \(\frac{0}{0}\text{:}\)
      \begin{equation*} \lim_{x\to 1}\frac{x-1}{(x-1)^3} = \frac{0}{0} \end{equation*}
      but we can cancel one of the factors of \(x-1\) and re-evaluate to get an infinite limit:
      \begin{equation*} \lim_{x\to 1}\frac{x-1}{(x-1)^3} = \lim_{x\to 1}\frac{1}{(x-1)^2} = \frac{1}{0^+} = \infty \end{equation*}
      The function \(g(x)\) has a vertical asymptote at \(x=1\text{.}\)
      ðŸ”—
      ðŸ”—
    3. Now consider \(h(x) = \frac{x-1}{x-1}\text{.}\) \(h(x)\) is undefined at \(x=1\) and the limit as \(x\to 1\) has the form \(\frac{0}{0}\text{.}\) However, when we simplify:
      \begin{equation*} \lim_{x\to 1}\frac{x-1}{x-1} = \lim_{x\to 1}1 = 1 \end{equation*}
      Thus \(h(x)\) is the constant function \(y=1\text{,}\) except for the hole at the point \((1,1)\text{.}\)
      ðŸ”—
      ðŸ”—
    4. Lastly, consider the function \(k(x) = \frac{x-1}{3x-3}\text{,}\) which is also undefined at \(x=1\text{.}\) We can factor a 3 from the denominator and then cancel:
      \begin{equation*} \lim_{x\to 1}\frac{x-1}{3x-3}= \lim_{x\to 1}\frac{x-1}{3(x-1)} = \lim_{x\to 1}\frac{1}{3} = \frac{1}{3} \end{equation*}
      So \(k(x)\) is equal to the constant function \(y=\frac{1}{3}\text{,}\) except for the hole at the point \(\left(1,\frac{1}{3}\right)\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    \(\frac{0}{0}\) is just one of several different indeterminate limit forms that we will encounter this semester. We summarize the indeterminate limit forms that arise from algebraic function below.

    Subsection 3.5.4 Hole or Vertical Asymptote?

    Recall that limits of the form \(\frac{0}{0}\) can be finite or infinite. These limits are indeterminate because we do not know whether they exist (and are finite) or not. We have seen that some can be evaluated by algebra. On the other hand, limits of the form \(\frac{1}{0}\) are always infinite, but we donâ€™t know whether they are positive or negative. Some of these limits must be broken into separate right and left limits in order to determine their signs. In this section, we will focus on identifying which limit form we are dealing with, and how to best proceed once we have identified the form. Then, once we have finished evaluating the limit, we will interpret graphically whether it describes a hole or vertical asymptote.
    ðŸ”—

    Example 3.5.11.

    Consider
    \begin{equation*} f(x) = \frac{(x-1)^2}{x-1} \end{equation*}
    ðŸ”—
    This function is undefined at \(x=1\text{,}\) with both the numerator and denominator being equal to 0, so the limit at \(x=1\) has the form \(\frac{0}{0}\text{.}\) For now, whenever we encounter this limit form, we attempt to apply the Cancellation Theorem. We can cancel and compute
    \begin{equation*} \lim_{x\to 1} \frac{(x-1)^2}{x-1} = \lim_{x\to 1} x-1 = 1-1 = 0\text{.} \end{equation*}
    ðŸ”—
    Thus \(f(x)\) has a hole at \((1,0)\text{,}\) rather than a vertical asymptote, as the limit is finite.
    ðŸ”—
    Here is a graph of \(f(x) = \frac{(x-1)^2}{x-1}\text{,}\) demonstrating the hole visually that we found via limits:
    ðŸ”—
    Note that the function \(f(x) = \frac{(x-1)^2}{x-1}\) is equal to the function \(y=x-1\text{,}\) except at the point \(x=1\) where \(f(x)\) is undefined. We can write
    \begin{equation*} f(x) = \begin{cases}x-1, \amp x\neq 1\\ \text{ undefined } , \amp x=1 \end{cases} \end{equation*}

    Now consider
    \begin{equation*} g(x) = \frac{x-1}{(x-1)^2} \end{equation*}
    ðŸ”—
    If we try to evaluate \(\ds\lim_{x\to 1}g(x)\) without canceling, we get
    \begin{equation*} \lim_{x\to 1}\frac{x-1}{(x-1)^2} = \frac{0}{0} \end{equation*}
    ðŸ”—
    Cancelling one of the factors of \(x-1\text{,}\) we can see that \(g(x) = \frac{x-1}{(x-1)^2} = \frac{1}{x-1}\) everywhere they are defined, meaning for \(x\neq 1\text{.}\) Now when we compute the limit as \(x\to 1\text{,}\) we see that we have the form \(\frac{1}{0}\text{,}\) so this limit is infinite:

    Example 3.5.21.

    Find the \(x\)-intercepts, holes, and vertical asymptotes of the function
    \begin{equation*} f(x) = \frac{2x^3-8x}{x^3+2x^2-x-2} \end{equation*}
    ðŸ”—
    Solution.
    First, we factor the numerator and denominator of \(f(x)\text{:}\)
    \begin{equation*} 2x^3-8x = 2x(x^2-4) = 2x(x-2)(x+2) \end{equation*}

    \begin{equation*} \begin{aligned}x^3+2x^2-x-2 \amp = x^2(x+2)-(x+2) \\ \amp = (x+2)(x^2-1) \\ \amp = (x+2)(x-1)(x+1) \end{aligned} \end{equation*}
    so that

    ðŸ”—
    \(f(x)=0\) when the numerator is equal to 0, provided that \(f(x)\) is defined, meaning that the denominator is not equal to 0. So \(f(x)=0\) for \(x=0\) and \(x=2\text{,}\) but not \(x=-2\) since \(f(-2)\) is undefined. Thus we have \(x\)-intercepts for \(x=0, 2\text{.}\)

    \begin{equation*} \begin{aligned}\lim_{x\to -2}f(x) \amp = \lim_{x\to -2}\frac{2x(x-2)(x+2)}{(x+2)(x-1)(x+1)}\\ \amp = \lim_{x\to -2}\frac{2x(x-2)}{(x-1)(x+1)}\\ \amp =\frac{2(-2)(-4)}{(-3)(-1)}\\ \amp = \frac{16}{3} \end{aligned} \end{equation*}
    and so the precise location of the hole is \((-2,16/3)\text{.}\)

    \begin{equation*} \lim_{x\to -1^-} f(x) = \lim_{x\to -1^-} \frac{2x(x-2)}{(x-1)(x+1)} = \frac{(-2)(-3)}{(-2)(0^-)} = +\infty \end{equation*}
    \begin{equation*} \lim_{x\to -1^+} f(x) = \lim_{x\to -1^+} \frac{2x(x-2)}{(x-1)(x+1)} = \frac{(-2)(-3)}{(-2)(0^+)} = -\infty \end{equation*}
    ðŸ”—
    At \(x=1\text{,}\) the factor \(x-1\) is slightly negative as \(x\to 1^-\) and slightly positive as \(x\to 1^+\text{:}\)
    \begin{equation*} \lim_{x\to 1^-} f(x) = \lim_{x\to 1^-} \frac{2x(x-2)}{(x-1)(x+1)} = \frac{(2)(-1)}{(0^-)(2)} = +\infty \end{equation*}
    \begin{equation*} \lim_{x\to 1^+} f(x) = \lim_{x\to 1^+} \frac{2x(x-2)}{(x-1)(x+1)} = \frac{(2)(-1)}{(0^+)(2)} = -\infty \end{equation*}
    ðŸ”—
    Thus we have
    \begin{equation*} \lim_{x\to-1^-}f(x) = \infty\text{ and } \lim_{x\to-1^+}f(x) = -\infty \end{equation*}
    and
    \begin{equation*} \lim_{x\to1^-}f(x) = \infty\text{ and } \lim_{x\to1^+}f(x) = -\infty \end{equation*}
    ðŸ”—
    To summarize, there are \(x\)-intercepts at \(x=0, 2\text{,}\) a hole at \(x=-2\text{,}\) and vertical asymptotes at \(x=-1, 1\text{.}\)

    Subsection 3.5.5 Summary

    The following terms and theorems were introduced in this section:
    ðŸ”—
    \(\frac{0}{0}\text{,}\)
    Key ideas: Limits of the form \(\frac{0}{0}\) are indeterminate and may be either infinite (in which case there is a vertical asymptote) or finite (in which case there is a hole). One way to evaluate limits of the form \(\frac{0}{0}\) is by cancellation. After cancelling, we may be able to evaluate the limit (either as a number or as an infinite limit of the form \(\frac{1}{0}\)).
    ðŸ”—
    Other ideas introduced: Instantaneous velocity is a limit of average velocities, and always has the form \(\frac{0}{0}\) before simplifying. We can tell quickly from factored formulas for rational functions where they have \(x\)-intercepts, holes, or vertical asymptotes.
