    Let \(f\) be a function with domain \(D\text{,}\) and suppose \(a\in D\) is either an interior point or an endpoint of \(D\text{.}\) The function \(f\) is continuous at \(a\) if the limit of \(f\) at \(a\) exists and is equal to the value of \(f\) at \(a\text{:}\) i.e., if
    1. \(\lim\limits_{x\to a}f(x)\) exists, and

    2. \(\lim\limits_{x\to a}f(x)=f(a)\text{.}\)

    The function \(f\) is discontinuous at \(a\) if it is not continuous there.

    Example 1.15.5. Continuity at endpoint.

    Let \(f(x)=\sqrt{x}\text{,}\) with domain \(D=[0,\infty)\text{.}\) Decide whether \(f\) is continuous at \(0\text{.}\)
    ğŸ”—
    Solution.
    First observe that since \(0\) is a left endpoint of \(D\text{,}\) we define
    \begin{equation*} \lim_{x\to 0}f(x)=\lim_{x\to 0^+}f(x)\text{.} \end{equation*}
    We compute
    \begin{align*} \lim_{x\to 0^+}f(x) \amp \lim_{x\to 0^+}\sqrt{x} \\ \amp \sqrt{\lim_{x\to 0^+}x} \amp \text{(root rule)}\\ \amp = \sqrt{0} \amp \text{(poly. eval.)}\\ \amp = 0 \end{align*}
    Thus \(\lim\limits_{x\to 0}f(x)\) exists and is equal to 0. Since \(f(0)=\sqrt{0}=0\text{,}\) we see further that
    \begin{equation*} \lim_{x\to 0^+}f(x)=\lim_{x\to 0^+}f(x)=f(0)=0 \end{equation*}
    and thus that \(f\) is continuous at \(0\text{.}\)

    Remark 1.15.6. Continuity: graphical.

    When scanning the graph of a function \(f\) for points of continuity or discontinuity, we look for â€œproblem pointsâ€. These are locations on the graph where either (a) the limit of the function appears not to exist, or (b) where the limit does exist but does not agree with the value of the function.
    ğŸ”—
    A well-loved adage in this regard is the phrase â€œa continuous function is one whose graph can be traced without lifting the pencilâ€. This is cute, and helpful for guidance, but should not be taken too literally. Indeed, there are some pretty wacky continuous functions out there. For example, the function
    \begin{equation*} f(x)=\begin{cases} x\sin(1/x)\amp \text{if } x> 0\\ 0 \amp \text{if } x=0 \end{cases} \end{equation*}
    is continuous everywhere on its domain, and yet there are some peculiar properties of its graph that make it unclear it what sense it could be â€œtraced by a pencilâ€.

    Example 1.15.9. Continuous everywhere.

    Let \(\displaystyle f(x)=\frac{x}{\sqrt{x^2-1}}\text{.}\) Prove, using DefinitionÂ 1.15.2 that \(f\) is continuous.
    ğŸ”—

    \begin{align*} \lim_{x\to a}f(x) \amp =\lim_{x\to a}\frac{x}{\sqrt{x^2-1}}\\ \amp =\frac{\lim_{x\to a}x}{\lim_{x\to a}\sqrt{x^2-1}}\\ \amp =\frac{a}{\sqrt{\lim_{x\to a}x^2-1}}\\ \amp = \frac{a}{\sqrt{a^2-1}}\\ \amp = f(a)\text{.} \end{align*}

    Example 1.15.13. Continuity rules.

    \begin{equation*} f(x)=\abs{\sqrt[5]{\frac{\sin^3 x}{\sqrt{\cos x}}}} \end{equation*}

    ğŸ”—
