    Subsection The Definition of a Function

    A function is a fundamental mathematical object that will form the basis of everything we study this year. In this section, we will explore their definition and its meaning, several ways of representing a function, and a physical example of how functions are manifest in real-world phenomena.

    Definition 1.1.1. Function, Independent and Dependent Variables.

    A function \(f\) is a rule that takes certain numbers \(a\) in the set \(A\) as inputs and assigns to each a single number \(b\) in the set \(B\) as the output.
    ðŸ”—
    The notation
    \begin{equation*} f: A\rightarrow B \end{equation*}
    represents a function \(f\) together with its input set \(A\) and output set \(B\text{.}\) It is pronounced "\(f\) from \(A\) to \(B\text{.}\)" The notation
    \begin{equation*} b=f(a)\text{,} \end{equation*}
    pronounced "\(b\) is equal to \(f\) of \(a\text{,}\)" represents the assignment of the output \(b\) to the input \(a\text{.}\)
    ðŸ”—
    The collection of symbols \(f(a)=b\) is called function notation, where the \(a\) in parentheses is called the independent variable and represents the input of the function. The output of the function is represented by either \(f(a)\) or \(b\text{,}\) which is called the dependent variable because its value depends on the choice of input.
    ðŸ”—
    ðŸ”—

    Remark 1.1.2.

    It is particularly important that each input has only one output. Think about the function as a machine, the input as a command, and the output as a task the machine will perform based on which command you give. When you give the machine an instruction, it should consistently perform a predictable task. You donâ€™t want it to be unpredictable and do different tasks based on the same instruction.
    ðŸ”—
    ðŸ”—

    Example 1.1.3.

    For example, the rule that takes a real number and squares it is a function. Since the result is also a real number, we write
    \begin{equation*} f:\mathbb{R}\to\mathbb{R} \end{equation*}
    where the symbol \(\mathbb{R}\) is used to represent the set of all real numbers. In this example, \(\mathbb{R}\) is both our input and our output set. This will be the case for many of the examples we study this semester.
    ðŸ”—
    Introducing some notation, letâ€™s call the number we are squaring \(x\) so that once we square it, we may write \(x^2\text{.}\) We can pick the name \(f\) to refer to this function by writing \(f(x) = x^2\text{,}\) which is pronounced "\(f\) of \(x\) is equal to \(x\) squared." Thus we have chosen \(x\) as the symbol we use to represent our independent variable. We can also choose to represent our outputs with the dependent variable \(y\) and can use the notation \(y=x^2\) or \(y=f(x)\) to represent the fact that the values of \(y\) are the result of squaring \(x\text{,}\) or equivalently, applying the function \(f\) to \(x\text{.}\)
    ðŸ”—
    For instance, if we choose the input value \(x=3\text{,}\) then we write
    \begin{equation*} f(3) = 3^2 = 9\text{,} \end{equation*}
    which is read "\(f\) of \(3\) is equal to \(3\) squared, which is equal to \(9\text{.}\)" The number \(9\) is the output of the function when the input is \(3\text{.}\)
    ðŸ”—

    \begin{equation*} f(-3)=(-3)^2 = 9\text{,} \end{equation*}
    which is read "\(f\) of negative \(3\) is equal to negative \(3\) squared, which is equal to \(9\text{.}\)" The number \(9\) is again the output of the function when the input is \(-3\text{.}\)

    Subsection Domain and Range of a Function

    The function \(f(x)=x^2\) in Examples ExampleÂ 1.1.3 and ExampleÂ 1.1.5 is fairly typical in that both the input and output sets are numbers; most of the functions we consider this year will have numbers as inputs and outputs. In order to be more precise in our discussion of functions, we introduce some additional terminology to describe the input and output sets:
    ðŸ”—

    Definition 1.1.9. Domain and Range of a Function.

    If \(f\) is a function between unspecified subsets of \(\mathbb{R}\text{,}\) then we will take the domain of \(f\) to be the largest subset of \(\mathbb{R}\) for which \(f\) is defined:
    \begin{equation*} \text{ Domain } (f) = \{\text{ real numbers } x\text{ such that } f(x)\text{ is defined } \}\text{.} \end{equation*}
    The range of such a function is the set of all possible outputs that it can attain:
    \begin{equation*} \text{ Range } (f) = \{\text{ real numbers } y\text{ such that we can find } x\text{ in Domain } (f)\text{ for which } f(x) = y\} \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 1.1.10.

    Returning to the squaring function \(f(x)=x^2\) in Example ExampleÂ 1.1.3. Because we can take any number \(x\) as an input and successfully square it, the domain of \(f(x)\) is indeed the set of all real numbers \(\mathbb{R}\text{.}\) However, notice that some real numbers can never serve as outputs, because squares of real numbers can never be negative. The range of the squaring function is real numbers that are greater than or equal to zero. Here are two different ways of describing that set symbolically:
    • With "set notation" \(\{y\in \mathbb{R}\mid y\geq 0\}\) where the symbol "\(\in\)" means "in" and the symbol "\(\mid\)" means "such that," so that this expression translates directly as "the set of \(y\) in the real numbers such that \(y\) is greater than or equal to zero." It is more common to omit the formal set notation and simply use the inequality \(y\geq 0\text{.}\)
      ðŸ”—
      ðŸ”—
    • With "interval notation" \([0,\infty)\) where the symbol "\([\)" means that we include the left endpoint \(0\) and the symbol "\()\)" means that we do not include right endpoint; together with the infinity symbol \(\infty\text{,}\) the right side of this interval extends forever.     Remark 1.1.11.

    While we will only refer to the domain and range of a function, there is another set that you may encounter relating to functions: the codomain. When we use the notation
    \begin{equation*} f:A\to B \end{equation*}
    to represent a function, the set \(A\) is the domain but the set \(B\) is called the codomain and does not have to be the same as the range of the function. The range of a function is either equal to or is a subset of the codomain. For example, when looking at the function \(f:\mathbb{R}\to\mathbb{R}\) defined by \(f(x) = x^2\) in Example ExampleÂ 1.1.3, the codomain is \(\mathbb{R}\) while the range is the set of numbers greater than or equal to \(0\text{.}\)

    Example 1.1.12. A Function with a Square Root.

    Consider the function \(g(x) = \sqrt{4-x}\text{.}\) We can successfully use the value \(x=0\) as an input because
    \begin{equation*} g(0) = \sqrt{4-0}=\sqrt{4}=2 \end{equation*}
    ðŸ”—
    So we see that the output \(g(0)=2\) is defined as the input of \(x=0\text{,}\) hence \(x=0\) is in the domain of \(g\text{.}\) Similarly, we can plug in the values \(x=1, 2, 3\text{,}\) and \(4\text{,}\) along with the values \(x=-1, -2,-3\) and \(-4\text{:}\)
    ðŸ”—
    However, if we try to plug in \(x=5\text{,}\) we do not get an output because we cannot take square roots of negative numbers:
    \begin{equation*} g(5) = \sqrt{4-5} = \sqrt{-1}\text{ is undefined }\text{.} \end{equation*}
    Thus \(x=-4, -3,\ldots, 3, 4\) ARE in the domain of \(g\) while \(x=5\) is NOT in the domain of \(g\text{.}\) How can we determine exactly where the domain begins and ends? Since we need the quantity inside the square root to be non-negative, meaning 0 or greater, we set up and solve the inequality
    \begin{equation*} \begin{aligned}4-x \amp \geq 0\\ +x \amp \amp +x\\ \hline 4 \amp \geq x \end{aligned} \end{equation*}
    ðŸ”—
    Hence any value of \(x\) that is equal or less than 4 is in the domain, which we write as either \(x\leq 4\) or \((-\infty,4]\text{.}\)

    Subsection Non-Functions and the Vertical Line Test

    So, what isnâ€™t a function? If a rule assigns a real-number input to more than one output, then that rule is not a function. In this subsection, we learn a theorem to help us determine graphically whether or not a relationship is a function.
    ðŸ”—

    Example 1.1.17. A Non-Example.

    For example, consider the relationship between the independent variable \(x\) and the dependent variable \(y\) where the square of \(y\) is equal to \(x\text{.}\) We could write this as \(y^2 = x\text{,}\) where we remember that the input is \(x\text{.}\) For example, if we take \(x=4\text{,}\) then the output \(y\) is a number that squares to \(4\text{.}\) This could be either \(y=2\) or \(y=-2\text{,}\) since \(2^2=4\) and \((-2)^2 = 4\text{.}\) Since we donâ€™t know which to choose, we say that "\(y\) is not defined as a function of \(x\text{.}\)"
    ðŸ”—
    Algebraically, we could try to "solve" the equation \(y^2=x\) for \(y\) by writing \(y=\pm\sqrt{x}\text{,}\) and we see that we have the same issue where the input \(x=4\) corresponds to two different \(y\)-values, both \(y=-2\) and \(y=2\text{.}\)
    ðŸ”—
    If you think of a function as a machine or algorithm that is required to return a single value, this ambiguity in having either \(y=2\) or \(y=-2\) as our output can be thought of as a program that crashes and fails to complete its task.
    ðŸ”—
    ðŸ”—
    Here is another example of a relationship between variables that does not describe a function.
    ðŸ”—

    Example 1.1.18. A relationship described by a formula and graph that is NOT a function.

    The relationship between the variables \(x\) and \(y\) given by the equation
    \begin{equation*} x^2+y^2=16 \end{equation*}
    is graphed in FigureÂ 1.1.19 below in red.
    ðŸ”—
    If we let \(x=1\text{,}\) we can solve for \(y\text{:}\)
    \begin{equation*} \begin{aligned}(1)^2+y^2 \amp =16\\ y^2 \amp =16-1\\ y^2 \amp =15\\ y \amp = \pm\sqrt{15}\approx \pm3.873 \end{aligned} \end{equation*}
    ðŸ”—
    The points \((1,\sqrt{15})\) and \((1,-\sqrt{15})\) are labeled in blue and green, respectively. Thus we see that when \(x=1\text{,}\) there are two \(y\)-values given by the equation instead of \(1\text{,}\) and without more information, we do not know which to use. So we see that \(y\) cannot be defined as a function of \(x\) from the equation \(x^2+y^2=16\text{,}\) even though it defines a very nice curve (a circle of radius 4).
    ðŸ”—
    Note that \(x=1\) was picked at random. You can see from the picture of the circle that there are many other points where a single \(x\)-value corresponds to two different \(y\)-values, in particular anywhere that a vertical line passes through the circle twice.

    ðŸ”—
    Thus, the usual way to demonstrate that a relationship or process is not a function is to find a particular input that is associated with two or more outputs. When the relationship is given graphically, we can use the vertical line test to determine whether or not the graph represents a function:
    ðŸ”—

    Theorem 1.1.20. The Vertical Line Test.

    A graph represents a function if and only if every vertical line intersects the graph in at most one point.
    ðŸ”—
    When the graph passes this test, the vertical coordinate of each point on the graph can be viewed as a function of the horizontal coordinate of the point. When the graph does not pass this test, the vertical coordinate cannot be viewed as a function of the horizontal coordinate.
    ðŸ”—

    Subsection Interval Notation for Sets

    When we talk about functions, they will always take some set of numbers as their domain and have some set of numbers as their range. In this section, we talk briefly about what a "set" is and some different notations for how to write them down.
    ðŸ”—
    You have already seen one mathematical shorthand for a commonly used set: \(\mathbb{R}\) denotes the set of all real numbers. Another useful notation is the empty set: \(\emptyset\) denotes the set that contains nothing.
    ðŸ”—

    Remark 1.1.33.

    Please do not confuse \(\emptyset\) with the number 0. They are VERY different and should NOT be used interchangeably.
    ðŸ”—
    ðŸ”—
    Another set we discussed was the domain and range of the water in the tank in Example ExampleÂ 1.1.28. We said that if the time \(t\) is between 0 and 5 minutes, then the volume \(V\) was between 0 and \(3.75\) ft\(^3\text{.}\) We can write these using inequalities:
    \begin{equation*} 0\leq t\leq 5 \qquad\text{ and } \qquad 0\leq V \leq 3.75\text{,} \end{equation*}
    which are read "0 is less than or equal to \(t\text{,}\) which is less than or equal to 5" and "0 is less than or equal to \(V\text{,}\) which is less than or equal to \(3.75\text{.}\)" An alternate notation for writing these two sets is called interval notation. The set \(0\leq t \leq 5\) is written
    \begin{equation*} [0,5] \end{equation*}
    and the set \(0\leq V\leq 3.75\) is written
    \begin{equation*} [0,3.75]\text{.} \end{equation*}
    This notation is nice and concise, but does not specify a variable and so can be confusing or ambiguous. If we wish to name the variable \(t\text{,}\) we would write \(t\in[0,5]\text{,}\) read "\(t\) is in the set of numbers from 0 (inclusive) to 5 (inclusive)." Likewise for \(V\text{,}\) we can write \(V\in[0,3.75]\text{,}\) read "\(V\) is in the set of numbers from 0 (inclusive) to 3.75 (inclusive)." The symbol \(\in\) means "in" and the square brackets indicate the the endpoints of the interval are included.
    ðŸ”—
    Other sets that you saw were the domain and range of the function \(g(x) = \sqrt{4-x}\) in ExampleÂ 1.1.12. The range \(y\geq 0\) can be written in interval notation as \(y\in [0,\infty)\text{.}\) The domain \(x\leq 4\) can be written in interval notation as \(x\in (-\infty,4]\text{.}\) The meaning of this set is best illustrated on a number line:

    The values of \(x\) are allowed to be any number below \(x=4\text{,}\) denoted by the open parentheses and \(-\infty\) on the left of the interval \((-\infty,4]\text{.}\)
    ðŸ”—
    Another set that you saw was the domain of the function \(h\) in Example ExampleÂ 1.1.15. We wrote this using the standard shorthand of \(x\neq -2,2\text{,}\) which means that \(x\) can take on any value except for \(-2\) and \(2\text{.}\) To write this using inequalities, we would need three different parts:
    \begin{equation*} x\lt -2\text{ or } -2\lt x\lt 2\text{ or } 2\lt x\text{.} \end{equation*}
    The corresponding interval notation also has three parts:
    \begin{equation*} (-\infty,-2)\cup(-2,2)\cup(2,\infty)\text{.} \end{equation*}
    ðŸ”—
    The round parentheses mean we do not include the endpoints at \(-2\) and \(2\text{,}\) and \(\cup\) is the "union" symbol, which means we are glueing the intervals together.
    ðŸ”—
    Finally, one more standard notation is an alternate way of writing the set of all real numbers: instead of writing \(\mathbb{R}\text{,}\) we can write \((-\infty,\infty)\text{.}\) There is no way to write the empty set \(\emptyset\) using interval notation; WeBWorK will typically have you enter the word "none" in cases where there is no solution.
