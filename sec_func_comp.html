    Subsection Motivation and Examples

    Recall that a function, by definition, is a process that takes a collection of inputs and produces a corresponding collection of outputs in such a way that the process produces one and only one output value for any single input value. Because every function is a process, it makes sense to think that it may be possible to take two function processes and do one of the processes first, and then apply the second process to the result.
    ðŸ”—
    Recall Dolbearâ€™s function, \(F=D(N)=40+0.25N\text{,}\) from Example ExampleÂ 1.4.4 that relates the number \(N\) of chirps per minute from a snowy cricket to the Fahrenheit temperature, \(F\text{.}\)
    ðŸ”—
    The Celsius and Fahrenheit temperature scales are connected by a linear function. Indeed, the function that converts Fahrenheit to Celsius is
    \begin{equation*} C = G(F) = \frac{5}{9}(F-32) \end{equation*}
    ðŸ”—
    For instance, a Fahrenheit temperature of 32 degrees corresponds to \(C=G(32) = 0\) degrees Celsius.
    ðŸ”—
    If we are given a number of chirps, \(N\text{,}\) we can combine the two processes given by Dolbearâ€™s function and the conversion from Fahrenheit to Celsius to obtain a new function \(C=H(N)\) that takes as input the number of chirps per minute and gives as output the temperature in degrees Celsius.
    ðŸ”—

    Example 1.9.1.

    Suppose that we observe 80 chirps per minute. What is the temperature in degrees Celsius?
    ðŸ”—
    First, we compute that the temperature in degrees Fahrenheit is
    \begin{equation*} F = D(80) = 40+0.25(80) = 60\text{.} \end{equation*}
    ðŸ”—
    Then, we compute that the temperature in degrees Celsius is
    \begin{equation*} C = G(60) = \frac{5}{9}(60-32) = \frac{140}{9} =15\frac{5}{9}^\circ C \end{equation*}
    ðŸ”—
    ðŸ”—
    We could do this individually for each value of \(N\text{,}\) but it would be more convenient to have a single formula that can go directly from number of chirps per minute to degrees Celsius. In the above example, we took the output \(F=D(80)\) and used it as the input for the temperature conversion \(C=G(F)\) from Fahrenheit to Celsius. We can do the same thing using the variable \(N\) instead of the explicit input \(N=80\text{:}\)
    ðŸ”—

    Example 1.9.2.

    Determine a formula for the new function \(C=H(N)\) that takes as input the number \(N\) of cricket chirps per minute and gives as output the temperature \(C\) in degrees Celsius.
    ðŸ”—
    Using \(F=D(N)\) as the input for the function \(C=G(F)\text{,}\) we simplify to find a formula directly from \(N\) to \(C\) that does not involve the variable \(F\text{:}\)
    \begin{equation*} \begin{aligned}C \amp = G(F)\\ \amp = G(D(N))\\ \amp = \frac{5}{9}(D(N)-32)\\ \amp = \frac{5}{9}((40+0.25N)-32)\\ \amp =\frac{5}{9}(8+0.25N) \end{aligned} \end{equation*}
    ðŸ”—
    So the function \(C=H(N)\) is given by
    \begin{equation*} C = \frac{5}{9}(8+0.25N) \end{equation*}
    ðŸ”—
    ðŸ”—
    There is, of course, a shorthand notation for this process of using the output of \(D\) as the input of \(G\text{:}\)
    \begin{equation*} H=G\circ D \end{equation*}
    ðŸ”—
    \(G\circ D\) is the function that first applies the process given by the function \(D\) and then applies the process given by the function \(G\text{.}\) Two natural questions we could begin by asking are:
    ðŸ”—
    \(H = G\circ D\text{?}\)Â Â Â Â \(H=G\circ D\text{?}\)

    Example 1.9.3.

    Find the domain and range of the function \(C=H(N)\text{.}\)
    ðŸ”—
    We established in Example ExampleÂ 1.4.8 that \(F=D(N)\) has a domain of \([40,180]\) cricket chirps per minute and a corresponding range of \([50,85]\) degrees Fahrenheit.
    ðŸ”—
    The domain for \(H=G\circ D\) is the inputs of \(H\text{,}\) so is also the inputs of \(D\) as that is the function that is applied first. Thus we can use the same set of \(N\)-values and the domain of \(H\) is also \([40,180]\text{.}\)
    ðŸ”—
    In order to determine the range, we must find the resulting degrees Celsius from the range of degrees Fahrenheit of \([50,85]\text{.}\)
    \begin{equation*} G(50) = \frac{5}{9}(50-32) = 10 \end{equation*}
    and
    \begin{equation*} G(85) = \frac{5}{9}(85-32) = \frac{265}{9}= 29\frac{4}{9}\text{.} \end{equation*}
    We know that a linear functionâ€™s range will stay within these two values based on the shape of the graph. Thus the range of \(H\) is in degrees Celsius instead of Fahrenheit, and is \([10,29\frac{4}{9}]\text{.}\)

    Subsection Definition of Composition of Functions

    Suppose we have two function, \(g\) and \(f\text{.}\) \(g\) has domain \(A\) and range \(B\text{,}\) while \(f\) has domain \(B\) and range \(C\text{.}\) We write \(g\) and \(f\) along with their domains and ranges as \(g:A\to B\) and \(f:B\to C\text{.}\) As long as the range of \(g\) matches the domain of \(f\text{,}\) it is possible to link the two processes together to create a new process that we call the composition of \(f\) and \(g\).
    ðŸ”—

    Definition 1.9.4. The Composition of Two Functions.

    If \(f\) and \(g\) are functions such that \(g:A\to B\) and \(f:B\to C\text{,}\) we define the composition of \(f\) and \(g\) to be the new function \(h:A\to C\) given by
    \begin{equation*} h(t) = f(g(x))\text{.} \end{equation*}
    We also sometimes use the notation \(h = f\circ g\text{,}\) where \(f\circ g\) is the single function defined by
    \begin{equation*} (f\circ g)(x) = f(g(x)) \end{equation*}
    ðŸ”—
    ðŸ”—
    We sometimes call \(g\) the â€œinner functionâ€ and \(f\) the â€œouter functionâ€. It is important to note that the inner function is actually the first function that gets applied to a given input, and then outer function is applied to the output of the inner function. In addition, in order for a composite function to make sense, we need to ensure that the range of the inner function lies within the domain of the outer function so that the resulting composite function is defined at every possible input.
    ðŸ”—
    In the following examples, we consider compositions of functions given by tables, then by graphs, and finally by formulas.
    ðŸ”—

    Example 1.9.6.

    Suppose that \(f\) and \(g\) are given by the table
    ðŸ”—
    ðŸ”—

    ðŸ”—
    We can calculate \((f\circ g)(2) = f(g(2))\text{.}\) Since \(g(2) = 0\) and \(f(0) = 6\text{,}\) \(f(g(2)) = f(0) = 6\text{.}\)
    ðŸ”—
    We can also calculate \((g\circ f)(2) = g(f(2))\text{.}\) Since \(f(2) = 3\) and \(g(3)=4\text{,}\) \(g(f(2)) = 4\text{.}\)

    Example 1.9.10.

    Suppose that
    \begin{equation*} f(x) = x^2-1 \end{equation*}
    and that
    \begin{equation*} g(x) = 3x-\frac{4}{x} \end{equation*}
    ðŸ”—
    The composition \(f\circ g\) means that we use \(g(x)\) as the inputs for the function \(f\text{.}\) Think of \(f\) as the process that squares its input, then subtracts 1:
    \begin{equation*} f(?) = (?)^2-1 \end{equation*}
    ðŸ”—
    If the input is \(g(x) = 3x-\frac{4}{x}\text{,}\) we can simplify:
    \begin{equation*} \begin{aligned}(f\circ g)(x) \amp = f(g(x))\\ \amp = (g(x))^2-1\\ \amp = \left(3x-\frac{4}{x}\right)^2-1 \end{aligned} \end{equation*}
    ðŸ”—
    We could simplify this further if we wanted to, thought it isnâ€™t really necessary:
    \begin{equation*} \begin{aligned}(f\circ g)(x) \amp = \left(3x-\frac{4}{x}\right)^2-1\\ \amp = \left(9x^2-24+\frac{16}{x^2}\right)-1\\ \amp = 9x^2+\frac{16}{x^2}-25 \end{aligned} \end{equation*}
    ðŸ”—
    We could also compose the other direction. The composition \(g\circ f\) means that we use \(f(x)\) as the inputs for the function \(g\text{.}\) Think of \(g\) as the function that multiplies 3 times its input, divides 4 by its input, then subtracts the latter from the former:
    \begin{equation*} g(?) = 3(?)-\frac{4}{?} \end{equation*}
    ðŸ”—
    If the input is \(f(x) = x^2-1\text{,}\)
    \begin{equation*} \begin{aligned}(g\circ f)(x) \amp = g(f(x))\\ \amp = 3(f(x))-\frac{4}{f(x)}\\ \amp = 3(x^2-1)-\frac{4}{x^2-1} \end{aligned} \end{equation*}
    ðŸ”—
    We could simplify a littler further if we wanted to by distributing the 3, but anything farther would be a waste of time (like writing it as a single fraction over a common denominator):
    \begin{equation*} \begin{aligned}(g\circ f)(x) \amp = 3(x^2-1)-\frac{4}{x^2-1}\\ \amp = 3x^2-3-\frac{4}{x^2-1} \end{aligned} \end{equation*}

    Subsection Domain of Compositions

    Suppose we have two functions, \(y=f(x)\) and \(x=g(t)\text{.}\) The domain of \(f\circ g\) is the set of all \(t\) in the domain of \(g\) such that \(x=g(t)\) is in the domain of \(f\text{.}\) In other words, \((f\circ g)(t)\) is defined whenever both \(g(t)\) and \(f(g(t))\) are defined.
    ðŸ”—

    Example 1.9.13.

    For \(y=f(x) = \frac{1}{x}\) and \(x=g(t) = \sqrt{t+1}\text{,}\) we find the domain of \(f\circ g\text{.}\)
    ðŸ”—
    For a value \(t\) to be in the domain of \(f\circ g\text{,}\) it must first be in the domain \([-1,\infty)\) of \(g\text{.}\) Then the value of \(g(t) = \sqrt{t+1}\) must be in the domain of \(f\text{,}\) so we must have \(\sqrt{t+1}\neq 0\text{,}\) or in other words, \(t\neq 1\text{.}\) Therefore the domain of \(f\circ g\) is \((-1,\infty)\text{.}\)
    ðŸ”—
    For values of \(t\) in this domain, we have
    \begin{equation*} (f\circ g)(t) = f(g(t)) = f(\sqrt{t+1}) = \frac{1}{\sqrt{t+1}}\text{.} \end{equation*}
    Notice that this equation is consistent with our calculation of the domain.
    ðŸ”—
    ðŸ”—

    Checkpoint 1.9.14.

    Example 1.9.15.

    Recall the functions \(f(x) = x^2-1\) and \(g(x) = 3x-\frac{4}{x}\) from Example ExampleÂ 1.9.10. Letâ€™s find the domains of \(f\circ g\) and \(g\circ f\text{.}\)
    ðŸ”—
    The domain of \(g(x)\) is \(x\neq 0\text{.}\) The domain of \(f(x)\) is all real numbers, so there is no need to limit the outputs of \(g\text{.}\) Hence the domain is \((-\infty,0)\cup (0,\infty)\text{.}\) We found a simplified formula of
    \begin{equation*} (f\circ g)(x) = 9x^2+\frac{16}{x^2}-25\text{,} \end{equation*}
    which agrees with our previous analysis.
    ðŸ”—
    In the other direction, the domain of \(f(x) = x^2-1\) has no restrictions, but when we plug \(f(x)\) into \(g(x) = 3x-\frac{4}{x}\text{,}\) we notice that this requires the input to be nonzero because of the \(x\) in the denominator, hence we must have \(f(x) = x^2-1\neq 0\) in order for \(g(f(x))\) to be defined. Thus the domain for \(g\circ f\) should be \(x\neq \pm1\text{.}\) Indeed, we found a simplified formula of
    \begin{equation*} (g\circ f)(x) = 3x^2-3-\frac{4}{x^2-1} \end{equation*}
    ðŸ”—
    Since \(x^2-1\) is in the denominator, we cannot have \(x=1\) or \(x=-1\text{.}\) Thus the domain is indeed \((-\infty,-1)\cup(-1,1)\cup(1,\infty)\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.9.16. A Cautionary Example.

    Letâ€™s consider the functions \(f(x) =g(x) = \frac{1}{x}\) and look at the domain of \(f\circ g\text{.}\) If we view \(g(x)\) as the first function applied to \(x\text{,}\) we know that we must have \(x\neq 0\text{.}\)
    ðŸ”—
    However, if we first simplify the formula for the composition:
    \begin{equation*} \begin{aligned}(f\circ g)(x) \amp = f(g(x))\\ \amp = \frac{1}{g(x)}\\ \amp =\frac{1}{1/x}\\ \amp = x \end{aligned} \end{equation*}

    Subsection Function Composition and Average Rate of Change

    In this section, we see how the idea of composition can be used in concert with the formula for average rate of change. You have already seen a few WeBWorK problems that involve doing this! Hopefully this section clears up any lingering confusion.
    ðŸ”—
    Recall that the average rate of change of a function \(f\) on the interval \([a,b]\) is given by
    \begin{equation*} AV_{[a,b]} = \frac{f(b)-f(a)}{b-a}\text{.} \end{equation*}
    In the figure below, we see the familiar representation of \(AV_{[a,b]}\) as the slope of the line joining the points \((a,f(a))\) and \((b,f(b))\) on the graph of \(f\text{.}\) In the study of calculus, we progress from the average rate of change on an interval to the instantaneous rate of change of a function at a single value (what we might want to call the "slope" of a nonlinear curve). The core idea that allows us to move from an average rate to an instantaneous one is letting the interval \([a,b]\) shrink in size.
    ðŸ”—
    To think about the interval \([a,b]\) shrinking while \(a\) stays fixed, we often change our perspective and think of \(b\) as \(a+h\text{,}\) where \(h\) measures the horizontal difference from \(b\) to \(a\text{.}\) See FigureÂ 1.9.17 below.
    ðŸ”—
    This allows us to eventually think about \(h\) getting closer and closer to \(0\text{,}\) and in that context we consider the equivalent expression
    \begin{equation*} AV_{[a,a+h]} = \frac{f(a+h)-f(a)}{(a+h)-a} =\frac{f(a+h)-f(a)}{h} \end{equation*}
    for the average rate of change of \(f\) on \([a,a+h]\text{.}\)
    ðŸ”—
    In this most recent expression for \(AV_{[a,a+h]}\text{,}\) we see the important role that the composite function "\(f(a+h)\)" plays. In particular, to understand the expression for \(AV_{[a,a+h]}\) we need to evaluate \(f\) at the quantity \((a+h)\text{.}\)
    ðŸ”—

    Example 1.9.18.

    Suppose that \(f(x) = x^2+1\text{.}\) Determine the simplest possible expression you can find for \(AV_{[3,3+h]}\text{,}\) the average rate of change of \(f\) on the interval \([3,3+h]\text{.}\)
    ðŸ”—
    Solution.
    By definition, we know that
    \begin{equation*} AV_{[3,3+h]} = \frac{f(3+h)-f(3)}{h} \end{equation*}
    ðŸ”—
    Using the formula for \(f\text{,}\) we have \(f(3+h) = (3+h)^2+1\) and \(f(3) = 3^2+1\text{,}\) so that
    \begin{equation*} AV_{[3,3+h]} = \frac{((3+h)^2+1)-(3^2+1)}{h} \end{equation*}
    ðŸ”—
    Expanding the numerator and combining like terms, it follows that
    \begin{equation*} \begin{aligned}AV_{[3,3+h]} \amp = \frac{9+6h+h^2+1-10}{h}\\ \amp = \frac{6h+h^2}{h}\\ \amp =\frac{h(6+h)}{h}\\ \amp = 6+h. \end{aligned} \end{equation*}
    ðŸ”—
    Hence \(AV_{[3,3+h]} = 6+h\text{,}\) which is the average rate of change of \(f(x) = x^2+1\) on the interval \([3,3+h]\text{.}\)
    ðŸ”—
    ðŸ”—
    ðŸ”—
