    Subsection 6.3.2 Defining Logarithmic Functions

    We saw at the end of the last section in TheoremÂ 6.2.14 that every exponential function \(f(x) = b^x\) passes the Horizontal Line Test, and so every exponential function has an inverse. In this section, we will define and explore inverse functions to the exponential functions. These inverse functions have a specific name: logarithmic functions.
    ðŸ”—

    Definition 6.3.1. Logarithmic Functions as Inverses of Exponential Functions.

    The inverse of the base-\(b\) exponential function \(b^x\) is the base-\(b\)logarithmic function
    \begin{equation*} \log_b x \end{equation*}
    which is the power that we raise \(b\) to to get \(x\text{.}\) In other words,
    \begin{equation*} y=\log_b x \Leftrightarrow b^y=x \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 6.3.2. Examples of Finding Values of Logarithms.

    The following examples demonstrate DefinitionÂ 6.3.1 in action:
    1. \(\log_2 8\) is another way of asking "To what power do we raise 2 to get 8?" Since \(2^3=8\text{,}\) we have
      \begin{equation*} 2^3=8 \Leftrightarrow \log_2 8=3\text{.} \end{equation*}
      ðŸ”—
      ðŸ”—
    2. \(\log_{36} 6\) is another way of asking "To what power do we raise 36 to get 6?" Since \(\sqrt{36} = 36^{1/2}=6\text{,}\) we have
      \begin{equation*} 36^{1/2}=6 \Leftrightarrow \log_{36} 6=\frac{1}{2}\text{.} \end{equation*}
      ðŸ”—
      ðŸ”—
    3. \(\log_7 \frac{1}{49}\) is another way of asking "To what power do we raise 7 to get \(\frac{1}{49}\text{?}\)" Since \(7^2=49\text{,}\) \(7^{-2} = \frac{1}{49}\text{,}\) thus we have
      \begin{equation*} 7^{-2} = \frac{1}{49} \Leftrightarrow \log_7 \frac{1}{49}=-2\text{.} \end{equation*}
      ðŸ”—
      ðŸ”—
    4. \(\log_b 1\) is another way of asking "To what power do we raise \(b\) to get 1?" Since \(b^0=1\) for any base \(b\text{,}\) we have
      \begin{equation*} b^0=1 \Leftrightarrow \log_b 1=0\text{.} \end{equation*}
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    \fbox{As a special case, the inverse of the base-\(10\) exponential function \(10^x\) is often written without an explicit base:
    \begin{equation*} \log_{10} x = \log x\text{,} \end{equation*}
    This is the power to which we raise \(10\) to get \(x\text{:}\)
    \begin{equation*} y=\log_{10}x \Leftrightarrow 10^y=x \end{equation*}
    ðŸ”—
    Why single out the base-\(10\) logarithm as the meaning of \(\log\) without an explicit base? This convention varies by discipline and is most common in the sciences where people make frequent use of scientific notation, where very large numbers are written in terms of powers of 10, such as
    \begin{equation*} 5,000,000,000 = 5\times10^9\text{.} \end{equation*}
    ðŸ”—
    In computer science, people are more likely to use base-\(2\) logarithms without an explicit base due to the use of powers of 2 for the size of data (a byte is \(2^8\text{,}\) a kilobyte is \(2^{10}\text{,}\) a megabyte is \(2^{20}\text{,}\) etc.). Mathematicians typically use a base of \(e\text{,}\) which you will learn about in a future section.
    ðŸ”—
    Note that on WeBWorK, the function \(\log\) denotes \(\log_e\text{,}\) rather than either \(\log_{10}\) or \(\log_{2}\text{.}\) On WeBWorK, you will need to enter \(\log_{10}(x)\) as "log10\((x)\text{.}\)"
    ðŸ”—
    We require that the base \(b\) satisfy \(b>0\) and \(b\neq 1\text{,}\) because these are exactly the conditions we must have for \(y=b^x\) to be an exponential function.

    Example 6.3.7.

    For instance,
    \begin{equation*} 10^2 = 100\text{ and } \log_{10}(100) = 2 \end{equation*}
    each say the same thing from two different perspectives. The first says "100 is 10 to the power 2", while the second says "2 is the power to which we raise 10 to get 100." Similarly
    \begin{equation*} 10^{-3} = \frac{1}{1000}\text{ and } \log_{10}\left(\frac{1}{1000}\right) = -3\text{.} \end{equation*}
    ðŸ”—
    The first says "10 to the power of \(-3\) is \(\frac{1}{1000}\)" while the second says "\(-3\) is the power to which we raise 10 to get \(\frac{1}{1000}\text{.}\)
    ðŸ”—
    ðŸ”—
    Remember that we began by defining inverse functions as reverse processes of each other via the identities
    \begin{equation*} f(f^{-1}(x))=x\qquad\text{ and } \qquad f(f^{-1}(x))=x \end{equation*}
    ðŸ”—
    This is also true for exponential functions and logarithms.
    ðŸ”—

    Example 6.3.8.

    If we rearrange the statements of the facts in \(10^2 = 100\) and \(\log_{10}(100) = 2\text{,}\) we can see yet another important relationship between the powers of 10 and base-10 logarithm function. Noting that \(\log_{10}(100) = 2\) and \(100 = 10^2\) are equivalent statements, and substituting the latter equation into the former, we see that
    \begin{equation*} \log_{10}(10^2) = 2\text{.} \end{equation*}
    ðŸ”—

    Subsection 6.3.4 The Logarithm Rules

    In this section, we cover a list of rules for manipulating logarithms. These can be memorized, if you like. However, it is far more satisfying (in my opinion) to have some idea why these rules are true. We state the rules first and give justifications for them in the next section.     For any positive real numbers \(c\) and \(d\) and any real number \(n\text{,}\)
    1. \(\displaystyle \log_b(c\cdot d) = \log_b(c)+\log_b(d)\)

      ðŸ”—
      ðŸ”—
    3. \(\log_b(c^n) = n\log_b(c)\text{.}\)

    Example 6.3.14. Simplifying Logarithmic Expressions.

    We can use the logarithm rules to simplify expressions:
    1. We simplify \(\log_64+2\log_63\) without needing a calculator. First observe that we can use rule (c) to rewrite
      \begin{equation*} 2\log_6 3 = \log_6 9 \end{equation*}
      and then use rule (a) to combine terms, multiplying the arguments of both logarithms:
      \begin{equation*} \log_6 4+\log_6 9 = \log_6(4\cdot 9) = \log_6 36\text{.} \end{equation*}
      We know that \(36 = 6^2\) and so by TheoremÂ 6.3.9 part (e),
      \begin{equation*} \log_6 36 = \log_6 6^2 = 2\text{.} \end{equation*}
      In this way, we fully simplify
      \begin{equation*} \log_64+2\log_63=2 \end{equation*}
      without needing to touch a calculator.

    Example 6.3.16.

    Hawksbill turtles are endangered and are extensively hunted as the traditional source of tortoiseshell. They currently have a population of 25,000 and the population is decreasing at a rate of 12% each year. How long will it be until there are under 5,000 hawksbill turtles left?

    \begin{equation*} P(t) = 25000(0.88)^t\text{,} \end{equation*}

    \begin{equation*} 25000(0.88)^t = 5000\text{.} \end{equation*}

    ðŸ”—

    Here is one final useful property of logarithms: the change of base formula:
    ðŸ”—

    Theorem 6.3.17. Change of Base Formula for Logarithms.

    For any positive real numbers \(d, a, b\) such that neither \(d\) nor \(b\) are \(1\text{,}\) we have
    \begin{equation*} \log_b a = \frac{\log_d a}{\log_d b} \end{equation*}
    ðŸ”—
    ðŸ”—
    This allows us to rewrite a logarithm in base \(b\) in terms of logarithms in any base \(d\text{.}\)
    ðŸ”—

    Example 6.3.18.

    The change of base formula is useful for simplifying certain gross computations involving logarithms. For example,
    \begin{equation*} \log_{1/4} 32\sqrt{2} \end{equation*}
    is a bit eye-melting. However, notice that all of the numbers involved are powers of \(2\text{.}\) Thus we might think to use the change of base formula with \(d=2\text{:}\)
    \begin{equation*} \log_{1/4} 32\sqrt{2} = \frac{\log_2 32\sqrt{2}}{\log_2(1/4)} \end{equation*}
    ðŸ”—
    We now have two simpler, more manageable logarithms to deal with:
    \begin{equation*} \log_2 32\sqrt{2}\qquad\text{ and } \qquad \log_2(1/4) \end{equation*}
    ðŸ”—
    We could convert \(32\sqrt{2}\) into a power of 2, or we could first use Logarithm Rule (a) to make this calculation even simpler:
    \begin{equation*} \log_2(32\sqrt{2}) = \log_2 32+\log_2\sqrt{2} = \log_2 2^5+\log_2 2^{1/2} = 5+\frac{1}{2} = \frac{11}{2} \end{equation*}
    ðŸ”—
    Similarly, we could either convert \(1/4 = 2^{-2}\) in order to see that \(\log_2(1/4) = -2\text{,}\) or we could use Logarithm Rule (b):
    \begin{equation*} \log_2\left(\frac{1}{4}\right) = \log_2 1-\log_2 4 = \log_2 2^0-\log_2 2^2 = 0-2 = -2\text{.} \end{equation*}
    ðŸ”—
    Putting all of this together, we see that:
    \begin{equation*} \begin{aligned}\log_{1/4} 32\sqrt{2} \amp = \frac{\log_2 32\sqrt{2}}{\log_2(1/4)}\\ \amp = \frac{11/2}{-2}\\ \amp = -\frac{11}{4} \end{aligned} \end{equation*}
    ðŸ”—
    Which method is best? That depends entirely on which you think of first and can do most accurately. Just make sure not to make up your own "rules."

    Subsection 6.3.5 Justifications for the Logarithm Rules

    Recall the Logarithm Rules from TheoremÂ 6.3.13: For any positive real numbers \(c\) and \(d\) and any real number \(n\text{,}\)
    1. \(\displaystyle \log_b(c\cdot d) = \log_b(c)+\log_b(d)\)
      ðŸ”—
      ðŸ”—
    2. \(\displaystyle \log_b\left(\dfrac{c}{d}\right) = \log_b(c)-\log_b(d)\)
      ðŸ”—
      ðŸ”—
    3. \(\log_b(c^n) = n\log_b(c)\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    Also recall the Change of Base Formula for Logarithms: For any positive real numbers \(d, a, b\) such that neither \(d\) nor \(b\) are \(1\text{,}\) we have
    \begin{equation*} \log_b a = \frac{\log_d a}{\log_d b} \end{equation*}
    ðŸ”—
    As mentioned earlier, you can memorize and use these without knowing why they are true. However, many people remember better with some kind of understanding of why a formula is valid. They are a result of the exponent rules. It is your choice whether you read the justifications and proofs below, and if you only have time to read one, I recommend the justification for part (c) of TheoremÂ 6.3.13.
    ðŸ”—
    Justification of TheoremÂ 6.3.13 part (a): Letâ€™s begin by working with a base of 10 instead of an arbitrary base \(b\text{.}\) Recall some of the rules for exponents in TheoremÂ 2.6.1. In particular, we know that
    \begin{equation*} 10^x\cdot 10^y = 10^{x+y}\text{.} \end{equation*}
    Applying \(\log_{10}\) to both sides, we have
    \begin{equation*} \begin{aligned}\log_{10}(10^x\cdot 10^y) \amp = \log_{10}(10^{x+y}) \\ \amp = x+y \end{aligned} \end{equation*}
    where the second equality is due to part (e) of TheoremÂ 6.3.9. Part (e) of TheoremÂ 6.3.9 also tells us that \(x = \log_{10}(10^x)\) and \(y = \log_{10}(10^y)\text{,}\) so that
    \begin{equation*} \begin{aligned}\log_{10}(10^x\cdot 10^y) \amp = \log_{10}(10^{x+y}) \\ \amp = x+y\\ \amp = \log_{10}(10^x)+\log_{10}(10^y) \end{aligned} \end{equation*}
    ðŸ”—
    Thus we have justified why, for any positive numbers \(c=10^x\) and \(d = 10^y\text{,}\)
    \begin{equation*} \log_b(c\cdot d) = \log_b(c)+\log_b(d)\text{.} \end{equation*}
    This argument is valid because \(c=10^x\) and \(d=10^y\) can indeed be equal to any positive number for suitable values of \(x\) and \(y\text{,}\) and the range of \(10^x\) or \(10^y\) is all positive real numbers.
    ðŸ”—
    Proof of TheoremÂ 6.3.13 part (b): In order to call this a "proof" rather than a "justification," we will use the arbitrary base of \(b\) instead of an explicit base. Let \(c=b^x\) and \(d=b^y\text{.}\) Recall this means that
    \begin{equation*} x=\log_b(c)\qquad\text{ and } \qquad y=\log_b(d)\text{.} \end{equation*}
    By nearly the same argument as in the justification for part (a), we can say that
    \begin{equation*} \begin{aligned}\log_b\left(\frac{c}{d}\right) \amp = \log_b\left(\frac{b^x}{b^y}\right)\\ \amp = \log_b(b^{x-y})\\ \amp = x-y\\ \amp =\log_b(c)-\log_b(d) \end{aligned} \end{equation*}
    ðŸ”—
    Justification of TheoremÂ 6.3.13 part (c): Because positive integer exponents are a shorthand way to express repeated multiplication, we can use the multiplication rule for logarithms to think about exponents as well. For example,
    \begin{equation*} \log_b(a^3) = \log_b(a\cdot a\cdot a) \end{equation*}
    and by repeated applications of part (a) of the Logarithm Rules, we see
    \begin{equation*} \log_b(a^3) = \log_b(a\cdot a\cdot a)= \log_b(a) + \log_b(a) + \log_b(a) = 3\log_b(a)\text{.} \end{equation*}
    A similar argument works to show that for every natural number \(n\text{,}\)
    \begin{equation*} \log_b(a^n) = n\log_b(a)\text{.} \end{equation*}
    More sophisticated mathematics can be used to prove that this property holds for every real number exponent, not just for natural numbers.
    ðŸ”—
    Proof of the Change of Base Formula: Let \(y = \log_b a\text{.}\) Then \(b^y = a\text{.}\) Take \(\log_d\) of both sides to get
    \begin{equation*} \log_d(b^y) = \log_d(a)\text{.} \end{equation*}
    By part (c) of TheoremÂ 6.3.13,
    \begin{equation*} y\log_db = \log_da\text{.} \end{equation*}
    Substituting for \(y\text{,}\)
    \begin{equation*} \log_ba\cdot\log_d b = \log_d a\text{.} \end{equation*}
    Finally, dividing both sides by \(\log_b a\text{,}\) we have the change of base formula:
    \begin{equation*} \log_b a = \frac{\log_d a}{\log_d b} \end{equation*}
