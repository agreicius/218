      🔗

    Subsection 5.4.2 The First Derivative Test

    Given the graph of a function, it is straightforward to visually locate any local minimum or maximum points. With our new knowledge from Theorem Theorem 5.3.12 in the last section, we even have a computational way to identify the precise \(x\)-value where a local minimum or maximum occurs via critical points. However, we would like to be able to work without a graph entirely. In order to accomplish this, we develop a method of using the sign of the first derivative to classify when a critical point is a local minimum, maximum, or neither.
    🔗
    We know from Theorem 5.3.12 that every local maximum or minimum must occur at a critical point, that is a point \(x=c\) in the domain of \(f\) where \(f'(c)=0\) or \(f'(c)\) does not exist. So critical points are candidates for where a function may have a local minimum or maximum. Remember that not all critical points are local extrema! See Example 5.3.13. The first derivative is our tool for determining whether a function actually has a local minimum or maximum at its critical points.
    🔗
    Before seeing how, exactly, we must collect some results connecting the sign of the first derivative with whether a function is increasing or decreasing. We have explored some of these ideas already when we first defined derivatives, but in order to justify them formally, we need the Mean Value Theorem. Theorem Theorem 5.4.1 is proven formally at the end of this section.
    🔗

    Theorem 5.4.1. The First Derivative Measures Where a Function is increasing or Decreasing.

    Let \(f\) be a function that is differentiable on an interval \(I\text{.}\)
    1. If \(f'\) is positive on \(I\text{,}\) then \(f\) is increasing on \(I\text{.}\)
      🔗
      🔗
    2. If \(f'\) is negative on \(I\text{,}\) then \(f\) is decreasing on \(I\text{.}\)
      🔗
      🔗
    3. If \(f'\) is zero on \(I\text{,}\) then \(f\) is constant on \(I\text{.}\)
      🔗
      🔗
    🔗
    🔗
    Intuitively, the first derivative \(f'\) in some sense measures the direction of the graph of a function \(f\) at each point, since \(f'\) is the associated slope function for \(f\text{.}\) In particular, if \(f'\) is positive at a point \(x=c\text{,}\) then the graph of \(f\) must be moving in an upwards direction, that is, increasing, as it passes \(x=c\text{.}\) Similarly, if \(f'(c)\) is negative, then the graph of \(f\) must be decreasing at \(x=c\text{.}\)
    🔗

    Example 5.4.2.

    For example, we can divide the real number line into intervals according to where the function \(f(x) = x^3-3x^2-9x+11\) is increasing or decreasing, as shown in the figure that follows at the left. This same division into subintervals descries where the derivative \(f'(x)=3x^2-6x-9\) is positive and negative, as shown at the right.
    🔗
    🔗
    The function \(f(x) = x^3-3x^2-9x+11\) is increasing on \((-\infty,-1)\cup (3,\infty)\) where \(f'(x) = 3x^2-6x-9\) is positive, and decreasing on \((-1,3)\) where \(f'(x) = 3x^2-6x-9\) is negative.
    🔗
    🔗
    Now let’s consider the case of a critical point where \(f'(c)=0\) and \(f\) is differentiable near \(c\text{.}\) Then as long as \(f\) is not constant, there are four different ways that \(f\) can behave near \(x=c\text{:}\)
    🔗
    🔗
    On the one hand, if \(f\) changes from increasing to decreasing at \(x=c\text{,}\) then \(f\) has a local maximum at \(x=c\text{.}\) If \(f\) changes from decreasing to increasing at the point \(x=c\text{,}\) then \(f\) has a local minimum at \(x=c\text{.}\) On the other hand, if \(f\) does not change direction at \(x=c\text{,}\) then \(f\) does not have a local extremum at \(x=c\text{.}\) Since a function \(f\) is increasing or decreasing according to whether its derivative is positive or negative, we can record the preceding four situations on sign charts for the derivative \(f'\text{,}\) with the corresponding information for \(f\) recorded as the following sign analyses:
    🔗
    🔗
    The first derivative test summarizes how sign changes in the first derivative (which can only occur at critical points thanks to the Intermediate Value Theorem) indicate the presence of a local maximum or minimum for a given function.
    🔗

    Theorem 5.4.3. The First Derivative Test.

    If \(c\) is a critical number of a continuous function \(f\) that is differentiable near \(c\) (except possibly at \(x=c\)), then
    1. \(f\) has a local maximum at \(c\) if and only if \(f'\) changes sign from positive to negative at \(c\text{.}\)
      🔗
      🔗
    2. \(f\) has a local minimum at \(c\) if and only if \(f'\) changes sign from negative to positive at \(c\text{.}\)
      🔗
      🔗
    3. \(f\) has neither a local minimum nor a local maximum at \(c\) if \(f'\) does not change sign at \(c\text{,}\) meaning \(f'\) is positive on both sides of \(c\) or \(f'\) is negative on both sides of \(c\text{.}\)
      🔗
      🔗
    🔗
    🔗

    Remark 5.4.4.

    Note that we are using the sign of the first derivative to give us information about whether the function is increasing or decreasing. The sign of the first derivative does not tell us anything about whether the original function itself is positive or negative.
    🔗
    🔗

    Example 5.4.5.

    Let’s determine where the function \(f(x) = x^2-2x+1\) is increasing and decreasing, and where it has a local maximum or minimum. (Note that \(f\) is a parabola so you may already have some ideas about whether it will have a max or a min.)
    🔗
    First, we locate the critical points of \(f\text{,}\) which are the candidates for local extrema. To do this, we take the derivative and check where it is equal to 0 or undefined:
    \begin{equation*} f'(x) = 2x-2 \end{equation*}
    and there are critical points where \(f'(x) = 2x-2=0\) (at \(x=1\)) or undefined (nowhere, since this expression is always defined).
    🔗
    Now we create a number line for the derivative \(f'(x) = 2x-2\text{,}\) marking the point \(x=1\) where it could change sign:
    🔗
    Recall our method for testing a point on each interval between tick marks: if we plug in, say, \(x=0\text{,}\) we have \(f'(0) = -2\lt 0\) and so \(f'(x)\lt 0\) for all \(x\) in \((-\infty,1)\text{.}\) If we plug in, say, \(x=2\text{,}\) we have \(f'(2) = 2>0\) and so \(f'(x)>0\) for all \(x\) in \((1,\infty)\text{.}\)
    🔗
    By Theorem 5.4.1, the information on the sign chart for \(f'(x)\) shows that \(f(x)\) is decreasing on \((-\infty,1)\) and increasing on \((1,\infty)\text{.}\)
    🔗
    By the First Derivative Test (Theorem 5.4.3), we know that \(f(x)\) has a local minimum at \(x=1\text{.}\) The \(y\)-value of the local minimum is \(y=f(1) = (1)^2-2(1)+1 = 0\text{,}\) so the minimum point is at \((1,0)\text{.}\)
    🔗
    We can check our answer using what we know about parabolas: \(f(x) = x^2-2x+1\) has a positive leading coefficient \(a=1\text{,}\) so \(f(x)\) is a concave up parabola and thus must have a minimum at its vertex.
    🔗
    🔗

    Example 5.4.6.

    Determine the critical points of
    \begin{equation*} g(x) = \frac{x^2+10x+1}{x-2} \end{equation*}
    and use the first derivative test to classify each as a local maximum, minimum, or neither.
    🔗
    Solution.
    First, let’s observe that \(g(x)\) is undefined at \(x=2\) since that is where the denominator is equal to 0. (Think about what kind of discontinuity \(g\) has at \(x=2\text{.}\))
    🔗
    In order to find critical points, we wish to know where \(g'(x)\) is either 0 or undefined. We find \(g'(x)\) using the quotient rule and simplify fully in order to determine where it is 0 or undefined:
    \begin{equation*} \begin{aligned}g'(x) \amp = \frac{(x-2)(2x+10) - (x^2+10x+1)(1)}{(x-2)^2}\\ \amp = \frac{2x^2+6x-20-(x^2+10x+1)}{(x-2)^2}\\ \amp = \frac{x^2-4x-21}{(x-2)^2}\\ \amp = \frac{(x+3)(x-7)}{(x-2)^2} \end{aligned} \end{equation*}
    🔗
    Note that the "simplest" form of the numerator is factored in order to be able to tell where \(g'(x)=0\text{.}\) We also left the denominator factored because it’s easier to see where the denominator is zero (where \(g'(x)\) is undefined) when it’s in this form.
    🔗
    Now we see that \(g'(x)=0\) when \(x=-3, 7\) and that \(g'(x)\) is undefined when \(x=2\text{.}\) But remember that the original function \(g(x)\) is undefined when \(x=2\) and that critical points must be in the domain of the function. So only \(x=-3\) and \(x=7\) are critical points of \(g(x)\text{.}\) In particular, since \(g(2)\) is undefined, \(x=2\) cannot be a local max or min. This is why non-domain points are not included in the list of critical points!
    🔗
    We would like to use the First Derivative Test to classify the critical points. We create a sign chart for \(g'\) in order to know where it is positive and negative. For the sign chart, we need to test intervals between all three of the \(x\)-values \(x=-3\text{,}\) \(x=7\text{,}\) and \(x=2\text{,}\) even though \(x=2\) is not a critical point. This is because we need to know where \(g'\) can change sign for the sign chart, which includes all of the critical points as well as any additional points where \(g'\) is undefined.
    🔗
    Since \(g'\) changes from increasing to decreasing at \(x=-3\text{,}\) there is a local maximum at \(x=3\text{.}\) The \(y\)-value on the graph is \(y=g(3)=40\text{,}\) so there is a local maximum at the point \((3,40)\text{.}\) Since \(g'\) changes from decreasing to increasing at \(x=7\text{,}\) there is a local minimum at \(x=7\text{.}\) The \(y\)-value on the graph is \(y=g(7)=24\text{,}\) so there is a local minimum at the point \((7,24)\text{.}\)

    Subsection 5.4.3 Concavity

    In addition to asking whether a function is increasing or decreasing, it is also natural to inquire how a function is increasing or decreasing. There are three basic behaviors that an increasing function can demonstrate on an interval, as pictured below: the function can increase more and more rapidly, it can increase at the same rate, or it can increase in a way that is slowing down. Fundamentally, we are beginning to think about how a particular curve bends, with the natural comparison being made to lines, which don’t bend at all. More than this, we want to understand how the bend in a function’s graph is tied to behavior characterized by the first derivative of the function.
    🔗
    For the leftmost curve above, we draw a sequence of tangent lines to the curve:
    🔗
    As we move from left at \(A\) to right at \(B\text{,}\) the slopes of those tangent lines will increase. Therefore, the slope, a.k.a. rate of change, of the pictured function is increasing, and we say this function is increasing at an increasing rate. In other words, \(f'\) is increasing.
    🔗
    For the rightmost graph above, observe that as \(x\) increases from \(A\) to \(B\text{,}\) the function increases, but the slopes of the tangent lines decrease.
    🔗
    This function is increasing at a decreasing rate, meaning \(g'\) is decreasing.
    🔗
    Similar options hold for how a function can decrease. Here we must be extra careful with our language, because decreasing functions involve negative slopes. Negative numbers present an interesting tension between common language and mathematical language. For example, it can be tempting to say that "\(-100\) is bigger than \(-2\text{.}\)" But we must remember that “greater than” describes how numbers lie on a number line: \(x>y\) provided that \(x\) lies to the right of \(y\text{.}\) So \(-2\) is actually bigger than \(-100\text{.}\) Informally, it might be helpful to say that "\(-100\) is more negative than \(-2\text{.}\)" When a function’s values are negative, and those values get more negative as the input increases, the function is actually decreasing.
    🔗
    From left to right below, we see three functions that are all decreasing, but doing so in different ways.
    🔗
    The middle graph depicts a function decreasing at a constant rate.
    🔗

    Checkpoint 5.4.14.

    Draw a sequence of tangent lines on the left-most and right-most graphs above. Which depicts \(f'\) increasing and which depicts \(f'\) decreasing?
    🔗
    🔗
    We now formally introduce the notion of concavity which provides simpler language to describe these behaviors. When a curve opens upward on a given interval, like the parabola \(y=x^2\text{,}\) we say that the curve is concave up on that interval. Likewise, when a curve opens down, like the parabola \(y=-x^2\text{,}\) we say that the function is concave down. Concavity is linked to both the first and second derivatives of the function; it describes the change in the first derivative.
    🔗
    In the pictures below, we see two functions and a sequence of tangent lines to each. On the left-hand plot, where the function is concave up, observe that the tangent lines always lie below the curve itself, and the slopes of the tangent lines are increasing as we move from left to right. In other words, the function \(f\) is concave up on the interval shown because its derivative, \(f'\text{,}\) is increasing on that interval. Similarly, on the right-hand plot below, where the function shown is concave down, we see that the tangent lines always lie above the curve, and the slopes of the tangent lines are decreasing as we move from left to right. The fact that its derivative, \(f'\) is decreasing makes \(f\) concave down on the interval.
    🔗
    🔗
    We state these most recent observations formally as the definitions of the terms concave up and concave down:
    🔗

    Definition 5.4.15. Concavity.

    Let \(f\) be a differentiable function on an interval \(I\text{.}\) Then \(f\) is concave up on \(I\) if and only if \(f'\) is increasing on \(I\text{.}\) \(f\) is concave down on \(I\) if and only if \(f'\) is decreasing on \(I\text{.}\)
    🔗
    Stated differently:
    • If \(f''\) is positive on \(I\text{,}\) then \(f\) is concave up on \(I\text{.}\)
      🔗
      🔗
    • If \(f''\) is negative on \(I\text{,}\) then \(f\) is concave down on \(I\text{.}\)
      🔗
      🔗
    • If \(f''\) is zero on \(I\text{,}\) then \(f\) is linear on \(I\text{.}\)
      🔗
      🔗
    🔗
    🔗
    Compare this to Theorem 5.4.1. While the first derivative gives us information about whether \(f\) is increasing or decreasing, the second derivative gives us information about whether \(f\) is concave up or concave down. The relationship between the function and its first and second derivatives is pictured below:
    🔗
    🔗
    The function \(f(x) = x^3\) is pictured at the left and is concave down on \((-\infty,0)\) and concave up on \((0,\infty)\text{.}\) The derivative \(f'(x) = 3x^2\) is pictured in the middle, and is decreasing on \((-\infty,0)\) and increasing on \((0,\infty)\text{.}\) The second derivative \(f''(x) = 6x\) is picture at the right, and is negative on \((-\infty,0)\) and positive on \((0,\infty)\text{.}\)
    🔗
    Just as a first derivative sign chart reveals all of the increasing and decreasing behavior of a function, we can construct a second derivative sign chart that demonstrates all of the important information involving concavity.

    Subsection 5.4.4 The Second Derivative Test

    We can also use the second derivative to determine whether some critical points are local minima or maxima. At a point where \(f'(c)=0\text{,}\) the sign of the second derivative determines whether \(f\) has a local minimum or local maximum at the critical point \(c\text{.}\)
    🔗
    🔗
    Above, we see the four possibilities for a function \(f\) that has a critical point at \(x=c\) at which \(f'(c)=0\text{,}\) provided \(f''(c)\) is not zero on an interval including \(c\) (except possibly at \(c\)). On either side of the critical point, \(f''\) can be either positive or negative, and hence \(f\) can be either concave up or concave down. In the first two graphs, \(f\) does not change concavity at \(c\text{,}\) and in those situations, \(f\) has either a local minimum or local maximum. In particular, if \(f'(c)=0\) and \(f''(c)\lt 0\text{,}\) then \(f\) is concave down at \(c\) with horizontal tangent line, so \(f\) has a local maximum there. This fact, along with the corresponding statement for when \(f''(c)\) is positive, is the substance of the second derivative test.
    🔗

    Theorem 5.4.17. The Second Derivative Test.

    If \(c\) is a critical number of a continuous function \(f\) such that \(f'(c)=0\text{,}\) then
    1. \(f\) has a local maximum at \(c\) if and only if \(f''(c)\lt 0\)
      🔗
      🔗
    2. \(f\) has a local minimum at \(c\) if and only if \(f''(c)>0\)
      🔗
      🔗
    3. if \(f''(c)=0\text{,}\) this test tells us nothing and we should use a different test (like the First Derivative Test).
      🔗
      🔗
    🔗
    🔗

    Remark 5.4.18.

    In the event that \(f''(c)=0\text{,}\) the second derivative test is "inconclusive," meaning that the test doesn’t provide us any information. This is because if \(f''(c)=0\text{,}\) it is possible that \(f\) has a local minimum, local maximum, or neither. Consider the functions \(f(x) = x^4, g(x) = -x^4\text{,}\) and \(h(x) =x^3\) at the critical point \(c=0\text{,}\) for example.
    🔗
    🔗

    Example 5.4.19.

    We saw in Example 5.4.16 that the first derivative of \(f(x) = 2x^3 -9x^2 +12x-4\) is \(f'(x) = 6x^2-18x+12\) and its second derivative is \(f''(x) = 12x-18\text{.}\) We find the critical points of \(f(x)\) by setting the derivative equal to zero and solving for \(x\text{:}\)
    \begin{equation*} f'(x) = 0 \end{equation*}
    \begin{equation*} 6x^2-18x+12 = 0 \end{equation*}
    \begin{equation*} 6(x^2-3x+2) = 0 \end{equation*}
    \begin{equation*} 6(x-2)(x-1) = 0 \end{equation*}
    and so the critical points are \(x=1, 2\text{.}\) Using the second derivative test, we compute the value of \(f''\) at each of the critical points:
    \begin{equation*} f''(1) = 12(1)-18 = -6\lt 0 \end{equation*}
    \begin{equation*} f''(2) = 12(2)-18 = 6>0 \end{equation*}
    🔗
    Thus we see that \(f\) has a local maximum at \(x=1\text{,}\) since \(f\) is concave down with a horizontal tangent line. \(f\) has a local minimum at \(x=2\text{,}\) since \(f\) is concave up with a horizontal tangent line.
    🔗
    🔗

    Subsection 5.4.5 Proof of Theorem \xautoref{thm_firstderivativeinfo} Using the Mean Value Theorem

    Theorem Theorem 5.4.1 tells us that a positive derivative corresponds to an increasing function, a negative derivative corresponds to a decreasing function, and a derivative of zero corresponds to a constant function, all on an interval that we will call \(I\text{.}\) We have been using this fact informally ever since intuitively defining the derivative as an associated slope function in Section Subsection 4.1.4.
    🔗
    The proof below is not necessary to read for your understanding of derivatives or functions and you can safely skip reading the rest of this section if you are not interested. However, it is a good example of how mathematicians encode intuitive ideas into rigorous definitions and then formally connect those ideas into theorems using deductive arguments. In order to prove that a positive/negative derivative corresponds to an increasing/decreasing function, we must first give a more formal definition of what it means for a function to be increasing or decreasing.
    🔗

    Definition 5.4.22. Formal Definition of Increasing and Decreasing.

    A function \(f\) is increasing on an interval \(I\) if
    🔗
    \(f(a)\lt f(b)\) for any \(a\) and \(b\) in \(I\) with \(a\lt b\text{.}\)
    In other words, if \(b\) is bigger than \(a\text{,}\) then \(f(b)\) is bigger than \(f(a)\text{.}\)
    🔗
    The reverse is true for a decreasing function: A function \(f\) is decreasing on an interval \(I\) if
    🔗
    \(f(a)>f(b)\) for any \(a\) and \(b\) in \(I\) with \(a\lt b\text{.}\)
    In other words, if \(b\) is bigger than \(a\text{,}\) then \(f(b)\) will be smaller than \(f(a)\) if \(f\) is decreasing.
    🔗
    🔗

    Proof.

    First we consider the case where \(f'\) is positive and show that \(f\) must be increasing. Suppose that \(f'(x)\) is positive for all values of \(x\) on the interval \(I\text{,}\) and that \(a\) and \(b\) are values of \(x\) that are in \(I\text{.}\) Since \(f'\) is positive, we know that \(f'\) is defined, thus \(f\) is differentiable and so also continuous by Theorem Theorem 4.3.16. Hence the Mean Value Theorem applies to \(f\) on the interval \([a,b]\text{.}\)
    🔗
    According to the Mean Value Theorem,
    \begin{equation*} \frac{f(b)-f(a)}{b-a} = f'(c) \end{equation*}
    for some \(c\) between \(a\) and \(b\text{.}\) We are assuming that \(f'(c)\) is positive. Observe that \(b-a\) is also positive since \(b\) is bigger than \(a\text{.}\) Hence in the equation above, the right-hand side is positive and the denominator of the left-hand side is positive, thus the numerator of the left-hand side must also be positive. Thus we see that
    \begin{equation*} f(b)-f(a)>0 \Longrightarrow f(b)>f(a)\text{.} \end{equation*}
    This means that \(f(b)\) is bigger than \(f(a)\text{,}\) showing that \(f\) is increasing on \([a,b]\text{.}\) Overall, if we know that \(f'\) is positive, the Mean Value Theorem tells us that \(f\) is increasing.
    🔗
    The same argument works to argue that \(f\) is decreasing if we know that \(f'\) is negative. Again, the Mean Value Theorem applies since we are assuming \(f'\) exists, thus \(f\) is differentiable and continuous. According to the Mean Value Theorem,
    \begin{equation*} \frac{f(b)-f(a)}{b-a} = f'(c) \end{equation*}
    for some \(c\) between \(a\) and \(b\text{.}\) If we know that \(f'(c)\) is negative, then since \(b-a\) is still positive (\(b\) is still bigger than \(a\)), we must have that \(f(b)-f(a)\) is negative so that the fraction on the left-hand side of the equation is negative. Then
    \begin{equation*} f(b)-f(a)\lt 0 \Longrightarrow f(b)\lt f(a)\text{,} \end{equation*}
    meaning that \(f(b)\) is smaller than \(f(a)\text{.}\) Thus, if we know that \(f'\) is negative, the Mean Value Theorem tells us that \(f\) is decreasing.
    🔗
    Lastly, we can show that if \(f'\) is zero, then \(f\) must be constant. In this case, the Mean Value Theorem still applies (since \(f'(x)=0\text{,}\) it does still exist, so \(f\) is differentiable and continuous). Thus
    \begin{equation*} \frac{f(b)-f(a)}{b-a} = f'(c)=0\text{.} \end{equation*}
    🔗
    Since \(f(b)-f(a)=0\text{,}\) it must be true that \(f(b)=f(a)\text{,}\) so \(f\) must have the same value for all \(a\) and \(b\) on the interval \(I\) (since we could pick any \(a\) and \(b\) we like on \(I\)). If the value of \(f\) is the same for all inputs, then \(f\) is constant.
    🔗
    Collecting all three arguments above, we see that when \(f'\) is positive, \(f\) is increasing, when \(f'\) is negative, \(f\) is decreasing, and when \(f'\) is zero, \(f\) is constant, all thanks to the Mean Value Theorem and the formal definitions of an increasing/decreasing function.
    🔗
    🔗
    🔗

    Subsection 5.4.6 Summary

    The following terms and theorems were introduced in this section:
    🔗
    Key ideas: A function is increasing where the derivative is positive and decreasing where the derivative is negative. This implies that a continuous function will have a local minimum where the derivative changes from negative to positive, and a local maximum where the derivative changes from positive to negative (the First Derivative Test). The second derivative is positive where a function is concave up and negative where a function is concave down. This implies that a critical point of a differentiable function with positive second derivative is a local minimum and a critical point with negative second derivative is a local maximum (the Second Derivative Test).
    🔗
    Other ideas reinforced/introduced: A critical point is where the first derivative is either zero or undefined, but must be in the domain of the function. Expressions can change sign whenever they are zero or undefined. The second derivative is the rate of change of the first derivative, so the first derivative is increasing where the second derivative is positive, and the first derivative is decreasing where the second derivative is negative.
