    We have already seen that the sine and cosine functions have many different relationships, leading to numerous identities. Now, we see that they are even further linked through calculus. The theorem below shows that the derivative of each involves the other:
    ðŸ”—

    Theorem 8.7.2. Derivatives of Sine and Cosine.

    For all real numbers \(x\text{,}\)
    \begin{equation*} \frac{d}{dx} [\sin x] = \cos x\qquad\text{ and } \qquad \frac{d}{dx}[\cos x] = -\sin x \end{equation*}

    We can now use these two derivative rules to calculate derivatives of more complicated expressions involving trigonometric functions. The first example uses some of our familiar derivative rules: the sum and constant multiple rules:
    ðŸ”—

    Example 8.7.6.

    Calculate the derivative of \(h(x) = 3\cos x-4\sin x\text{.}\)
    ðŸ”—
    Solution.
    \begin{equation*} \begin{aligned}h'(x) \amp = \frac{d}{dx}[3\cos x-4\sin x]\\ \amp = 3\frac{d}{dx}[\cos x]-4\frac{d}{dx}[\sin x]\\ \amp = 3(-\sin x)-4(\cos x)\\ \amp = -3\sin x-4\cos x \end{aligned} \end{equation*}
    ðŸ”—
    ðŸ”—
    We can also find equations of tangent lines to functions involving trigonometric functions the same way that we are used to doing: by taking the derivative and plugging in the \(x\)-value to find the slope of the line, using the original function to find the \(y\)-value of the point, and then using point-slope form to write the equation:
    ðŸ”—

    Example 8.7.7.

    Find the equation of the tangent line to \(y=g(x) = x^2+2\cos x\) at the point where \(x=\frac{\pi}{2}\text{.}\)
    ðŸ”—
    Solution.
    The derivative of \(g(x) = x^2+2\cos x\) is
    \begin{equation*} \begin{aligned}g'(x) \amp = \frac{d}{dx}[x^2]+2\frac{d}{dx}[\cos x]\\ \amp =2x+2(-\sin x) \\ \amp = 2x-2\sin x \end{aligned} \end{equation*}
    and so the slope of the tangent line at \(x=\frac{\pi}{2}\) is
    \begin{equation*} g'\left(\frac{\pi}{2}\right) = 2\left(\frac{\pi}{2}\right)-2\sin\left(\frac{\pi}{2}\right) = \pi-2(1) = \pi-2\text{.} \end{equation*}
    ðŸ”—
    The \(y\)-value at the point where \(x=\frac{\pi}{2}\) is
    \begin{equation*} y=g\left(\frac{\pi}{2}\right) = \left(\frac{\pi}{2}\right)^2+2\cos\left(\frac{\pi}{2}\right) = \frac{\pi^2}{4}+2(0) = \frac{\pi^2}{4} \end{equation*}
    ðŸ”—
    Thus the equation of the tangent line has slope \(m = \pi-2\) and passes through the point \(\left(\frac{\pi}{2},\frac{\pi^2}{4}\right)\text{.}\) Plugging these into \(y = m(x-x_1)+y_1\text{,}\) we get the equation
    \begin{equation*} y =(\pi-2)\left(x-\frac{\pi}{2}\right)+\frac{\pi^2}{4}\text{.} \end{equation*}

    Example 8.7.10.

    The function \(f(x) = \sin(4x)\) has an "inside" function of \(4x\text{,}\) and so we need to use the chain rule to take its derivative. The derivative of \(f(x) = \sin(4x)\) is
    \begin{equation*} f'(x) = \cos(4x)\cdot\frac{d}{dx}(4x) = \cos(4x)\cdot 4 = 4\cos(4x)\text{.} \end{equation*}
    Note that the "inside" function is still \(4x\text{,}\) and if we want to find the second derivative, we need to use the chain rule again. The second derivative is
    \begin{equation*} f''(x) = 4(-\sin(4x)\cdot4) = -16\sin(4x)\text{.} \end{equation*}
    Note again that the "inside" function has not changed. We can keep going! The third derivative is
    \begin{equation*} f'''(x) = -16(\cos(4x)\cdot 4) = -64\cos(4x) \end{equation*}
    and the fourth derivative is
    \begin{equation*} f^{(4)}(x) = -64(-\sin(4x)\cdot 4) = 256\sin(4x)\text{.} \end{equation*}
    What would the fifth derivative be? And the sixth? Seventh? Eighth?

    Did you spot the pattern? For each additional derivative, we multiply by another 4 from the chain rule, while the sines and cosines alternate and the sign changes with every other derivative.

    Example 8.7.13.

    \begin{equation*} P'(t) = 8\cos t \end{equation*}

    \begin{equation*} P'(2) = 8\cos 2\approx -3.329\text{.} \end{equation*}

    Subsection 8.7.3 Derivatives of the Other Trigonometric Functions

    Recall that we can define the four other trigonometric functions in terms of the sine and cosine functions:
    \begin{equation*} \tan \theta = \frac{\sin\theta}{\cos\theta} \qquad \cot\theta = \frac{\cos\theta}{\sin\theta} \end{equation*}
    \begin{equation*} \sec\theta = \frac{1}{\cos\theta}\qquad \csc\theta = \frac{1}{\sin \theta} \end{equation*}
    ðŸ”—
    Because we know the derivatives of the sine and cosine function, we can now develop rules for how to differentiate these other functions.
    ðŸ”—

    Example 8.7.14.

    For example, we can use the quotient rule to take the derivative of the tangent function:
    \begin{equation*} \begin{aligned}\frac{d}{dx}(\tan x) \amp = \frac{d}{dx}\left(\frac{\sin x}{\cos x}\right) \amp \\ \amp = \frac{\frac{d}{dx}(\sin x)\cdot(\cos x) - (\sin x)\cdot\frac{d}{dx}(\cos x)}{(\cos x)^2} \amp \leftarrow \text{ quotient rule } \\ \amp =\frac{(\cos x)(\cos x) - (\sin x)(-\sin x)}{\cos^2 x} \amp \leftarrow \text{ derivatives of } \sin x\text{ and } \cos x\\ \amp = \frac{\cos^2x+\sin^2x}{\cos^2 x} \amp \leftarrow\text{ algebra } \\ \amp = \frac{1}{\cos^2x} \amp \leftarrow\text{ Pythagorean Identity } \cos^2x+\sin^2x=1\\ \amp = \sec^2 x \amp \leftarrow\text{ rewriting } \frac{1}{\cos x} = \sec x \end{aligned} \end{equation*}
    ðŸ”—
    Thus we see that
    \begin{equation*} \frac{d}{dx}(\tan x) = \sec^2 x \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 8.7.15.

    Similarly, we can use the chain rule to take the derivative of the secant function:
    \begin{equation*} \begin{aligned}\frac{d}{dx}(\sec x) \amp = \frac{d}{dx}\left[\frac{1}{\cos x}\right] \amp \leftarrow\text{ definition of } \sec x\\ \amp = \frac{d}{dx}\big[(\cos x)^{-1}\big] \amp \leftarrow\text{ rewriting to use the chain rule } \\ \amp = -(\cos x)^{-2} \frac{d}{dx}[\cos x]\amp \leftarrow\text{ chain rule } \\ \amp = -\frac{1}{\cos^2 x}(-\sin x) \amp \leftarrow\text{ algebra, derivative of } \sin x\\ \amp =\frac{\sin x}{\cos ^2x}\amp \leftarrow\text{ algebra } \\ \amp =\frac{1}{\cos x} \frac{\sin x}{\cos x}\amp \leftarrow\text{ algebra } \\ \amp = \sec x\tan x \amp \leftarrow\text{ rewriting in terms of tangent and secant } \end{aligned} \end{equation*}
    ðŸ”—
    ðŸ”—
    Note that in the last steps of both of the above calculations, we rewrite the derivatives in terms of tangent and secant. These are the typical forms of these derivatives as seen in any standard textbook, but there is nothing technically wrong with leaving them in terms of the sine and cosine functions.
    ðŸ”—
    The following theorem summarizes the derivatives of tangent, cotangent, secant, and cosecant. Of course, you can use these derivatives without knowing how they are derived, but it is useful to know where they come from in case you are stranded on a desert island without your notes and need to do calculus with trigonometry in order to tell time by tracking the movement of the stars (for example):
    ðŸ”—

    Theorem 8.7.16. Derivatives of Tangent, Cotangent, Secant, Cosecant.

    For all values of \(x\) at which the functions below are defined,
    \begin{equation*} \frac{d}{dx}(\tan x) = \sec^2 x\qquad \frac{d}{dx}(\cot x) = -\csc^2 x \end{equation*}
    \begin{equation*} \frac{d}{dx}(\sec x) = \sec x\tan x\qquad \frac{d}{dx}(\csc x) = -\csc x\cot x \end{equation*}
    ðŸ”—
    ðŸ”—
    Note that each derivative is defined only where the original function is defined. For instance, \(\tan x\) is undefined where \(\cos x = 0\text{,}\) and likewise \(\sec^2x\) is undefined where \(\cos x = 0\text{,}\) namely \(x=\frac{\pi}{2}+k\pi\text{,}\) where \(k\) is any integer.
