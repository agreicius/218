    Subsection Definitions of Power Functions

    Definition 1.6.3.

    A power function is a function that can be written in the form
    \begin{equation*} f(x) = Ax^k \end{equation*}
    for some nonzero real number \(A\) and some rational number \(k\text{.}\)
    ðŸ”—
    ðŸ”—
    For example, \(f(x)=8x^3\text{,}\) \(g(x) = 8x^{17/5}\text{,}\) and \(h(x) = 1\) are all power functions. Note that \(h(x)\) can be rewritten in the form
    \begin{equation*} h(x) = 1\cdot x^0\text{,} \end{equation*}
    which is why it is a power function (with \(A=1\) and \(k=0\)).
    ðŸ”—
    In this definition, the constant \(k\) is called the power or exponent, and the constant \(A\) is called the coefficient. Note that the exponent \(k\) must be a rational number. A rational number is a real number that can be written as a quotient of the form \(\frac{p}{q}\) for some integers \(p\) and \(q\) with \(q\neq 0\text{.}\) The variable must be in the base; this means that, for example, \(f(x) = x^{\pi}\) and \(f(x) = 10^x\) are not considered power functions.
    ðŸ”—

    Subsection Using Square Roots to Find Distance

    Power functions are extremely useful because square roots are used in many practical calculations. We have already seen the square root used when applying the quadratic formula to find the \(x\)-intercepts of quadratic functions. Another use of square roots is in calculating distance.
    ðŸ”—

    Definition 1.6.27. Distance Between Two Real Numbers.

    The distance between two real numbers \(a\) and \(b\) is \(\text{ dist } (a,b) = \sqrt{(b-a)^2}\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.6.28.

    For example, we know that the distance between 5 and 3 is 2. Using the above formula:
    \begin{equation*} \text{ dist } (3,5) = \sqrt{(5-3)^2} = \sqrt{(2)^2} = \sqrt{4} = 2 \end{equation*}
    ðŸ”—
    Note that it doesnâ€™t matter whether we compute \(\text{ dist } (3,5)\) or \(\text{ dist } (5,3)\text{:}\)
    \begin{equation*} \text{ dist } (5,3) = \sqrt{(3-5)^2} = \sqrt{(-2)^2} = \sqrt{4} = 2 \end{equation*}
    since once we square \(5-3 = 2\) or \(3-5 = -2\text{,}\) they are both equal to positive 4.
    ðŸ”—

    Subsection A Summary of Algebraic Functions

    A function is algebraic if it can be expressed in terms of constants and a variable \(x\) by using only arithmetic operations \((+, -, \times\text{,}\) and \(\div)\) and rational constant powers of the variables. For example
    \begin{equation*} f(x) = \frac{3x^2-5}{1-x} \text{ and } g(x) = (\sqrt{x}+\sqrt[3]{x})^{7/2} \end{equation*}
    are algebraic functions, but \(h(x) = 2^x\) and \(k(x)=\sin x\) are not.
    ðŸ”—
