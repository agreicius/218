    Subsection 6.5.2 Exponential and Logarithmic Functions are Continuous on their Domains

    Similar to the algebraic functions we worked with in the fall, exponential and logarithmic functions are continuous on their domains. Note in particular that their graphs do not have any jumps or holes on their domains.
    ðŸ”—

    Theorem 6.5.1. Continuity of Exponential and Logarithmic Functions.

    All exponential and logarithmic functions are continuous on their domains.

    Subsection 6.5.3 Limits at Infinity of Exponential Functions

    We have already established that any exponential function of the form \(f(t) = ab^t\) where \(a\) and \(b\) are positive real numbers with \(b\neq 1\) is always concave up and is either always increasing or always decreasing. We next introduce precise language to describe the behavior of an exponential functionâ€™s value as \(t\) gets bigger and bigger. To start, letâ€™s consider the two basic exponential functions \(p(t) = 2^t\) and \(q(t) = \left(\frac{1}{2}\right)^t\) and their respective values at \(t=10, t=20\text{,}\) and \(t=30\text{,}\) as displayed below:
    ðŸ”—
    ðŸ”—
    For the increasing function \(p(t) = 2^t\text{,}\) we see that the output of the function gets very large very quickly. In addition, there is no upper bound to how large the function can be. Indeed, we can make the value of \(p(t)\) as large as weâ€™d like by taking \(t\) sufficiently big. We thus say that as \(t\) increases, \(p(t)\) increases without bound and write
    \begin{equation*} \lim_{t\to\infty}2^t = \infty \end{equation*}
    ðŸ”—
    For the decreasing function \(q(t) = \left(\frac{1}{2}\right)^t\text{,}\) we see that the output \(q(t)\) is always positive but getting closer and closer to 0. Indeed, because we can make \(2^t\) as large as we like, it follows that we can make its reciprocal \(\frac{1}{2^t} = \left(\frac{1}{2}\right)^t\) as small as weâ€™d like. We thus say that as \(t\) increases, \(q(t)\) approaches 0. We write
    \begin{equation*} \lim_{t\to\infty}\left(\frac{1}{2}\right)^t = 0\text{.} \end{equation*}
    ðŸ”—
    Weâ€™ve seen several times how the value of \(b\) affects the steepness of the graph of \(f(t) = ab^t\text{,}\) as well as how all graphs with \(b>1\) have the similar increasing behavior, and all graphs with \(0\lt b\lt 1\) have similar decreasing behavior. For instance, by taking \(t\) sufficiently large, we can make \((1.01)^t\) as large as we want; it just takes much larger values of \(t\) to make \((1.01)^t\) big in comparison to \(2^t\text{.}\) In the same way, we can make \((0.99)^t\) as close to \(0\) as we wish by taking \(t\) sufficiently big, even though it takes longer for \((0.99)^t\) to get close to \(0\) in comparison to \(\left(\frac{1}{2}\right)^t\text{.}\) For an arbitrary choice of \(b\text{,}\) we can say the following:

    Let \(f(t) = b^t\) with \(b>0\) and \(b\neq 1\text{.}\)
    1. If \(0\lt b\lt 1\text{,}\) then \(\ds\lim_{t\to\infty}b^t = 0\text{.}\)
      ðŸ”—
      ðŸ”—
    2. If \(b>1\text{,}\) then \(\ds\lim_{t\to\infty}b^t = \infty\text{.}\)
      ðŸ”—
      ðŸ”—

    In addition, we make a key observation about the use of exponents. For the function \(q(t) = \left(\frac{1}{2}\right)^t\text{,}\) there are three equivalent ways we may write the function:
    \begin{equation*} \left(\frac{1}{2}\right)^t = \frac{1}{2^t} = 2^{-t}\text{.} \end{equation*}
    Now we can see that \(q(t) = 2^{-t} = q(-t)\text{.}\) Recall that the graph of a function \(y=h(-t)\) is the reflection of the graph of \(y=h(t)\) across the \(y\)-axis. Therefore, we can say that the graphs of \(p(t) = 2^t\) and \(q(t) = \left(\frac{1}{2}\right)^t = 2^{-t}\) are reflections of one another in the \(y\)-axis since \(p(-t) = 2^{-t} = q(t)\text{.}\) We see this fact verified in the picture below:
    ðŸ”—
    Similar observations hold for the relationship between the graphs of \(b^t\) and \(\frac{1}{b^t} = b^{-t}\) for any \(b\neq 1\text{.}\) This allows us to analyze the limits at \(-\infty\) of functions \(f(t) = b^t\) quickly, using what we already know. For example, if \(b>1\text{,}\) we know that \(0\lt \frac{1}{b}\lt 1\) and hence
    \begin{equation*} \lim_{t\to\infty}\left(\frac{1}{b}\right)^t = \lim_{t\to\infty} b^{-t} = 0 \end{equation*}
    ðŸ”—
    Reflecting this across the \(y\)-axis, we see then that
    \begin{equation*} \lim_{t\to-\infty} b^t = 0 \end{equation*}
    ðŸ”—
    A similar statement holds for \(0\lt b\lt 1\text{,}\) with the infinite limit of the exponential growth at infinity reflecting over the \(y\)-axis to the infinite limit of exponential decay at negative infinity (see the graphs in FigureÂ 6.5.4 to visualize this). Summarizing these observations:

    1. If \(0\lt b\lt 1\text{,}\) then \(\ds\lim_{t\to-\infty}b^t = \infty\text{.}\)

    Subsection 6.5.4 Limits of Logarithmic Functions

    We know that we can calculate limits of logarithmic functions at domain points simply by evaluation. What about non-domain points? Logarithmic functions have domain \((0,\infty)\text{,}\) so we need to examine their limits as \(x\to0^+\) and as \(x\to\infty\text{.}\)
    ðŸ”—
    Consider \(b=2\text{.}\) Because the graph of \(f(x) = 2^x\) increases more and more rapidly as \(x\) increases, the graph of \(\log_2{x}\) increases more slowly as \(x\) increases. Even though the logarithmic function grows very slowly, it grows without bound because we can make \(\log_2 x\) as large as we want by making \(x\) sufficiently large. For instance, if we want \(x\) such that \(\log_2(x) = 100\text{,}\) we choose \(x=2^{100}\text{,}\) since \(\log_2(2^{100}) = 100\text{.}\)
    ðŸ”—
    \(x\) \(2^{10}\) \(2^{100}\) \(2^{1,000}\) \(2^{10,000}\) \(2^{100,000}\) \(2^{1,000,000}\) \(\to\infty\)
    \(\log_2(x)\) 10 100 1,000 10,000 100,000 1,000,000 \(\to\infty\)
    Using this table, we can see that as \(x\to\infty\text{,}\) \(\log_2(x)\to\infty\text{.}\) Using limit notation, this means that
    \begin{equation*} \displaystyle\lim_{x\to\infty}\log_2(x)=\infty \end{equation*}
    ðŸ”—
    On the other hand, we saw that \(\ds\lim_{x\to-\infty}2^x = 0\text{,}\) meaning that when the input of \(f(x) = 2^x\) becomes large and negative, the output becomes closer and closer to 0. Recall that since \(2^x\) is always positive, the output is approaching 0 and is positive, which we can write as \(\ds\lim_{x\to\infty} 2^x = 0^+\text{.}\)
    ðŸ”—
    Using our understanding of inverse functions as swapping input and output, this tells us that as the inputs of \(f^{-1}(x) = \log_2 x\) approach 0 and are positive, the values of the output will become large and negative. Written as a limit:
    \begin{equation*} \lim_{x\to 0^+}\log_2 x = -\infty \end{equation*}
    ðŸ”—
    In words, this tells us that we can make the values of \(\log_2 x\) as large and negative as we want by making \(x\) sufficiently small, meaning sufficiently close to 0. For instance, if we want \(x\) such that \(\log_2(x) = -100\text{,}\) we choose \(x = 2^{-100}\text{,}\) since \(\log_2(2^{-100}) = -100\text{.}\) If we want \(x\) such that \(\log_2(x) = -1,000\text{,}\) we choose \(x=2^{-1000}\text{,}\) since \(\log_2(2^{-1000}) = -1000\text{.}\) (Recall that \(2^{-100} = \frac{1}{2^{100}}\) is positive and very, very close to zero. \(2^{-1000}\) is even close to 0.)     \begin{equation*} \lim_{x\to\infty} \left(\frac{1}{2}\right)^x = 0^+ \qquad\text{ and so } \qquad \lim_{x\to0^+}\log_{1/2}x = \infty \end{equation*}
    and
    \begin{equation*} \lim_{x\to-\infty} \left(\frac{1}{2}\right)^x =\infty\qquad\text{ and so } \qquad \lim_{x\to\infty}\log_{1/2}x = -\infty \end{equation*}

    Subsection 6.5.5 Calculating limits involving exponential and logarithmic functions

    The ideas covered so far in this section allow you to calculate many limits that involve exponential and logarithmic functions. We can use algebraic methods and logical reasoning from last semester in order to determine the behavior of these limits. Itâ€™s important to note that limits involving exponential and logarithmic functions can still be indeterminate, but many of the same methods and ideas can be applied with small modifications.
    ðŸ”—

    Example 6.5.18.

    Calculate \(\displaystyle\lim_{x\to0}\frac{3^{2x}+3^x-2}{3^x-1}\text{.}\)
    ðŸ”—
    Solution.
    First, we try to evaluate the limit. If we find that the limit is indeterminate, then we will use our limit-evaluating skills to rewrite the limit until it can be evaluated.
    ðŸ”—
    As \(x\to0\text{,}\) both the numerator and the denominator approach zero, so this limit is of the form \(\frac{0}{0}\text{,}\) which is indeterminate. So, we need to rewrite the limit, likely using algebra.
    ðŸ”—
    Notice that we can factor the numerator in the following way: call \(z=3^x\text{.}\) Then, \(3^{2x}+3^x-2=(3^{x})^2+3^x-2=z^2+z-2\text{,}\) which can be factored as \((z+2)(z-1)\text{.}\) We then replace \(z\) with \(3^x\) to obtain a factored numerator: \(3^{2x}+3^x-2=(3^x-1)(3^x+2)\text{.}\)
    ðŸ”—
    Returning to the limit, we use the factored form of the numerator:
    \begin{equation*} \displaystyle\lim_{x\to0}\frac{3^{2x}+3^x-2}{3^x-1}=\displaystyle\lim_{x\to0}\frac{(3^x-1)(3^x+2)}{3^x-1}\text{.} \end{equation*}
    How does this help us? Well, we now see that both the denominator and numerator have a factor of \(3^x-1\text{.}\) Using the cancelation property for limits, we can cancel these factors, and then finish the limit via evaluation:
    \begin{equation*} \displaystyle\lim_{x\to0}\frac{(3^x-1)(3^x+2)}{3^x-1}=\displaystyle\lim_{x\to0}(3^x+2)=3^0+2=3\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Example 6.5.19.

    Calculate \(\displaystyle\lim_{x\to\infty}\frac{3^{x}-2^x}{1+4^x}\text{.}\)
    ðŸ”—
    Solution.
    By TheoremÂ 6.5.3, we know that as \(x\to\infty\text{,}\) \(2^x\text{,}\) \(3^x\text{,}\) and \(4^x\) all approach \(\infty\text{.}\) Therefore, the limit in question is an indeterminate form, with \(\infty-\infty\) in the numerator and another \(\infty\) in the denominator. An analog of the method of dividing by the highest power works in this case. Because \(4^x\) grows faster than either \(3^x\) or \(2^x\) as \(x\to\infty\text{,}\) \(4^x\) is the dominant term, hence we divide through by it:
    \begin{equation*} \displaystyle\lim_{x\to\infty}\frac{3^{x}-2^x}{1+4^x}=\displaystyle\lim_{x\to\infty}\frac{3^{x}-2^x}{1+4^x}\cdot \frac{(1/4)^x}{(1/4)^x}=\displaystyle\lim_{x\to\infty}\frac{(3/4)^{x}-(2/4)^x}{(1/4)^x+1}=\frac{0-0}{0+1}=0/1=0\text{.} \end{equation*}
    ðŸ”—
    Notice that by dividing by the exponential function with the largest base, all of the bases are now between \(0\) and \(1\text{,}\) so their limits approach 0.

    ðŸ”—
