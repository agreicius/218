Example 5.1.11.

For the curve given implicitly by \(x^3+y^3-2xy = 2\text{,}\) find the derivative \(\frac{dy}{dx}\) if we view \(y\) as an implicit function of \(x\text{.}\)

Solution.
We begin by differentiating the curveâ€™s equation implicitly, meaning by treating \(y\) as an implicit function of \(x\text{.}\) Taking the derivative of each side with respect to \(x\text{,}\)
\begin{equation*} \frac{d}{dx}[x^3+y^2-2xy] = \frac{d}{dx}[2]\text{,} \end{equation*}
by the sum rule and the fact that the derivative of a constant is zero, we have
\begin{equation*} \frac{d}{dx}[x^3]+\frac{d}{dx}[y^2]-\frac{d}{dx}[2xy] = 0\text{.} \end{equation*}

We have three derivatives to compute. \(\frac{d}{dx}[x^3]\) uses the power rule applied to the variable \(x\text{.}\) \(\frac{d}{dx}[y^2]\) requires the chain rule since \(y\) is an implicit function of \(x\text{.}\) And \(\frac{d}{dx}[2xy]\) requires the product rule since \(y\) is a function of \(x\) and we are multiplying \(2x\) by \(y\text{.}\) Applying these rules, we now find that
\begin{equation*} 3x^2+2y\frac{dy}{dx}-\left(2x\frac{dy}{dx}+2y\right) = 0\text{.} \end{equation*}

Now we want to solve this equation for \(\frac{dy}{dx}\text{.}\) To do so, we first collect all of the terms involving \(\frac{dy}{dx}\) on one side of the equation, factor \(\frac{dy}{dx}\text{,}\) then divide to isolate \(\frac{dy}{dx}\text{:}\)
\begin{equation*} \begin{aligned}3x^2+2y\frac{dy}{dx}-\left(2x\frac{dy}{dx}+2y\right) \amp = 0\\ 3x^2+2y\frac{dy}{dx}-2x\frac{dy}{dx}-2y \amp = 0\\ 2y\frac{dy}{dx}-2x\frac{dy}{dx} \amp = 2y-3x^2\\ \frac{dy}{dx}(2y-2x) \amp = 2y-3x^2\\ \frac{dy}{dx} \amp = \frac{2y-3x^2}{2y-2x}. \end{aligned} \end{equation*}
