Example 3.2.18.
Describe the continuity of the piecewise-defined function
\begin{equation*} g(x) = \begin{cases}4-x^2, \amp \text{ if } x\leq 1\\ x-1, \amp \text{ if } x>1 \end{cases} \end{equation*}
at the value \(x=1\) by using the graph in FigureÂ 3.2.19 below to determine the left, right, and two-sided limits at \(x=1\text{.}\) Then describe the intervals on which \(g(x)\) is continuous.

Solution.
We can see that
\begin{equation*} \ds\lim_{x\to 1^-} g(x) = 3, \ds\lim_{x\to 1^+}g(x) = 0, \text{ and } g(1)=3\text{.} \end{equation*}
ðŸ”—
\(g(x)\) is continuous from the left at \(x=1\) because
\begin{equation*} \ds\lim_{x\to 1^-} g(x) = 3=g(1) \end{equation*}
ðŸ”—
\(g(x)\) is not continuous from the right at \(x=1\) because
\begin{equation*} \ds\lim_{x\to 1^-} g(x) = 0\neq g(1)=3\text{.} \end{equation*}
ðŸ”—
The two-sided limit \(\ds\lim_{x\to 1}g(x)\) does not exist, and so \(g(x)\) is not (both-sided) continuous at \(x=1\text{.}\)
ðŸ”—
\(g(x)\) is continuous at every other point, and so overall is continuous on the intervals
\begin{equation*} (-\infty, 1]\text{ and } (1,\infty)\text{.} \end{equation*}
Remember that the closed endpoint on the interval \((-\infty,1]\) means that the function is continuous from the left at \(x=1\text{.}\) The open endpoint on the interval \((1,\infty)\) means that the function is not continuous from the right at \(x=1\text{.}\)
