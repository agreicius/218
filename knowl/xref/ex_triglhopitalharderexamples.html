Example 9.3.6.

Calculate each of the following limits:
  1. \(\displaystyle \ds\lim_{x\to 0^+}\frac{\csc x}{\ln x}\)
  2. \(\displaystyle \ds\lim_{x\to \pi/2}(\tan x-\sec x)\)
Solution.
  1. As \(x\to 0^+\text{,}\) \(\csc x\) approaches \(\infty\) and \(\ln x\) approaches \(-\infty\text{;}\) therefore this is a limit of the form \(\frac{\infty}{\infty}\text{,}\) which is indeterminate. LH么pitals rule applied and gives us
    \begin{equation*} \lim_{x\to0^+}\frac{\csc x}{\ln x} \stackrel[L'H]{}= \lim_{x\to 0^+}\frac{-\csc x\cot x}{1/x} = \lim_{x\to 0^+}(-x\csc x\cot x)\text{.} \end{equation*}
    As \(x\to 0^+\) we have \(-x\to 0\text{,}\) \(\csc x\to \infty\text{,}\) and \(\cot x\to \infty\text{,}\) and therefore this limit is of the indeterminate form \(0\cdot \infty\text{.}\) After converting to sines and cosines and simplifying, we will be able to apply LH么pitals rule again:
    \begin{equation*} \begin{aligned}\lim_{x\to 0^+}(-x\csc x\cot x) \amp = \lim_{x\to 0^+} (-x)\left(\frac{1}{\sin x}\right)\left(\frac{\cos x}{\sin x}\right)\amp \leftarrow\text{ trigonometric identities } \\ \amp = \lim_{x\to 0^+} \frac{-x\cos x}{\sin^2 x}\amp \leftarrow\text{ algebra, limit now of form } \frac{0}{0}\\ \amp \stackrel[L'H]{}= \lim_{x\to 0^+}\frac{(-1)\cos x+(-x)(-\sin x)}{2\sin x\cos x}\amp \leftarrow\text{ note use of product and chain rules } \\ \amp = \lim_{x\to 0^+}\frac{-\cos x+x\sin x}{2\sin x\cos x}\amp \leftarrow\text{ simplify } \end{aligned} \end{equation*}
    Now as \(x\to 0^+\) we have \(\sin x\to 0^+\) and \(\cos x\to 1\text{,}\) and therefore the denominator in the remaining expression approaches \(0^+\text{.}\) Since the numerator approaches \(-\cos 0+0(\sin 0) = -1\text{,}\) the limit in question is of the form \(\frac{-1}{0^+}\) and therefore
    \begin{equation*} \lim_{x\to 0^+}\frac{\csc x}{\ln x} = \lim_{x\to 0^+}\frac{-\cos x+x\sin x}{2\sin x\cos x} = -\infty \end{equation*}
  2. At first glance this does not look like a LH么pitals rule problem, since \(\tan x-\sec x\) is not in the form of a quotient. Worse, the limit is of the indeterminate form \(\infty-\infty\text{,}\) so we cant solve it simply by evaluating the function at \(x=\frac{\pi}{2}\text{.}\) After using a bit of algebra, we see that LH么pitals rule does apply here:
    \begin{equation*} \begin{aligned}\lim_{x\to \pi/2} (\tan x-\sec x) \amp = \lim_{x\to \pi/2}\left(\frac{\sin x}{\cos x}-\frac{1}{\cos x}\right)\amp \leftarrow\text{ definition of tangent and secant } \\ \amp = \lim_{x\to \pi/2}\frac{\sin x-1}{\cos x}\amp \leftarrow\text{ this is now in the form } \frac{0}{0}\\ \amp \stackrel[L'H]{}= \lim_{x\to \pi/2}\frac{\cos x}{-\sin x}\amp \leftarrow\text{ L'H\^opital's rule } \\ \amp =\frac{\cos(\pi/2)}{-\sin(\pi/2)} = \frac{0}{-1} = 0 \amp \leftarrow\text{ evaluate limit } \end{aligned} \end{equation*}
