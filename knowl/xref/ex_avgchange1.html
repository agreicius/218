Example 1.3.4.

For example, the function graphed below in FigureÂ 1.3.5 is increasing until \(x=-2\text{,}\) decreasing from \(x=-2\) to \(x=1\text{,}\) increasing again from \(x=1\) until \(x=3\text{,}\) and then decreasing after \(x=3\text{.}\)
ðŸ”—
The function is not linear and so has no "slope." In order to discuss how much it increases or decreases, we can look at each interval separately.
ðŸ”—
On the interval \([-2,1]\text{,}\) we calculate the average rate of change of \(y\) between \(x=-2\) and \(x=1\text{.}\) First, we observe that \(f(-2)=4\) and \(f(1)=-2\) in order to use these quantities in the formula for Average Rate of Change with \(a=-2\) and \(b=1\text{:}\)
\begin{equation*} AV_{[-2,1]} = \frac{f(1)-f(-2)}{1-(-2)} = \frac{-2-4}{1+2} = \frac{-6}{3} = -2 \end{equation*}
and so we see that, on average, the function is changing by \(-2\) units of \(y\) per unit of \(x\) between \(x=-2\) and \(x=1\text{.}\) We could also say that the function is decreasing by 2 units of \(y\) per unit of \(x\text{,}\) which matches what we see in the picture.
ðŸ”—
Similarly, we calculate the average rate of change of \(y\) on the interval \([1,3]\text{.}\) Observe that \(f(1)=-2\) and \(f(3)=0\) will be used in the formula for Average Rate of Change, using \(a=1\) and \(b=3\text{:}\)
\begin{equation*} AV_{[1,3]} = \frac{f(3)-f(1)}{3-1} = \frac{0-(-2)}{3-1} = \frac{2}{2} = 1 \end{equation*}
so, on average, the function is increasing by 1 unit of \(y\) per unit of \(x\) between \(x=1\) and \(x=3\text{.}\)
ðŸ”—
We can also measure average rate of change over intervals where the function both increases and decreases. For example, on the interval \([-1,2]\) the function first decreases and then increases. Note that \(f(-1) = 2\) and \(f(2) = -1\text{.}\) Computing the average rate of change from \(x=-1\) to \(x=2\text{:}\)
\begin{equation*} AV_{[-1,2]} = \frac{f(2)-f(-1)}{2-(-1)} = \frac{-1-2}{2+1} = \frac{-3}{3} = -1 \end{equation*}
so, on average, the function is decreasing by 1 (or changing by \(-1\)) unit of \(y\) per unit of \(x\) between \(x=-1\) and \(x=2\text{.}\)
ðŸ”—
As another example, the average rate of change on the interval \([0,2]\) is
\begin{equation*} AV_{[0,2]} = \frac{f(2)-f(0)}{2-0} = \frac{-1-(-1)}{2} = \frac{0}{2} = 0 \end{equation*}
where we observe that since \(f(0) = -1\) and \(f(2) = -1\text{,}\) the function has not changed on average from \(x=0\) to \(x=2\text{.}\)
