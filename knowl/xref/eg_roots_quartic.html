Example 1.6.19. Roots of a quartic.

Find all roots of \(f(x)=x^4-4x^3+5x^2-4x+4\text{.}\)

Solution.
Since the coefficients of \(f\) are integers, we first try and find an integer root of \(f\) using TheoremÂ 1.6.15. Evaluating \(f(a)\) for \(a=\pm 1, \pm 2, \pm 4\) (the factors of the constant term of \(f\)), we see that \(f(2)=0\text{,}\) and thus that \((x-2)\) is a factor of \(f\text{.}\) Using polynomial longdivision (we skip the steps here), we see that
\begin{equation*} f(x)=(x-2)(x^3-2x^2+x-2)\text{.} \end{equation*}
Now continue the process by factoring \(g(x)=x^3-2x^2+x-2\text{.}\) Again using TheoremÂ 1.6.15, we try \(\pm 1, \pm 2\) as potential roots. It turns out that \(g(2)=0\text{,}\) and thus \(x-2\) can be factored out (again). Using polynomial longdivision (again we omit the steps), we find that
\begin{equation*} g(x)=(x-2)(x^2+1)\text{,} \end{equation*}
and thus that
\begin{equation*} f(x)=(x-2)g(x)=(x-2)(x-2)(x^2+1)=(x-2)^2(x^2+1)\text{.} \end{equation*}
Lastly, we apply TheoremÂ 1.2.27 to the remaining factor \(x^2+1\text{,}\) concluding that it has no roots. By ProcedureÂ 1.6.18, we conclude that the only root of \(f\) is \(2\text{.}\)
