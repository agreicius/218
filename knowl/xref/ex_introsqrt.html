Example 1.1.12. A Function with a Square Root.

Consider the function \(g(x) = \sqrt{4-x}\text{.}\) We can successfully use the value \(x=0\) as an input because
\begin{equation*} g(0) = \sqrt{4-0}=\sqrt{4}=2 \end{equation*}
ðŸ”—
So we see that the output \(g(0)=2\) is defined as the input of \(x=0\text{,}\) hence \(x=0\) is in the domain of \(g\text{.}\) Similarly, we can plug in the values \(x=1, 2, 3\text{,}\) and \(4\text{,}\) along with the values \(x=-1, -2,-3\) and \(-4\text{:}\)
ðŸ”—
However, if we try to plug in \(x=5\text{,}\) we do not get an output because we cannot take square roots of negative numbers:
\begin{equation*} g(5) = \sqrt{4-5} = \sqrt{-1}\text{ is undefined }\text{.} \end{equation*}
Thus \(x=-4, -3,\ldots, 3, 4\) ARE in the domain of \(g\) while \(x=5\) is NOT in the domain of \(g\text{.}\) How can we determine exactly where the domain begins and ends? Since we need the quantity inside the square root to be non-negative, meaning 0 or greater, we set up and solve the inequality
\begin{equation*} \begin{aligned}4-x \amp \geq 0\\ +x \amp \amp +x\\ \hline 4 \amp \geq x \end{aligned} \end{equation*}
ðŸ”—
Hence any value of \(x\) that is equal or less than 4 is in the domain, which we write as either \(x\leq 4\) or \((-\infty,4]\text{.}\)
