Example 4.2.8.

Suppose \(f(x)=\frac{2}{x}\text{.}\) Find the equation of the tangent line to \(f\) at \(x=1\text{.}\)
ðŸ”—
Solution.
By definition, the tangent line to \(f\) at \(x=1\) has slope \(f\,'(1)\) and passes through the point \((1,f(1)) = \left(1,\frac{2}{1}\right) = (1,2)\text{.}\) Before we can find the equation of this line, we must calculate the value of \(f\,'(1)\text{.}\) A lot of algebra will be needed before we can cancel a common factor of \(h\) and solve the limit:
\begin{equation*} \begin{aligned}f\,'(1) \amp = \lim_{h\to 0}\frac{f(1+h)-f(1)}{h} = \lim_{h\to 0}\frac{\frac{2}{1+h}-\frac{2}{1}}{h} \amp \leftarrow\text{ definition of the derivative with } x=1\\ \amp = \lim_{h\to 0}\frac{\frac{2}{1+h}-\frac{2(1+h)}{1+h}}{h} = \lim_{h\to 0}\frac{\left(\frac{2-2(1+h)}{1+h}\right)}{h}\amp \leftarrow\text{ common denominator in the numerator } \\ \amp = \lim_{h\to 0} \left(\frac{2-2-2h}{1+h}\right)\cdot \frac{1}{h}\amp \leftarrow\text{ distribute, multiply by the reciprocal of } h\\ \amp =\lim_{h\to 0}\frac{-2h}{(1+h)h} = \lim_{h\to 0}\frac{-2}{1+h} = \frac{-2}{1+0} = -2 \amp \leftarrow\text{ cancellation, limit rules } \end{aligned} \end{equation*}
ðŸ”—
So we have our derivative \(f\,'(1) = -2\) at last. Now we use the point-slope form of a line to find the equation of the tangent line, using \(m=-2\) and the point \((1,f(1)) = (1,2)\text{,}\) then simplifying to slope-intercept form:
\begin{equation*} y=-2(x-1)+2 = -2x+4 \end{equation*}
ðŸ”—
We can verify the reasonableness of the answer we just found by graphing the function \(f(x) = \frac{2}{x}\) and the line \(y=-2x+4\) on the same set of exes and checking that the line appears to be tangent to the graph of \(f\) at the point \((1,f(1))\text{:}\)
