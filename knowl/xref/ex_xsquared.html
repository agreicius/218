Example 1.1.3.

For example, the rule that takes a real number and squares it is a function. Since the result is also a real number, we write
\begin{equation*} f:\mathbb{R}\to\mathbb{R} \end{equation*}
where the symbol \(\mathbb{R}\) is used to represent the set of all real numbers. In this example, \(\mathbb{R}\) is both our input and our output set. This will be the case for many of the examples we study this semester.

Introducing some notation, letâ€™s call the number we are squaring \(x\) so that once we square it, we may write \(x^2\text{.}\) We can pick the name \(f\) to refer to this function by writing \(f(x) = x^2\text{,}\) which is pronounced "\(f\) of \(x\) is equal to \(x\) squared." Thus we have chosen \(x\) as the symbol we use to represent our independent variable. We can also choose to represent our outputs with the dependent variable \(y\) and can use the notation \(y=x^2\) or \(y=f(x)\) to represent the fact that the values of \(y\) are the result of squaring \(x\text{,}\) or equivalently, applying the function \(f\) to \(x\text{.}\)

For instance, if we choose the input value \(x=3\text{,}\) then we write
\begin{equation*} f(3) = 3^2 = 9\text{,} \end{equation*}
which is read "\(f\) of \(3\) is equal to \(3\) squared, which is equal to \(9\text{.}\)" The number \(9\) is the output of the function when the input is \(3\text{.}\)

We could also use the input value \(x=-3\text{,}\) and then we would write
\begin{equation*} f(-3)=(-3)^2 = 9\text{,} \end{equation*}
which is read "\(f\) of negative \(3\) is equal to negative \(3\) squared, which is equal to \(9\text{.}\)" The number \(9\) is again the output of the function when the input is \(-3\text{.}\)
