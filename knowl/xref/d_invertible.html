Definition 2.10.12. Invertible function.

A function \(f\colon A\rightarrow B\) is invertible if there is a function \(g\colon B\rightarrow A\) satisfying the following properties:
\begin{align} g(f(a)) \amp = a \text{ for all } a\in A\tag{2.24}\\ f(g(b)) \amp = b \text{ for all } b\in B\text{.}\tag{2.25} \end{align}
The function \(g\) in this case is called the inverse function of \(f\text{,}\) and we write \(g=f^{-1}\text{.}\)
