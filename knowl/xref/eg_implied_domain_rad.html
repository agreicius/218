Example 1.1.20. Implied domain: principal square root.

Let \(f\) be the function defined as \(f(x)=\sqrt{x}\text{.}\) Following FiatÂ 1.1.19, give a full definition of this function by determining the codomain and implied domain.

Solution.
Since \(f\) is only given as a formula, following FiatÂ 1.1.19, the codomain of \(f\) is \(\R\) and the implied domain is the set of all real numbers \(x\) for which the radical formula \(\sqrt{x}\) makes sense. By definition, we only compute \(\sqrt{x}\) for nonnegative real numbers \(x\text{.}\) Thus the implied domain of \(f\) is
\begin{equation*} D=\{x\in \R\mid x\geq 0\}\text{,} \end{equation*}
and we can state the full definition of \(f\) as the function \(f\colon D\rightarrow \R\) defined as \(f(x)=\sqrt{x}\text{.}\)
