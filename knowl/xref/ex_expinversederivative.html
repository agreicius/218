Example 8.8.10.

For another example, the function \(f(x) = e^x\) has inverse \(f^{-1}(x) = \ln x\text{.}\) Recall that \(f'(x) = e^x\) is the derivative of \(e^x\text{.}\) Using the formula developed earlier, we see that
\begin{equation*} (f^{-1})'(x) = \frac{1}{f'\left(f^{-1}(x)\right)} = \frac{1}{e^{\ln x}} = \frac{1}{x} \end{equation*}
since we know \(e^{\ln x} = x\text{.}\) This is another way of showing that \(\frac{d}{dx}(\ln x) = \frac{1}{x}\text{.}\) Note that mathematics is always consistent with itself; you have now seen two different ways to find the derivative of \(\ln x\text{,}\) and they both agree that \(\frac{d}{dx}[\ln x] = \frac{1}{x}\text{.}\)
