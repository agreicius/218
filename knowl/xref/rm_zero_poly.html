Remark 1.4.14. Zero function and zero polynomial.

A function \(f\colon D\rightarrow \R\) is called a zero function if \(f(x)=0\) for all \(x\in D\text{.}\) Observe that a zero function is also a polynomial according to DefinitionÂ 1.4.11. Indeed we have
\begin{equation*} f(x)=a_0 \end{equation*}
where \(a_0=0\text{.}\) Because of this, zero functions are also called zero polynomials.
