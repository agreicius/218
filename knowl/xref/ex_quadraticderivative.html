Example 4.2.3.

For the function \(f(x) = x-x^2\text{,}\) use the \(h\to 0\) definition of the derivative to compute \(f\,'(2)\text{.}\) In addition, discuss the meaning of this value and draw a labeled graph that supports your explanation.
ðŸ”—
Solution.
From the limit definition, we know that
\begin{equation*} f\,'(2) = \lim_{h\to 0}\frac{f(2+h)-f(2)}{h}\text{.} \end{equation*}
Now we use the rule for \(f\text{,}\) and observe that
\begin{equation*} f(2) = 2-2^2=-2 \end{equation*}
and
\begin{equation*} \begin{aligned}f(2+h)\amp =(2+h)-(2+h)^2\\ \amp = (2+h)-(4+4h+h^2)\\ \amp = -2-3h-h^2 \end{aligned} \end{equation*}
ðŸ”—
Substituting these values into the limit definition, we have that
\begin{equation*} f\,'(2) = \lim_{h\to 0}\frac{(-2-3h-h^2)-(-2)}{h}\text{.} \end{equation*}
this limit has the form \(\frac{0}{0}\text{.}\) In order to evaluate \(\ds\lim h\to 0\text{,}\) we must simplify the quotient. Combining like terms, we have
\begin{equation*} f\,'(2) = \lim_{h\to 0}\frac{-3h-h^2}{h}\text{.} \end{equation*}
Next, we remove a common factor of \(h\) in both the numerator and denominator and find that
\begin{equation*} f\,'(2) = \lim_{h\to 0}\frac{h(-3-h)}{h} = \lim_{h\to 0}(-3-h)=-3\text{.} \end{equation*}
ðŸ”—
Finally, we are able to conclude that \(f\,'(2) = -3\text{.}\) We note that \(f\,'(2)\) is the instantaneous rate of change of \(f\) at the point \((2,-2)\text{.}\) It is also the slope of the tangent line to the graph of \(y=x-x^2\) at the point \((2,-2)\text{.}\) The picture below shows both the function and the line through \((2,-2)\) with slope \(m=f\,'(2)=-3\text{.}\)
