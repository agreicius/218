Example 3.6.18.

Consider the function
\begin{equation*} g(x) = \frac{7x^2+14x+7}{x^2-2x+1} \end{equation*}
ðŸ”—
We can begin to evaluate \(\ds\lim_{x\to\infty} g(x)\) and \(\ds\lim_{x\to-\infty}g(x)\) by evaluating the limits of the numerators and denominators separately, force-factoring when necessary. For the limit as \(x\to\infty\text{:}\)
\begin{equation*} \begin{aligned}\lim_{x\to\infty}7x^2+14x+7 \amp = \infty+\infty+7 = \infty\\ \text{ and } \amp \\ \lim_{x\to\infty} x^2-2x+1 \amp = \infty-\infty+1\\ \amp \text{ indeterminate, so force-factor } \\ \amp = \lim_{x\to\infty} x^2(1-\frac{2}{x}+\frac{1}{x^2})\\ \amp = \infty\cdot(1-0+0) = \infty \end{aligned} \end{equation*}
ðŸ”—
This tells us that
\begin{equation*} \lim_{x\to\infty}g(x) = \frac{\infty}{\infty}\text{,} \end{equation*}
and so is indeterminate.
ðŸ”—
For the limit as \(x\to-\infty\text{:}\)
\begin{equation*} \begin{aligned}\lim_{x\to-\infty}7x^2+14x+7 \amp = \infty-\infty+7 \\ \amp \text{ indeterminate, so force-factor } \\ \amp = \lim_{x\to-\infty} x^2(7+\frac{14}{x}+\frac{7}{x^2})\\ \amp = \infty\cdot(7+0+0) = \infty\\ \text{ and } \amp \\ \lim_{x\to-\infty} x^2-2x+1 \amp = \infty+\infty+1 = \infty \end{aligned} \end{equation*}
ðŸ”—
This tells us that
\begin{equation*} \lim_{x\to-\infty}g(x) = \frac{\infty}{\infty} \end{equation*}
and so is also indeterminate.

But how can we be sure about the value of either of these limits? We saw in the previous example that we may be able to use algebra to convert such limits from an indeterminate form to a non-indeterminate form. Here are two different algebraic methods for doing so
ðŸ”—
Method 1: Divide the numerator and denominator by \(x^2\text{.}\) This is equivalent to multiplying the numerator and denominator by \(1/x^2\text{:}\)
\begin{equation*} \begin{aligned}\lim_{x\to\pm\infty}\frac{7x^2+14x+7}{x^2-2x+1} \amp = \lim_{x\to\pm\infty}\frac{7x^2+14x+7}{x^2-2x+1}\cdot\frac{1/x^2}{1/x^2}\\ \amp = \lim_{x\to \pm\infty}\frac{7+\frac{14}{x}+\frac{7}{x^2}}{1-\frac{2}{x}+\frac{1}{x^2}}\\ \amp = \frac{7+0+0}{1-0+0}\\ \amp = \frac{7}{1}\\ \amp =7 \end{aligned} \end{equation*}
ðŸ”—
Method 2: Force-factor \(x^2\) from both the numerator and denominator. The result is the same as in the previous method, since after factoring \(x^2\) from both the numerator and denominator, the \(x^2\)â€™s will cancel:
\begin{equation*} \begin{aligned}\lim_{x\to\pm\infty}\frac{7x^2+14x+7}{x^2-2x+1} \amp = \lim_{x\to \pm\infty}\frac{x^2\left(7+\frac{14}{x}+\frac{7}{x^2}\right)}{x^2\left(1-\frac{2}{x}+\frac{1}{x^2}\right)}\\ \amp = \lim_{x\to \pm\infty}\frac{7+\frac{14}{x}+\frac{7}{x^2}}{1-\frac{2}{x}+\frac{1}{x^2}}\\ \amp = \frac{7+0+0}{1-0+0}\\ \amp = \frac{7}{1}\\ \amp =7 \end{aligned} \end{equation*}
ðŸ”—
