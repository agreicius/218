Example 1.1.25. Range: principal square-root function.

Let \(f\) be defined as \(f(x)=\sqrt{x}\text{.}\) Compute \(\range f\text{,}\) expressed using interval notation.

Solution.
Firstly, we observe that the implied domain of \(f\) is \(D=[0,\infty)\text{,}\) since the expression \(\sqrt{x}\) is only defined for nonnegative real numbers \(x\text{.}\) We claim that
\begin{equation*} \range f=[0,\infty)\text{.} \end{equation*}
Indeed, since by definition \(\sqrt{x}\) is the nonnegative square root of \(x\text{,}\) we have \(f(x)=\sqrt{x}\in [0,\infty)\) for all \(x\in [0,\infty)\text{,}\) and hence that \(\range f\subseteq [0,\infty)\text{.}\) For the other direction, note that given any \(y\in [0,\infty)\text{,}\) we have \(y=\sqrt{y^2}\text{.}\) Thus for all \(y\in [0,\infty)\text{,}\) we have \(y=f(x)\text{,}\) where \(x=y^2\text{,}\) and thus \(y\in \range f\text{.}\) We conclude that \(\range f=[0,\infty)\text{,}\) as claimed.
