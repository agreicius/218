Remark 1.1.4. Independent and dependent variables.

The essential detail in the definition of a function \(f\colon A\rightarrow B\text{,}\) is that for each input \(a\) in our domain \(A\) the functions assigns a unique output \(b\) in our codomain \(B\text{.}\) This detail introduces a fundamental asymmetry between the roles of the variable \(a\) and the variable \(b\) in the context of the function \(f\colon A\rightarrow B\text{.}\) We codify this asymmetry with the terms independent variable and dependent variable.
