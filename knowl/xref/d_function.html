Definition 1.1.3. Function.

Let \(A\) and \(B\) be sets. A function \(f\) from \(A\) to \(B\text{,}\) denoted
\begin{equation} f\colon A\rightarrow B\text{,}\tag{1.3} \end{equation}
is a rule which, given any input \(a\in A\text{,}\) returns an output \(b\in B\text{.}\) We write \(b=f(a)\) in this case, and call \(b\) the image of \(a\) under \(f\text{,}\) or the value of \(f\) at \(a\text{.}\) Similarly, we say \(f\) maps (or sends) the input \(a\) to the output \(b=f(a)\text{.}\)

The set \(A\) is called the domain of \(f\text{;}\) the set \(B\) is called the codomain of \(f\text{.}\)
