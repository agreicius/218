Example 1.5.17. Polynomial roots.

Find all roots of \(f(x)=x^3+4x^2+4x+3\text{.}\)

Solution.
From ExampleÂ 1.5.14, we know that \(f\) factors as
\begin{equation*} f(x)=(x+3)(x^2+x+1)\text{,} \end{equation*}
and thus that \(-3\) is a root of \(f\text{.}\) Are there any other roots? Suppose \(a\ne -3\text{.}\) Evaluating \(f\) at \(a\) yields
\begin{equation*} f(a)=(a+3)(a^2+a+1)\text{.} \end{equation*}
Since \(a\ne -3\text{,}\) \(a+3\ne 0\text{.}\) By TheoremÂ 1.2.24, we have \(f(a)=0\) if and only if \(a^2+a+1=0\text{.}\) Thus, \(a\ne -3\) is a root of \(f\) if and only if it is a root of \(g(x)=x^2+x+1\text{.}\) But using TheoremÂ 1.2.33, we see that the quadratic function \(g(x)\) has not roots! We conclude that \(-3\) is the only root of \(f\text{.}\)
