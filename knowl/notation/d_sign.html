Definition 1.10.1. Sign function.

The sign function, \(\sgn\text{,}\) is the function \(\sgn\colon \R\rightarrow \R\) defined as
\begin{equation} \operatorname{sgn}(x) = \begin{cases}1, \amp \text{if } x> 0\\ 0, \amp \text{if } x=0\\ -1, \amp \text{if } x< 0. \end{cases}\tag{1.59} \end{equation}
In other words, \(\sgn(x)\) is 1 if \(x\) is positive, -1 if \(x\) is negative, and 0 otherwise.
