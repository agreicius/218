Definition 1.1.22. Range.

The range of a function \(f\colon A\rightarrow B\text{,}\) denoted \(\range f\text{,}\) is the set of all outputs of \(f\text{.}\) Using set-builder notation, we have
\begin{equation} \range f=\{b\in B\colon b=f(a) \text{ for some } a\in A\}\text{,}\tag{1.6} \end{equation}
or equivalently,
\begin{equation} \range f=\{f(a)\mid a\in A\}\text{.}\tag{1.7} \end{equation}
