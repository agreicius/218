We want to minimize the gas consumption per mile, not the gas consumption per hour. So we do not want to minimize the function \(g(v)\text{.}\) It has the wrong units. However, notice that
\begin{equation*} \begin{aligned}\text{ Units of } \frac{g(v)}{v} \amp = \frac{\text{ gallons/hour } }{\text{ miles/hour } } \\ \amp = \frac{\text{ gallons } }{\text{ hour } }\cdot\frac{\text{ hour } }{\text{ miles } } \\ \amp = \frac{\text{ gallons } }{\text{ miles } }, \end{aligned} \end{equation*}
suggesting we instead minimize the function \(G(v)=g(v)/v\text{.}\) We can still use the picture in FigureÂ 7.3.23 of gas consumption versus velocity to estimate the velocity which minimizes \(G(v)=g(v)/v\text{.}\)

We want to find the minimum value of \(G=g/v\) when \(g\) and \(v\) are related by the graph above. We could use the picture to sketch a graph of \(G\) against \(v\) and estimate a critical point. But there is an easier way. Because \(g\) is the vertical measure of the graph in FigureÂ 7.3.23 and \(v\) is the horizontal measure of the graph, \(g/v\) is the slope of the line from the origin \((0,0)\) to the point \(P=(v,g)\text{,}\) viewed as rise over run:
\begin{equation*} \text{ slope } = \frac{\text{ rise } }{\text{ run } } \frac{g-0}{v-0} = \frac{g}{v} \end{equation*}
