Solution.
  1. We solve
    \begin{align*} 3-2x \amp > 7\\ -2x \amp > 4 \amp (\text{add } -3)\\ x \amp < -2 \amp (\text{multiply by } -1/2)\text{.} \end{align*}
    Thus the set of solutions is the interval \((-\infty,-2)\text{.}\)

  2. Clearly, we want to use the reciprocal rule here, but we need to be careful since to do so, the signs of both sides of the inequality have to agree! We handle this by separately treating the cases where \(5/(x+1)\) is negative and positive.
    ðŸ”—
    Note that \(5/(x+1)< 0\) if and only if \(x+1< 0\) if and only if \(x< -1\text{.}\) In this case we certainly have \(5/(x+1)\leq 8\text{.}\) Thus the interval \((-\infty, -1)\) is part of our solution.
    ðŸ”—
    Now assume \(x> -1\text{.}\) In this case \(5/(x+1)\) is positive, allowing us to use the reciprocal rule. We solve:
    \begin{align*} \frac{5}{x+1}\amp\leq 8 \\ \frac{1}{8} \amp \leq \frac{x+1}{5} \amp (\text{reciprocal rule})\\ \frac{5}{8} \amp \leq x+1 \\ -\frac{3}{8} \amp \leq x\text{.} \end{align*}
    Thus \([-3/8,\infty)\) is the set of all \(x\) satisfying the inequality with \(x\geq -1\text{.}\)
    ðŸ”—
    Combining our two cases, we conclude that the set of solutions to the inequality is
    \begin{equation*} (-\infty, -1)\cup [-3/8,\infty)\text{.} \end{equation*}
