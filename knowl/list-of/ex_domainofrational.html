Example 1.5.30.

Find the domain of \(f(x) = \dfrac{x+5}{x^3+x^2-2x}\text{.}\)
ðŸ”—
Solution.
We know that the domain of a rational function is all \(x\)-values that are not zeros of the denominator. Thus we set the denominator \(x^3+x^2-2x\) equal to 0 and solve for \(x\text{,}\) which will give us the \(x\)-values to exclude from the domain.
ðŸ”—
Factoring, we have
\begin{equation*} \begin{aligned}x^3+x^2-2x \amp = x(x^2+x-2)\\ \amp = x(x+2)(x-1) \amp =0 \end{aligned} \end{equation*}
when
\begin{equation*} x=0 x+2=0 x-1=0 \end{equation*}
and so when \(x=0, -2, 1\text{.}\)
ðŸ”—
Thus the domain is \(x\neq -2, 0, 1\) since those are the \(x\)-values where the denominator is equal to 0. Using interval notation we can write the domain as
\begin{equation*} (-\infty,-2)\cup(-2,0)\cup(0,1)\cup(1,\infty) \end{equation*}
