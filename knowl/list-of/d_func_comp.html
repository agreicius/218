Definition 1.7.3. Function composition.

Given functions
\begin{align*} g\colon A \amp \rightarrow B \amp f\colon B \amp \rightarrow C \end{align*}
satisfying \(\range f\subseteq B\text{,}\) their composition \(h=f\circ g\) is the function \(h\colon A\rightarrow C\) defined as
\begin{equation} h(a)=f(g(a))\tag{1.52} \end{equation}
for all \(a\in A\text{.}\)
