Theorem 2.9.18. Invertibility equivalence.

Let \(f\colon A\rightarrow B\) be a function.
  1. If \(f\) is invertible, then \(f\) is one-to-one.

Proof.

  1. Assume \(f\) is invertible, and let \(f^{-1}\colon B\rightarrow A\) be its inverse. We have
    \begin{align*} f(x_1)=f(x_2) \amp \implies f^{-1}(f(x_1))=f^{-1}(f(x_2))\\ \amp \implies x_1=x_2 \amp (f^{-1}(f(a))=a \text{ for all } a\in A)\text{,} \end{align*}
    showing that \(f\) is one-to-one.

  2. Assume \(f\) is one-to-one. By definition of \(C=\range f\text{,}\) for every \(c\in C\) there is some \(a\in A\text{,}\) such that \(f(a)=c\text{.}\) Since furthermore \(f\) is one-to-one, there is in fact exactly one \(a\in A\) such that \(f(a)=c\text{.}\) Thus we can define a function \(f^{-1}\colon C\rightarrow A\) as follows: given \(c\in C\text{,}\) \(f^{-1}(c)=a\) where \(a\) is the unique element of \(A\) satisfying \(f(a)=c\text{.}\) It is now straightforward to verify that \(f^{-1}\) satisfies the inverse equations (2.13) and (2.14).
