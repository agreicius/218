Example 6.1.29. Visually and Algebraically NOT Invertible.

Consider the function \(y=s(t)=3-\dfrac{1}{5}(t-1)^2\) defined by the graph below:

We can see that \(s(t)\) does NOT pass the horizontal line test; in particular, it looks like \(s(2)=s(0)\text{,}\) and using the formula \(s(t) = 3-\frac{1}{5}(t-1)^2\text{,}\) we can indeed calculate that \(s(2)=s(0)=\dfrac{14}{5}\text{.}\) Therefore \(s(t)\) does not have an inverse function.
ðŸ”—
What if we did not have a picture? Would we still be able to determine that \(y=s(t)\) is not invertible? If we were to try to solve for \(t\) in terms of \(y\) algebraically, we could again subtract \(3\) from both sides to get
\begin{equation*} y-3=-\dfrac{1}{5}(t-1)^2\text{.} \end{equation*}
ðŸ”—
Next, multiplying both sides by \(-5\text{,}\) it follows that
\begin{equation*} -5(y-3)=(t-1)^2\text{.} \end{equation*}
ðŸ”—
Here, however, we encounter a crucial issue. Recall from ExampleÂ 6.1.15 that the function \(f(x)=x^2\) does not have an inverse. It takes any nonzero number and its opposite to the same output (e.g. \((-5)^2=25=(5)^2\)), this means that we have to account for both possible inputs that result in the same output. Based on our last equation, this means that either
\begin{equation*} t-1=\sqrt{-5(y-3)}\qquad\textrm{ or } \qquad t-1=-\sqrt{-5(y-3)} \end{equation*}
ðŸ”—
As such, we find not a single equation that expresses \(t\) as a function of \(y\text{,}\) but rather two:
\begin{equation*} t=1+\sqrt{-5(y-3)}\qquad\textrm{ or } \qquad t=1-\sqrt{-5(y-3)} \end{equation*}
ðŸ”—
It appears that \(t\) cannot be expressed as a single function of \(y\text{;}\) and indeed, \(y=s(t)=3-\dfrac{1}{5}(t-1)^2\) does not have an inverse function.
