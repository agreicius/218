Example 6.3.18.

The change of base formula is useful for simplifying certain gross computations involving logarithms. For example,
\begin{equation*} \log_{1/4} 32\sqrt{2} \end{equation*}
is a bit eye-melting. However, notice that all of the numbers involved are powers of \(2\text{.}\) Thus we might think to use the change of base formula with \(d=2\text{:}\)
\begin{equation*} \log_{1/4} 32\sqrt{2} = \frac{\log_2 32\sqrt{2}}{\log_2(1/4)} \end{equation*}
ðŸ”—
We now have two simpler, more manageable logarithms to deal with:
\begin{equation*} \log_2 32\sqrt{2}\qquad\text{ and } \qquad \log_2(1/4) \end{equation*}
ðŸ”—
We could convert \(32\sqrt{2}\) into a power of 2, or we could first use Logarithm Rule (a) to make this calculation even simpler:
\begin{equation*} \log_2(32\sqrt{2}) = \log_2 32+\log_2\sqrt{2} = \log_2 2^5+\log_2 2^{1/2} = 5+\frac{1}{2} = \frac{11}{2} \end{equation*}
ðŸ”—
Similarly, we could either convert \(1/4 = 2^{-2}\) in order to see that \(\log_2(1/4) = -2\text{,}\) or we could use Logarithm Rule (b):
\begin{equation*} \log_2\left(\frac{1}{4}\right) = \log_2 1-\log_2 4 = \log_2 2^0-\log_2 2^2 = 0-2 = -2\text{.} \end{equation*}
ðŸ”—
Putting all of this together, we see that:
\begin{equation*} \begin{aligned}\log_{1/4} 32\sqrt{2} \amp = \frac{\log_2 32\sqrt{2}}{\log_2(1/4)}\\ \amp = \frac{11/2}{-2}\\ \amp = -\frac{11}{4} \end{aligned} \end{equation*}
ðŸ”—
Which method is best? That depends entirely on which you think of first and can do most accurately. Just make sure not to make up your own "rules."
