Definition 1.10.11. Absolute value (piecewise).

Given a real number \(x\text{,}\) we definte its absolute value \(\abs{x}\) as follows:
\begin{equation} \abs{x}=\begin{cases} x\amp \text{if } x\geq 0\\ -x\amp \text{if } x< 0 \text{.}\end{cases}\tag{1.60} \end{equation}
The function \(f\colon \R\rightarrow \R\) defined as \(f(x)=\abs{x}\) is called the absolute value function.
