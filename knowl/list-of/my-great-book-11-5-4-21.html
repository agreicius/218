Example 9.4.15.

The limit
\begin{equation*} \lim_{x\to 0^+}(\sin x)^x \end{equation*}
has the form \(0^0\) since as \(x\to 0^+\) we have \(\sin x\to 0\) and \(x\to 0\text{.}\) Taking \(\ln\text{:}\)
\begin{equation*} \begin{aligned}\lim_{x\to 0^+}\ln\left((\sin x)^x\right) \amp = \lim_{x\to 0^+} x\ln(\sin x) \amp \leftarrow \sin x\to 0^+\text{ so } \ln(\sin x)\to -\infty\text{ ; form } 0\cdot\infty\\ \amp = \lim_{x\to 0^+}\frac{\ln(\sin x)}{1/x} \amp \leftarrow\text{ form } \frac{\infty}{\infty}\\ \amp \stackrel[L'H]{}= \lim_{x\to 0^+} \frac{(\cos x)/(\sin x)}{-1/x^2} \amp \leftarrow\text{ L'H\^opital's } \\ \amp = \lim_{x\to0^+}\frac{-x^2\cos x}{\sin x}\amp \leftarrow\text{ simplify; form } \frac{0}{0}\\ \amp \stackrel[L'H]{}= \lim_{x\to 0^+}\frac{-2x\cos x+x^2\sin x}{\cos x}\amp \leftarrow\text{ L'H\^opital's again; product rule } \\ \amp = \frac{0+0}{1} = 0\amp \leftarrow\text{ evaluate } \end{aligned} \end{equation*}
