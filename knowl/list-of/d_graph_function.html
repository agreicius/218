Definition 1.1.28. Graph of a function.

Let \(f\colon D\rightarrow \R\) be a function, where \(D\subseteq \R\text{.}\) The graph of \(f\text{,}\) denoted \(\operatorname{graph} f\text{,}\) is the set of all pairs \((x,f(x))\text{,}\) where \(x\in D\text{:}\) i.e.,
\begin{equation*} \operatorname{graph} f=\{(x,y)\in \R^2\mid x\in D \text{ and } y=f(x)\}=\{(x,f(x))\mid x\in D\}\text{.} \end{equation*}
