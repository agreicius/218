Example 8.4.16.

We can calculate the exact value of \(\sin^{-1}\left(\frac{1}{2}\right)\) using the unit circle. First, we translate from inverse trigonometric function to trigonometric function:
\begin{equation*} \theta = \sin^{-1}\left(\frac{1}{2}\right) \qquad \Longleftrightarrow\qquad \sin\theta = \frac{1}{2}\text{ with } -\frac{\pi}{2}\leq \theta\leq \frac{\pi}{2} \end{equation*}
