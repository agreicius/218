Definition 1.4.23. The Profit Function.

The profit function, written as \(\pi\) to distinguish it from the price, \(p\text{,}\) is the difference
\begin{equation*} \text{ Profit } = \text{ Revenue } -\text{ Cost } \text{ so } \pi=R-C \end{equation*}
