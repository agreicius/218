Example 5.6.2.

For example, consider the function \(f(x) = x^3\text{.}\) Calculating the first two derivatives,
\begin{equation*} f'(x) = 3x^2 \text{ and } f''(x) = 6x\text{.} \end{equation*}
\(f''(x)=6x=0\) only when \(x=0\) and is defined for all \(x\)-values, hence \(f''(x)\) can only change sign at \(x=0\text{.}\) Completing a sign analysis for \(f''(x) = 6x\text{:}\)
