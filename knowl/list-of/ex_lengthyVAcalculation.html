Example 3.4.33.

Consider the function
\begin{equation*} f(x) = \frac{x}{3-x}\text{.} \end{equation*}
Its domain is \(x\neq 3\) and so the only limit that we cannot compute by plugging in is \(\ds\lim_{x\to 3}\frac{x}{3-x}\text{.}\) If we do try to evaluate the limit by plugging in, we have
\begin{equation*} \lim_{x\to 3}\frac{x}{3-x} = \frac{3}{0}\text{.} \end{equation*}
As a value, this is undefined. However, we are attempting to describe the behavior of the function near \(x=3\text{.}\) Instead we recognize that this limit "has the form \(\frac{1}{0}\)" since its numerator approaches a non-zero number (3) and its denominator approaches zero. Thus we expect this limit to be infinite and to describe a vertical asymptote of \(f(x) = \frac{x}{3-x}\) at \(x=3\text{.}\)
