Example 8.3.20.

Determine algebraically whether the function
\begin{equation*} f(t) = \frac{\sin t}{1+\cos t} \end{equation*}
is even, odd, or neither.

Solution.
The function \(f(t)\) is even only if \(f(-t) = f(t)\) and is odd only if \(f(-t) = -f(t)\text{.}\) If neither of these identities are true, then the function is neither even nor odd.
