Example 1.10.22. Transformed absolute value function.

Consider the function \(f(x)=\abs{2x-1}\text{.}\) We have \(f(x)=\abs{g(x)}\text{,}\) where \(g(x)=2x-1\text{.}\) Since \(g(x)< 0\) if and only if \(x< 1/2\text{,}\) we obtain the graph of \(f\) by flipping the part of the line \(y=2x-1\) that lies below the \(x\)-axis across the \(x\)-axis.
