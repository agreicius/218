Example 1.15.21. IVT for an algebraic function.

Prove that the equation
\begin{equation} \sqrt[3]{x-1}+x=2\tag{1.81} \end{equation}
has a solution.
ðŸ”—
Solution.
Define \(f(x)=\sqrt[3]{x-1}+x-2\text{.}\) The function \(f\) is algebraic and hence continuous on its domain, which is \(\R\text{.}\) Observe that
\begin{align*} f(0) \amp =\sqrt[3]{-1}-2=-1-2=-3\\ f(2) \amp =\sqrt[3]{1}+2-2=1\text{.} \end{align*}
Since \(f\) is continuous on \([0,2]\) and since \(f(0)< 0 < f(2)\text{,}\) the intermediate value theorem implies that there is an element \(c\in (0,2)\) satisfying \(f(c)=0\text{.}\) Setting \(x=c\) gives us a solution to the original equation (1.81).
