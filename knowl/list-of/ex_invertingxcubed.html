Example 6.1.5.

For example, the functions \(f(x) = x^3\) and \(g(x) = \sqrt[3]{x}\) are inverses of each other. Intuitively, taking the cube of a number is undone by taking the cube root, and vice versa. Its easy to verify this algebraically by converting \(\sqrt[3]{x} = x^{1/3}\text{:}\)
\begin{equation*} g(f(x))=\sqrt[3]{x^3}=(x^3)^{1/3} = x^{3\cdot(1/3)} = x^1 = x, \textrm{ and} \end{equation*}
\begin{equation*} f(g(x))=(\sqrt[3]{x})^3=(x^{1/3})^3 = x^{(1/3)\cdot 3} = x^1=x\text{.} \end{equation*}
Thus we can write \(f^{-1}(x) = \sqrt[3]{x}\) for the inverse of \(f(x) = x^3\text{.}\) Likewise, we can write \(g^{-1}(x) = x^3\) for the inverse of \(g(x) = \sqrt[3]{x}\text{.}\)
