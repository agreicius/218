Solution.
As in the picture above, call the vertical height of the rocket \(y\text{.}\) The problem tells us to find \(\frac{dy}{dt}\) when \(\theta = \frac{\pi}{3}\) and \(\frac{d\theta}{dt} = 0.9\text{,}\) given that the base of the triangle is \(6\text{.}\) Note that we have no information at all about the hypotenuse, so do not wish to use either the sine or cosine function. Thus we relate \(y\) and \(\theta\) with the tangent function:
\begin{equation*} \tan\theta = \frac{y}{6}\text{.} \end{equation*}
Solving for \(y\text{,}\) we have
\begin{equation*} y = 6\tan\theta \end{equation*}
and taking the derivative in order to relate \(\frac{dy}{dt}\) to \(\frac{d\theta}{dt}\text{,}\) we get
\begin{equation*} \frac{dy}{dt} = 6\sec^2\theta\frac{d\theta}{dt} \end{equation*}

When \(\theta = \frac{\pi}{3}\text{,}\) \(\cos\theta = \cos(\pi/3) = 1/2\) and so \(\sec(\pi/3) = 2\text{.}\) Plugging this in along with \(\frac{d\theta}{dt} = 0.9\text{,}\) we have
\begin{equation*} \frac{dy}{dt}\Big|_{\substack{\theta = \pi/3\\ \frac{d\theta}{dt} = 0.9} } = 6\sec^2(\pi/3)\cdot(0.9) = 6(2)^2(0.9) = 21.6 \end{equation*}

Therefore the rocketâ€™s velocity is 21.6 km/min.
