Example 1.10.3.

Let \(f\) be the function defined piecewise as
\begin{equation*} f(x) = \begin{cases}x^2\amp \text{if } x\leq -1\\ 2x \amp \text{if } x> -1 \end{cases}\text{.} \end{equation*}
To graph \(f\) we graph the function \(g(x)=x^2\) on the domain \((-\infty, -1]\) and the function \(h(x)=2x\) on the domain \((-1,\infty)\text{.}\) Note that we must be careful about the value of \(f\) at the cutoff input \(x=-1\text{.}\) Since this value is included in the first case of the definition, we have \(f(-1)=(-1)^2=1\text{,}\) corresponding to the point \(P=(-1,1)\) on the graph below.
