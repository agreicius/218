Theorem 9.2.1. The Squeeze Theorem for Limits.

If \(l(x)\leq f(x)\leq u(x)\) for all \(x\) near \(c\text{,}\) not not necessarily at \(x=c\text{,}\) and if \(\ds\lim_{x\to c}l(x)\) and \(\ds\lim_{x\to c} u(x)\) are both equal to \(L\text{,}\) then \(\ds\lim_{x\to c}f(x) = L\text{.}\)
