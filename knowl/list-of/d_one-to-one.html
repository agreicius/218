Definition 2.10.1. One-to-one function.

A function \(f\) is one-to-one on its domain \(D\) if \(f(x_1)\ne f(x_2)\) for all \(x_1, x_2\in D\) with \(x_1\ne x_2\text{.}\) Using logical shorthand:
\begin{align} x_1\ne x_2 \amp \implies f(x_1)\ne f(x_2)\text{,}\tag{2.19} \end{align}
or equivalently, using the contrapositive,
\begin{align} f(x_1)=f(x_2) \amp \implies x_1=x_2\text{.}\tag{2.20} \end{align}
Given a subset \(I\subseteq D\text{,}\) we say that \(f\) is one-to-one on \(I\) if the implication (2.19) holds for all \(x_1, x_2\in I\text{.}\)
