Theorem 1.10.14. Absolute value function is algebraic.

For all \(x\in \R\) we have
\begin{equation} \sqrt{x^2}=\abs{x}\text{.}\tag{1.61} \end{equation}
It follows that the absolute value function \(f(x)=\abs{x}\) is algebraic.
