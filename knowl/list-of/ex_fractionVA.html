Example 3.6.27.

For example, consider the limit
\begin{equation*} \lim_{x\to 0}\frac{1}{x^2}-\frac{1}{x^4}\text{.} \end{equation*}
Even though this limit is not being taken at infinity, it results in a limit of the form \(\infty-\infty\text{:}\)
\begin{equation*} \lim_{x\to 0}\frac{1}{x^2}-\frac{1}{x^4} = \frac{1}{0^+}-\frac{1}{0^+} = \infty-\infty \end{equation*}
ðŸ”—
We try to resolve this by using algebra via finding a common denominator:
\begin{equation*} \lim_{x\to 0}\left(\frac{1}{x^2}-\frac{1}{x^4}\right) = \lim_{x\to 0}\left(\frac{x^2}{x^4}-\frac{1}{x^4}\right) =\lim_{x\to 0}\frac{x^2-1}{x^4} = \frac{0-1}{0^+} = \frac{-1}{0^+} = -\infty \end{equation*}
ðŸ”—
Thus we see that this limit describes a vertical asymptote that becomes infinitely negative to both the right and left of \(x=0\text{,}\) as illustrated in FigureÂ 3.6.28.
ðŸ”—
Alternate solution: Could we instead have approached this calculation in the same way as suggested for \(\infty-\infty\) limits: factor out the largest power of \(x\text{?}\) In this case the "largest" power is \(x^{-2}\text{:}\)
\begin{equation*} \frac{1}{x^2}-\frac{1}{x^4} = x^{-2}-x^{-4} = x^{-2}(1-x^{-2}) = \frac{1}{x^2}\left(1-\frac{1}{x^2}\right) \end{equation*}
ðŸ”—
Having done this, we can reevaluate the limit:
\begin{equation*} \lim_{x\to 0}\frac{1}{x^2}-\frac{1}{x^4} = \lim_{x\to 0} \frac{1}{x^2}\left(1-\frac{1}{x^2}\right) = (\infty)(1-\infty) = (\infty)(-\infty) = -\infty \end{equation*}
ðŸ”—
It worked!!!
