Definition 6.3.1. Logarithmic Functions as Inverses of Exponential Functions.

The inverse of the base-\(b\) exponential function \(b^x\) is the base-\(b\)logarithmic function
\begin{equation*} \log_b x \end{equation*}
which is the power that we raise \(b\) to to get \(x\text{.}\) In other words,
\begin{equation*} y=\log_b x \Leftrightarrow b^y=x \end{equation*}
