Definition 1.1.18. Domain and Range of a Function.

If \(f\) is a function between unspecified subsets of \(\mathbb{R}\text{,}\) then we will take the domain of \(f\) to be the largest subset of \(\mathbb{R}\) for which \(f\) is defined:
\begin{equation*} \text{ Domain } (f) = \{\text{ real numbers } x\text{ such that } f(x)\text{ is defined } \}\text{.} \end{equation*}
The range of such a function is the set of all possible outputs that it can attain:
\begin{equation*} \text{ Range } (f) = \{\text{ real numbers } y\text{ such that we can find } x\text{ in Domain } (f)\text{ for which } f(x) = y\} \end{equation*}
