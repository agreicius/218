Example 6.5.18.

Calculate \(\displaystyle\lim_{x\to0}\frac{3^{2x}+3^x-2}{3^x-1}\text{.}\)
ðŸ”—
Solution.
First, we try to evaluate the limit. If we find that the limit is indeterminate, then we will use our limit-evaluating skills to rewrite the limit until it can be evaluated.
ðŸ”—
As \(x\to0\text{,}\) both the numerator and the denominator approach zero, so this limit is of the form \(\frac{0}{0}\text{,}\) which is indeterminate. So, we need to rewrite the limit, likely using algebra.
ðŸ”—
Notice that we can factor the numerator in the following way: call \(z=3^x\text{.}\) Then, \(3^{2x}+3^x-2=(3^{x})^2+3^x-2=z^2+z-2\text{,}\) which can be factored as \((z+2)(z-1)\text{.}\) We then replace \(z\) with \(3^x\) to obtain a factored numerator: \(3^{2x}+3^x-2=(3^x-1)(3^x+2)\text{.}\)
ðŸ”—
Returning to the limit, we use the factored form of the numerator:
\begin{equation*} \displaystyle\lim_{x\to0}\frac{3^{2x}+3^x-2}{3^x-1}=\displaystyle\lim_{x\to0}\frac{(3^x-1)(3^x+2)}{3^x-1}\text{.} \end{equation*}
How does this help us? Well, we now see that both the denominator and numerator have a factor of \(3^x-1\text{.}\) Using the cancelation property for limits, we can cancel these factors, and then finish the limit via evaluation:
\begin{equation*} \displaystyle\lim_{x\to0}\frac{(3^x-1)(3^x+2)}{3^x-1}=\displaystyle\lim_{x\to0}(3^x+2)=3^0+2=3\text{.} \end{equation*}
