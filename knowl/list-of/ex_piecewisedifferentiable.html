Example 4.3.20.

Is it possible for the function
\begin{equation*} f(x) = \begin{cases}3x+c, \amp \text{ if } x\lt 1\\ 6-x, \amp \text{ if } x\geq 1 \end{cases} \end{equation*}
to be differentiable at \(x=1\) for an appropriate choice of \(c\text{?}\) Why or why not?
ðŸ”—
Solution.
In order for the function to be differentiable at \(x=1\text{,}\) it must first be continuous at \(x=1\text{.}\) Thus we find the limits to the right and left of \(x=1\text{:}\)
\begin{equation*} \lim_{x\to 1^-}f(x) = \lim_{x\to 1^-}(3x+c) = 3(1)+c = 3+c \end{equation*}
\begin{equation*} \lim_{x\to 1^+}f(x) = \lim_{x\to 1^+}(6-x) =6-1 = 5 \end{equation*}
ðŸ”—
We know that to be continuous at \(x=1\) we cannot have a jump, and need
\begin{equation*} 3+c = 5\Longrightarrow c = 2 \end{equation*}
ðŸ”—
We discovered that for \(c=2\text{,}\) the function is continuous. We can ask if it is differentiable. The slope of the function for \(x\lt 1\) is \(m=3\text{,}\) while the slope of the function for \(x>1\) is \(m=-1\text{.}\) So even if the function is continuous, the derivatives to the left and right at 1 will change suddenly from \(m=3\) to \(m=-1\text{.}\) That is a corner, hence the function is not differentiable at \(x=1\text{.}\)
