Procedure 1.13.4. Epsilon-delta proof for limits.

To prove \(\lim_{x\to a}f(x)=L\) for a function \(f\) and real numbers \(a,L\in \R\text{,}\) proceed as follows.
  1. Treating \(\epsilon\) as an arbitrary positive constant, solve the inequality
    \begin{equation} \abs{f(x)-L}< \epsilon\tag{1.80} \end{equation}
    for \(x\text{.}\) That is, find the set \(S\) of all \(x\) satisfying (1.80). This set is typically a union of one or more intervals.
