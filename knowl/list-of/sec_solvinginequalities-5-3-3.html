When \(x^2-1\) is positive or zero, its absolute value will remain \(x^2-1\text{.}\) When \(x^2-1\) is negative, its absolute value will be \(-(x^2-1)\text{.}\) Therefore we have
\begin{equation*} f(x) = |x^2-1| = \begin{cases}x^2-1, \amp \text{ for all } x\text{ with } x^2-1\geq 0\\ -(x^2-1), \amp \text{ for all } x\text{ with } x^2-1\lt 0 \end{cases} \end{equation*}
