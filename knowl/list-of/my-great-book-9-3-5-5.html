Example 7.2.17.

Use LH么pitals rule to calculate the limit of \(\displaystyle\lim_{x\to0^+}x\ln x\text{.}\)
Solution.
Note that we consider only the limit from the right of 0, since \(\ln x\) is not defined for negative numbers.
As \(x\to0^{+}\text{,}\) we have \(x\to0\) and \(\ln x\to-\infty\text{,}\) and therefore the limit \(\displaystyle\lim_{x\to0^+}x\ln x\) is in the indeterminate form \(0(-\infty)\text{:}\)
\begin{equation*} \displaystyle\lim_{x\to0^+}x\ln x = 0(-\infty) \end{equation*}
In order to apply LH么pitals rule we must rewrite the limit in the form \(\frac{0}{0}\) or \(\frac{\infty}{\infty}\text{.}\) In this case, it is easier to leave \(\ln x\) in the numerator; as \(x\to0^+\) we have
\begin{align*} \displaystyle\lim_{x\to0^+}x\ln x \amp =\displaystyle\lim_{x\to0^+}\frac{\ln x}{x^{-1}} = \frac{-\infty}{\infty}\amp \leftarrow\amp \textrm{ limit is now in form } \frac{-\infty}{\infty}\\ \amp \stackrel[L'H]{}=\displaystyle\lim_{x\to0^+}\frac{1/x}{-1/x^2} = \frac{\infty}{-\infty}\amp \leftarrow\amp \textrm{ apply L'H\^opital's rule } \end{align*}
This last expression still has an indeterminate form, but now it can be simplified using algebra:
\begin{equation*} \frac{1/x}{-1/x^2} = \frac{1}{x}\cdot\frac{-x^2}{1} = \frac{-x^2}{x} = -x \end{equation*}
Lastly, we finish computing the limit
\begin{equation*} \displaystyle\lim_{x\to0^+}-x=0 \end{equation*}
Thus we have found that
\begin{equation*} \lim_{x\to0^+}x\ln x = 0\text{.} \end{equation*}
Note: We could have applied LH么pitals rule a second time in this problem. Immediately after the first application of the rule, the limit \(\displaystyle\lim_{x\to0^+}\frac{1/x}{-1/x^2}\) was again in the indeterminate form \(\frac{\infty}{-\infty}\text{.}\) However, simplifying via algebra instead resulted in a very easy limit that we could evaluate. You should always look for the simplest possible means of evaluating any limit, applying algebraic methods before using a fancy theorem like LH么pitals rule.
