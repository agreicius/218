Example 1.11.10.

Consider the function
\begin{equation*} f(x) = \frac{1}{(x-1)^2}\text{.} \end{equation*}
We can see that the domain of \(f(x)\) is \(x\neq 1\) since the denominator is equal to zero when \(x-1=0\text{,}\) hence when \(x=1\text{.}\) Graphing \(f(x)\text{,}\) we see that the values of \(f(x)\) become infinitely large as \(x\) gets close to \(x=1\text{:}\)
