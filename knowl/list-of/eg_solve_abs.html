  2. We solve as follows:
    \begin{align*} \abs{x} \amp > \frac{3}{\abs{x-2}}\\ \abs{x}\cdot\abs{x-2} \amp > 3 \amp (\abs{x-2}> 0)\\ \abs{x^2-2x} \amp > 3 \amp \knowl{./knowl/xref/eq_abs_prod.html}{\text{(1.62)}}\\ x^2-2x > 3 \amp \text{ or } x^2-2x < -3 \amp \knowl{./knowl/xref/eq_abs_lesser.html}{\text{(1.65)}} \\ x^2-2x-3> 0 \amp \text{ or } x^2-2x+3< 0\\ (x-3)(x+1)> 0 \amp \text{ or } (x-1)^2+2< 0 \\ x> 3 \text{ or } x< -1 \amp \text{.} \end{align*}
    Here the last line follows from a sign analysis of \(f(x)=(x-3)(x+1)\) (or alternatively, by constructing a sign diagram) and the fact that \((x-1)^2+2\geq 2\) for all \(x\text{,}\) and hence never satisfies \((x-1)^2+2< 0\text{.}\) We conclude that the solution set is \((-\infty, -1)\cup (3,\infty)\text{.}\)
