Definition 3.4.20. Horizontal Asymptote.

A function \(f\) has a horizontal asymptote at \(y=L\) if one or both of the following are true:
\begin{equation*} \lim_{x\to\infty}f(x) = L \text{ or } \lim_{x\to-\infty}f(x) = L\text{.} \end{equation*}
If \(\ds\lim_{x\to\infty}f(x) = L\text{,}\) then \(f(x)\) has a right horizontal asymptote at \(y=L\text{,}\) and if \(\ds\lim_{x\to-\infty}f(x) = L\text{,}\) then \(f(x)\) has a left horizontal asymptote at \(y=L\text{.}\)
