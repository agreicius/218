Example 1.1.15. Implied domain: reciprocal function.

Let \(f\) be the function defined as \(f(x)=1/x\text{.}\) Following FiatÂ 1.1.11, give a full definition of this function by determining the codomain and implied domain.

Solution.
Since \(f\) is only given as a formula, following FiatÂ 1.1.11, the codomain of \(f\) is \(\R\) and the implied domain is the set of all real numbers \(x\) for which the formula \(1/x\) makes sense. Since the fraction \(1/x\) is well-defined if and only of \(x\ne 0\text{,}\) we conclude that the domain \(D\) of \(f\) is the set of all nonzero real numbers: i.e.,
\begin{equation*} D=\{x\in \R\mid x\ne 0\}\text{.} \end{equation*}
Thus, in all its glorious detail, \(f\) is the function \(f\colon D\rightarrow \R\) defined as \(f(x)=1/x\text{,}\) where \(D=\{x\in \R\mid x\ne 0\}\text{.}\)
