From this graph it seems reasonable that \(\ds\lim_{x\to0}x^2\sin \frac{1}{x} = 0\text{.}\) We can use the Squeeze Theorem to find this limit algebraically. The sine function has outputs that are always between \(-1\) and \(1\text{,}\) regardless of the input. Therefore,
\begin{equation*} -1\leq \sin\frac{1}{x}\leq 1 \end{equation*}
where we are substituting \(\theta = \frac{1}{x}\) in the inequality \(-1\leq \sin \theta\leq 1\text{.}\) Multiplying both sides of the above inequality by \(x^2\text{,}\)
\begin{equation*} -x^2\leq x^2\sin\frac{1}{x}\leq x^2\text{.} \end{equation*}
