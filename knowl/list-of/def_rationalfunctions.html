Definition 1.5.23. Rational Functions are Quotients of Polynomials.

A rational function is a function that can be written as the quotient of two polynomial functions
\begin{equation*} f(x) = \frac{p(x)}{q(x)} = \frac{a_nx^n+a_{n-1}x^{n-1}+\ldots+a_1x+a_0}{b_mx^m+b_{m-1}x^{m-1}+\ldots+b_1x+b_0} \end{equation*}
for any \(x\) such that \(q(x)\neq 0\text{.}\)
