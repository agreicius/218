For example, the function \(f(x) = \frac{x}{\sqrt{x+1}}\) likely has a vertical asymptote at \(x=-1\) since that is where the denominator is equal to 0. Note as well that \(\sqrt{x+1}\) is undefined for \(x\lt -1\) since we must have \(x+1\geq 0\) inside the square root.

The denominator is positive and the numerator is negative, hence the fraction is negative overall for values of \(x\) near \(-1\text{.}\) Thus we conclude that the values of the expression become very large and are negative, which we summarize using limit notation as
\begin{equation*} \lim_{x\to -1^+}\frac{x}{\sqrt{x+1}} = -\infty \end{equation*}
