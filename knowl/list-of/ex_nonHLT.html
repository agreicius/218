For example, consider the squaring function \(f(x) = x^2\) with the domain \(\mathbb{R}\text{.}\) How could we reverse the process defined by \(f\text{?}\)

Observe that \(f(2) = 4\) and \(f(-2)=4\text{.}\) An inverse of \(f\) would have to send the input 4 back to both \(2\) and \(-2\) in order to reverse \(f\text{.}\) This violates our rules for a function since an input must have one and only one output. Thus \(f(x) = x^2\) does not have an inverse.
