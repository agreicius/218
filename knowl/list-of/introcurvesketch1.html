Example 5.6.10.

Consider the function
\begin{equation*} f(x) = x^3-3x+2\text{.} \end{equation*}
  1. The domain of \(f(x) = x^3-3x+2\) is all real numbers, since \(f(x)\) is a polynomial.

  3. \(f(x)\) is cubic and does not factor easily, so we will not find its \(x\)-intercepts or where the function is positive and negative. (After finishing our sketch, we will likely have a better idea of where the roots are.)

  4. The first derivative of \(f(x) = x^3-3x+2\) is
    \begin{equation*} f'(x) = 3x^2-3\text{.} \end{equation*}
    This derivative is always defined and continuous, so the critical points of \(f\) are just the places where \(f'(x)=0\text{:}\)
    \begin{equation*} \begin{aligned}3x^2-3 \amp =0\\ 3x^2 \amp =3\\ x^2 \amp =1\\ x \amp =\pm 1 \end{aligned} \end{equation*}
    These critical points divide the real-number line into three intervals and we test the sign of \(f'\) on each interval:
    Thus we see that \(f\) has a local maximum at \(x=-1\text{,}\) where \(f'\) changes from positive to negative, and a local minimum at \(x=1\text{,}\) where \(f'\) changes from negative to positive. \(f\) is increasing on \((-\infty,-1)\cup(1,\infty)\) and decreasing on \((-1,1)\text{.}\) The local maximum at \(x=-1\) has \(y\)-value \(y=f(-1) = (-1)^3-3(-1)+2 = 4\text{,}\) so the point where the maximum occurs is \((-1,4)\text{.}\) The local minimum at \(x=1\) has \(y\)-value \(y=f(1) = (1)^3-3(1)+2 = 0\text{,}\) so the point where the minimum occurs is \((1,0)\text{.}\) Note that we have found one of the \(x\)-intercepts that we could not find directly via factoring! This is a happy coincidence.

  5. Moving on to the second derivative, we have \(f''(x) = 6x\text{.}\) This is zero when \(x=0\) and is never undefined. Drawing a sign chart:
    Thus we see that \(f\) has an inflection point at \(x=0\text{,}\) where \(f''\) changes from negative to positive. \(f\) is concave down on \((-\infty,0)\) and concave up on \((0,\infty)\text{.}\) The inflection point at \(x=0\) has \(y\)-value \(y=f(0) = (0)^3-3(0)+2 = 2\text{,}\) so the point where the graph changes concavity is \((0,2)\text{.}\) This also happens to be the \(y\)-intercept of the function (though that is a coincidence).
