Example 3.5.8.

For example, the four limits that follow are all initially of the indeterminate form \(\frac{0}{0}\text{.}\) After some algebra, we see that three limits exist (and describe holes) and one limit becomes infinite (and describes a vertical asymptote). The three that exist are each equal to different real numbers.
  1. Consider \(f(x) = \frac{x^2-2x+1}{x-1}\) at \(x=1\text{.}\) Initially,
    \begin{equation*} \ds\lim_{x\to 1}\frac{x^2-2x+1}{x-1} = \frac{1-2+1}{1-1} = \frac{0}{0} \end{equation*}
    However, we can factor the numerator: \(x^2-2x+1 = (x-1)(x-1) = (x-1)^2\) and then evaluate the limit by cancelling:
    \begin{equation*} \lim_{x\to 1}\frac{(x-1)^2}{x-1} = \lim_{x\to 1}\frac{x-1}{1} = \frac{0}{1} = 0 \end{equation*}
    The function \(f(x)\) has a hole at the point \((1,0)\text{.}\)

  2. Now consider \(g(x) = \frac{x-1}{(x-1)^3}\) at \(x=1\text{.}\) Again, technically originally the limit has the form \(\frac{0}{0}\text{:}\)
    \begin{equation*} \lim_{x\to 1}\frac{x-1}{(x-1)^3} = \frac{0}{0} \end{equation*}
    but we can cancel one of the factors of \(x-1\) and re-evaluate to get an infinite limit:
    \begin{equation*} \lim_{x\to 1}\frac{x-1}{(x-1)^3} = \lim_{x\to 1}\frac{1}{(x-1)^2} = \frac{1}{0^+} = \infty \end{equation*}
    The function \(g(x)\) has a vertical asymptote at \(x=1\text{.}\)

  3. Now consider \(h(x) = \frac{x-1}{x-1}\text{.}\) \(h(x)\) is undefined at \(x=1\) and the limit as \(x\to 1\) has the form \(\frac{0}{0}\text{.}\) However, when we simplify:
    \begin{equation*} \lim_{x\to 1}\frac{x-1}{x-1} = \lim_{x\to 1}1 = 1 \end{equation*}
    Thus \(h(x)\) is the constant function \(y=1\text{,}\) except for the hole at the point \((1,1)\text{.}\)

  4. Lastly, consider the function \(k(x) = \frac{x-1}{3x-3}\text{,}\) which is also undefined at \(x=1\text{.}\) We can factor a 3 from the denominator and then cancel:
    \begin{equation*} \lim_{x\to 1}\frac{x-1}{3x-3}= \lim_{x\to 1}\frac{x-1}{3(x-1)} = \lim_{x\to 1}\frac{1}{3} = \frac{1}{3} \end{equation*}
    So \(k(x)\) is equal to the constant function \(y=\frac{1}{3}\text{,}\) except for the hole at the point \(\left(1,\frac{1}{3}\right)\text{.}\)
