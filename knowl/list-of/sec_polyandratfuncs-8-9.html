Example 1.4.30.

For example, if you can cancel a common factor \(x-c\) from the numerator and denominator of a fraction, the cancelled expression will equal the original expression for all values of \(x\) except \(x=c\text{.}\) One illustration of this theorem is the cancellation
\begin{equation*} \frac{(x-3)}{(x-3)(x-1)} = \begin{cases}\frac{1}{x-1},\amp \text{ if } x\neq 3\\ \text{ undefined } , \amp \text{ if } x=3 \end{cases} \end{equation*}
