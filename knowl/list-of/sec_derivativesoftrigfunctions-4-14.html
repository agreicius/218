So we see that \(h(x)\) is a composition of three functions: the outer-most function is squaring, next is secant, and the inner-most function is \(e^x\text{.}\) Thus we need to apply the chain rule twice:
\begin{equation*} \begin{aligned}\frac{d}{dx}(\sec^2 e^x) \amp = \frac{d}{dx}\big[ \left(\sec(e^x)\right)^2\big] \amp \leftarrow\text{ rewrite so compositions are clear } \\ \amp = 2(\sec(e^x))^{2-1}\cdot\frac{d}{dx}\left[\sec(e^x)\right] \amp \leftarrow\text{ first application of the chain rule to } (\cdot)^2\\ \amp = 2(\sec e^x)\cdot\sec(e^x)\tan(e^x)\cdot\frac{d}{dx}\left[e^x\right] \amp \leftarrow\text{ second application of the chain rule to } \sec(\cdot)\\ \amp = 2(\sec e^x)^2\tan e^x\cdot e^x\amp \leftarrow\text{ derivative of } e^x\\ \amp = 2e^x\sec^2e^x\tan e^x \amp \leftarrow\text{ rewriting without extra parentheses } \end{aligned} \end{equation*}
