Example 8.7.14.

For example, we can use the quotient rule to take the derivative of the tangent function:
\begin{equation*} \begin{aligned}\frac{d}{dx}(\tan x) \amp = \frac{d}{dx}\left(\frac{\sin x}{\cos x}\right) \amp \\ \amp = \frac{\frac{d}{dx}(\sin x)\cdot(\cos x) - (\sin x)\cdot\frac{d}{dx}(\cos x)}{(\cos x)^2} \amp \leftarrow \text{ quotient rule } \\ \amp =\frac{(\cos x)(\cos x) - (\sin x)(-\sin x)}{\cos^2 x} \amp \leftarrow \text{ derivatives of } \sin x\text{ and } \cos x\\ \amp = \frac{\cos^2x+\sin^2x}{\cos^2 x} \amp \leftarrow\text{ algebra } \\ \amp = \frac{1}{\cos^2x} \amp \leftarrow\text{ Pythagorean Identity } \cos^2x+\sin^2x=1\\ \amp = \sec^2 x \amp \leftarrow\text{ rewriting } \frac{1}{\cos x} = \sec x \end{aligned} \end{equation*}
