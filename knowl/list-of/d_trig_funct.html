Definition 2.18.9. Trigonometric functions.

The trigonometric functions are the six functions consisting of \(\cos\) and \(\sin\text{,}\) together with the following four functions.
  1. Tangent.
    The tangent function, is defined as
    \begin{align} \tan \theta \amp =\frac{\sin\theta}{\cos\theta}\text{.}\tag{2.80} \end{align}
    The domain of \(\tan\) is
    \begin{align*} D\amp =\{\theta\in \R\mid \theta\ne \pi/2+\pi n \text{ for } n\in \Z\} \amp \\ \amp =\dots (-3\pi/2, -\pi/2)\cup (-\pi/2, \pi/2) \cup (\pi/2, 3\pi/2)\cup \dots\text{.} \end{align*}
    🔗
    🔗
  2. Cotangent.
    The cotangent function is defined as
    \begin{align} \cot\theta \amp =\frac{1}{\tan\theta}=\frac{\cos\theta}{\sin\theta}\text{.}\tag{2.81} \end{align}
    The domain of \(\cot\theta\) is
    \begin{align*} D\amp =\{\theta\in \R\mid \theta\ne \pi n \text{ for } n\in \Z\} \amp \\ \amp =\dots (-2\pi, -\pi)\cup (-\pi, 0)\cup (0, \pi)\cup (\pi, 2\pi)\cup \dots \end{align*}
    🔗
    🔗
  3. Secant.
    The secant function is defined as
    \begin{align} \sec\theta \amp =\frac{1}{\cos\theta}\text{.}\tag{2.82} \end{align}
    The domain of \(\sec\theta\) is
    \begin{align*} D\amp =\{\theta\in \R\mid \theta\ne \pi/2+\pi n \text{ for } n\in \Z\} \amp \\ \amp =\dots (-3\pi/2, -\pi/2)\cup (-\pi/2, \pi/2) \cup (\pi/2, 3\pi/2) \dots\text{.} \end{align*}
    🔗
    🔗
  4. Cosecant.
    The cosecant function is defined as
    \begin{align} \csc\theta \amp =\frac{1}{\sin\theta}\text{.}\tag{2.83} \end{align}
    The domain of \(\csc\theta\) is
    \begin{align*} D\amp =\{\theta\in \R\mid \theta\ne \pi n \text{ for } n\in \Z\} \amp \\ \amp =\dots (-2\pi, -\pi)\cup (-\pi, 0)\cup (0, \pi)\cup (\pi, 2\pi)\cup \dots\text{.} \end{align*}
