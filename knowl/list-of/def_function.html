Definition 1.1.1. Function, Independent and Dependent Variables.

A function \(f\) is a rule that takes certain numbers \(a\) in the set \(A\) as inputs and assigns to each a single number \(b\) in the set \(B\) as the output.
ðŸ”—
The notation
\begin{equation*} f: A\rightarrow B \end{equation*}
represents a function \(f\) together with its input set \(A\) and output set \(B\text{.}\) It is pronounced "\(f\) from \(A\) to \(B\text{.}\)" The notation
\begin{equation*} b=f(a)\text{,} \end{equation*}
pronounced "\(b\) is equal to \(f\) of \(a\text{,}\)" represents the assignment of the output \(b\) to the input \(a\text{.}\)
ðŸ”—
The collection of symbols \(f(a)=b\) is called function notation, where the \(a\) in parentheses is called the independent variable and represents the input of the function. The output of the function is represented by either \(f(a)\) or \(b\text{,}\) which is called the dependent variable because its value depends on the choice of input.
