Example 7.2.1.

Consider the limits
\begin{equation*} \lim_{x\to 0}\frac{\ln(1+x)}{x}\qquad\text{ and } \qquad\lim_{x\to\infty}\frac{x}{e^x} \end{equation*}

The first limit evaluates to
\begin{equation*} \lim_{x\to 0}\frac{\ln(1+x)}{x} = \frac{\ln(1)}{0} = \frac{0}{0} \end{equation*}
and thus is indeterminate. There is no algebra that will let us alter this expression so that it is not indeterminate.

\begin{equation*} \lim_{x\to\infty}\frac{x}{e^x} = \frac{\infty}{\infty} \end{equation*}
and thus is also indeterminate. Again, there is no algebra that will let us alter this expression so that it is not indeterminate.
