Example 1.5.31.
Find the domain of the algebraic function
\begin{equation*} f(x) = \sqrt{\frac{3}{x}-2} \end{equation*}
ðŸ”—
Solution.
To find the domain of \(f\text{,}\) we ask which values of \(x\) can be plugged into the equation that defines \(f(x)\) and produce a real number when the expression is simplified. In order for the value of \(f\) to be defined for a real number \(x\text{,}\) that value of \(x\) must make the expression underneath the square-root sign nonnegative (meaning \(\frac{3}{x}-2\geq 0\)) and the denominator appearing in \(\frac{3}{x}\) nonzero (meaning \(x\neq 0\)). Thus the domain of \(f(x)\) is the set of real numbers that satisfy:
\begin{equation*} \frac{3}{x}-2\geq 0\text{ and } x\neq 0 \end{equation*}
ðŸ”—
In order to tackle the inequality, we need to rewrite the left-hand side as a single fraction:
\begin{equation*} \begin{aligned}\frac{3}{x}-2 \amp \geq 0\amp \amp \leftarrow \text{ original inequality } \\ \frac{3}{x}-\frac{2x}{x} \amp \geq 0 \amp \amp \leftarrow \text{ find common denominator } \\ \frac{3-2x}{x}\amp \geq 0 \amp \amp \leftarrow \text{ combine fractions } \end{aligned} \end{equation*}
ðŸ”—
The fraction can change sign when the numerator is zero:
\begin{equation*} 3-2x=0\text{ when } x=\frac{3}{2} \end{equation*}
or when the denominator is zero:
\begin{equation*} x=0\text{.} \end{equation*}

Thus \(\frac{3-2x}{x}\) is positive on \(\left(0,\frac{3}{2}\right)\text{,}\) and we know that it is equal to zero at \(x=\frac{3}{2}\text{;}\) it is undefined at \(x=0\text{.}\) The equivalent inequality \(\frac{3}{x}-2\geq 0\) thus has solution \(\left(0,\frac{3}{2}\right]\text{.}\) Note that the second condition, \(x\neq 0\text{,}\) on the domain is also satisfied.
ðŸ”—
Hence overall we have found the domain of the function \(f(x) = \sqrt{\frac{3}{x}-2}\) by determining that \(\frac{3}{x}-2\geq 0\) and \(x\neq 0\) on the interval
\begin{equation*} \left(0,\frac{3}{2}\right] \end{equation*}
