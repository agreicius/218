Example 4.5.10.

Let \(h(t) = f(t)\cdot g(t)\text{.}\) Use the figures below to find the exact value of \(h'(1)\text{.}\)
ðŸ”—
Solution.
First, observe that
\begin{equation*} h'(t) = f'(t)g(t)+f(t)g'(t) \end{equation*}
so that
\begin{equation*} h'(1) = f'(1)g(1)+f(1)g'(1)\text{.} \end{equation*}
Thus we will need the values \(f(1), g(1), f'(1)\text{,}\) and \(g'(1)\) from the graphs.
ðŸ”—
On the graph of \(y=f(t)\text{,}\) we can see that \(f(1) = 0\text{.}\) On the graph of \(y=g(t)\text{,}\) we cannot see the value of \(g(1)\) without determining the equation of the line segment containing \(x=1\text{.}\) The slope looks to be \(\frac{\text{ rise } }{\text{ run } } = \frac{2}{3/2} = \frac{4}{3}\) and we can see that the \(y\)-intercept is \(0\text{,}\) so that \(g(t) = \frac{4}{3}x\text{.}\) Thus \(g(1) = \frac{4}{3}\text{.}\)
ðŸ”—
Since \(f(t)\) and \(g(t)\) are piecewise-linear functions, their derivatives at any point is given by the slope of the line segment at that point. On the graph of \(y=f(t)\text{,}\) the slope of the line segment containing \(x=1\) looks to be \(\frac{\text{ rise } }{\text{ run } } = \frac{1}{3/2} = \frac{2}{3}\text{.}\) Thus \(f'(1) = \frac{2}{3}\text{.}\) On the graph of \(y=g(t)\text{,}\) we already observed that the slope of the line segment containing \(x=1\) looks to be \(\frac{4}{3}\text{.}\) Thus \(g'(1) = \frac{4}{3}\text{.}\)
ðŸ”—
Putting these values together, we have
\begin{equation*} h'(1) = f'(1)g(1)+f(1)g'(1) = \left(\frac{2}{3}\right)\left(\frac{4}{3}\right)+(0)\left(\frac{4}{3}\right) = \frac{8}{9} \end{equation*}
