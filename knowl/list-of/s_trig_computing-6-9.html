Definition 2.17.20. Trigonometric functions for acute angles.

Given an acute angle \(\theta\text{,}\) the six trigonometric functions are defined as the following six ratios of side lengths:
  1. \(\displaystyle \sin\theta = \frac{\text{ opp } }{\text{ hyp } }\)
    🔗
    🔗
  2. \(\displaystyle \csc\theta = \frac{\text{ hyp } }{\text{ opp } } = \frac{1}{\sin\theta}\)
    🔗
    🔗
  3. \(\displaystyle \cos\theta = \frac{\text{ adj } }{\text{ hyp } }\)
    🔗
    🔗
  4. \(\displaystyle \sec\theta = \frac{\text{ hyp } }{\text{ adj } } = \frac{1}{\cos\theta}\)
    🔗
    🔗
  5. \(\displaystyle \tan\theta = \frac{\text{ opp } }{\text{ adj } } = \frac{\sin\theta}{\cos\theta}\)
    🔗
    🔗
  6. \(\displaystyle \cot\theta = \frac{\text{ adj } }{\text{ opp } } = \frac{\cos\theta}{\sin\theta}\)
    🔗
    🔗
🔗
It is worth noting in particular that \(\cot\theta = \frac{1}{\tan\theta}\) for most values of \(\theta\text{.}\)
