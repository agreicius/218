Theorem 1.9.32. Cancellation when Variables are Involved.

Given any real expression \(A\) that depends on a variable \(x\text{,}\)
\begin{equation*} \frac{A}{A}= \begin{cases}1, \amp \text{ for values of } x\text{ that make } A\neq 0\\ \text{ undefined } ,\amp \text{ for values of } x\text{ that make } A=0 \end{cases} \end{equation*}
