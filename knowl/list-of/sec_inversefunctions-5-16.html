Example 6.1.8.

Suppose \(f\) is an invertible function with inverse \(f^{-1}\text{.}\) If \(f(3) = 7\text{,}\) what does this tell us about \(f^{-1}\text{?}\)

The function \(f^{-1}\) reverses the process of \(f\text{,}\) so that inputs become outputs and vice versa. Thus if \(f\) takes the input of 3 to the output of 7:
\begin{equation*} f(3)=7 \end{equation*}
then \(f^{-1}\) should reverse this process and take the input of 7 to the output of 3:
\begin{equation*} f^{-1}(7)=3\text{.} \end{equation*}
