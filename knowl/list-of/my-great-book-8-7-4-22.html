Example 6.6.22.

Find the derivative of
\begin{equation*} h(x) = 3\sqrt{e^x}-x^2 \end{equation*}

There are two possible ways to approach this problem. The first is to recognize that the exponential function is on the inside of a composition, since \(e^x\) appears inside the square root. Thus we have
\begin{align*} \frac{d}{dx}(3\sqrt{e^x}-x^2) \amp = \frac{d}{dx}(3(e^x)^{1/2}-x^2) \amp \leftarrow\amp \text{ rewrite so differentiation will be easier }\\ \amp = 3\left(\frac{1}{2}\right)(e^x)^{-1/2}(e^x)-2x \amp \leftarrow \amp \text{ don't forget the chain rule! }\\ \amp = \frac{3}{2}e^x(e^x)^{-1/2}-2x \amp \leftarrow\amp \text{ some algebra } \end{align*}

Here is another, arguably better way to approach this computation: we could have saved ourselves some differentiation work by doing some algebra first, as is often the case. In particular, we can combine the exponent of the exponential function with the square root:
\begin{align*} \frac{d}{dx}(3\sqrt{e^x}-x^2) \amp = \frac{d}{dx}(3(e^x)^{1/2}-x^2) \amp \leftarrow \amp \text{ rewrite so differentiation will be easier }\\ \amp =\frac{d}{dx}(3e^{x/2}-x^2) \amp \leftarrow\amp \text{ combine exponents of \(e\) }\\ \amp = 3\left(\frac{1}{2}\right)e^{x/2}-2x\amp \leftarrow\amp \text{ differentiation is easier now } \end{align*}

The answer is equivalent to the answer we got in the first calculation. (Do some algebra to see why.) Notice that the differentiation steps used in the second calculation were easier than the differentiation steps used in the first calculation.
