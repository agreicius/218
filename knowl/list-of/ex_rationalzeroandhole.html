Example 1.4.38.

Find the domain of the function \(g(x) = \dfrac{x^2-1}{x+1}\text{.}\) What are the zeros, if any, of \(f(x)\text{?}\)

Solution.
The domain of \(g(x) = \dfrac{x^2-1}{x+1}\) is where the denominator \(x+1\neq 0\text{,}\) so \(x\neq -1\text{.}\)
ðŸ”—
The zeros of \(g(x) = \dfrac{x^2-1}{x+1}\) are where the numerator \(x^2-1=0\) and the denominator \(x+1\neq 0\text{.}\) Using the difference of squares,
\begin{equation*} x^2-1 = (x-1)(x+1) = 0\text{ when } x=\pm1 \end{equation*}
ðŸ”—
But we already know that the denominator is 0 when \(x=-1\text{,}\) so the only zero of the function \(g(x)\) is
\begin{equation*} x=1 \end{equation*}
