Example 1.7.1.

For example, the solution to \(x-1=0\) is the single number \(x=1\text{.}\) The solution to \(x-1>0\) is the interval of numbers \(x>1\text{,}\) or, written using interval notation, \((1,\infty)\text{.}\) This means that if we take any number greater than one and subtract one, the result will be positive (meaning greater than 0).
