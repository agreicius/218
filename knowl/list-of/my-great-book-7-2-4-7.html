Example 5.1.10. The Circle.

Letâ€™s consider again the example of the circle given by \(x^2+y^2 = 16\text{.}\) How can we find a formula for the derivative \(\frac{dy}{dx}\text{?}\)
ðŸ”—
By viewing \(y\) as an implicit function of \(x\text{,}\) we think of \(y\) as some function whose formula \(f(x)\) is unknown, but which we can differentiate. Just as \(y\) represents an unknown formula, we can treat its derivative \(\frac{dy}{dx}\) as an unknown formula; we use it purely as a symbol.
ðŸ”—
Viewing \(y\) as an unknown differentiable function of \(x\text{,}\) we take the derivative of both sides of the equation with respect to \(x\text{:}\)
\begin{equation*} \frac{d}{dx}[x^2+y^2] = \frac{d}{dx}[16] \end{equation*}
ðŸ”—
On the right, the derivative of the constant \(16\) is \(0\text{,}\) and on the left we can apply the sum rule, so it follows that
\begin{equation*} \frac{d}{dx}[x^2]+\frac{d}{dx}[y^2] =0\text{.} \end{equation*}
ðŸ”—
Pay attention to the different roles being played by \(x\) and \(y\text{.}\) Because \(x\) is the independent variable, \(\frac{d}{dx}[x^2]=2x\) as usual. But \(y\) is the dependent variable, meaning \(y\) is an implicit function of \(x\text{.}\) Recall ExampleÂ 5.1.9, where we computed \(\frac{d}{dx}[(f(x))^2] = 2f(x)\, f'(x)\text{.}\) Computing \(\frac{d}{dx}[y^2]\) is exactly the same, and requires the chain rule, by which we find that \(\frac{d}{dx}[y^2] = 2y\,\frac{dy}{dx}\text{.}\)
ðŸ”—
The derivative of the equation \(x^2+y^2=16\) is then
\begin{equation*} 2x+2y\,\frac{dy}{dx} = 0 \end{equation*}
once we put together the left and right sides computed above.
ðŸ”—
Now, we can solve this equation for \(\frac{dy}{dx}\) by subtracting \(2x\) from both sides and dividing by \(2y\text{,}\) then canceling the 2. Some people refer to this as "isolating \(\frac{dy}{dx}\text{.}\)"
\begin{equation*} \begin{aligned}2x+2y\,\frac{dy}{dx} \amp = 0\\ 2y\frac{dy}{dx} \amp = -2x\\ \frac{dy}{dx} \amp =\frac{-2x}{2y}\\ \frac{dy}{dx} \amp = -\frac{x}{y} \end{aligned} \end{equation*}
ðŸ”—
And voila! We have managed to compute the derivative \(\frac{dy}{dx}\) without ever knowing an explicit formula for \(y\) as a function of \(x\text{.}\)
