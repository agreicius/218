Example 1.1.2. Set membership.

Let \(A=\{0,1,4\}\text{.}\) Since \(0\text{,}\) \(1\text{,}\) and \(4\) are elements of \(A\text{,}\) we have
\begin{align*} 0\amp \in A \amp 1\amp\in A \amp 4\amp\in A \text{.} \end{align*}
Since \(-1\) is not an element of \(A\text{,}\) we have
\begin{equation*} -1\notin A\text{.} \end{equation*}
