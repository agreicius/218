Example 3.5.13.

Now consider
\begin{equation*} g(x) = \frac{x-1}{(x-1)^2} \end{equation*}
ðŸ”—
If we try to evaluate \(\ds\lim_{x\to 1}g(x)\) without canceling, we get
\begin{equation*} \lim_{x\to 1}\frac{x-1}{(x-1)^2} = \frac{0}{0} \end{equation*}
ðŸ”—
Cancelling one of the factors of \(x-1\text{,}\) we can see that \(g(x) = \frac{x-1}{(x-1)^2} = \frac{1}{x-1}\) everywhere they are defined, meaning for \(x\neq 1\text{.}\) Now when we compute the limit as \(x\to 1\text{,}\) we see that we have the form \(\frac{1}{0}\text{,}\) so this limit is infinite:
\begin{equation*} \lim_{x\to 1}g(x) = \lim_{x\to 1}\frac{1}{x-1} = \frac{1}{0} \end{equation*}
ðŸ”—
Since \(x-1\) changes sign at \(x=1\text{,}\) we compute the left and right limits separately and find that
\begin{equation*} \lim_{x\to 1^-}g(x) = \lim_{x\to 1^-}\frac{1}{x-1} = \frac{1}{0^-} = -\infty \end{equation*}
and
\begin{equation*} \lim_{x\to 1^+}g(x) = \lim_{x\to 1^-}\frac{1}{x-1} =\frac{1}{0^+} = -\infty \end{equation*}
ðŸ”—
Thus \(g(x)\) has a vertical asymptote at \(x=1\text{,}\) despite originally having the form \(\frac{0}{0}\text{,}\) since we could cancel and arrive at the form \(\frac{1}{0}\) instead.
