Example 4.6.8.

Take the derivative of \(y=\dfrac{1}{(4x^2+3x)^9}\text{.}\)
ðŸ”—
Solution.
First we notice that we can rewrite
\begin{equation*} y=\dfrac{1}{(4x^2+3x)^9} = (4x^2+3x)^{-9} \end{equation*}
so that we DO NOT have to use the quotient rule. Yay!
ðŸ”—
The inside function is \(u(x) = 4x^2+3x\) while the outside function is \(f(u) = u^{-9}\text{.}\) \(u'(x) = 8x+3\) and \(f'(u) = -9u^{-10}\text{,}\) so that \(f'(u(x)) = f'(4x^2+3x) = -9(4x^2+3x)^{-10}\text{.}\) Then
\begin{equation*} \begin{aligned}y' \amp = f'(u(x)) u'(x)\\ \amp =-9(u(x))^{-10} (8x+3)\\ \amp = -9(4x^2+3x)^{-10}(8x+3)\\ \amp = \frac{-9(8x+3)}{(4x^2+3x)^{10}} \end{aligned} \end{equation*}
