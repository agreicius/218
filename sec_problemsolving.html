      Notice this is an exponential decay situation. We are keeping \(\frac{1}{3}\)(\(\approx\) 33%) of the remaining susceptible trees each year, meaning \(\frac{1}{3}^t (0.4)(5)\) trees. We can add this to the other non-suspectible trees which remain constant, \(0.6(5)\) to get the formula for total trees:
      \begin{equation*} T(t) = (0.6)(5) + (\frac{1}{3})^t (0.4)(5)\text{.} \end{equation*}
      Now \(T(10) = (0.6)(5) + (\frac{1}{3})^{10} (0.4)(5) \approx 3.00003\) million trees. We can also calculate \(T(100) = (0.6)(5) + (\frac{1}{3})^{100} (0.4)(5)=3\) million trees and that \(T(1000) = (0.6)(5) + (\frac{1}{3})^{1000} (0.4)(5) = 3\) million trees as well (according to a calculator), so there appear to be 3 million trees remaining after a long time.
