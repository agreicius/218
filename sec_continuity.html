    Subsection 3.2.2 Definition of Continuity

    What do you think of when you hear the word "continuous?" A quick Google search will tell you that continuous is an adjective that means
    ðŸ”—
    forming an unbroken whole; without interruption.
    In this section, we define what it means for a function to be continuous, or to "have continuity."
    ðŸ”—

    Example 3.2.1.

    Consider the graph of the function \(f(x) = \frac{1}{2}x+3\) below and the limit \(\ds\lim_{x\to 2} f(x)\text{.}\)
    ðŸ”—
    The limit asks us to consider what value the function \(f(x)= \frac{1}{2}x+3\) approaches as \(x\) approaches 2. We can see in the picture that the \(y\)-values of the function approach 4, and that this is also what we get when we evaluate the function at 2 since \(f(2) = \frac{1}{2}(2)+3 = 1+3 = 4\text{.}\) For this limit and this function, it is the case that
    \begin{equation*} \lim_{x\to 2}f(x) = f(2)=4 \end{equation*}
    so that taking the limit is the same as evaluating the function. There is a particular name for this situation where the value of the limit is the same as the functionâ€™s value: such a function is called continuous.
    ðŸ”—
    ðŸ”—
    Visually, a function is continuous if its graph has no breaks, jumps, or holes. Loosely speaking, you can sketch the graph of a continuous function "without picking up your pencil."
    ðŸ”—

    Example 3.2.3. Non-examples.

    The three graphs in [cross-reference to target(s) "fig_TaalmanPage125b-d" missing or not unique] are not continuous at \(x=1\text{.}\)
    ðŸ”—
    Visually, we notice the "gaps" in all three graphs at \(x=1\text{.}\) The left-most function \(g(x)\) in FigureÂ 3.2.4 has a "jump" at \(x=1\text{.}\) The left limit approaches 1 while the right limit approaches 2. The middle function \(h(x)\) in [cross-reference to target(s) "fig_TaalmanPage125c" missing or not unique] and the right function \(k(x)\) in [cross-reference to target(s) "fig_TaalmanPage125d" missing or not unique] have "holes" at \(x=1\text{.}\) All three graphs have "interruptions" at \(x=1\text{,}\) and so all three are not continuous at \(x=1\text{.}\)
    ðŸ”—
    ðŸ”—
    The algebraic notation of limits makes the idea of continuity more precise. A function is continuous at a point \(x=c\) if its limit as \(x\to c\) is equal to a real number that is the same as the value of the function at \(x=c\text{.}\)
    ðŸ”—

    Definition 3.2.5. Continuity of a Function.

    A function \(f\) is continuous at \(x=c\) if
    \begin{equation*} \ds\lim_{x\to c}f(x) = f(c) \end{equation*}
    ðŸ”—
    ðŸ”—
    \fbox{The following three things need to be true about \(f(x)\) in order for it to be continuous at \(x=c\)
    • The function must be defined at \(x=c\text{.}\) In other words, \(x=c\) must be in the domain of \(f\text{.}\)
      ðŸ”—
      ðŸ”—
    • The limit of \(f(x)\) must exist at \(x=c\text{.}\) In other words, \(\ds\lim_{x\to c}f(x)\) must be equal to a number.
      ðŸ”—
      ðŸ”—
    • The limit of \(f(x)\) at \(x=c\) must be equal to the value of the function at \(x=c\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—

    Example 3.2.6.

    In ExampleÂ 3.2.1, \(f(x) = \frac{1}{2}x+3\) is continuous at \(x=2\) because \(\ds\lim_{x\to 2} f(x) = 4\) and \(f(2)=4\text{,}\) and these two things are equal. Note that \(f(x)\) is continuous at every point pictured since at every point, the limit will equal the value of the function.
    ðŸ”—
    ðŸ”—

    Example 3.2.8.

    By contrast, \(g(x)\text{,}\) \(h(x)\text{,}\) and \(k(x)\) in Example ExampleÂ 3.2.3 are not continuous at \(x=1\) because the limits at \(x=1\) are not equal to the values of the functions at \(x=1\text{.}\)
    ðŸ”—
    For \(g(x)\text{,}\) the limit does not exist and so cannot equal \(g(1)=2\text{.}\) For \(h(x)\text{,}\) the limit at \(x=1\) is \(1\neq h(1)=2\text{.}\) For \(k(x)\text{,}\) the limit exists and is \(\ds\lim_{x\to 1}k(x)=1\) while \(k(1)\) is undefined, so cannot equal the limit.
    ðŸ”—
    However, \(g(x)\text{,}\) \(h(x)\text{,}\) and \(k(x)\) are continuous at every other \(x\)-value pictured. It is only at \(x=1\) that these functions are not continuous.
    ðŸ”—
    ðŸ”—
    The functions that we will work with are almost always continuous, meaning that they are continuous at almost all points except for a few isolated points where they are discontinuous. Thus it makes sense for us to define what it means for a function to be continuous on an interval of points, rather than just point-by-point.
    ðŸ”—

    Definition 3.2.10. Continuity of a Function on an Interval.

    A function \(f\) is continuous on an interval \((a,b)\) if it is continuous for every point \(x=c\) with \(a\lt c\lt b\text{.}\)
    ðŸ”—
    We say that a function is continuous on its domain if it is continuous on every interval on which it is defined.
    ðŸ”—
    ðŸ”—

    Example 3.2.11.

    For instance, the function \(h(x)\) from [cross-reference to target(s) "fig_TaalmanPage125cagain" missing or not unique] is continuous on the interval \((1,4)\) because it is continuous at all points \(x=c\) with \(1\lt c\lt 4\text{.}\) (Note the strict inequalities.) However, it is not continuous on \((0,4)\) because it is not continuous at the point \(x=1\) contained within the interval \((0,4)\text{.}\)

    Subsubsection 3.2.2.1 Left and Right Continuity

    Example 3.2.13.
    Note that in ExampleÂ 3.2.3, the right limit of \(g(x)\) at \(x=1\) is equal to the value of the function:
    ðŸ”—
    The left limit of \(g(x)\text{,}\) on the other hand, goes to 1, which is not the same as the value of the function at \(x=1\text{:}\) \(\ds\lim_{x\to 1^-} g(x) = 1\neq 2 = g(1)\text{.}\)

    ðŸ”—

    ðŸ”—
    Definition 3.2.15. Left and Right Continuity at a Point.
    A function \(f\) is continuous from the left at \(x=c\), a.k.a. left-continuous, if
    \begin{equation*} \lim_{x\to c^-}f(x) = f(c) \end{equation*}
    ðŸ”—
    A function \(f\) is continuous from the right at \(x=c\), a.k.a right-continuous, if
    \begin{equation*} \lim_{x\to c^+}f(x) = f(c) \end{equation*}
    ðŸ”—
    ðŸ”—
    So the function \(g(x)\) in Example ExampleÂ 3.2.13 above is continuous from the right at \(x=1\) but is not continuous from the left at \(x=1\text{.}\)
    ðŸ”—
    We can use interval notation to express whether a function is right or left continuous at the edges of an interval.

    Subsection 3.2.3 Types of Discontinuities

    When a function is not continuous at a point \(x=c\text{,}\) we say that it is discontinuous at \(x=c\text{.}\) In terms of limits this means that
    ðŸ”—
    \(\ds\lim_{x\to c}f(x)\)\(f(c)\text{.}\)
    The three types of discontinuities that a function can have are illustrated as follows:
    ðŸ”—
    In the left-most picture, we see that the function is discontinuous at \(x=2\) because \(\ds\lim_{x\to 2}f(x) = 1\) while \(f(2)=2\text{.}\) This is called a removable discontinuity. Intuitively, we say that a discontinuity is removable if we could get rid of the discontinuity just by changing one value of the function. This type of discontinuity is also called a hole.
    ðŸ”—
    In the middle picture, we can see that the function is discontinuous at \(x=2\) because the left and right limits are different, so \(\ds\lim_{x\to 2}f(x)\) does not exist. When the left and right limits both exist but are not equal, this is called a jump discontinuity. Here, \(\ds\lim_{x\to 2^-} f(x) = 1\) and \(\ds\lim_{x\to 2^+}f(x) = 2\) both exist, but are not equal. At a jump discontinuity, the function jumps from one value to another.
    ðŸ”—
    In the right-most picture, we see that the function is discontinuous at \(x=2\) because \(f(2)\) is undefined and the limit is not finite, and so does not exist. Anytime a left or right limit is infinite, we call this an infinite discontinuity. Infinite discontinuities are also called vertical asymptotes.
    ðŸ”—
    To summarize, these types of discontinuities can be described precisely in terms of limits as follows:
    ðŸ”—

    Definition 3.2.23. Removable, Jump, and Infinite Discontinuities.

    Suppose \(f\) is discontinuous at \(x=c\text{.}\) We say that \(x=c\) is a
    1. removable discontinuity or hole if \(\ds\lim_{x\to c}f(x)\) exists but is not equal to \(f(c)\text{,}\) including the case where \(f(c)\) is undefined.
      ðŸ”—
      ðŸ”—
    2. jump discontinuity if \(\ds\lim_{x\to c^-}f(x)\) and \(\ds\lim_{x\to c^+} f(x)\) both exist and are finite but are not equal. It does not matter whether \(f(c)\) is defined.
      ðŸ”—
      ðŸ”—
    3. infinite discontinuity or vertical asymptote if one or both of \(\ds\lim_{x\to c^-} f(x)\) and \(\ds\lim_{x\to c^+} f(x)\) is infinite.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—

    Subsection 3.2.4 Rules for Calculating Limits

    Now we know that limits of continuous function are easy to evaluate: we simply plug in the \(x\)-value of the limit to the formula, the same way we would if we just wanted to know the value of the function. In this section, we build up a list of functions that we know to be continuous. We begin with the simplest functions: constant and linear functions:
    ðŸ”—

    Theorem 3.2.27. Continuity of Linear Functions.

    Constant and linear functions are continuous for all \(x\)-values:
    • For any real numbers \(k\) and \(c\text{,}\)
      \begin{equation*} \lim_{x\to c}k = k \end{equation*}
      ðŸ”—
      ðŸ”—
    • For any real numbers \(c, m\text{,}\) and \(b\text{,}\)
      \begin{equation*} \lim_{x\to c}(mx+b) = mc+b \end{equation*}
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    Now we know that for any constant or any linear function, evaluating a limit is no different than evaluating the function.
    ðŸ”—

    Theorem 3.2.28. Continuity of Power Functions.

    Power functions are continuous on their domains:
    • For any real number \(A\) and rational number \(k\text{,}\) then for any values \(x=c\) for which \(x^k\) is defined,
      \begin{equation*} \lim_{x\to c} Ax^k = Ac^k\text{.} \end{equation*}
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    They key difference in Theorem TheoremÂ 3.2.28 is that power functions are sometimes undefined, and so cannot be continuous where they are not defined, meaning outside of their domain.
    ðŸ”—
    These are powerful theorems, because it tells us that we can calculate limits of certain simple functions at domain points just by evaluating the functions at those points.
    ðŸ”—

    Example 3.2.29.

    For example, \(f(x) = \frac{1}{x} = x^{-1}\) is a power function, so by Theorem TheoremÂ 3.2.28, it is continuous on its domain, which is \((-\infty,0)\cup(0\infty)\text{.}\) This means that at any point \(c\neq 0\) we can calculate \(\ds\lim_{x\to c}\frac{1}{x}\) by simply calculating the value \(f(c) = \frac{1}{c}\text{.}\)
    ðŸ”—
    For instance,
    \begin{equation*} \lim_{x\to 5}\frac{1}{x} = \frac{1}{5} \end{equation*}
    and
    \begin{equation*} \lim_{x\to -17}\frac{1}{x} = \frac{1}{-17} \end{equation*}
    ðŸ”—
    However, we cannot yet compute \(\ds\lim_{x\to 0}\frac{1}{x}\text{.}\) We will examine this limit in depth in a later section.
    ðŸ”—
    ðŸ”—

    Example 3.2.30.

    The function \(f(x) = 2\) is constant. Hence using Theorem TheoremÂ 3.2.27, we know that
    \begin{equation*} \lim_{x\to 3} 2 = 2\text{.} \end{equation*}
    ðŸ”—
    The function \(g(x) = x\) is linear, so again using Theorem TheoremÂ 3.2.27, we know that
    \begin{equation*} \lim_{x\to 5} x = 5 \end{equation*}
    ðŸ”—
    The function \(h(x) = \sqrt{x} = x^{1/2}\) is a power function, so using Theorem TheoremÂ 3.2.28, we know that
    \begin{equation*} \lim_{x\to 16} \sqrt{x} = \lim_{x\to 16} x^{1/2} = (16)^{1/2} = \sqrt{16} = 4\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—
    In Theorem TheoremÂ 3.2.28, the limit may sometimes be only one-sided.
    ðŸ”—

    Example 3.2.31.

    For example, \(f(x) = \sqrt{x} = x^{1/2}\) is defined at \(x=0\) and for \(x>0\text{,}\) meaning to the right of \(x=0\text{.}\) However it is not defined for \(x\lt 0\text{,}\) meaning to the left of \(x=0\text{.}\) Therefore the corresponding limit statement is one-sided:
    \begin{equation*} \ds\lim_{x\to 0^+} \sqrt{x} = 0 \text{ but } \ds\lim_{x\to 0^-}\sqrt{x} \text{ DNE, } \text{ hence } \lim_{x\to 0}\sqrt{x}\text{ DNE. } \end{equation*}

    Subsubsection 3.2.4.1 Limits of Combinations of Functions

    What about more complicated functions? Can we put together two continuous functions and still have a continuous function?
    ðŸ”—
    Example 3.2.32.
    For example, we already know from the continuity of power functions that
    \begin{equation*} \lim_{x\to 2}x^2 = 2^2 = 4\text{ and } \lim_{x\to 2}x^3 = 2^3 = 8\text{.} \end{equation*}
    ðŸ”—
    Can we use these results to say something about
    \begin{equation*} \lim_{x\to 2} (x^2+x^3)? \end{equation*}
    ðŸ”—
    ðŸ”—
    The key theorem that follows in Theorem TheoremÂ 3.2.33 will help us answer this question; it says that limits can be evaluated within all of the arithmetic operations between function and even sometimes within a composition of functions.
    ðŸ”—
    Theorem 3.2.33. Rules for Calculating Limits of Combinations.
    If \(\ds\lim_{x\to c} f(x)\) and \(\ds\lim_{x\to c} g(x)\) exist and are finite, then the following rules hold for their combinations:
    ðŸ”—
    Constant Multiple Rule: \(\ds\lim_{x\to c}k f(x) = k\lim_{x\to c} f(x)\) for any real number \(k\)
    Sum Rule: \(\ds\lim_{x\to c} (f(x)+g(x)) = \ds\lim_{x\to c}f(x) +\ds\lim_{x\to c}g(x)\)
    Difference Rule: \(\ds\lim_{x\to c}(f(x)-g(x)) = \ds\lim_{x\to c}f(x)-\ds\lim_{x\to c}g(x)\)
    Product Rule: \(\ds\lim_{x\to c}(f(x)\cdot g(x)) = \left(\ds\lim_{x\to c}f(x)\right)\cdot\left(\ds\lim_{x\to c}g(x)\right)\)
    Quotient Rule: \(\ds\lim_{x\to c}\dfrac{f(x)}{g(x)} = \dfrac{\ds\lim_{x\to c}f(x)}{\ds\lim_{x\to c}g(x)}\text{,}\) if \(\ds\lim_{x\to c}g(x)\neq 0\)
    Composition Rule: \(\ds\lim_{x\to c}f(g(x)) = f\left(\ds\lim_{x\to c}g(x)\right)\text{,}\) if \(f\) is continuous at \(\ds\lim_{x\to c}g(x)\text{.}\)
    ðŸ”—
    This theorem is a powerful tool for calculating limits, since it tells us how to find limits of compound functions in terms of the limits of their components.
    ðŸ”—
    Example 3.2.34.
    For example, we can now calculate the limit of the sum \(x^2+x^3\) as \(x\to 2\) by taking the sum of the limits of \(x^2\) and \(x^3\) as \(x\to 2\text{:}\)
    \begin{equation*} \lim_{x\to 2}(x^2+x^3) = \lim_{x\to 2}x^2+\lim_{x\to 2}x^3 = 2^2+2^3 = 4+8=12 \end{equation*}
    ðŸ”—
    ðŸ”—
    Example 3.2.35.
    We can calculate \(\ds\lim_{x\to 4}\sqrt{x-1}\) using the Composition Rule. For \(f(x) = \sqrt{x}\) and \(g(x) = x-1\text{,}\) we have
    \begin{equation*} f(g(x)) = \sqrt{x-1}\text{.} \end{equation*}
    Since \(\ds\lim_{x\to 4}x-1 = 3\) is in the domain of \(f(x) = \sqrt{x}\text{,}\) we know that \(f(x)\) is continuous at \(x=3\) and so
    \begin{equation*} \lim_{x\to 4}\sqrt{x-1} = \sqrt{\lim_{x\to 4}(x-1)} = \sqrt{4-1} = \sqrt{3} \end{equation*}
    ðŸ”—
    ðŸ”—
