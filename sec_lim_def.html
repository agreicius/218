    Subsection The Idea of a Limit

    There are many instances when studying natural phenomena where we wish to know what happens near a point, but not specifically at a point. Alternately, we may wish to know about the long-term behavior of a quantity, but not specifically at some time in the future, no matter how distant. The mathematical idea of a "limit" is precisely designed to describe such things using the language of functions. Instead of looking at the output value of a function when plugging in a specific input, we look at the pattern of the output values over a sequence of input values.
    ðŸ”—

    Example 1.11.1. Using Limits to Describe Zenoâ€™s Paradox.

    Zenoâ€™s paradox
    â€‰8â€‰
    See a delightful Numberphile video at https://youtu.be/u7Z9UnWOJNY for more on Zenoâ€™s Paradox and infinite processes.
    is a wonderful example of a limit. Consider that in moving from one place to another, you must first travel half the distance, then half of the remaining distance, then half of that remaining distance, etc. Since there will never be zero distance left to travel, you will never reach your destination. We can express these distances as a sequence of fractions of the total distance:
    \begin{equation*} \left\{\frac{1}{2}, \frac{1}{4}, \frac{1}{8}, \frac{1}{16}, \ldots\right\} \end{equation*}
    ðŸ”—
    Note that each term in the sequence has the form
    \begin{equation*} \frac{1}{2^n} \end{equation*}
    for a positive integer \(n\text{.}\) Note as well that as \(n\) increases, the denominator \(2^n\) increases, with the result that the fractions \(\frac{1}{2^n}\) become smaller and smaller as \(n\) gets bigger and bigger. Eventually, the fractions \(\frac{1}{2^n}\) are so small as to be indistinguishable from zero; in a theoretical way we have not yet reached our destination, but in a practical sense, it makes no difference.
    ðŸ”—
    With the "limit" notation that you will be learning in this section, we would express this idea as
    \begin{equation*} \lim_{n\to\infty}\frac{1}{2^n} = 0\text{.} \end{equation*}
    ðŸ”—
    What it is saying is that we will eventually have 0 distance left to travel, though it will take us infinitely long to get there. So do we ever actually get where we are going?
    ðŸ”—

    Example 1.11.2. Instantaneous Velocity as a Limit of Average Velocity.

    Recall that, for an object moving in a straight line with position function \(s(t)\text{,}\) the average velocity of the object on the interval from \(t=a\) to \(t=b\text{,}\) denoted \(AV_{[a,b]}\text{,}\) is given by the formula
    \begin{equation*} AV_{[a,b]} = \frac{s(b)-s(a)}{b-a} \end{equation*}
    where \(s(t)\) is the position of the object. Average velocity is the same as average rate of change of position.

    Subsection The Definition of a Limit

    Letâ€™s make these ideas more precise and also explain what this "limit notation" actually means. The basic idea is that we ask what output values the function gets close to near an input value.
    ðŸ”—

    Definition 1.11.5. Definition of a Limit.

    Given a function \(f\text{,}\) a fixed input \(x=c\text{,}\) and a real number \(L\text{,}\) we say that \(f\) has limit \(L\) as \(x\) approaches \(c\) and write
    \begin{equation*} \lim_{x\to c}f(x) = L \end{equation*}
    provided that we can make \(f(x)\) as close to \(L\) as we like by taking \(x\) sufficiently close (but not equal) to \(c\text{.}\)
    ðŸ”—
    If we cannot make \(f(x)\) as close to a single value as we would like as \(x\) approaches \(c\text{,}\) then we say that \(f\) does not have a limit as \(x\) approaches \(c\text{.}\)Another way to say this is that the limit â€œdoes not exist,â€ and we abbreviate with
    \begin{equation*} \ds \lim_{x\to c}f(x) \text{ DNE }\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—
    \fbox{ The symbols \(\ds\lim_{x\to c} f(x) = L\) are read, "the limit as \(x\) approaches \(c\) of \(f(x)\) is equal to \(L\text{.}\)" }
    ðŸ”—
    Here is an example of finding the limit of a function at a point that is not in the domain of a rational function:
    ðŸ”—

    Example 1.11.6.

    For example, the function \(g(x) = \dfrac{x^2-1}{x-1}\) can be simplified by factoring the numerator as \(\dfrac{x^2-1}{x-1} = \dfrac{(x-1)(x+1)}{x-1}\) so that
    \begin{equation*} g(x) = \begin{cases}x+1, \amp \text{ for } x\neq 1\\ \text{ undefined } , \amp \text{ for } x=1 \end{cases} \end{equation*}
    ðŸ”—

    \begin{equation*} \lim_{x\to 1} g(x) = 2 \end{equation*}
    because the limit does not care what happens at the point where \(x=1\text{,}\) only close to it.
    ðŸ”—

    Subsection One-Sided Limits

    Weâ€™ve seen that limits can be quite useful for describing what happens when a function is undefined. They are also useful for describing what happens when two parts of a piecewise-defined function do not line up smoothly, as occurred at the point \(x=2\) in ExampleÂ 1.11.8.
    ðŸ”—

    Definition 1.11.14. Definition of One-Sided Limits.

    Given a function \(f\text{,}\) a fixed input \(x=c\) and real numbers \(L\) and \(R\text{,}\) we say that \(f\) has limit \(L\) as \(x\) approaches \(c\) from the left and write
    \begin{equation*} \lim_{x\to c^-}f(x) = L \end{equation*}
    provided that we can make \(f(x)\) as close to \(L\) as we like by taking \(x\) sufficiently close (but not equal) to \(c\text{,}\) while keeping \(x\) less than \(c\text{.}\)
    ðŸ”—
    Similarly, we say that \(f\) has limit \(R\) as \(x\) approaches \(c\) from the right and write
    \begin{equation*} \lim_{x\to c^+}f(x) = R \end{equation*}
    provided that we can make \(f(x)\) as close to \(R\) as we like by taking \(x\) sufficiently close (but not equal) to \(c\text{,}\) while keeping \(x\) greater than \(c\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.11.15.

    Recall the graph of the function \(g\) from Example ExampleÂ 1.11.8:
    ðŸ”—
    ðŸ”—
    On the left of the jump at \(x=1\text{,}\) the \(y\)-values approached 3. Since \(x\lt 1\) on that side, we write
    \begin{equation*} \lim_{x\to 1^-} g(x) = 3 \end{equation*}
    and we say ``the limit from the left at 1 of \(g(x)\) is equal to 3". On the right of the jump, the \(y\)-values approached 2. Since \(x>1\) on that side, we write
    \begin{equation*} \lim_{x\to 1^+} g(x) = 2 \end{equation*}
    and we say ``the limit from the right at 1 of \(g(x)\) is equal to 2."
    ðŸ”—
    ðŸ”—
    The terminology of â€œleftâ€ and â€œrightâ€ limit refers to the orientation around \(x=c\) on the \(x\)-axis and can be visualized:
    ðŸ”—
    \fbox{The regular limit \(\ds\lim_{x\to c}f(x)\) is sometimes called the â€œtwo-sidedâ€ limit of \(c\text{,}\) and requires that \(f(x)\) approaches one number for values of \(x\) on both sides of \(c\text{.}\) This means that in order for \(\ds\lim_{x\to c}f(x)\) to exist, the left and right limits at \(c\) must exist and be equal:
    ðŸ”—
    \(\ds\lim_{x\to c}f(x)=L\)\(\ds\lim_{x\to c^-}f(x) =L\)\(\ds\lim_{x\to c^+}f(x) =L\)

    Remark 1.11.17.

    Remember that the phrase "if and only if" is shorthand for saying that one thing can only be true if the other is also true. In particular, if either right or left limit does not exist, or the left and right limits are equal to different numbers, then the two-sided limit does not exist.
    ðŸ”—
    ðŸ”—

    Subsection Summary

    The following terms were introduced in this section:
    ðŸ”—
