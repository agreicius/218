    Subsection Definition of Piecewise-Defined Functions

    So far, you have learned about the different types of algebraic functions. In this section we discuss functions that are NOT algebraic functions. Instead, they are pieced together by parts of algebraic functions.
    ðŸ”—
    A piecewise-defined function is a function that is defined in multiple pieces, with different formulas on different parts of its domain.
    ðŸ”—

    Example 1.7.1.

    Let \(f\) be the function defined piecewise by
    \begin{equation*} f(x) = \begin{cases}x^2,\amp \text{ if } x\leq -1\\ 2x, \amp \text{ if } x>-1 \end{cases} \end{equation*}
    ðŸ”—
    This means that if we want to plug in an \(x\)-value that is \(-1\) or less, we use the formula \(x^2\) in the first row. If we want to plug in an \(x\)-value that is greater than \(-1\text{,}\) we use the formula \(2x\text{.}\) So the domain for the function is still \((-\infty,\infty)\text{,}\) but we use \(x^2\) on \((-\infty,-1]\) and use \(2x\) on \((-1,\infty)\text{.}\)
    ðŸ”—
    If we want to find the value \(f(-5)\text{,}\) we observe that \(x=-5\leq -1\text{,}\) and so we use the formula \(x^2\) for this value of \(x\text{:}\)
    \begin{equation*} f(-5) = (-5)^2 = 25\text{.} \end{equation*}
    ðŸ”—
    If we want to find the value \(f(3)\text{,}\) we observe that \(x=3>-1\text{,}\) and so we use the formula \(2x\) for this value of \(x\text{:}\)
    \begin{equation*} f(3) = 2(3) = 6\text{.} \end{equation*}
    ðŸ”—
    If we want to find the value \(f(-1)\text{,}\) we observe that \(x=-1\leq -1\text{,}\) because of the equal sign, and so we once again use the formula \(x^2\) for this value of \(x\text{:}\)
    \begin{equation*} f(-1) = (-1)^2 = 1 \end{equation*}
    ðŸ”—
    To graph \(f\text{,}\) we begin by graphing the functions \(y=x^2\) and \(y=2x\) that are used in the definition of \(f\text{,}\) as shown in [cross-reference to target(s) "fig_TaalmanPage46b" missing or not unique]:
    ðŸ”—
    To graph \(f\text{,}\) we use the graph of \(y=x^2\) on the interval \((-\infty,-1]\) and use \(y=2x\) on \((-1,\infty)\text{.}\) Whether these intervals are open or closed is important. To find \(f(-1)\) we used the first equation \(y=x^2\text{:}\) \(f(-1)=(-1)^2 = 1\text{.}\) Note that when we sketch the graph of \(f\) below, we use open and closed dots to represent the function values corresponding to the ends of open and closed intervals in the domain, respectively.

    Subsection An Application of Piecewise-defined Functions

    Piecewise-defined functions are useful for many real-word problems where the relationship between the quantities involved are different on different time-frames. For example, a personâ€™s salary changes when they get a raise, which changes the amount that they earn each year.
    ðŸ”—

    Example 1.7.5.

    Suppose that in your first job after graduating from college you make $36,000 a year before taxes. After four years you get a raise of $2,500. Two years after that you change jobs and go to work for a company that pays you $49,000 a year.
    ðŸ”—
    We can construct a piecewise-defined function that describes your pretax income in the year that is \(t\) years after you graduate from college as follows. Note that this is just the function that describes your income rate per year at a time \(t\) (the units are dollars per year).
    \begin{equation*} \text{ income } (t) = \begin{cases}36000,\amp \text{ if } 0\leq t\lt 4\\ 38500,\amp \text{ if } 4\leq t\lt 6\\ 49000,\amp \text{ if } t\geq6 \end{cases} \end{equation*}
    ðŸ”—
    What may be more interesting is to describe that amount that you will have earned \(t\) years after graduating from college. During the first four years, you make $36,000 per year, and so the amount that you earn after \(t\) years is \(36000t\text{.}\) After four years, you have already earned $\(36,000\cdot 4 = 144,000\text{.}\) Then for the next two years, you earn $38,500 per year, so your additional income for \(4\leq t\lt 6\) is given by \(38500(t-4)\) (because we have to measure in years since \(t=4\)). Thus overall for \(4\leq t\lt 6\text{,}\) you have earned \(144000+38500(t-4)\) by time \(t\text{.}\) Lastly, after 6 years you have earned $\(36,000\cdot 4+38,500\cdot2 = 221,000\text{,}\) and for each additional year you earn $49,000, so by time \(t\) you have earned \(21000+4900(t-6)\) when \(t\geq 6\text{.}\)
    ðŸ”—
    Summarizing this, the function for total earnings at time \(t\) years since graduation is
    \begin{equation*} \text{ earnings } (t) = \begin{cases}36000t,\amp \text{ if } 0\leq t\lt 4\\ 144000+38500(t-4),\amp \text{ if } 4\leq t\lt 6\\ 221000+49000(t-6),\amp \text{ if } t\geq 6 \end{cases} \end{equation*}
    ðŸ”—
    If you continue at that pay rate, how long will it have taken you to earn a million dollars? Well, we know we did not make that much by \(t=6\text{,}\) so we use the equation for \(t\geq 6\) and solve for \(t\text{:}\)
    \begin{equation*} \begin{aligned}221000+49000(t-6) \amp = 1000000\\ 49000(t-6) \amp = 779000\\ t-6 \amp = \frac{779000}{49000}\approx 15.8980\\ t \amp \approx 21.8980 \end{aligned} \end{equation*}
    ðŸ”—
    So in your 22nd year after graduating you will have earned one million pretax dollars.
    ðŸ”—
    ðŸ”—

    Checkpoint 1.7.6.

    Subsection Absolute Value and Distance

    The absolute value function measures the size of a quantity, disregarding whether it is positive or negative. For a number visualized on the real number line, this also describes the distance of that number from 0. Said in another way, the absolute value of a real number \(a\) is the magnitude, or size, of \(a\) and is denoted \(|a|\text{.}\)
    ðŸ”—

    Example 1.7.8.

    For example, \(|3|=3\) and \(|-3|=3\text{.}\)
    ðŸ”—
    ðŸ”—
    The absolute value of \(a\) "makes the number \(a\) positive." More precisely, if \(a\) is positive or zero, then taking the absolute value should do nothing: in this case \(|a|=a\text{.}\) However, if \(a\) is negative, then taking the absolute value should change the sign: in this case \(|a|=-a\text{.}\)
    ðŸ”—

    Example 1.7.9.

    For example, \(|-3|=-(-3) = 3\text{.}\)
    ðŸ”—
    ðŸ”—
    To define the absolute value function rigorously, we must use a piecewise-defined function. As just discussed, for positive numbers and zero, we must apply one rule (do nothing), and for negative numbers, we must apply another rule (multiply by \(-1\)). Here is the formal definition, which says precisely that.
    ðŸ”—

    Definition 1.7.10. Absolute Value.

    The absolute value of a real number \(x\) is
    \begin{equation*} |x| = \begin{cases}x, \amp \text{ if } x\geq 0\\ -x, \amp \text{ if } x\lt 0. \end{cases} \end{equation*}
    ðŸ”—
    ðŸ”—

    Remark 1.7.11.

    The quantity \(-x\) is not necessarily negative; it depends on whether the number \(x\) is positive or negative. For example, if \(x=-3\text{,}\) then \(-x = -(-3)=3\) is positive. In DefinitionÂ 1.7.10, \(|x|=-x\) only if \(x\) is negative, in which case \(-x\) is a positive number.
    ðŸ”—
    ðŸ”—
    Absolute values distribute over products and quotients; for example, the absolute value of a product is simply the product of the absolute values.
    ðŸ”—

    Theorem 1.7.12. Absolute Values Distribute over Products and Quotients.

    Given any real numbers \(a\) and \(b\text{,}\) \(|ab| = |a||b|\) and \(\left|\dfrac{a}{b}\right| = \dfrac{|a|}{|b|}\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.7.13.

    For example, \(|(-2)(3)| = |-6| = 6\) is equal to \(|-2||3| = (2)(3) = 6\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.7.14. Non-Example.

    Note that absolute values do not distribute over sums and differences. For example, \(|-2+3| = |1| = 1\) is not equal to \(|-2|+|3| = 2+3 = 5\text{.}\)
    ðŸ”—
    ðŸ”—
    The absolute value \(|a|\) of any real number \(a\) is the distance between \(a\) and 0 on the real number line. Geometrically, the distance between any two real numbers \(a\) and \(b\) is the length of the line segment from \(a\) to \(b\) on the number line. Algebraically, this distance is given by \(|b-a|\text{,}\) regardless of the signs of \(a\) and \(b\text{.}\)
    ðŸ”—

    Example 1.7.15.

    For example, the diagram that follows shows that the distance between \(a=2\) and \(b=-1\) is 3 units. Algebraically, \(|b-a| = |-1-2| = |-3| = 3\text{.}\)
    ðŸ”—
    ðŸ”—

    Definition 1.7.16. Distance Between Two Real Numbers.

    The distance between two real numbers \(a\) and \(b\) is \(\text{ dist } (a,b) = |b-a|\text{.}\)
    ðŸ”—
    ðŸ”—

    You may remember that we gave a different definition for the distance between two real numbers in DefinitionÂ 1.6.27. These two definitions are the same, because of the following alternate definition of the absolute value function:
    ðŸ”—

    Definition 1.7.18. Alternate Definition of Absolute Value.

    For any real number \(x\text{,}\)
    \begin{equation*} |x| = \sqrt{x^2} \end{equation*}
    ðŸ”—
    ðŸ”—
    When \(x\) is positive, the square root of \(x^2\) returns us to the original \(x\)-value. When \(x\) is negative, \(x^2\) is positive and the square root denotes only the positive square root, so the square root of \(x^2\) will give us \(|x| = -(x)\text{.}\) (Remember, if \(x\) is negative, \(-x\) is positive!) So this different definition is exactly the same as DefinitionÂ 1.7.10, just stated in a different way.
    ðŸ”—
    This idea can help us in solving the following types of common equations:
    ðŸ”—

    Example 1.7.19.

    Find the solution of \(x^2=25\text{.}\)
    ðŸ”—
    Solution.
    If we take the square root of both sides, we have
    \begin{equation*} \begin{aligned}x^2 \amp = 25\\ \sqrt{x^2} \amp =\sqrt{25}\\ |x| \amp = 5\\ x \amp =\pm 5 \end{aligned} \end{equation*}
    ðŸ”—
    The \(\pm\) comes from the following logic: if the magnitude of \(x\) is 5, \(x\) is 5 units away from 0 on the number line to either the right or the left. So \(x\) could be either positive or negative 5 units from 0, meaning that \(x=5, -5\text{.}\)
    ðŸ”—
    Alternately, we could solve via factoring. Approached in this way, we have
    \begin{equation*} \begin{aligned}x^2 \amp = 25\\ x^2-25 \amp = 0\\ (x-5)(x+5) \amp =0\\ x-5 = 0 \amp \text{ or } \amp x+5 = 0\\ x \amp = 5, -5 \end{aligned} \end{equation*}
    ðŸ”—
    Thus using either method, we can see that both \(x=-5\) and \(x=5\) are solutions to the equation \(x^2=25\text{.}\) Verify this by plugging in both numbers!
    ðŸ”—
    Important! It is very common for students to forget to include the \(\pm\) when taking the square root of a square. The main lesson to be learned from this example is that equations \(x^2 = r^2\) have two solutions: \(x=\pm r\text{.}\)
    ðŸ”—
    ðŸ”—
    ðŸ”—
    Letâ€™s reconcile the two definitions for the distance between real numbers. Definition DefinitionÂ 1.6.27 tell us that
    \begin{equation*} \text{ dist } (a,b) = \sqrt{(b-a)^2} \end{equation*}
    while Definition DefinitionÂ 1.7.16 combined with Definition DefinitionÂ 1.7.18 tells us that
    \begin{equation*} \text{ dist } (a,b) = |b-a| = \sqrt{(b-a)^2}\text{.} \end{equation*}
    Hooray! The two definitions agree.
    ðŸ”—

    Checkpoint 1.7.20.

    Which definition of distance do you prefer? \(\text{ dist } (a,b) = \sqrt{(b-a)^2}\) or \(\text{ dist } (a,b) = |b-a|\text{?}\) Why? You can use either one. As we showed, they agree with each other.

    Subsection Graphs of Absolute Values

    The graph of \(y=|x|\) is a combination of the graph of \(y=x\) on \([0,\infty)\) and the graph of \(y=-x\) on \((-\infty,0)\text{:}\)
    ðŸ”—
    Of course, in general we might wish to take the absolute value of a more complicated expression. In the more general case, we do exactly the same thing: the absolute value will leave positive quantities untouched, but flip the sign of negative quantities.
    ðŸ”—

    Definition 1.7.22. The Absolute Value of a Function.

    The absolute value of a function \(g(x)\) is
    \begin{equation*} |g(x)| = \begin{cases}g(x), \amp \text{ for all } x\text{ with } g(x)\geq 0\\ -g(x),\amp \text{ for all } x\text{ with } g(x)\lt 0 \end{cases} \end{equation*}
    ðŸ”—
    ðŸ”—

    Example 1.7.23.

    For example, consider the function \(g(x) = 2x-1\) and its absolute value \(f(x) = |g(x)| = |2x-1|\text{.}\) When \(2x-1\) is positive or zero (meaning when \(x\geq 1/2\)), the absolute value remains \(2x-1\text{.}\) But when \(2x-1\) is negative (meaning when \(x\lt 1/2\)) the absolute value of \(g\) is \(-(2x-1)\text{.}\) The graph of \(y=|2x-1|\) is a combination of the graphs of \(y=2x-1\) and \(y=-(2x-1)\text{,}\) switching between graphs at \(x=1/2\) as shown in [cross-reference to target(s) "fig_TaalmanPage55b" missing or not unique].

    Subsection Summary

    The following terms were introduced in this section:
    ðŸ”—
    Key ideas: Piecewise-defined functions are not algebraic functions. They use different formulas on different intervals and their graphs are found by piecing together the graphs of the different formulas. Absolute value is an example of a piecewise-defined function and can be used to find distance. The graph of the absolute value of a function reflects the negative parts of the graph of the original function up over the horizontal axis.
    ðŸ”—
    Other ideas introduced: It would be great to have an algebraic way to determine when an expression is positive and negative, so we donâ€™t have to use a graph to find the absolute value of a function.
