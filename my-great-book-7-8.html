    Subsubsection 5.7.2.1 A Brief Review of Symmetry

    It has been several chapters since we thought about even and odd symmetry, so here is a quick reminder. For a more thorough review, you could reread SubsectionÂ 2.10.3.
    ðŸ”—
    Remember that a function \(f(x)\) is called even if \(f(-x)=f(x)\text{.}\) This means that it has reflectional symmetry across the \(y\)-axis. A function \(f(x)\) is called odd if \(f(-x)=-x\text{.}\) An odd function has \(180^\circ\) rotational symmetry about the origin, or equivalently is the same if reflected over both the \(x\)- and \(y\)-axes.

    \begin{equation*} g(-x) = (-x)^3 = -x^3 = -g(x)\text{.} \end{equation*}

    Example 5.7.3.

    Consider the function
    \begin{equation*} g(x) = x^5-15x^3\text{.} \end{equation*}
    1. The domain of any polynomial function is all real numbers.
      ðŸ”—
      ðŸ”—
    2. The \(y\)-intercept of \(g(x)\) is \(y=g(0) = 0^5-15(0)^3 = 0\text{,}\) meaning the point \((0,0)\text{.}\)
      ðŸ”—
      ðŸ”—
    3. \(g(x)\) factors by hand with a little persistence and a difference of "squares" involving the observation that \((\sqrt{15})^2 = 15\text{:}\)
      \begin{equation*} g(x) = x^5-15x^3 = x^3(x^2-15) = x^3(x-\sqrt{15})(x+\sqrt{15}) \end{equation*}
      and we see that there are \(x\)-intercepts at \(x=0\) and \(x=\pm\sqrt{15}\approx \pm 3.87\text{.}\) Notice that \(x=0\) was also the \(y\)-intercept. We can additionally draw a number line for where the function \(g\) is positive and negative, which will be useful when we go to draw the function:
      ðŸ”—
      ðŸ”—
    4. \(g(x)\) is odd since
      \begin{equation*} g(-x) = (-x)^5-15(-x)^3 = -x^5+15x^3 = -(x^5-15x^3) = -g(x)\text{.} \end{equation*}
      This confirms the accuracy of the previous sign analysis for \(g(x)\text{:}\) if \(g(x)\) is positive at a certain \(x\)-value \(x=a\text{,}\) it will be negative at the value \(x=-a\text{,}\) and vice versa. For instance, \(g(-1)\) is positive while \(g(1)\) is negative. Additionally, if we had not found both \(x\)-intercepts \(x=\pm \sqrt{15}\text{,}\) this would have told us that we were missing one. (Finding only \(x=+\sqrt{15}\) is a fairly common error.)
      ðŸ”—
      ðŸ”—
    5. There are no non-domain points of \(g(x)\) since \(g(x) = x^5-15x^3\) is a polynomial. In particular, there are no holes or vertical asymptotes in the graph of \(g(x)\text{.}\)
      ðŸ”—
      ðŸ”—
    6. Recall that for limits of the form \(\infty-\infty\text{,}\) we can determine the precise behavior by force-factoring the highest power of \(x\text{.}\) Thus

      and
      \begin{equation*} \lim_{x\to-\infty} (x^5-15x^3) = \lim_{x\to-\infty} x^5\left(1-\frac{15}{x^2}\right) = (-\infty)^5(1-0) = -\infty \end{equation*}
      ðŸ”—

    Subsection 5.7.4 Some general observations about graphs of polynomials

    We discussed in SectionÂ 2.5 that the degree of a polynomial determines an upper bound for the number of its \(x\)-intercepts and also of its turning points, a.k.a. local maxima and minima; revisit Theorem TheoremÂ 2.5.13 if you do not remember. It turns out that the \(x\)-intercepts of the polynomial can give us a lower bound for the number of turning points.
    ðŸ”—
    By the Mean Value Theorem (TheoremÂ 5.3.2), between every two \(x\)-intercepts of a polynomial \(f(x)\) must lie at least one zero of its derivative (where \(f'(c)=0\)). In fact, we can guarantee even more: Between each adjacent pair of \(x\)-intercepts of \(f(x)\) there must be some zero of \(f'(x)\) at which the graph of \(f(x)\) actually turns around and has a local maximum or minimum value. This observation is summarized in the following useful theorem:
    ðŸ”—

    Theorem 5.7.12. Turning Points Between Zeros of Polynomials.

    If \(f\) is a polynomial function of degree \(n\) with \(k\) real zeros, then \(f\) has at least \(k-1\) turning points, with at least one turning point between each zero.
    ðŸ”—
    ðŸ”—
    As a corollary (in other words, an immediate consequence) to this theorem, notice that if a polynomial of degree \(n\) splits into \(n\) distinct linear factors, then it will have the maximum number \(n\) of \(x\)-intercepts and also the maximum number \(n-1\) of turning points, one between and/or at each \(x\)-intercept. If the polynomial splits into \(k\) linear factors, some of which are repeated, then it will have exactly \(k-1\) turning points, one between each \(x\)-intercept.
    ðŸ”—
    You do not need to memorize these results, but they are useful in checking that your sign analyses for your first derivatives are correct.
