    Subsection 6.7.2 Computing Derivatives of Logarithmic Functions

    Since logarithms are defined as the inverse of exponential functions, we can use our knowledge of the derivatives of exponential functions to determine derivatives of logarithms, rather than having to start with the limit definition of the derivative. Here is how we can do this:
    ðŸ”—
    Since \(y=\log_b x\) and \(b^y=x\) are inverses, we know that
    \begin{equation*} b^{\log_bx}=x\text{.} \end{equation*}
    This is also part (f) of TheoremÂ 6.3.9. We can take the derivative of both sides of this equation:
    \begin{equation*} \frac{d}{dx}\left(b^{\log_bx} \right)=\frac{d}{dx}( x) \end{equation*}
    ðŸ”—
    The derivative of the right side is \(1\) since \(\frac{d}{dx}(x) = 1\text{.}\) On the left side, we know that the derivative of the composition \(b^{g(x)}\) will be \((\ln b) b^{g(x)} g'(x)\text{,}\) by the chain rule. If we let \(g(x) = \log_bx\text{,}\) then \(g'(x) = \frac{d}{dx}(\log_bx)\) is precisely the derivative that we are trying to find:
    \begin{equation*} (\ln b)b^{\log_b x}\cdot \frac{d}{dx}(\log_bx) = 1 \end{equation*}
    ðŸ”—
    Now, we can solve for \(\frac{d}{dx}(\log_bx)\text{:}\)
    \begin{equation*} \frac{d}{dx}(\log_bx) = \frac{1}{(\ln b)b^{\log_b x}} \end{equation*}
    ðŸ”—
    We are technically done, but because we know \(b^{\log_b x} = x\text{,}\) we can simplify further to:
    \begin{equation*} \frac{d}{dx}(\log_bx) = \frac{1}{(\ln b)x} \end{equation*}
    ðŸ”—
    Thus we have discovered that the derivative of \(\log_b x\) is the fraction \(\frac{1}{(\ln b)x}\text{:}\) Notice that when \(b=e\text{,}\) \(\log_bx=\log_ex = \ln x\text{,}\) and \(\ln(e)=1\text{.}\) This means that
    \begin{equation*} \frac{d}{dx}\left(\ln x \right)=\frac{1}{x} \end{equation*}
    ðŸ”—

    Remark 6.7.1.

    Up to this point, derivatives have been the same basic type as the original function from which they come. For example, derivatives of power functions are power functions, derivatives of polynomials are polynomials, and derivatives of exponential functions are exponential functions. Now something surprising happens: derivatives of logarithmic functions are not logarithmic! Even more surprisingly, logarithmic functions are transcendental, but their derivatives are algebraic.
    ðŸ”—
    ðŸ”—

    For any constant \(b>0\) with \(b\neq1\text{,}\) and for all positive values of \(x\text{,}\)
    1. \(\displaystyle \frac{d}{dx}\left(\log_b(x) \right)=\frac{1}{\ln(b)x}\)
      ðŸ”—
      ðŸ”—
    2. \(\displaystyle \frac{d}{dx}\left(\ln (x) \right)=\frac{1}{x}\)

    Subsubsection 6.7.2.1 A Detail About Domains and Absolute Value:

    Because \(\ln x\) has domain \((0,\infty)\text{,}\) when we say that \(\frac{d}{dx}(\ln x) = \frac{1}{x}\text{,}\) we are also restricting \(\frac{1}{x}\) to the domain \((0,\infty)\text{;}\) we cannot have a derivative where the original function is undefined! However, it would be a pity to let half of the derivative go to waste. Letâ€™s see how to use the entire domain \((-\infty,0)\cup (0,\infty)\) of \(\frac{1}{x}\text{.}\)
    ðŸ”—
    Consider the function \(f(x) = \ln|x|\text{.}\) We know from our previous study that
    \begin{equation*} |x| = \begin{cases}x, \amp \text{ when } x\geq 0\\ -x, \amp \text{ when } x\lt 0 \end{cases} \end{equation*}
    ðŸ”—
    In plain language, \(|x|\) "makes \(x\) positive", so for \(x\lt 0\text{,}\) \(\ln|x|\) is now defined. For example, when \(x=-1\text{,}\) we have that \(|-1| = -(-1) = 1\) and so now, \(f(-1) =\ln|-1| = \ln(1) = 0\) is indeed defined.
    ðŸ”—
    However, when \(x=0\text{,}\) \(|0|=0\) and so \(f(0) = \ln |0| = \ln(0)\) is still undefined. Thus the domain of \(f(x) = \ln|x|\) is \((-\infty,0)\cup(0,\infty)\) and we can write
    \begin{equation*} f(x) = \ln|x| = \begin{cases}\ln(x), \amp \text{ when } x> 0\\ \ln(-x), \amp \text{ when } x\lt 0 \end{cases} \end{equation*}
    ðŸ”—
    Now consider the derivative of \(f(x) = \ln|x|\text{.}\) When \(x>0\text{,}\) we can simplify \(|x|=x\) so that \(f(x) = \ln|x|=\ln x\) and so
    \begin{equation*} f'(x) = \frac{d}{dx}(\ln x) = \frac{1}{x}\text{.} \end{equation*}
    On the other hand, when \(x\lt 0\text{,}\) we can simplify \(f(x) =\ln( -x)\text{.}\) Now the function \(y = -x\) is inside the logarithm, so using the chain rule, we have
    \begin{equation*} f'(x) = \frac{d}{dx}(\ln(-x)) = \frac{1}{-x}\cdot \frac{d}{dx}(-x) = -\frac{1}{x}\cdot (-1) = \frac{1}{x} \end{equation*}
    ðŸ”—
    Notice that we got \(f'(x) = \frac{1}{x}\) for both \(x>0\) and \(x\lt 0\text{.}\) Thus we have discovered that
    \begin{equation*} \frac{d}{dx}(\ln|x|) = \frac{1}{x}\text{.} \end{equation*}
    The benefit to this is that now both the original function and its derivative have the full domain \((-\infty,0)\cup(0,\infty)\text{.}\)
    ðŸ”—

For positive values of \(x\text{,}\) as we move from left to right, the slopes of \(\ln x\) are positive but getting smaller while the heights of \(\frac{1}{x}\) do the same.

    \begin{equation*} \frac{d}{dx}(\ln|x|) = \frac{1}{x} \end{equation*}

    Subsection 6.7.3 The Chain Rule with Logarithmic Functions

    By combining our new differentiation rules for logarithmic functions with the chain rule, we obtain the following rules for derivatives of logarithmic compositions:
    \begin{equation*} \begin{aligned}\frac{d}{dx}\left(\log_b(g(x))\right) \amp =\frac{1}{(\ln b)g(x)}\cdot g'(x)\\ \amp =\frac{g'(x)}{(\ln b)g(x)} \end{aligned} \end{equation*}
    and
    \begin{equation*} \begin{aligned}\frac{d}{dx}\left(\ln g(x)\right )\amp =\frac{1}{g(x)}\cdot g'(x)\\ \amp =\frac{g'(x)}{g(x)}. \end{aligned} \end{equation*}
    ðŸ”—
    Because the chain rule looks sufficiently different for logarithmic functions, we state it below with this new possible "outside" function of \(\log_b x\text{:}\)
    ðŸ”—

    Theorem 6.7.7. The chain rule for logarithmic functions.

    \begin{equation*} \frac{d}{dx}\left(\log_b(g(x))\right) =\frac{g'(x)}{(\ln b)g(x)} \end{equation*}
    \begin{equation*} \frac{d}{dx}\left(\ln g(x)\right)=\frac{g'(x)}{g(x)} \end{equation*}
    ðŸ”—

    Example 6.7.8.

    For example,
    \begin{equation*} \frac{d}{dx}\left(\log_5(x^2+1)\right)=\frac{1}{(\ln 5)(x^2+1)}\cdot (2x)=\frac{2x}{(\ln 5)(x^2+1)} \end{equation*}
    and
    \begin{equation*} \frac{d}{dx}\left(\ln (x^2+1)\right)=\frac{1}{(x^2+1)}\cdot (2x)=\frac{2x}{x^2+1} \end{equation*}

    Subsection 6.7.4 Logarithmic differentiation

    The calculation of derivatives of complicated functions involving products, quotients, or powers can often be simplified by taking logarithms. We saw a few examples of this in the last section. In this section, we will codify a new method of taking derivatives, called logarithmic differentiation, that takes advantage of the properties of logarithmic functions, and will also see how it can be used to take derivatives of yet another category of functions.
    ðŸ”—
    We can use logarithm rules to make taking derivatives easier, even if there are no logarithms involved in a problem. We cannot simply introduce a logarithm; that would change the function we are trying to differentiate. However, we can take the logarithm of both sides of the equation:
    ðŸ”—

    Example 6.7.16.

    In order to take the derivative of
    \begin{equation*} y = \frac{x^{3/4}\sqrt{x^2+1}}{(3x+2)^5} \end{equation*}
    we would need to first apply the quotient rule, then use the product rule for the derivative of the numerator, which involves using the chain rule for the derivative of \(\sqrt{x^2+1}\text{.}\) We would also need the chain rule for the derivative of the denominator. Instead, we take the natural logarithm of both sides and simplify using algebra:
    \begin{equation*} \begin{aligned}\ln y \amp = \ln\left(\frac{x^{3/4}\sqrt{x^2+1}}{(3x+2)^5}\right)\\ \amp = \ln(x^{3/4}\sqrt{x^2+1})-\ln((3x+2)^5)\\ \amp = \ln(x^{3/4})+\ln(\sqrt{x^2+1})-5\ln(3x+2)\\ \amp = \frac{3}{4} \ln x+\frac{1}{2}\ln (x^2+1) - 5\ln(3x+2) \end{aligned} \end{equation*}
    ðŸ”—
    Now we move on to taking derivatives. Pay very close attention to the left-hand side of this equation! Since \(y\) is a function of \(x\text{,}\) we must use the chain rule/implicit differentiation to take its derivative. However, the right-hand side of the equation is now much simpler to differentiate as there are no quotients or products.
    \begin{equation*} \begin{aligned}\frac{d}{dx}(\ln y) \amp = \frac{d}{dx}\left(\frac{3}{4} x+\frac{1}{2}\ln (x^2+1) - 5\ln(3x+2)\right)\\ \frac{1}{y}\frac{dy}{dx} \amp = \frac{3}{4}\cdot\frac{1}{x}+\frac{1}{2}\frac{2x}{x^2+1}-5\cdot\frac{3}{3x+2} \end{aligned} \end{equation*}
    ðŸ”—
    We can now solve for \(dy/dx\) by multiplying both sides of the equation by \(y\text{:}\)
    \begin{equation*} \frac{dy}{dx} = y\left(\frac{3}{4x}+\frac{x}{x^2+1}-\frac{15}{3x+2}\right) \end{equation*}
    ðŸ”—
    Because we started with an explicit expression for \(y\text{,}\) we would like to end with an explicit derivative \(\frac{dy}{dx}\text{,}\) meaning one that involves only the independent variable \(x\text{.}\) So we substitute \(y\) from the beginning of the problem:
    \begin{equation*} \frac{dy}{dx} = \frac{x^{3/4}\sqrt{x^2+1}}{(3x+2)^5}\left(\frac{3}{4x}+\frac{x}{x^2+1}-\frac{15}{3x+2}\right) \end{equation*}
    ðŸ”—
    This expression is not pretty, but it gives us a formula for the derivative \(y'=\frac{dy}{dx}\text{.}\)
    ðŸ”—
    ðŸ”—
    \fbox{Here is a summary of the steps taken in this method, called logarithmic differentiation
    • Take the natural logarithm of both sides of an equation \(y=f(x)\) to get \(\ln y = \ln f(x)\)
      ðŸ”—
      ðŸ”—
    • Use the logarithm rules to simplify \(\ln f(x)\) as much as possible, bringing exponents down and splitting up products and quotients into addition and subtraction.
      ðŸ”—
      ðŸ”—
    • Differentiate implicitly with respect to \(x\text{.}\) The derivative of \(\ln y\) is \(\frac{1}{y}\frac{dy}{dx}\text{.}\)
      ðŸ”—
      ðŸ”—
    • Solve the resulting equation for \(\frac{dy}{dx}\) by multiplying both sides by \(y\text{.}\)
      ðŸ”—
      ðŸ”—
    • Replace \(y\) by the original expression for \(f(x)\) in order to write your answer in terms of only the variable \(x\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—

    Remark 6.7.17.

    If \(f(x)\lt 0\) for some values of \(x\text{,}\) then \(\ln (f(x))\) is not defined, but we can write \(|y| = |f(x)|\) and use the derivative rule
    \begin{equation*} \frac{d}{dx}\ln|x| = \frac{1}{x}\text{.} \end{equation*}
    We can also distribute absolute values over products, quotients, and powers, so the computations are exactly the same:
    \begin{equation*} |ab| = |a||b|, \left|\frac{a}{b}\right| = \frac{|a|}{|b|}, |a^b| = |a|^b \end{equation*}

    ðŸ”—

    Example 6.7.18.

    Subsection 6.7.5 Functions with Variables in the Base and Exponent

    Consider the function
    \begin{equation*} f(x) = x^x \end{equation*}
    ðŸ”—
    None of the derivative rules we know apply to this function! The power rule \(\frac{d}{dx}(x^n) = nx^{n-1}\) applies only to functions where the exponent is a number. The rule \(\frac{d}{dx}(b^x) = (\ln b)b^x\) applies only to functions where the base is a number. In general, we do not yet have a way to take derivatives of functions where the variable appears in both the base and the exponent.
    ðŸ”—
    Logarithmic differentiation provides a method for taking derivatives of such functions. The key step in this method is to use the logarithm rule for moving the exponent in front of the logarithm, so that there is no longer an exponent with a variable:
    ðŸ”—

    Example 6.7.20.

    Use logarithmic differentiation to calculate the derivative of the function
    \begin{equation*} f(x)=x^x \end{equation*}
    ðŸ”—
    Solution.
    First, letâ€™s take the natural log of both sides, and use algebra:
    \begin{equation*} \begin{aligned}\ln(f(x)) \amp =\ln\left( x^x\right)\\ \amp =x\ln x. \end{aligned} \end{equation*}
    ðŸ”—
    Now, lets take the derivative of both sides:
    \begin{equation*} \begin{aligned}\frac{d}{dx}(\ln(f(x))) \amp =\frac{d}{dx}\left(x\ln x\right)\\ \frac{f'(x)}{f(x)} \amp = x(1/x)+\ln x \end{aligned} \end{equation*}
    ðŸ”—
    So, multiplying both sides by \(f(x)\text{,}\) we obtain:
    \begin{equation*} \begin{aligned}f'(x)\amp =f(x)( x(1/x)+\ln x)\\ \amp =x^x( 1+\ln x)\\ \amp = x^x+x^x\ln x. \end{aligned} \end{equation*}

    ðŸ”—

    Subsection 6.7.6 Summary

    The following methods were introduced in this section:
    ðŸ”—
    Key ideas: Derivatives of logarithms are algebraic functions: \(\frac{d}{dx}(\ln x) = \frac{1}{x}\) and \(\frac{d}{dx}(\log_b x) = \frac{1}{(\ln b)x}\text{.}\) The chain rule looks strange when combined with logarithmic functions and is worth paying special attention to. Applying a logarithm to (both sides of) a function can allow us to break apart the function and avoid using complicated derivative rules. In order to take the derivative of a function with variables in both the base and the exponent, we must use logarithmic differentiation.
    ðŸ”—
    Other ideas introduced: We can use an absolute value within a logarithm to extend the domain of the function. In the case of \(\ln|x|\text{,}\) this does not change the derivative:
    \begin{equation*} \frac{d}{dx}(\ln |x|) = \frac{1}{x} \end{equation*}
    ðŸ”—
    Other ideas reinforced: The logarithm rules are extremely useful.
