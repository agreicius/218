    Subsection 3.4.2 Definitions: Infinite Limits and Limits at Infinity

    The wonderful thing about continuity is that it lets us calculate limits by evaluation. But what are we supposed to do with the limits that we cannot just evaluate, such as at \(x\)-values that are not in the domain of a function? We begin to investigate with a simple example:
    ðŸ”—

    Example 3.4.1.

    For example, the function \(f(x) = \frac{1}{x}\) is undefined when \(x=0\text{,}\) so the only limit of this function that we cannot calculate by simply plugging in is
    \begin{equation*} \lim_{x\to 0}\frac{1}{x} \end{equation*}
    ðŸ”—
    The graph of \(f(x) = \frac{1}{x}\) is in FigureÂ 3.4.2 below:
    ðŸ”—
    Based on FigureÂ 3.4.2, you may be able to see what \(\ds\lim_{x\to0}\frac{1}{x}\) should be. In case it isnâ€™t clear to you, here is a table of values for \(f(x) = \frac{1}{x}\) as \(x\to 0\) from both the left and right.
    ðŸ”—
    We can see that as \(x\to 0^+\text{,}\) the values of \(\frac{1}{x}\) become larger and larger. We write this using limit notation as
    \begin{equation*} \lim_{x\to 0^+}\frac{1}{x} = \infty\text{.} \end{equation*}
    Similarly, we can see that as \(x\to 0^-\text{,}\) the values of \(\frac{1}{x}\) become larger and larger, and are negative. We write this using limit notation as
    \begin{equation*} \lim_{x\to0^-}\frac{1}{x} = -\infty\text{.} \end{equation*}
    Since the right and left limits do not agree, we cannot make any two-sided limit statement, other than to say that
    \begin{equation*} \lim_{x\to 0}\frac{1}{x}\text{ DNE, because the right and left limits are not equal. } \end{equation*}
    ðŸ”—
    This last answer is not particularly informative or useful, and so our REAL answer for the limit as \(x\to 0\) is to describe both the left and right limits as precisely as we can:
    \begin{equation*} \lim_{x\to 0^+}\frac{1}{x} = \infty\text{ and } \lim_{x\to0^-}\frac{1}{x} = -\infty \end{equation*}
    ðŸ”—
    ðŸ”—
    Example ExampleÂ 3.4.1 demonstrates the necessity of defining a notation that describes the values of a function becoming infinitely large, either positively or negatively. This is stated more carefully in Definition DefinitionÂ 3.4.4 below:
    ðŸ”—

    Definition 3.4.4. Definition of Infinite Limits.

    Given a function \(f\) and a fixed input \(x=c\text{,}\) we say that \(f\) approaches infinity as \(x\) approaches \(c\) and write
    \begin{equation*} \lim_{x\to c}f(x) = \infty \end{equation*}
    provided that we can make the values of \(f(x)\) as large as we like by taking \(x\) sufficiently close (but not equal) to \(c\text{.}\)
    ðŸ”—
    Similarly, we say that \(f\) approaches negative infinity as \(x\) approaches \(c\) and write
    \begin{equation*} \lim_{x\to c}f(x) = -\infty \end{equation*}
    provided that we can make the values of \(f(x)\) as large and negative as we like by taking \(x\) sufficiently close (but not equal) to \(c\text{.}\)
    ðŸ”—
    Either of these statements can be modified appropriately for left and right limits.
    ðŸ”—
    ðŸ”—
    \fbox{ Recall that in Definition DefinitionÂ 3.1.5, a limit does not exist if there is no single value that \(f(x)\) approaches as \(x\to c\text{.}\) That is the case for every infinite limit: there is no finite value that the function approaches. Even though we write that the limit is "equal to" \(\infty\) or \(-\infty\text{,}\) these limits do not exist because \(\infty\) and \(-\infty\) are not real numbers. }
    ðŸ”—

    Example 3.4.5.

    We can see from the graph of \(g(x) = \frac{1}{x^2}\) in FigureÂ 3.4.6 that
    \begin{equation*} \lim_{x\to 0^-}\frac{1}{x^2} = \infty\text{ and } \lim_{x\to 0^+}\frac{1}{x^2} = \infty \end{equation*}
    so that
    \begin{equation*} \lim_{x\to 0}\frac{1}{x^2} = \infty\text{.} \end{equation*}
    This means only that the values of \(\frac{1}{x^2}\) become infinitely large as \(x\to 0\text{.}\) The limit still does not exist. However, we do NOT write \(\lim_{x\to 0}\frac{1}{x^2}\) DNE because that does not give us any useful information. Remember: limits are used to describe the behavior of functions, not simply spit out an answer, so we want our answers to be as descriptive as possible.

    Subsection 3.4.3 Vertical and Horizontal Asymptotes

    Infinite limits (Definition DefinitionÂ 3.4.4) and finite limits at infinity (Definition DefinitionÂ 3.4.12) correspond to two different visual features of graphs: vertical and horizontal asymptotes.
    ðŸ”—
    A function has a vertical asymptote at the point \(x=c\) if the values of the function become infinitely large, either positively or negatively, to either the left or the right (or both) as \(x\) approaches \(c\text{.}\)
    ðŸ”—

    Definition 3.4.17. Vertical Asymptote.

    A function \(f\) has a vertical asymptote at \(x=c\) if one or more of the following are true:
    \begin{equation*} \lim_{x\to c^+} f(x) = \infty, \lim_{x\to c^-} f(x) = \infty, \lim_{x\to c^+} f(x) = -\infty, \lim_{x\to c^-} f(x) = -\infty \end{equation*}

    ðŸ”—

    ðŸ”—
    A function has a horizontal asymptote if the values of the function approach a finite real number as \(x\) becomes infinitely large, either positively or negatively.

    Subsection 3.4.4 Limits of the form \(\frac{1}{0}\)

    As we saw in Example ExampleÂ 3.4.1 and Example ExampleÂ 3.4.5 the graphs of \(y= \frac{1}{x}\) and \(y= \frac{1}{x^2}\) have vertical asymptotes at \(x=0\text{.}\) These two examples motivate a category of limits that "have the form \(\frac{1}{0}\)" and that result in vertical asymptotes.
    ðŸ”—
    As we saw in Example ExampleÂ 3.4.5 in the case of \(y=\frac{1}{x^2}\text{,}\) when the denominator approaches \(0\) and is positive, the limit will approach \(\infty\text{.}\) We denote this behavior as having the form \(\frac{1}{0^+}\) since the denominator is always positive. However, as we saw in Example ExampleÂ 3.4.1 in the case of \(y=\frac{1}{x}\text{,}\) it is possible for the denominator to approach \(0\) and be positive on one side of the limit (which we denote by the form \(\frac{1}{0^+}\)) but approach \(0\) and be negative on the other side of the limit (which we denote by the form \(\frac{1}{0^-}\)). Thus with limits of the form \(\frac{1}{0}\text{,}\) it is necessary to determine whether the denominator is positive or negative on either side of the limit. Only then will we be able to determine whether the limit approaches \(\infty\) or \(-\infty\text{.}\)
    ðŸ”—
    The above discussion is summarized in Theorem TheoremÂ 3.4.32 below:
    ðŸ”—

    Theorem 3.4.32. Limits Whose Denominators Approach Zero from the Right or the Left.

    For any real number \(c\text{,}\)
    1. If \(\ds\lim_{x\to c}\frac{f(x)}{g(x)}\) is of the form \(\frac{1}{0^+}\text{,}\) then \(\ds\lim_{x\to c}\frac{f(x)}{g(x)} = \infty\text{.}\)
      ðŸ”—
      ðŸ”—
    2. If \(\ds\lim_{x\to c}\frac{f(x)}{g(x)}\) is of the form \(\frac{1}{0^-}\text{,}\) then \(\ds\lim_{x\to c}\frac{f(x)}{g(x)} = -\infty\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    The above statements also apply to one-sided limits.
    ðŸ”—
    ðŸ”—
    Of course, the numerator of the limit need not be exactly equal to 1. Any nonzero finite number divided by a very small number will become very large. For instance,
    \begin{equation*} \frac{2}{0.000001} = 2,000,000\text{.} \end{equation*}
    ðŸ”—
    For ease of notation, we say that limits that evaluate to \(\dfrac{\text{ "non-zero" } }{\text{ zero } }\) have the form \(\mathbf{\frac{1}{0}}\), even though the numerator may be any nonzero number. As we shall see later, the sign of the numerator does have an influence on whether the overall limit is positive or negative, even though the size of the nonzero numerator does not matter.
    ðŸ”—

    Subsection 3.4.5 Summary

    The following terms were introduced in this section:
    ðŸ”—
    \(\frac{1}{0}\text{.}\)
    Key ideas: Limits that have infinite \(y\)-values for finite \(x\)-values are called vertical asymptotes. Limits that have finite \(y\)-values for infinite \(x\)-values are called horizontal asymptotes. Limits of the form \(\frac{1}{0}\) are always infinite, and often need to be computed separately from the left and right to determine whether the limit is positive or negative from either side.
    ðŸ”—
    Other ideas introduced: We still have a long way to go to be able to calculate limits algebraically. We do not yet know how to calculate limits at infinity, except in very simple cases. We also do not yet know how to manage limits that evaluate to \(\frac{0}{0}\text{.}\)
