    Subsection 4.3.2 When is a Function Differentiable?

    We now know that the instantaneous rate of change of a function \(f(x)\) at \(x=a\text{,}\) or equivalently the slope of the tangent line to the graph of \(y=f(x)\) at \(x=a\text{,}\) is given by the value \(f\,'(a)\text{.}\) The process of "taking the derivative" generates a new function, \(f\,'(x)\text{,}\) derived from the original function \(f(x)\text{.}\) In this section, we look at the domain of the function \(f\,'(x)\) and what can cause it to be undefined. The key observation is that the derivative arises from a limit, and it is possible for a limit to fail to exist for several different reasons.
    ðŸ”—

    Example 4.3.1.

    The derivative of the function \(f(x) = 4x-x^2\) is
    \begin{equation*} f\,'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h} = \lim_{h\to 0}\frac{(4(x+h)-(x+h)^2)-(4x-x^2)}{h}\text{.} \end{equation*}
    The numerator simplifies as follows:
    \begin{equation*} \begin{aligned}(4(x+h)-(x+h)^2)-(4x-x^2) \amp = 4x+4h-a^2-2hx-h^2-4x+x^2\\ \amp = 4h-2hx-h^2 \end{aligned} \end{equation*}
    ðŸ”—
    Then returning to the limit, we can factor an \(h\) in the numerator and cancel, before evaluating the limit:
    \begin{equation*} \lim_{h\to 0}\frac{4h-2hx-h^2}{h} = \lim_{h\to 0}\frac{h(4-2x-h)}{h} = \lim_{h\to 0} (4-2x-h) = 4-2x-0 = 4-2x \end{equation*}
    ðŸ”—
    Thus we have \(f\,'(x) = 4-2x\text{.}\) As a function, \(f\,'(x)\) is defined for all values of \(x\text{.}\) Below, we see the graph of \(f(x)\) on the left, with several tangent lines, and the graph of \(f\,'(x)\) on the right, with several values corresponding to the marked tangent lines. We see that where \(f(x)\) is increasing, \(f\,'(x)\) is positive. Where \(f(x)\) is decreasing, \(f\,'(x)\) is negative. We also notice that the graph of \(f(x)\) has a tangent line at every point, so the graph of \(f\,'(x)\) has a value at every point, meaning that it is defined for all \(x\)-values. Indeed, the function \(f\,'(x) = 4-2x\) is defined for all real numbers \(x\text{.}\)
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Remark 4.3.2.

    Remember that when taking the derivative, it is always the variable \(h\) that approaches 0, not \(x\text{.}\)
    ðŸ”—
    ðŸ”—
    The function \(f(x)\) in ExampleÂ 4.3.1 is a polynomial, one of the most well-behaved functions we have looked at so far, and so we expect its derivative to be similarly well-behaved. Letâ€™s look at a more complicated function and see what happens with its derivative.
    ðŸ”—

    Example 4.3.3.

    The function \(f(x)=|x|\) is actually a piecewise-defined function
    \begin{equation*} f(x) = \begin{cases}x,\amp x\geq 0\\ -x, \amp x \lt 0 \end{cases} \end{equation*}
    ðŸ”—
    Away from the break point at \(x=0\text{,}\) \(f(x)\) is linear. For \(x\lt 0\text{,}\) \(f(x) = -x\text{,}\) which has derivative \(f\,'(x) = -1\text{.}\) For \(x>0\text{,}\) \(f(x) = x\text{,}\) which has derivative \(f\,'(x) = 1\text{.}\)
    ðŸ”—
    Letâ€™s look specifically at the break point \(x=0\) and calculate \(f\,'(0)\text{.}\)
    \begin{equation*} f\,'(0) = \lim_{h\to 0}\frac{f(0+h)-f(0)}{h} = \lim_{h\to 0} \frac{|0+h|-0}{h} = \lim_{h\to 0} \frac{|h|}{h} \end{equation*}
    ðŸ”—
    In order to simplify further, we have to know what to do with \(|h|\text{.}\) When \(h>0\text{,}\) \(|h|=h\text{,}\) and when \(h\lt 0\text{,}\) \(|h| = -h\text{.}\) This means that we need to split up the limit into right and left limits in order to finish the calculation, since we get a different formula for \(|h|\) depending on whether \(h\to 0^-\) or \(h\to 0^+\text{:}\)
    \begin{equation*} \begin{aligned}\lim_{h\to 0^-} \frac{|h|}{h} \amp = \lim_{h\to 0^-} \frac{-h}{h} = \lim_{h\to 0^-} -1 = -1\\ \lim_{h\to 0^+} \frac{|h|}{h} \amp = \lim_{h\to 0^+} \frac{h}{h} = \lim_{h\to 0^+} 1 = 1 \end{aligned} \end{equation*}
    ðŸ”—
    Since the right and left limits as \(h\to 0^{\pm}\) are not equal, we conclude that the two-sided limit as \(h\to 0\) does not exist:
    \begin{equation*} f\,'(0) = \lim_{h\to 0} \frac{|h|}{h} \text{ DNE } \end{equation*}
    ðŸ”—
    This means that the derivative \(f\,'(x)\) is undefined at \(x=0\text{.}\) Summarizing all of the above, we have
    \begin{equation*} f\,'(x) = \begin{cases}1,\amp x>0\\ \text{ DNE } , \amp x=0\\ -1, \amp x\lt 0 \end{cases} \end{equation*}
    ðŸ”—
    As a function, \(f\,'(x)\) has domain \(x\neq 0\text{,}\) or \((-\infty,0)\cup(0,\infty)\text{.}\) Itâ€™s interesting to note that this is different than the domain of the original function \(f(x)= |x|\text{,}\) which is \(\mathbb{R} = (-\infty,\infty)\text{.}\)
    ðŸ”—
    Below, we see the graph of \(f(x) = |x|\) with slopes marked for \(h\lt 0\) (left) and \(h>0\) (middle), along with the graph of \(f\,'(x)\) (right). Note that where \(f(x)\) is increasing, \(f\,'(x)\) is positive, and where \(f(x)\) is decreasing, \(f\,'(x)\) is negative. At \(x=0\text{,}\) the graph of \(f(x)\) has a "corner" where there is no clearly defined tangent line, while the graph of \(f\,'(x)\) has a jump and is undefined at \(x=0\text{.}\)

    ðŸ”—

    Subsection 4.3.3 Differentiable vs. Continuous

    The examples we have seen so far have all been of functions that were continuous for all values of \(x\) but were not differentiable at a single point. Another way that a function \(f\) can fail to be differentiable at a point \(x=a\) is if \(f\) fails to be continuous at \(x=a\text{.}\)
    ðŸ”—
    If a function is not continuous, then it has absolutely no chance of being differentiable. Think about the different ways that a function can be discontinuous: a jump, removable discontinuity (hole), or infinite discontinuity (vertical asymptote).
    ðŸ”—
    At a vertical asymptote or at some holes, the function is undefined at \(x=a\text{.}\) If the function is undefined at \(x=a\text{,}\) it cannot have a tangent line at \(x=a\text{.}\) To see this, think of the definition of \(f\,'(a)\text{:}\)
    \begin{equation*} f\,'(a) = \lim_{h\to 0}\frac{f(a+h)-f(a)}{h} \end{equation*}
    ðŸ”—
    In order to even begin to calculate this limit, we have to be able to use the value \(f(a)\) in the numerator. If \(f\) is undefined at \(x=a\text{,}\) then \(f(a)\) does not exist, and we canâ€™t even write down this definition!

    Subsection 4.3.4 Local Linearizations and Approximating Functions

    Letâ€™s use this notion of differentiability to take another look at tangent lines. The tangent line to \(f\) at a point \(x=a\) is defined in terms of the derivative \(f\,'(a)\text{:}\)
    ðŸ”—

    Definition 4.3.22. Equation of the Tangent Line to a Function at a Point.

    The tangent line to the graph of a function \(f\) at a point \(x=a\) is defined to be the line passing through \((a,f(a))\) with slope \(f\,'(a)\text{,}\) provided that the derivative \(f\,'(a)\) exists. This line has equation
    \begin{equation*} y=f\,'(a)(x-a)+f(a) \end{equation*}
    ðŸ”—
    ðŸ”—
    The equation in DefinitionÂ 4.3.22 comes from the point-slope equation of a line
    \begin{equation*} y=m(x-x_0)+y_0 \end{equation*}
    where we use the slope \(m=f'(a)\) and the point \((x_0,y_0) = (a,f(a))\) so that \(x_0=a\) and \(y_0 = f(a)\text{.}\)
    ðŸ”—

    Example 4.3.23.

    For example, we have seen that the derivative of \(f(x)=x^2\) is \(f'(x) = 2x\text{,}\) and so \(f\,'(3)=6\text{.}\) This means that the tangent line to the graph of \(f(x)=x^2\) at \(x=3\) has slope \(f\,'(3)=6\) and passes through the point where \(x=3\) and \(y=f(3) = 3^2 = 9\text{.}\) This line has equation \(y=6(x-3)+9\text{,}\) which we can simplify to slope-intercept form as \(y=6x-9\text{.}\)
    ðŸ”—
    ðŸ”—
    When it exists, the tangent line to a function \(f\) at a point \(x=a\) is the unique line that "agrees with" both the height of the function and the slope of the function at \(x=a\text{.}\) This means that near \(x=a\) the graph of a function is very close to the graph of its tangent line; the graph is "locally linear," meaning linear near the point. Therefore we can use the tangent line as a rough approximation to the function \(f\) itself near \(x=a\text{.}\) When we use the tangent line to approximate the function near a point, we call it the local linearization of \(f\) around \(x=a\) and we write
    \begin{equation*} f(x)\approx L(x) = f\,'(a)(x-a)+f(a) \end{equation*}
    ðŸ”—
    Why do we not just use the name "tangent line?" The perspective here is that we are "linearizing", meaning making linear, the function \(f(x)\) by approximating it by its tangent line. The word "local" here is used to mean nearby, or in a neighborhood of, the point \(x=a\text{.}\)
    ðŸ”—
    The definition of the tangent line does not tell us how good an approximation one can make by using the tangent line. What constitutes "near" and what constitutes "good" will be determined by the context of the problem at hand.
    ðŸ”—
