    Subsection 9.2.2 Statement of the Squeeze Theorem

    Another useful tool for calculating new types of limits is the Squeeze Theorem. This theorem says that if the output of a function \(f(x)\) is always bounded between a "lower function" \(l(x)\) and an "upper function" \(u(x)\text{,}\) and if the lower and upper functions approach the same value \(L\) as \(x\to c\text{,}\) then \(f(x)\) gets squeezed between the lower and upper functions and also approaches \(L\) as \(x\to c\)
    ðŸ”—

    Theorem 9.2.1. The Squeeze Theorem for Limits.

    If \(l(x)\leq f(x)\leq u(x)\) for all \(x\) near \(c\text{,}\) not not necessarily at \(x=c\text{,}\) and if \(\ds\lim_{x\to c}l(x)\) and \(\ds\lim_{x\to c} u(x)\) are both equal to \(L\text{,}\) then \(\ds\lim_{x\to c}f(x) = L\text{.}\)
    ðŸ”—
    ðŸ”—
    The theorem holds for \(x\to\pm\infty\) and \(x\to c^\pm\text{.}\)
    ðŸ”—
    ðŸ”—

    Remark 9.2.2.

    The Squeeze Theorem is also called the Sandwich Theorem. You can think of the lower and upper functions as pieces of bread that make a sandwich with the middle function \(f(x)\text{.}\)
    ðŸ”—
    ðŸ”—

    Remark 9.2.6.

    There are three important hypotheses that must be satisfied in order to use the Squeeze Theorem:
    • We must have the inequality \(l(x)\leq f(x)\leq u(x)\) for \(x\) near \(c\text{.}\)
      ðŸ”—
      ðŸ”—
    • The limits \(\ds\lim_{x\to c} l(x)\) and \(\ds\lim_{x\to c}u(x)\) must exist (and be finite).
      ðŸ”—
      ðŸ”—
    • The limits \(\ds\lim_{x\to c} l(x)\) and \(\ds\lim_{x\to c}u(x)\) must be equal to each other.

    Subsection 9.2.3 Using the Squeeze Theorem

    In general, finding the inequality \(l(x)\leq f(x)\leq u(x)\) is very difficult. However, there are three inequalities that are particularly useful as a starting point for applying the Squeeze Theorem:
    \begin{equation*} -1\leq \sin\theta\leq 1 \end{equation*}
    \begin{equation*} -1\leq \cos\theta\leq 1 \end{equation*}
    \begin{equation*} -\frac{\pi}{2}\lt \arctan\theta\lt \frac{\pi}{2} \end{equation*}
    ðŸ”—
    For limits involving sine, cosine, or arctangent, we can use these inequalities as a starting point for building up to an inequality involving the function we wish to know the limit of.
    ðŸ”—

    ðŸ”—
    Solution.
    We wish to determine \(\ds\lim_{x\to\infty} \frac{\sin x}{x}\) and \(\ds\lim_{x\to-\infty} \frac{\sin x}{x}\text{.}\) Starting with \(x\to\infty\text{,}\) we begin with the inequality
    \begin{equation*} -1\leq \sin x\leq 1\text{.} \end{equation*}
    ðŸ”—
    We are interested in the function \(\frac{\sin x}{x}\text{,}\) so we can divide by \(x\) in order to have the middle function equal to \(\frac{\sin x}{x}\text{;}\) this is what we want in order to apply the Squeeze Theorem. As \(x\to\infty\text{,}\) we know that \(x>0\) and so dividing by \(x\) does not flip the inequality:
    \begin{equation*} -\frac{1}{x}\leq \frac{\sin x}{x}\leq\frac{1}{x}\text{.} \end{equation*}
    If we take the limit of the bounding functions, they are both equal to zero:
    \begin{equation*} \lim_{x\to\infty}-\frac{1}{x} = 0\text{ and } \lim_{x\to\infty}\frac{1}{x} = 0 \end{equation*}
    ðŸ”—
    Hence by the Squeeze Theorem, the limit of the middle function \(\frac{\sin x}{x}\) is the same:
    \begin{equation*} \lim_{x\to\infty}\frac{\sin x}{x} = 0 \end{equation*}
    ðŸ”—
    Letâ€™s now consider \(x\to-\infty\text{.}\) We can still start with
    \begin{equation*} -1\leq \sin x\leq 1 \end{equation*}
    but now \(x\lt 0\) so when we divide by \(x\text{,}\) we must flip the inequalities:
    \begin{equation*} -\frac{1}{x}\geq \frac{\sin x}{x}\geq\frac{1}{x}\text{.} \end{equation*}
    We still have that the limits of both bounding function are equal to zero:
    \begin{equation*} \lim_{x\to-\infty}-\frac{1}{x} = 0\text{ and } \lim_{x\to-\infty}\frac{1}{x} = 0 \end{equation*}
    so by the Squeeze Theorem, the limit of the middle function \(\frac{\sin x}{x}\) is the same:
    \begin{equation*} \lim_{x\to-\infty}\frac{\sin x}{x} = 0 \end{equation*}
    ðŸ”—
    Thus overall, the function \(f(x) = \frac{\sin x}{x}\) has left and right horizontal asymptotes at \(y=0\text{.}\)

    Subsection 9.2.4 The Derivatives of Sine and Cosine: Rigorous Proofs

    In SectionÂ 8.7, we inferred that the derivative of the sine function is the cosine function by observing the behavior of the associated slope function of sine, and likewise that the derivative of the cosine function is the negative of the sine function. However, that is not a mathematically rigorous proof. In this section, we formally prove what you already know:
    \begin{equation*} \frac{d}{dx}(\sin x) = \cos x\text{ and } \frac{d}{dx}(\cos x) = -\sin x\text{.} \end{equation*}
    There is a very complicated application of the Squeeze Theorem necessary for this proof, which is why we include it here. However, it is much harder than anything we will ask you to do. It is worth reading through this section to see how all of the different ideas weâ€™ve developed fit together, but donâ€™t feel like you have to follow every single detail. In particular, the two special limits calculated via the Squeeze Theorem will be found using different methods in the next section.
    ðŸ”—
    In order to calculate the derivative of \(\sin x\text{,}\) we start with the limit definition of the derivative:
    \begin{equation*} \begin{aligned}\frac{d}{dx}(\sin x) \amp = \lim_{h\to 0}\frac{\sin (x+h)-\sin x}{h} \amp \leftarrow\text{ definition of the derivative } \\ \amp = \lim_{h\to 0}\frac{(\sin x\cos h+\sin h\cos x)-\sin x}{h}\amp \leftarrow\text{ sum identity for sine } \\ \amp = \lim_{h\to 0}\frac{\sin x(\cos h-1)+\sin h\cos x}{h}\amp \leftarrow\text{ rearrange and factor } \\ \amp = \lim_{h\to 0}\left(\sin x\frac{\cos h-1}{h}+\cos x\frac{\sin h}{h}\right)\amp \leftarrow\text{ factor more } \\ \amp =\sin x\left(\lim_{h\to 0}\frac{\cos h-1}{h}\right)+\cos x\left(\lim_{h\to 0}\frac{\sin h}{h}\right)\amp \leftarrow\text{ use limit rules and factor again } \end{aligned} \end{equation*}
    ðŸ”—
    In the last step above, notice that since the variable in the limit is \(h\text{,}\) not \(x\text{,}\) we can factor the \(\sin x\) and \(\cos x\) out of the limits as "constants."
    ðŸ”—
    We now need to calculate the limits \(\ds\lim_{h\to 0}\frac{\cos h-1}{h}\) and \(\ds\lim_{h\to 0}\frac{\sin h}{h}\text{,}\) which both have the form \(\frac{0}{0}\text{.}\) However, we cannot use Lâ€™HÃ´pitalâ€™s rule! Why??? This is a very important piece of logic: we are trying to prove the formula for the derivative of the sine function, so we cannot use the derivative of the sine function while proving it. That would be what is called "circular logic:" when we rely on the thing we are trying to prove in order to prove it.
    ðŸ”—

    Checkpoint 9.2.12.

    Give an example from your own experience that demonstrates circular logic, meaning an argument that uses a "fact" in order to justify that same "fact."

    In order to evaluate \(\ds\lim_{h\to 0}\frac{\sin h}{h}\text{,}\) we use the Squeeze Theorem. However, as observed in ExerciseÂ 9.2.9, we cannot use the inequality \(-\frac{1}{h}\leq \frac{\sin h}{h}\leq \frac{1}{h}\) for this limit. (In that exercise, you were asked to explain why.)
    ðŸ”—
    Instead, consider the picture below that shows a triangle appropriate for small, positive values of \(\theta\) (so values close to \(0^+\)).
    ðŸ”—
    ðŸ”—
    In the smaller triangle, the hypotenuse is 1, the height of the side opposite \(\theta\) is \(\sin\theta\text{,}\) and the length of the adjacent side is \(\cos\theta\text{.}\) In the larger triangle, the adjacent side has length 1. We can find the height of the opposite side using similar triangles:
    \begin{equation*} \frac{\text{ opp } }{\text{ adj } } = \frac{\sin\theta}{\cos\theta} = \frac{\text{?} }{1} \end{equation*}
    and so the height of the unknown opposite side of the larger triangle must be \(\frac{\sin\theta}{\cos\theta} = \tan\theta\text{.}\) Lastly, the length of the arc from the bottom right corner of the larger triangle to the top right corner of the smaller triangle has length \(\theta\) because the hypotenuse of the smaller triangle is \(1\text{;}\) radians are specifically defined to be equal to the arc length around the circle of radius 1. Comparing these three lengths, we see that \(\sin\theta\) is the smallest, \(\tan\theta\) is the largest, and \(\theta\) is between these two. Hence we have the inequality
    \begin{equation*} \sin\theta\leq \theta\leq \tan\theta\text{.} \end{equation*}
    Dividing through by \(\sin\theta\text{,}\) which is positive as long as \(\theta\to0^+\text{,}\) and remembering that \(\tan\theta = \frac{\sin\theta}{\cos\theta}\text{,}\)
    \begin{equation*} 1\leq \frac{\theta}{\sin\theta}\leq \frac{1}{\cos\theta}\text{.} \end{equation*}
    Since all of the expressions in this chain of inequalities are positive, we can take reciprocals and flip the inequalities to obtain:
    \begin{equation*} 1\geq \frac{\sin\theta}{\theta}\geq \cos\theta\text{.} \end{equation*}
    We justified this inequality for \(\theta\to0^+\) but it is also true for \(\theta\to0^-\text{,}\) only we must draw the picture in the fourth quadrant and the arithmetic has a few more flippings of inequalities since the quantities involved are negative.
    ðŸ”—
    Now by applying the Squeeze Theorem to this inequality, we have \(\ds\lim_{\theta\to 0}1 =\ds\lim_{\theta\to0}\cos\theta = 1\text{,}\) hence
    \begin{equation*} \lim_{\theta\to 0}\frac{\sin\theta}{\theta} = 1\text{.} \end{equation*}
    ðŸ”—
    In particular, \(\ds\lim_{h\to0}\frac{\sin h}{h} = 1\text{.}\) We also needed to compute \(\ds\lim_{h\to 0}\frac{\cos h-1}{h}\text{.}\) In order to do this, we let \(h=2\theta\) and use the double-angle identity \(\cos2\theta = 1-\sin^2\theta\text{:}\)
    \begin{equation*} \begin{aligned}\lim_{h\to 0}\frac{1-\cos h}{h} \amp = \lim_{\theta\to 0}\frac{1-\cos2\theta}{2\theta}\amp \leftarrow\text{ let } h=2\theta\\ \amp = \lim_{\theta\to 0} \frac{1-(1-2\sin^2\theta)}{2\theta}\amp \leftarrow\text{ use } \cos2\theta = 1-\sin^2\theta\\ \amp =\lim_{\theta\to 0}\frac{\sin^2\theta}{\theta}\amp \leftarrow\text{ simplify } \\ \amp =\lim_{\theta\to 0}\theta\cdot\frac{(\sin\theta)^2}{\theta^2}\amp \leftarrow\text{ multiply and divide by an extra } \theta\\ \amp =\left(\lim_{\theta\to 0}\theta\right)\cdot\left(\lim_{\theta\to 0}\frac{\sin\theta}{\theta}\right)^2\amp \leftarrow\text{ limit rules } \\ \amp = (0)(1)^2\amp \leftarrow\text{ use that } \lim_{\theta\to0}\frac{\sin\theta}{\theta}=1\\ \amp = 0 \amp \leftarrow\text{ Voila! } \end{aligned} \end{equation*}
    ðŸ”—
    Thus we see that
    \begin{equation*} \lim_{h\to 0}\frac{1-\cos h}{h} = \lim_{h\to 0}-\frac{\cos h-1}{h} = 0 \end{equation*}
    and returning to our calculation of the derivative:
    \begin{equation*} \begin{aligned}\frac{d}{dx}(\sin x) \amp =\sin x\left(\lim_{h\to 0}\frac{\cos h-1}{h}\right)+\cos x\left(\lim_{h\to 0}\frac{\sin h}{h}\right)\amp \leftarrow\text{ where we left off before } \\ \amp = \sin x(0)+\cos x(1)\amp \leftarrow\text{ using the limits we calculated } \\ \amp = \cos x\amp \leftarrow\text{ simplifying } \end{aligned} \end{equation*}
    ðŸ”—
    We can use the sum identity for cosine and the same two limits in order to calculate the derivative of cosine using the limit definition of the derivative:
    \begin{equation*} \begin{aligned}\frac{d}{dx}(\cos x) \amp = \lim_{h\to 0}\frac{\cos (x+h)-\cos x}{h} \amp \leftarrow\text{ definition of the derivative } \\ \amp = \lim_{h\to 0}\frac{(\cos x\cos h-\sin x\sin h)-\cos x}{h}\amp \leftarrow\text{ sum identity for cosine } \\ \amp = \lim_{h\to 0}\frac{\cos x(\cos h-1)-\sin x\sin h}{h}\amp \leftarrow\text{ rearrange and factor } \\ \amp = \lim_{h\to 0}\left(\cos x\frac{\cos h-1}{h}-\sin x\frac{\sin h}{h}\right)\amp \leftarrow\text{ factor more } \\ \amp =\cos x\left(\lim_{h\to 0}\frac{\cos h-1}{h}\right)-\sin x\left(\lim_{h\to 0}\frac{\sin h}{h}\right)\amp \leftarrow\text{ use limit rules and factor again } \\ \amp = \cos x(0)-\sin x(1)\amp \leftarrow\text{ using the limits we calculated } \\ \amp = -\sin x\amp \leftarrow\text{ simplifying } \end{aligned} \end{equation*}
