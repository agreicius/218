    Subsection Rules for Inequalities and Solving Linear Inequalities

    As we have seen over the last few sections, in order to determine when a function is positive and negative, find the domain of a function involving an even root, and rewrite an absolute value, we will need to be able to work with inequalities. We will need to solve inequalities in order to explore functions in general and answer many interesting questions, and so we summarize the relevant rules and develop the necessary techniques for solving inequalities. Let’s begin by defining what we mean by an "inequality."
    🔗
    An inequality consists of two mathematical expressions related by a greater-than \((>)\text{,}\) less-than \((\lt )\text{,}\) greater-than-or-equal-to \((\geq)\text{,}\) or less-than-or-equal-to \((\leq)\) sign. The solution set of an inequality is the set of all values that make the inequality true. Wheres the solution set of an equation often consists of a finite number of points, the solution set of an inequality is often an interval of real numbers.
    🔗

    Example 1.5.1.

    For example, the solution to \(x-1=0\) is the single number \(x=1\text{.}\) The solution to \(x-1>0\) is the interval of numbers \(x>1\text{,}\) or, written using interval notation, \((1,\infty)\text{.}\) This means that if we take any number greater than one and subtract one, the result will be positive (meaning greater than 0).
    🔗
    🔗
    The next theorem describes what happens to inequalities when they are inverted or multiplied by a nonzero constant.
    🔗

    Theorem 1.5.2. Algebraic Rules for Inequalities.

    Suppose \(a, b\text{,}\) and \(c\) are nonzero real numbers.
    1. If \(a\lt b\text{,}\) then \(a+c\lt b+c\text{.}\) In other words, adding a number to both sides of an inequality does not reverse the inequality.
      🔗
      🔗
    2. If \(a\lt b\) and \(c>0\text{,}\) then \(ac\lt bc\text{.}\) In other words, multiplying both sides of an inequality by a positive number does not reverse the inequality.
      🔗
      🔗
    3. If \(a\lt b\) and \(c\lt 0\text{,}\) then \(ac>bc\text{.}\) In other words, multiplying both sides of an inequality by a negative number reverses the inequality.
      🔗
      🔗
    4. If \(0\lt a\lt b\text{,}\) then \(\frac{1}{a}>\frac{1}{b}\text{.}\) In other words, taking reciprocals of two positive numbers reverses the inequality between the numbers.
      🔗
      🔗
    🔗
    🔗
    In particular, notice that part (c) implies that multiplying an inequality by a negative number "flips" the inequality; for example, if \(a\lt b\text{,}\) then \(-a>-b\text{.}\) Part (a) illustrates that we may add any number to, or subtract any number from, both sides of an inequality without changing the direction of the inequality. Theorem 1.5.2 is written in terms of strict inequalities \(>\) and \(\lt\text{,}\) but similar results apply for the inequalities \(\geq\) and \(\leq\text{.}\)
    🔗

    Checkpoint 1.5.3.

    Generate Examples: Create your own examples for parts (a)-(d) of Theorem Theorem 1.5.2 by picking explicit, nonzero number for \(a, b\text{,}\) and \(c\text{.}\) Note that for part (b), \(c\) must be positive and for part (c), \(c\) must be negative (so you will need to pick different values of \(c\) to illustrate those two parts). For part (d), \(a\) and \(b\) must both be positive.
    🔗
    🔗
    We can use the rules in Theorem Theorem 1.5.2 to solve linear inequalities in almost the same way that we solve equalities. The only exception is multiplying (or dividing) by negative numbers, or taking reciprocals.
    🔗

    Example 1.5.4. Solving a simple inequality.

    Find the solution set of the inequality \(3-2x>7\text{.}\)
    🔗
    Solution.
    First, we can use part (a) of Theorem Theorem 1.5.2 to add \(-3\) to both sides, which is the same as subtracting \(3\) from both sides:
    \begin{equation*} \begin{aligned}3-2x \amp >7\\ (3-2x)-3 \amp >7-3\\ -2x\amp >4 \end{aligned} \end{equation*}
    🔗
    Next, we can use part (c) of Theorem Theorem 1.5.2 to multiply both sides by \(-\frac{1}{2}\text{,}\) which is the same as dividing both sides by \(-2\text{.}\) Remember that this requires flipping the inequality!
    \begin{equation*} \begin{aligned}-2x\amp >4\\ -\frac{1}{2}(-2x)\amp \lt -\frac{1}{2}(4) \\ x\amp \lt -2 \end{aligned} \end{equation*}
    🔗
    Therefore the solution set of the original inequality is \((-\infty,-2)\text{.}\)
    🔗
    🔗
    🔗

    Example 1.5.5. Checking the Answer.

    You can check the solution we just found by graphing \(y=3-2x\) (in red) and \(y=7\) (in green), as shown in Figure 1.5.6. The values of \(x\) for which the line described by \(3-2x\) has \(y\)-coordinate greater than \(7\) are those to the left of \(x=-2\text{.}\)
    🔗
    🔗

    Checkpoint 1.5.7.

    Consider the problem:
    🔗
    Determine where the function \(f(x) = 1-3x\) is positive.
    Translate the phrase "\(f(x) = 1-3x\) is positive" into an inequality. Solve the inequality and then check your answer by graphing \(f(x) = 1-3x\) on Desmos or another graphing calculator.
    🔗
    🔗
    \fbox{ Important Note! Since multiplying both sides of an inequality by a negative number "flips" the inequality, you cannot multiply both sides of an inequality by a quantity whose sign you do not know. The following example illustrates a calculation where you must be especially careful about the algebraic steps you follow for precisely this reason. }
    🔗

    Example 1.5.8.

    Suppose we wish to solve \(\frac{3}{x}>2\text{.}\) You cannot multiply both sides of the inequality \(\frac{3}{x}>2\) by \(x\text{,}\) because \(x\) may be positive or negative. Therefore, you do not know whether or not multiplying both sides of an inequality by \(x\) will change the direction of the inequality. We will look at how to solve such an inequality later in this section.

    Subsection Solving Inequalities That Involve Quadratics

    If we want to know where a quadratic function \(f(x) = ax^2+bx+c\) is positive and negative, we need to be able to solve the inequalities \(ax^2+bx+c>0\) and \(ax^2+bx+c\lt 0\text{.}\) However, this comes with significantly more complication than solving a quadratic equality, which we see in the next example.
    🔗

    Example 1.5.9.

    If we try to solve the inequality \(x^2-x\lt 0\) by adding \(x\) to both sides, we get
    \begin{equation*} x^2\lt x\text{.} \end{equation*}
    We may be tempted to divide by \(x\) here, but remember, we do not know whether \(x\) is positive or negative, so we do not know whether to flip the inequality! We must use an alternate method for this problem, which is motivated by Theorem 1.5.10 below.
    🔗
    🔗
    Solving quadratic inequalities is best accomplished by factoring and using a key property of the real numbers: a product of two real numbers is positive if the numbers have the same sign and negative if the numbers have opposite signs. This property is stated carefully in the following theorem:
    🔗

    Theorem 1.5.10. The Sign of a Product depends on the Signs of Its Factors.

    If \(A\) and \(B\) are any real expressions, then
    1. \(AB>0\) if and only if both \(A\) and \(B\) are positive (\(A>0\) and \(B>0\)) or both \(A\) and \(B\) are negative (\(A\lt 0\) and \(B\lt 0\)).
      🔗
      🔗
    2. \(AB\lt 0\) if and only if \(A\) is positive and \(B\) is negative (\(A>0\) and \(B\lt 0\)) or \(A\) is negative and \(B\) is positive (\(A\lt 0\) and \(B>0\)).
      🔗
      🔗
    🔗
    🔗
    An easy corollary (meaning an immediate logical consequence that is interesting on its own) of this theorem is that \(AB\geq 0\) if and only if either \(A\) and \(B\) have the same sign or one of \(A\) and \(B\) is zero.
    🔗
    Theorem 1.5.10 reduces the problem of solving a factored inequality to the problem of solving a collection of related simpler inequalities.
    🔗

    Example 1.5.11.

    For example, the expression in the inequality \(x^2-x\lt 0\) factors as \(x(x-1)\lt 0\text{.}\) The product of \(x\) and \(x-1\) is negative exactly when \(x\) and \(x-1\) have opposite signs. It’s difficult to recognize exactly which values of \(x\) satisfies this condition, but it looks like it depends on whether \(x\) is below 0, between 0 and 1, or above 1. (Note that \(x-1\lt 0\) when \(x\lt 1\text{,}\) and \(x-1>0\) when \(x>1\text{.}\)) In other words, the zeros of \(x^2-x\) seem to break up the intervals where \(x^2-x\) could be positive or negative.
    🔗
    In Table 1.5.12, we look at the sign of each factor of \(x(x-1)\) on the intervals between its zeros in order to determine the sign of the product. If we plug in a sample value of \(x\) on each of these intervals, it becomes easier to determine the signs:
    🔗
    For example, on the interval \(0\lt x\lt 1\text{,}\) we can plug in a sample value of \(x=0.5\text{.}\) Then \(x=0.5\) is positive, \(x-1 = 0.5-1 = -0.5\) is negative, so the product \(x(x-1) = (0.5)(-0.5) = -0.25\) is negative. From this we conclude that \(x(x-1)\) is negative on the entire interval \(0\lt x\lt 1\text{.}\) We will see in a later section why we can use a sample value on this interval to draw a conclusion about the sign on the entire interval.
     5 
    The theorem that tells us this is called the Intermediate Value Theorem.
    🔗
    We can summarize the information from Table 1.5.12 in a sign chart on a number line by making tick marks at each \(x\)-value that made the expression zero and writing \(+\) (for positive) or \(-\) (for negative) above each interval we tested, as shown in Figure 1.5.13. Note that we include tick marks only at the potential sign-changing points.
    🔗
    From our sign analysis, we see that \(x(x-1)\) is positive for \(x\lt 0\) and \(x>1\text{,}\) and negative for \(0\lt x\lt 1\text{.}\) We wished to find the solution to the inequality
    \begin{equation*} x^2-x\lt 0\text{ meaning } x(x-1)\lt 0 \end{equation*}
    and so we see that the solution is values of \(x\) on the interval \(0\lt x\lt 1\text{,}\) which we write using interval notation as
    \begin{equation*} x\in(0,1) \end{equation*}
    🔗
    🔗
    In Example Example 1.5.11 above, we can see that the quadratic function \(q(x) = x^2-x\) { changes sign} at \(x=0\) and \(x=1\text{,}\) where it is equal to zero. Visually, this is where the graph fully crosses the \(x\)-axis:
    🔗
    🔗
    As a rule, quadratic functions (and polynomials in general) are only able to change sign only where their graph crosses the \(x\)-axis:
    🔗

    Theorem 1.5.14. When Can a Polynomial Change Sign?

    A polynomial can change sign only at values where it is equal to zero.
    🔗
    🔗
    This theorem is a special case of the Intermediate Value Theorem, which we will explore in Chapter 2.
    🔗
    Important! Just because a polynomial can change sign does not mean that it does change sign. The following example illustrates this fact.
    🔗

    Example 1.5.15.

    The quadratic function \(q(x) = x^2+2x+1\) factors as \((x+1)(x+1)\) and so has a single zero at \(x=-1\) where it can change sign. We test points above and below \(x=-1\) to determine the sign of the expression by determining the sign of each factor:
    🔗
    Summarizing this on a number line:
    🔗
    We see that despite having an \(x\)-intercept at \(x=-1\text{,}\) \(q(x)=x^2+2x+1\) does not change sign at \(x=-1\text{.}\) \(q(x)\) is positive (meaning \(q(x) = x^2+2x+1>0\)) on the intervals \((-\infty,-1)\cup(-1,\infty)\text{,}\) is zero at \(x=-1\text{,}\) and is never negative. You should graph \(q(x)\) to confirm.

    Subsection Solving Inequalities That Involve Products and Quotients

    Recall Theorem 1.5.10: The Sign of a Product \(AB\) Depends on the Signs of its Factors. This theorem also holds for quotients: \(\frac{A}{B}\) is greater than zero if \(A\) and \(B\) have the same sign and is less than zero if \(A\) and \(B\) have opposite signs. We can also generalize this theorem to products with more than two factors:
    🔗

    Theorem 1.5.22. The Sign of a Product or Quotient Depends on the Number of Its Negative Factors.

    A product or quotient of nonzero real numbers is positive if and only if it has an even number of negative factors and is negative if and only if it has an odd number of negative factors.
    🔗
    🔗
    For example, the product \((-2)(-1)(2)(-6)\) is negative because it has an odd number of negative factors. In contrast, \(\frac{(-3)(-2)(-8)}{(7)(-4)}\) is positive because it has an even number of negative factors.
    🔗

    Example 1.5.23.

    As an illustration of the power of Theorem 1.5.22, consider the problem of solving the inequality
    \begin{equation*} (x^2+2)(x+1)(x-1)\lt 0\text{.} \end{equation*}
    🔗
    Applying Theorem 1.5.26, the expression \((x^2+2)(x+1)(x-1)\) is zero only when \(x=-1\) and \(x=1\text{.}\) (Note that \(x^2+2\) is never zero; it is irreducible.) Therefore, these are the only two \(x\)-values at which the expression can change sign. That means that the expression has the same sign on the entire interval \((-\infty, -1)\text{,}\) and the same sign on all of \((-1,1)\text{,}\) and the same sign on all of \((1,\infty)\text{.}\) Applying Theorem 1.5.22, we can test the sign of each factor and multiply to find the sign of the entire product:

    🔗
    From this, we can conclude that \((x^2+2)(x+1)(x-1)\) is less than zero exactly when \(x\in(-1,1)\text{,}\) and we have solved the inequality.
    🔗
    🔗
    There is also a version of Theorem 1.5.14 that holds for rational functions:
    🔗

    Theorem 1.5.26. When Can a Quotient of Polynomials Change Sign?

    A quotient of polynomials can change sign only at values where its numerator or denominator is equal to zero.
    🔗
    🔗
    The next example applies this theorem to solving an inequality involving a rational function.
    🔗

    Example 1.5.27.

    Determine where
    \begin{equation*} h(x) = \frac{1-2x}{x+1} \end{equation*}
    is positive and negative.
    🔗
    Solution.
    The expression \(\frac{1-2x}{x+1}\) can change sign only when either the numerator or denominator is equal to zero, by Theorem 1.5.26.
    \begin{equation*} \text{ The numerator } 1-2x = 0\text{ when } x=\frac{1}{2} \end{equation*}
    and
    \begin{equation*} \text{ the denominator } x+1 = 0\text{ when } x=-1\text{.} \end{equation*}
    🔗
    Thus we perform a sign analysis on the intervals \((-\infty,-1)\text{,}\) \(\left(-1,\frac{1}{2}\right)\text{,}\) and \(\left(\frac{1}{2},\infty\right)\text{,}\) testing the sign of each factor in the numerator and denominator and combining to determine the sign of the quotient according to Theorem 1.5.22.
    🔗
    Summarizing this on a number line:
    🔗
    So we see that \(h(x) = \frac{1-2x}{x+1}\) is positive on \(\left(-1,\frac{1}{2}\right)\) and negative on \((-\infty,-1)\cup \left(\frac{1}{2},\infty\right)\text{.}\)
    🔗
    🔗
    🔗
    In the next example, we see how to solve an inequality where neither side is 0: we must rewrite the inequality so that one side is 0 and then perform a sign analysis. This requires further simplifying the nonzero expression to be a quotient of fully factored expressions.
    🔗

    Example 1.5.30.

    Solve the inequality
    \begin{equation*} \frac{3}{x+1}\geq 2 \end{equation*}
    🔗
    Solution.
    Although it is tempting, recall that we can’t multiply both sides of the inequality by \(x+1\) because \(x+1\) may be positive or may be negative (depending on the value of \(x\)). Instead, we will use some algebra to turn this inequality into one that involves a product or quotient of factors on one side and zero on the other:
    \begin{equation*} \begin{aligned}\frac{3}{x+1} \amp \geq2 \amp \amp \leftarrow \text{ original inequality } \\ \frac{3}{x+1}-2 \amp \geq 0 \amp \amp \leftarrow \text{ collect terms so 0 is on the right-hand side } \\ \frac{3}{x+1}-\frac{2(x+1)}{x+1}\amp \geq 0 \amp \amp \leftarrow \text{ find common denominator } \\ \frac{3-2(x+1)}{x+1}\amp \geq 0 \amp \amp \leftarrow \text{ combine fractions } \\ \frac{3-2x-2}{x+1}\amp \geq 0 \amp \amp \leftarrow \text{ simplify numerator } \\ \frac{1-2x}{x+1}\amp \geq 0 \amp \amp \leftarrow \text{ simplify numerator some more } \end{aligned} \end{equation*}
    🔗
    Note that none of the above steps has changed the values of \(x\) that can satisfy the inequalities. In other words, the inequality \(\frac{3}{x+1} \geq2\) has the same solution set as the inequality \(\frac{1-2x}{x+1}\geq 0\text{.}\)
    🔗
    Notice that the latter inequality \(\frac{1-2x}{x+1}\geq 0\) is now a problem we have already solved! In Example Example 1.5.27, we found that the function \(h(x) = \frac{1-2x}{x+1}\) is positive for \(x\in \left(-1,\frac{1}{2}\right)\text{.}\) Additionally, the fraction \(\frac{1-2x}{x+1}\) is equal to zero when its numerator is zero, so for \(x = \frac{1}{2}\text{.}\) Thus
    \begin{equation*} \frac{1-2x}{x+1}\geq 0 \text{ for } x\in \left(-1,\frac{1}{2}\right]\text{.} \end{equation*}
    Remembering that the two inequalities have the same solution sets, we realize that we have now solved the original problem, and conclude
    \begin{equation*} \frac{3}{x+1}\geq 2\text{ for } x\in \left(-1,\frac{1}{2}\right] \end{equation*}
    🔗

    Subsubsection Application: Finding the Domain of a Function with an Even Root

    Example 1.5.31.
    Find the domain of the algebraic function
    \begin{equation*} f(x) = \sqrt{\frac{3}{x}-2} \end{equation*}
    🔗
    Solution.
    To find the domain of \(f\text{,}\) we ask which values of \(x\) can be plugged into the equation that defines \(f(x)\) and produce a real number when the expression is simplified. In order for the value of \(f\) to be defined for a real number \(x\text{,}\) that value of \(x\) must make the expression underneath the square-root sign nonnegative (meaning \(\frac{3}{x}-2\geq 0\)) and the denominator appearing in \(\frac{3}{x}\) nonzero (meaning \(x\neq 0\)). Thus the domain of \(f(x)\) is the set of real numbers that satisfy:
    \begin{equation*} \frac{3}{x}-2\geq 0\text{ and } x\neq 0 \end{equation*}
    🔗
    In order to tackle the inequality, we need to rewrite the left-hand side as a single fraction:
    \begin{equation*} \begin{aligned}\frac{3}{x}-2 \amp \geq 0\amp \amp \leftarrow \text{ original inequality } \\ \frac{3}{x}-\frac{2x}{x} \amp \geq 0 \amp \amp \leftarrow \text{ find common denominator } \\ \frac{3-2x}{x}\amp \geq 0 \amp \amp \leftarrow \text{ combine fractions } \end{aligned} \end{equation*}
    🔗
    The fraction can change sign when the numerator is zero:
    \begin{equation*} 3-2x=0\text{ when } x=\frac{3}{2} \end{equation*}
    or when the denominator is zero:
    \begin{equation*} x=0\text{.} \end{equation*}
    🔗
    Performing a sign analysis for \(\frac{3-2x}{x}\text{:}\)
    🔗
    Summarizing this on a number line:
    🔗
    Thus \(\frac{3-2x}{x}\) is positive on \(\left(0,\frac{3}{2}\right)\text{,}\) and we know that it is equal to zero at \(x=\frac{3}{2}\text{;}\) it is undefined at \(x=0\text{.}\) The equivalent inequality \(\frac{3}{x}-2\geq 0\) thus has solution \(\left(0,\frac{3}{2}\right]\text{.}\) Note that the second condition, \(x\neq 0\text{,}\) on the domain is also satisfied.
    🔗
    Hence overall we have found the domain of the function \(f(x) = \sqrt{\frac{3}{x}-2}\) by determining that \(\frac{3}{x}-2\geq 0\) and \(x\neq 0\) on the interval
    \begin{equation*} \left(0,\frac{3}{2}\right] \end{equation*}

    Subsection Summary

    The following terms and methods were introduced in this section:
    🔗
    Key ideas: The solutions of inequalities are intervals of numbers. Multiplying or dividing an inequality by a negative number requires flipping the inequality. The sign of a product or quotient depends on the number of its negative factors. A polynomial can only change sign where it is equal to zero. A quotient of polynomials can only change sign where its numerator or denominator is equal to zero. In order to solve an inequality involving a product or quotient, algebraically manipulate the inequality so that one side is 0, then perform a sign analysis on the nonzero side.
    🔗
    Other ideas reinforced: Finding the domain of a function with an even root can involve solving an inequality. Writing a function involving an absolute value as a piecewise-defined function can involve solving an inequality.
