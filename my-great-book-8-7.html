    Subsection 6.6.2 Defining \(e\) via Derivatives of Exponential Functions

    We have already made some observations about the derivatives of exponential functions \(f(x) = b^x\) without ever having calculated them:
    • When \(b>1\text{,}\) because \(b^x\) models percent growth, \(b^x\) is always increasing, thus its derivative is always positive.
      ðŸ”—
      ðŸ”—
    • When \(0\lt b\lt 1\text{,}\) because \(b^x\) models percent decay, \(b^x\) is always decreasing, thus its derivative is always negative.
      ðŸ”—
      ðŸ”—
    • Because it models percent change,
      â€‰1â€‰
      The change in \(b^x\) depends on its size, so an increasing exponential function will increase faster as \(b^x\) grows, and a decreasing exponential function will decrease more slowly as \(b^x\) decays, both of which translate to a concave up shape.
      \(b^x\) is always concave up, regardless of whether it models exponential growth or decay, thus its second derivative is always positive.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    But how do we actually find a formula for the derivative of \(f(x)=b^x\text{?}\) The power rule tells us that the derivative of \(x^k\) is \(kx^{k-1}\text{,}\) but this rule only works for power functions, where the base is the variable \(x\) and the exponent is a constant \(k\text{.}\) It does not tell us how to differentiate an exponential function like \(2^x\) or \((1.07)^{3t}\text{,}\) where the variable is in the exponent!!!
    ðŸ”—
    To determine the derivative of an exponential function \(f(x) = b^x\text{,}\) we must return to the limit definition of the derivative:
    \begin{equation*} f'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h} \end{equation*}
    ðŸ”—
    Letâ€™s apply this definition to the function \(f(x) = b^x\text{.}\) Note that since the original limit has the form \(\frac{0}{0}\text{,}\) we anticipate needing to use some algebraic manipulation to evaluate the limit.
    \begin{align*} \frac{d}{dx}(b^x) \amp = \lim_{h\to 0}\frac{b^{x+h} - b^x}{h}\amp \leftarrow\amp \text{ apply the definition } f'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h}\\ \amp = \lim_{h\to 0}\frac{b^{x}b^h - b^x}{h}\amp \leftarrow\amp \text{ split up the exponent }\\ \amp = \lim_{h\to 0}\frac{b^{x}(b^h - 1)}{h}\amp \leftarrow\amp \text{ factor out } b^x\\ \amp = \left(\lim_{h\to 0}\ b^x\cdot \frac{b^h - 1}{h}\right) \end{align*}
    ðŸ”—
    Although we have simplified as much as possible, this is a limit that we do not yet know how to calculate. The expression
    \begin{equation*} \frac{b^h - 1}{h} \end{equation*}
    will still evaluate to \(\frac{0}{0}\) as \(h\to 0\) and so is still indeterminate. Letâ€™s numerically estimate (using a table) a few values of this expression as \(h\to0\) to see if we can anticipate what it will be equal to:

    Definition 6.6.6. A Characterization of the Number $e$.

    The number \(e\) satisfies the following limit statement:
    \begin{equation*} \lim_{h\to 0}\frac{e^h-1}{h} = 1\text{.} \end{equation*}
    ðŸ”—
    ðŸ”—
    Therefore, when \(b=e\text{,}\) the limit as \(h\to 0\) of \(\frac{b^h-1}{h}\) in our derivative calculation becomes 1 and "disappears," hence the derivative of the exponential function \(f(x) = e^x\) is:
    \begin{align*} \frac{d}{dx}(e^x) \amp = \lim_{h\to 0}\frac{e^{x+h} - e^x}{h}\amp \amp\\ \amp \vdots\amp \amp\\ \amp = e^x\left(\lim_{h\to 0}\frac{e^h - 1}{h}\right)\\ \amp = e^x(1) \amp \leftarrow\amp \text{ since } \ds\lim_{h\to 0}\frac{e^h-1}{h} = 1\\ \amp =e^x \end{align*}
    ðŸ”—
    We have just shown that the function \(f(x) = e^x\) is its own derivative! As illustrated in FigureÂ 6.6.7 below, \(y=e^x\) is its own associated slope function, meaning that \(y=e^x\) grows in such a way that its height is exactly how fast its height is increasing.

    Other exponential functions of the form \(b^x\) have graphs similar to the graph of \(e^x\text{,}\) but only \(e^x\) is scaled in exactly the right way to be its own derivative. The exponential function \(e^x\) is called the natural exponential function.
    ðŸ”—
    From the point of view of derivatives, the base \(e\) is extremely special. The number \(e\) (named in homage to the great Swiss mathematician Leonard Euler (1707-1783)) is complicated to define. Like \(\pi\text{,}\) \(e\) is an irrational number that cannot be represented exactly by a ratio of integers and whose decimal expansion never repeats. Because of our numerical estimates in TableÂ 6.6.2 and TableÂ 6.6.3, we know that \(e\) must be between \(2\) and \(3\) in order for the slope of the tangent line to \(e^x\) at \(x=0\) to be 1. Here is another definition for the number \(e\text{,}\) but showing that it is equivalent to DefinitionÂ 6.6.6 requires advanced mathematics that is beyond the scope of this course.

    Subsection 6.6.3 Derivatives of Other Exponential Functions

    In order to take derivatives of an arbitrary exponential function \(f(x) = Ab^x\text{,}\) we will need to know that every exponential function can be written so that its base is the number \(e\text{.}\) In order to do this, we note that
    \begin{equation*} b^x = e^{(\log_e b) x} \end{equation*}
    because of the cancellation properties of logarithms (part (f) of TheoremÂ 6.3.9). Using a base of \(e\) in a logarithm is so common that there is a special name and notation for such logarithmic functions: the base-\(e\) logarithmic function is the natural logarithmic function \fbox{\(\log_e x = \ln x\text{.}\)}
    ðŸ”—

    Theorem 6.6.10. Natural Exponential Functions.

    Every exponential function \(f(x) = Ab^x\) can be written in the form
    \begin{equation*} f(x) = Ae^{kx} \end{equation*}
    for \(k=\log_e b =\ln b\text{.}\)
    ðŸ”—
    ðŸ”—
    Similar to when we rewrote functions in terms of a base of \(2\) while exploring doubling time and half-life, when \(b>1\text{,}\) the value of \(k\) is positive, and when \(0\lt b\lt 1\text{,}\) the value of \(k\) is negative:
    ðŸ”—
    With the chain rule, we can use the derivative of \(e^x\) to find the derivatives of general exponential functions.
    ðŸ”—
    The previous discussion showed us that \(\frac{d}{dx}(e^x) = e^x\text{.}\) To find the derivative of \(b^x\text{,}\) we rewrite \(b^x\) as \((e^{\ln b})^x\) and then apply the chain rule:
    \begin{align*} \frac{d}{dx}(b^x) \amp = \frac{d}{dx}((e^{\ln b})^x) \amp \leftarrow \amp \ b = e^{\ln b},\text{ remember that } \ln b = \log_e b\\ \amp = \frac{d}{dx}(e^{(\ln b)x}) \amp \leftarrow\amp \text{ algebra of exponents }\\ \amp = e^{(\ln b) x}\cdot\frac{d}{dx}((\ln b)x) \amp \leftarrow\amp \text{ chain rule and derivative of } e^x\\ \amp = (e^{\ln b})^x(\ln b) \amp \leftarrow \amp \text{ algebra and derivative of linear function }\\ \amp = (\ln b) b^x.\amp \leftarrow \amp \text{ algebra } \end{align*}

    We have now discovered the derivatives of the natural exponential function \(e^x\) along with any base-\(b\) exponential function \(b^x\text{:}\)

    ðŸ”—

    Theorem 6.6.16. The chain rule for exponential functions.

    \begin{equation*} \frac{d}{dx}\left(e^{g(x)}\right) = g'(x)e^{g(x)} \end{equation*}
    \begin{equation*} \frac{d}{dx}\left(b^{g(x)}\right) = (\ln b)g'(x)b^{g(x)} \end{equation*}
    ðŸ”—

    Proof.

    The "outside" function is \(f(u) = e^u\text{,}\) and the "inside" function is \(u=g(x)\text{,}\) so that \(e^{g(x)} = f(g(x))\text{.}\) The chain rule tells us that
    \begin{equation*} \frac{d}{dx}(f(g(x))) = f'(g(x))g'(x) \end{equation*}
    and in this case we have
    \begin{equation*} f'(u) = e^u\text{.} \end{equation*}
    Combining these,
    \begin{equation*} \begin{aligned}\frac{d}{dx}\left(e^{g(x)}\right) \amp = \frac{d}{dx}\left(f(g(x))\right)\\ \amp =f'(g(x))g'(x)\\ \amp = e^{g(x)} g'(x) \amp \amp \text{ since } f'(u) = e^u \end{aligned} \end{equation*}
    ðŸ”—
    Similarly, if \(f(x) = b^x\text{,}\) then \(f'(x) = (\ln b)b^x\) and so
    \begin{equation*} \begin{aligned}\frac{d}{dx}\left(b^{g(x)}\right) \amp = \frac{d}{dx}\left(f(g(x))\right)\\ \amp =f'(g(x))g'(x)\\ \amp = (\ln b)b^{g(x)} g'(x) \amp \amp \text{ since } f'(u) = (\ln b)b^u \end{aligned} \end{equation*}
    ðŸ”—
    ðŸ”—

    Find the derivative of
    \begin{equation*} h(x) = 3\sqrt{e^x}-x^2 \end{equation*}
    ðŸ”—
    There are two possible ways to approach this problem. The first is to recognize that the exponential function is on the inside of a composition, since \(e^x\) appears inside the square root. Thus we have
    \begin{align*} \frac{d}{dx}(3\sqrt{e^x}-x^2) \amp = \frac{d}{dx}(3(e^x)^{1/2}-x^2) \amp \leftarrow\amp \text{ rewrite so differentiation will be easier }\\ \amp = 3\left(\frac{1}{2}\right)(e^x)^{-1/2}(e^x)-2x \amp \leftarrow \amp \text{ don't forget the chain rule! }\\ \amp = \frac{3}{2}e^x(e^x)^{-1/2}-2x \amp \leftarrow\amp \text{ some algebra } \end{align*}
    ðŸ”—
    We could simplify the answer further, but thereâ€™s no need to do that right now. Here is another, arguably better way to approach this computation: we could have saved ourselves some differentiation work by doing some algebra first, as is often the case. In particular, we can combine the exponent of the exponential function with the square root:
    \begin{align*} \frac{d}{dx}(3\sqrt{e^x}-x^2) \amp = \frac{d}{dx}(3(e^x)^{1/2}-x^2) \amp \leftarrow \amp \text{ rewrite so differentiation will be easier }\\ \amp =\frac{d}{dx}(3e^{x/2}-x^2) \amp \leftarrow\amp \text{ combine exponents of \(e\) }\\ \amp = 3\left(\frac{1}{2}\right)e^{x/2}-2x\amp \leftarrow\amp \text{ differentiation is easier now } \end{align*}
    ðŸ”—
    The answer is equivalent to the answer we got in the first calculation. (Do some algebra to see why.) Notice that the differentiation steps used in the second calculation were easier than the differentiation steps used in the first calculation.

    Subsection 6.6.4 Summary

    The following terms were introduced in this section:
    ðŸ”—
    \(e\text{,}\)\(e^x\text{,}\)\(\ln x = \log_e x\)
