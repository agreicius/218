    Subsection Definitions of linear and quadratic functions

    Linear and quadratic functions are important for modeling relationships between many different quantities. For example, in statistics, the method of Linear Regression seeks to find a line that best describes the relationship between quantities that appear to change at a constant rate relative to each other. Quadratic functions are used to model the height of an object falling under the force of gravity. We will explore several models that use linear and quadratic functions in SectionÂ 1.4 after we develop our mathematical understanding of them.
    ðŸ”—
    We begin with the mathematical definitions of linear and quadratic functions. Recall that a mathematical definition is "used to give a precise meaning to a new term, by describing a condition which unambiguously qualifies what a mathematical term is and is not."
    â€‰8â€‰
    https://en.wikipedia.org/wiki/Definition accessed August 4, 2022.
    Thus the definitions below give us the precise meaning of a linear or quadratic function. Remember that we generally need a few examples to make sense of definitions, and these follow below each of the definitions.
    ðŸ”—

    Definition 1.2.1. Linear Functions.

    A linear function is a function that can be written in the form \(f(x) = mx+b\) for some real numbers \(m\) and \(b\text{.}\) Alternately, a linear function can be written as a linear equation \(y=mx+b\text{,}\) where \(y\) is a dependent variable of the independent variable \(x\text{.}\)

    Definition 1.2.3. Quadratic Function.

    A quadratic function is one that may be written in the form
    \begin{equation*} f(x)=ax^2+bx+c\text{,} \end{equation*}
    where \(a, b\text{,}\) and \(c\) are real numbers with \(a\neq 0\text{.}\) Alternately, a quadratic function can be written as a quadratic equation
    \begin{equation*} y=ax^2+bx+c\text{,} \end{equation*}
    where \(y\) is a dependent variable of the independent variable \(x\text{.}\)

    Subsubsection Parameters

    It is important to note that when we write an expression like \(f(x) = mx+b\) or \(f(x) = ax^2+bx+c\text{,}\) only \(x\) and \(y\) are variables. The extra letters \(a, b, c\text{,}\) and \(m\) that are not variables are called parameters of the function. How should you think about these parameters and what they represent?
    ðŸ”—
    While the word parameter may be new to you, the concept is probably not so unfamiliar. We can think of a parameter as a "dial" on the function machine. Setting the dials to different values creates different functions.

    The table that follows gives us vocabulary and precise mathematical definitions for various types of graphical behavior. Rows 1, 2, 5, 6, and 8 describe behaviors that a function could exhibit at a specific point. Rows 3, 4, and 7 describe graphical behaviors that occur over an interval \(I\) of real numbers.

    Remark 1.2.10.

    TableÂ 1.2.11 uses \(x\) as the independent variable and \(y\) as the dependent variable. The notation \(x_1, x_2, y_1,\ldots\) is used to indicate particular values of \(x\) and \(y\text{.}\) We are using these instead of other letters because we have been using other letters to indicate parameters. A subscript is called an "index" (plural "indices"). The benefit to using these is that it is easier to keep track of whether they represent a value of the independent variable (for example, \(x=x_1\) or \(x=x_2\)) or the dependent variable (for example, \(y=y_1\) or \(y=y_2\)).

    There are similar definitions for negative, decreasing, and concave down behavior, as well as local and global minima.
    ðŸ”—
    Notice that we describe extrema (maxima and minima) by where on the \(x\)-axis they occur, since we can always find the corresponding \(y\)-values from these \(x\)-values. The concept of "near" in the description of a local maximum will be made more precise in Chapters 2 and 3. Inflection points and concavity cannot be precisely defined until we learn about derivatives in Chapter 3. In that chapter we will also learn ways for algebraically calculating the locations of extrema and inflection points. For now we present them just to set terminology and to familiarize ourselves with various types of function behavior; we will have to be content with examining such things graphically.

    Subsection Different Equations of Lines

    A line in the plane is completely described by its slope \(m\) and \(y\)-intercept \(b\text{.}\) You could tell someone the values of \(m\) and \(b\) in a text message and they would know exactly which line you were referring to, without needing to attach a picture. We can also describe a line in the plane with its slope and the coordinates of any point on the line, or with the coordinates of any two points on the line.
    â€‰15â€‰
    This last is how Euclidâ€™s Elements begin to develop geometry. Book I Postulate 1 states that given any two points, there is a line that joins them. https://mathcs.clarku.edu/Â djoyce/elements/bookI/post1.html accessed August 17, 2022.
    ðŸ”—
    These three ways of describing lines give us three forms of equations for lines found in TableÂ 1.2.18: slope-intercept form, point-slope form, and two-point form. Note that here we are again using indices to represent different values of the variables \(x\) and \(y\text{.}\) \(x_0, x_1\text{,}\) and \(x_2\) represent values of the independent variable \(x\text{,}\) while \(y_0, y_1\text{,}\) and \(y_2\) represent values of the dependent variable \(y\text{.}\)
    ðŸ”—
    There is an advantage to slope-intercept form in that we can tell from looking at any two functions in that form whether they represent the same line. However, the other forms are sometimes easier to use based on our given information.
    ðŸ”—
    Letâ€™s explore the point-slope and two-point forms a bit more. The slope \(m\) of a line can be interpreted as how much the line "rises" (increases or decreases) given a "run" of one unit increase in \(x\text{.}\) Thus the slope can be calculated by the formula \fbox{\(m =\frac{\text{ rise } }{\text{ run } }=\dfrac{y_2-y_1}{x_2-x_1}\)} for any two points \((x_1,y_1)\) and \((x_2,y_2)\) on the line. From this we see that the two-point form of the equation of a line is the same as the point-slope form, as long as we calculate the slope \(m\) from the two points ahead of time. Note that when we do that, we may use either point \((x_1,y_1)\) or \((x_2,y_2)\) to determine the equation, but we must stick with one or the other.
    ðŸ”—

    Subsection Finding \(x\)-intercepts of Quadratic Functions

    In this section, we develop algebraic methods for finding the \(x\)-intercepts of quadratic functions. This is where the graph of the function crosses the \(x\)-axis. As we can see in FigureÂ 1.2.22, by shifting the graph of a quadratic function vertically, we can make its graph cross the \(x\)-axis 0 times (as in the graph labeled \(p\)), exactly 1 time ( as in the graph labeled \(q\)), or twice (as in the graph labeled \(r\)). These points are the \(x\)-intercepts, sometimes referred to as zeros of the graph, because these are the points where the quadratic function is equal to 0.
    ðŸ”—
    If we want to find the zeros of a quadratic equation, we must find the values of \(x\) that make the quadratic function \(f(x) = ax^2+bx+c\) equal to zero, meaning we must solve solve the quadratic equation \(ax^2+bx+c=0\) for the variable \(x\text{.}\) Some quadratic equations can be solved easily by factoring the quadratic expression \(ax^2+bx+c\text{.}\) To factor an algebraic expression, you write that expression as a product of simpler expressions. There are a few different common methods for factoring that we will discuss here:
    ðŸ”—

    Example 1.2.23.

    For example, the expression \(3x^2-6x\) can be factored by recognizing that both terms share common factors of 3 and \(x\) that can be un-distributed from the expression:
    \begin{equation*} 3x^2-6x = 3x\cdot x-3x\cdot 2 = 3x(x-2) \end{equation*}
    ðŸ”—
    We can think of this factoring as "reverse distributing."
    ðŸ”—
    ðŸ”—

    Theorem 1.2.26. Formulas for Factoring Differences of Squares.

    For all real numbers \(n\) and \(m\text{,}\)
    \begin{equation*} n^2-m^2 = (n-m)(n+m) \end{equation*}

    Example 1.2.27.

    The reason that factoring can help solve an equation is that a basic property of the real numbers is that a product can be equal to zero only if one of the factors in the product is zero. In the following theorem we use the term real expressions, which will denote any expressions that when calculated are real numbers.
    ðŸ”—

    Theorem 1.2.28. A Product Is Zero If and Only If One of the Factors Is Zero.

    For any real expressions \(A\) and \(B\text{,}\) \(AB=0\) if and only if \(A=0\) or \(B=0\text{.}\)

    Remark 1.2.31.

    If you try to solve \(x^2-4=0\) by adding 4 to both sides and taking the square root of both sides, you may think that there is only one solution when \(x=2\text{.}\) For this reason, factoring is a more reliable method for solving this type of equation!     The phrase "if and only if" in TheoremÂ 1.2.28 is shorthand for saying that the implication goes both ways. The forward direction of this implication says that if \(AB\) is zero, then either \(A\) or \(B\) (or both) is zero. the backwards direction says that if \(A\) or \(B\) is zero, then \(AB\) is also zero. This theorem can be generalized to products involving more than two factors; for example, a product \(ABCD=0\) if and only if one or more of the factors \(A\text{,}\) \(B\text{,}\) \(C\text{,}\) and \(D\) is zero.
    ðŸ”—
    Warning! TheoremÂ 1.2.28 applies only to equations that are set to zero, that is, equations of the form \(AB=0\text{.}\) Theorem does not say, for example, that \(3x(x-2)=2\) if and only if \(3x=2\) or \(x-2=2\text{.}\)

    Example 1.2.32.

    To find the solutions of the equation \(x^2-3x=4\text{,}\) we CANNOT factor the left-hand side and then set each factor equal to a divisor of \(4\text{.}\) For example, the following calculation is INCORRECT:
    \begin{equation*} \begin{aligned}x^2-3x \amp = 4\\ x(x-3) \amp = 2\cdot 2\\ x=2 \amp \text{ or } x-3=2\\ x=2 \amp \text{ or } x=5 \end{aligned} \end{equation*}
    ðŸ”—
    However, when \(x=2\text{,}\)
    \begin{equation*} x^2-3x = (2)^2-3(2) = 4-6=-2\neq 4\text{,} \end{equation*}
    and when \(x=5\text{,}\)
    \begin{equation*} x^2-3x = (5)^2-3(5) = 25-15 = 10\neq 4\text{.} \end{equation*}
    ðŸ”—
    The correct method is to first subtract the 4 from both sides so that the right side is 0, and then factor the left-hand side:
    \begin{equation*} \begin{aligned}x^2-3x \amp = 4\\ x^2-3x-4 \amp = 0\\ (x-4)(x+1) \amp = 0\\ x-4 = 0 \amp \text{ or } x+1 = 0\\ x = 4 \amp \text{ or } x = -1 \end{aligned} \end{equation*}
    ðŸ”—
    You should check that \(x^2-3x = 4\) when either \(x=4\) or \(x=-1\) by plugging in those values and simplifying.
