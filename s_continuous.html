    Objectives

    In this section we explore the notion of continuity of a function. The definition of continuity is most clearly stated in terms of limits: a function \(f\) is continuous at a point \(a\) in its domain if its limit \(\lim\limits_{x\to a}f(x)\) exists there and is equal to its value \(f(a)\text{.}\) Graphically, this means, roughly speaking that the graph of \(f\) has no untoward behavior at \(a\text{:}\) e.g., there are no jumps or asymptotes at \(a\text{,}\) nor any wild oscillations near \(a\text{.}\) As we will see, the property of being continuous at points has many important and useful consequences, including the famous Intermediate value theorem (IVT).

    Subsection Continuous functions

    Before getting to the official definition of a continuity, we quickly indulge in some technicalities around the notions of interior points and endpoints of domains.
    ðŸ”—

    Definition 1.15.1. Interior points and endpoints.

    Let \(f\) be a function with domain \(D\text{.}\)
    • Interior point.
      An element \(a\in D\) is an interior point of \(D\) if \(D\) contains an open interval that contains \(a\text{:}\) i.e., \(a\in (c,d)\) and \((c,d)\subseteq D\text{.}\)
      ðŸ”—
      ðŸ”—
    • Endpoints.
      An element \(a\in D\) is a left endpoint of \(D\) if it is not an interior point of \(D\) and \(D\) contains an interval of the form \([a,d)\text{.}\)
      ðŸ”—
      An element \(a\in D\) is a right endpoint of \(D\) if it is not an interior point of \(D\) and \(D\) contains an interval of the form \((c,a]\text{.}\) with \(a\in (c,d)\text{.}\)
      ðŸ”—
      An element \(a\in D\) is an endpoint of \(D\) if it is either left or right endpoint.
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—

    Definition 1.15.2. Continuity.

    Let \(f\) be a function with domain \(D\text{,}\) and suppose \(a\in D\) is either an interior point or an endpoint of \(D\text{.}\) The function \(f\) is continuous at \(a\) if the limit of \(f\) at \(a\) exists and is equal to the value of \(f\) at \(a\text{:}\) i.e., if
    1. \(\lim\limits_{x\to a}f(x)\) exists, and
      ðŸ”—
      ðŸ”—
    2. \(\lim\limits_{x\to a}f(x)=f(a)\text{.}\)
      ðŸ”—
      ðŸ”—
    The function \(f\) is discontinuous at \(a\) if it is not continuous there.
    ðŸ”—
    For a subset \(I\subseteq D\text{,}\) we say \(f\) is continuous on \(I\) if \(f\) is continuous at all interior or endpoint points of \(D\) lying in \(I\text{.}\) We say \(f\) is continuous if it is continuous on its entire domain.
    ðŸ”—
    ðŸ”—

    Remark 1.15.3. Continuity at \(a\).

    Let \(f\) be a function with domain \(D\text{.}\) Observe that our definition of continuity (and discontinuity) only applies to elements of the domain. (In fact, it only applies to elements of the domain that are either interior points or endpoints.)
    ðŸ”—
    For example, consider the function \(f(x)=1/x\) with domain
    \begin{equation*} D=\R-\{0\}=(-\infty, 0)\cup (0,\infty)\text{.} \end{equation*}
    Since \(0\) is not an element of the domain of \(f\text{,}\) we do not say that \(f\) is continuous or discontinuous at \(0\text{.}\)
    ðŸ”—
    ðŸ”—

    Remark 1.15.4. Endpoints.

    Let \(f\) be a function with domain \(D\text{,}\) and let \(a\in D\) be a right endpoint of \(D\text{.}\) Since by definition \(D\) contains an interval of the form \((c,a]\) but not a full open interval containing \(a\text{,}\) we are in a situation where only the left limit of \(f\) is defined at \(a\text{.}\) Recall (DefinitionÂ 1.11.16) that we define
    \begin{equation*} \lim_{x\to a}f(x)=\lim_{x\to a^-}f(x) \end{equation*}
    in this case. A similar remark applies to left endpoints of \(D\text{.}\)
    ðŸ”—
    ðŸ”—

    Example 1.15.5. Continuity at endpoint.

    Let \(f(x)=\sqrt{x}\text{,}\) with domain \(D=[0,\infty)\text{.}\) Decide whether \(f\) is continuous at \(0\text{.}\)
    ðŸ”—
    Solution.
    First observe that since \(0\) is a left endpoint of \(D\text{,}\) we define
    \begin{equation*} \lim_{x\to 0}f(x)=\lim_{x\to 0^+}f(x)\text{.} \end{equation*}
    We compute
    \begin{align*} \lim_{x\to 0^+}f(x) \amp \lim_{x\to 0^+}\sqrt{x} \\ \amp \sqrt{\lim_{x\to 0^+}x} \amp \text{(root rule)}\\ \amp = \sqrt{0} \amp \text{(poly. eval.)}\\ \amp = 0\text{.} \end{align*}
    Thus \(\lim\limits_{x\to 0}f(x)\) exists and is equal to 0. Since \(f(0)=\sqrt{0}=0\text{,}\) we see further that
    \begin{equation*} \lim_{x\to 0^+}f(x)=\lim_{x\to 0^+}f(x)=f(0)=0 \end{equation*}
    and thus that \(f\) is continuous at \(0\text{.}\)

    ðŸ”—
    ðŸ”—

    Remark 1.15.6. Continuity: graphical.

    When scanning the graph of a function \(f\) for points of continuity or discontinuity, we look for â€œproblem pointsâ€. These are locations on the graph where either (a) the limit of the function appears not to exist, or (b) where the limit does exist but does not agree with the value of the function.
    ðŸ”—
    A well-loved adage in this regard is the phrase â€œa continuous function is one whose graph can be traced without lifting the pencilâ€. This is cute, and helpful for guidance, but should not be taken too literally. Indeed, there are some pretty wacky continuous functions out there. For example, the function
    \begin{equation*} f(x)=\begin{cases} x\sin(1/x)\amp \text{if } x> 0\\ 0 \amp \text{if } x=0 \end{cases} \end{equation*}
    is continuous everywhere on its domain, and yet there are some peculiar properties of its graph that make it unclear it what sense it could be â€œtraced by a pencilâ€. (Not having defined the sine function yet, we have gotten a little ahead of ourselves. The reader is invited to graph the function above using technology.)

    Example 1.15.9. Continuous everywhere.

    Let \(\displaystyle f(x)=\frac{x}{\sqrt{x^2-1}}\text{.}\) Prove, using DefinitionÂ 1.15.2 that \(f\) is continuous.
    ðŸ”—
    Solution.
    Let \(a\) be any interior element in the implied domain of \(f\text{.}\) By definition of this domain, this means \(a^2-1> 0\text{.}\) It follows that
    \begin{align*} \lim_{x\to a}f(x) \amp =\lim_{x\to a}\frac{x}{\sqrt{x^2-1}}\\ \amp =\frac{\lim_{x\to a}x}{\lim_{x\to a}\sqrt{x^2-1}}\\ \amp =\frac{a}{\sqrt{\lim_{x\to a}x^2-1}}\\ \amp = \frac{a}{\sqrt{a^2-1}}\\ \amp = f(a)\text{.} \end{align*}
    This computation shows both that (i) \(\lim\limits_{x\to a}f(x)\) exists, and (ii) \(\lim\limits_{x\to a}f(x)=f(a)\text{,}\) and thus by DefinitionÂ 1.15.2, that \(f\) is continuous at \(a\text{.}\) Since \(a\) was an arbitrary element of the domain of \(f\text{,}\) we conclude that \(f\) is continuous at all points of its domain, and hence continuous.

    Next we state some theorems that facilitate our (non-graphical) analysis of continuity for a given function. The first result establishes the continuity of some familiar families, proving us with a basic collection of continuous functions.

    Theorem 1.15.10. Continuous functions.

    1. Absolute value.
      The function \(f(x)=\abs{x}\) is continuous everywhere on its domain. In other words, the absolute value function is a continuous function.
      ðŸ”—
      ðŸ”—
    2. Polynomials and rational functions.
      If \(f\) is either a polynomial or a rational function, then \(f\) is continuous everywhere on its domain. In other words, polynomials and rational functions are continuous functions.

    By contrast, our next two theorems give us rules for building new continuous functions out of old ones, allowing us to build up our collection of continuous functions.
    ðŸ”—

    Theorem 1.15.11. Continuity rules.

    Assume \(f\) and \(g\) are both continuous at the point \(a\text{.}\)
    1. Linear combination.
      For all constants \(c,d\in \R\text{,}\) the function \(h(x)=cf(x)+g(x)\) is continuous at \(a\text{.}\)
      ðŸ”—
      ðŸ”—
    2. Product.
      The function \(h(x)=f(x)g(x)\) is continuous at \(a\text{.}\)
      ðŸ”—
      ðŸ”—
    3. Quotient.
      Assume further that \(g(a)\ne 0\text{.}\) The function \(h(x)=f(x)/g(x)\) is continuous at \(a\text{.}\)
      ðŸ”—
      ðŸ”—
    4. Power.
      Let \(n\) be a positive integer. The function \(h(x)=[f(x)]^n\) is continuous at \(a\text{.}\)
      ðŸ”—
      ðŸ”—
    5. Root rule.
      Let \(n\) be a positive integer. If \(n\) is even, assume further that \(f(a)\geq 0\text{.}\) The function \(h(x)=\sqrt[n]{f(x)}\) is continuous at \(a\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—

    Theorem 1.15.12. Continuity composition rule.

    Assume \(f\) and \(g\) are functions, and suppose the range of \(g\) is a subset of the domain of \(f\text{.}\) If \(\lim\limits_{x\to a}g(x)=b\text{,}\) and if \(f\) is continuous at \(b\text{,}\) then \(\lim\limits_{x\to a}f(g(x))=f(b)\text{.}\)
    ðŸ”—
    It follows that if \(g\) is continuous at \(a\text{,}\) then the composition \(f\circ g\) is continuous at \(a\text{.}\)
    ðŸ”—
    ðŸ”—
    Whereas TheoremÂ 1.15.10 establishes the continuity of some basic families of functions, TheoremÂ 1.15.11 and TheoremÂ 1.15.12 tell us that continuity is preserved by all of our familiar function operations (sum, product, quotient, composition, etc.)! As a result, just about any function we build from our starting collection of polynomials and rational functions, is continuous on its entire domain. Since the set family of functions that can be build up from polynomials in this way is precisely the set of algebraic functions, we conclude that all algebraic functions are continuous!
    ðŸ”—

    Theorem 1.15.13. Algebraic functions are continuous.

    All algebraic functions are continuous. In more detail, if \(f\) is an algebraic function with domain \(D\text{,}\) then \(f\) is continuous at all interior points and endpoints of \(D\text{.}\)
    ðŸ”—
    ðŸ”—
    By definition of continuity, \(f\) is continuous at \(a\) if and only if \(\lim\limits_{x\to a}f(x)=f(a)\text{.}\) Now that we know that any algebraic functions is continuous everywhere on their domain, we obtain a sort of super-powered evaluation formula generalizing TheoremÂ 1.12.6: namely, the algebraic evaluation formula.
    ðŸ”—

    Theorem 1.15.14. Algebraic function evaluation.

    If \(f\) is an algebraic function with domain \(D\text{,}\) then
    \begin{equation} \lim\limits_{x\to a}f(x)=f(a)\tag{1.80} \end{equation}
    for all \(a\in D\text{.}\)
    ðŸ”—
    ðŸ”—

    Subsection Intermediate value theorem

    Recall the cute informal description of a continuous function as one whose graph can be traced without lifting your pencil. This can be further interpreted as saying that as we trace a segment of the graph lying over some interval in the \(x\)-axis, it should not be possible to â€œskipâ€ over a particular \(y\)-value. Indeed, to skip this value, we would have to lift our pencil. This seems plausible, starting from our cutesy informal description of continuity, but can we prove it rigorously? Yes, we can as witnessed by the intermediate value theorem.

    Theorem 1.15.18. Intermediate value theorem (IVT).

    Let \(f\) be continuous on the closed interval \([a,b]\text{.}\) If \(f(a)\ne f(b)\text{,}\) then given any value \(d\in \R\) lying strictly between \(f(a)\) and \(f(b)\text{,}\) there is an element \(c\in (a,b)\) such that \(f(c)=d\text{.}\)

    Remark 1.15.19. IVT and existence of solutions.

    Remark 1.15.22. Sign diagram technique.

    At long last we are able to justify TheoremÂ 1.8.11, which was the basis of ProcedureÂ 1.8.12, our sign diagram technique for solving inequalities. The crucial claim was that given an algebraic function \(f\) and an interval \(I\) on which \(f\) is defined and never equal to zero, \(f\) is either always positive or always negative on \(I\text{.}\) Letâ€™s see how the intermediate value theorem implies this fact.
    ðŸ”—
    Assume \(f\) is an algebraic function, and that the interval \(I\) is a subset of the domain of \(f\text{:}\) i.e., \(f\) is defined at all points in \(I\text{.}\) If \(f\) assumes both positive and negative values on \(I\text{,}\) then we can find points \(a,b\in I\) such that \(a< b\text{,}\) and \(f(a)\) and \(f(b)\) have different signs (i.e., one is positive and the other is negative). But since \(f\) is algebraic, it is continuous, which means in particular that it is continuous at all points in \(I\text{,}\) and hence also on the interval \([a,b]\subseteq I\text{.}\) We now apply the intermediate value theorem: since the signs of \(f(a)\) and \(f(b)\) are different, \(0\) lies between \(f(a)\) and \(f(b)\text{,}\) and thus there is a \(c\in (a,b)\) such that \(f(c)=0\text{.}\) We have shown that if an algebraic function changes sign on an interval in its domain, then it must have a zero in that interval. Equivalently, an algebraic function cannot change sign on an interval in its domain on which it is never equal to zero.
