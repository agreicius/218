    Section 2.10 One-to-one and invertible functions

    Objectives

    1. Use algebraic and geometric methods to determine when a function is one-to-one and invertible and how to deduce information about its inverse.
      üîó
      üîó
    2. Understand inverse functions as a ‚Äúreversing‚Äù or ‚Äúundoing‚Äù operation.
      üîó
      üîó
    3. Compute formulas for inverse functions.
      üîó
      üîó
    4. Determine the graph of an inverse function from the graph of the original function.
      üîó
      üîó
    üîó

    Subsection One-to-one functions

    We saw in Theorem¬†2.8.21 that exponential growth functions are increasing on their entire domain, and that exponential decay functions are decreasing. As a result, an exponential function \(f\) satisfies an important property: given two distinct inputs \(x_1\) and \(x_2\) (i.e., \(x_1\ne x_2\)), we the outputs \(f(x_1)\) and \(f(x_2)\) are distinct (i.e., \(f(x_1)\ne f(x_2)\)). Why is this true? Take the case where \(f\) is an exponential growth function, so that in particular it is increasing on its domain. If \(x_1\ne x_2\text{,}\) then one of the two inputs is greater than the other. Without loss of generality, we may assume \(x_1< x_2\text{.}\) But then since \(f\) is increasing, we have \(f(x_1)< f(x_2)\text{,}\) which implies \(f(x_1)\ne f(x_2)\text{.}\)
    üîó
    A function that satisfies this property of sending distinct inputs to distinct outputs is called one-to-one, as we now make official.
    üîó

    Definition 2.10.1. One-to-one function.

    A function \(f\) is one-to-one on its domain \(D\) if \(f(x_1)\ne f(x_2)\) for all \(x_1, x_2\in D\) with \(x_1\ne x_2\text{.}\) Using logical shorthand:
    \begin{align} x_1\ne x_2 \amp \implies f(x_1)\ne f(x_2)\text{,}\tag{2.19} \end{align}
    or equivalently, using the contrapositive,
    \begin{align} f(x_1)=f(x_2) \amp \implies x_1=x_2\text{.}\tag{2.20} \end{align}
    Given a subset \(I\subseteq D\text{,}\) we say that \(f\) is one-to-one on \(I\) if the implication (2.19) holds for all \(x_1, x_2\in I\text{.}\)
    üîó
    üîó
    Before proceeding in our investigation of one-to-one functions, we hasten to point out that not all functions satisfy this property, as the next example illustrates.
    üîó

    Example 2.10.2. One-to-one functions.

    Use algebraic methods to determine whether the function \(f\) with given domain \(D\) is one-to-one.
    1. \(f(x)=3x+5\text{,}\) \(D=\R\)
      üîó
      üîó
    2. \(f(x)=x^2\text{,}\) \(D=\R\)
      üîó
      üîó
    3. \(f(x)=x^2\text{,}\) \(D=[0,\infty)\)
      üîó
      üîó
    üîó
    Solution.
    1. The function \(f(x)=3x+5\) is one-to-one on its domain \(D=\R\text{.}\) We have
      \begin{align} f(x_1)=f(x_2) \amp \implies 3x_1+5=3x_2+5 \tag{2.21}\\ \amp \implies 3x_1=3x_2 \tag{2.22}\\ \amp \implies x_1=x_2 \tag{2.23} \end{align}
      for all \(x_1,x_2\in \R\text{.}\)
      üîó
      üîó
    2. The function \(f(x)=x^2\) is not one-to-one on its domain \(D=\R\text{,}\) since \(-1\ne 1\text{,}\) and yet
      \begin{align*} f(-1) \amp = (-1)^2\\ \amp = 1\\ \amp = 1^2\\ \amp = f(1)\text{.} \end{align*}
      üîó
      üîó
    3. The function \(f(x)=x^2\) with domain \(D=[0,\infty)\text{,}\) by contrast, is one-to-one! To see why we examine more carefully what it means for \(f(x_1)=f(x_2)\) for arbitrary elements \(x_1,x_2\in \R\text{.}\) We have
      \begin{align*} f(x_1)=f(x_2) \amp \iff x_1^2=x_2^2 \\ \amp \iff x_1^2-x_2^2=0\\ \amp \iff (x_1-x_2)(x_1+x_2)=0\\ \amp \iff x_1=x_2 \text{ or } x_1=-x_2 \text{.} \end{align*}
      Thus \(f(x_1)=f(x_2)\) if and only if \(x_1=\pm x_2\text{.}\) It follows that given distinct elements \(x_1,x_2\in D=[0,\infty)\text{,}\) we cannot have \(f(x_1)=f(x_2)\text{,}\) because we cannot have \(x_1=-x_2\text{.}\)
      üîó
      üîó
    üîó
    üîó
    üîó

    Remark 2.10.3. Importance of domain.

    As the previous illustrates, the role of the domain \(D\) of a function is essential in determining whether the function is one-to-one or not. As we saw there, the function \(f(x)=x^2\) with domain \(D=\R\) is not one-to-one (e.g., \(f(-1)=f(1))\text{,}\) but it is one-to-one on the subset \(I=[0,\infty)\text{.}\)
    üîó
    üîó
    As the next example illustrates, knowing that a function is one-to-one allows us to solve certain equations more easily.
    üîó

    Example 2.10.4. Algebra with one-to-one functions.

    Find all \(x\) satisfying \(2^{3x}=2^{x^2-10}\)
    üîó
    Solution.
    To use properties of one-to-one functions, we first observe that the equation can be expressed as
    \begin{align*} f(3x) \amp =f(x^2-10) \end{align*}
    where \(f(x)=2^x\text{.}\) Since \(f\) is exponential, we know it is one-to-one on its domain \(\R\text{.}\) Thus we have
    \begin{align*} 2^{3x}=2^{x^2-10}\amp \iff f(3x)=f(x^2-10)\\ \amp \iff 3x=x^2-10 \amp (f \text{ one-to-one}) \\ \amp \iff x^2-3x-10=0\\ \amp \iff (x-5)(x+2)=0\\ \amp \iff x=5,-2\text{.} \end{align*}
    Thus the solutions to the equation are \(x=5\) and \(x=-2\text{.}\)
    üîó
    üîó
    üîó
    In Example¬†2.10.2 we used algebra to decide whether a given function was one-to-one. The horizontal line test provides a convenient graphical method for doing the same. The simple principle of this test can be summarized as follows.
    • If \(f\) is not one-to-one, then we can find two distinct inputs \(x_1\) and \(x_2\) whose outputs are equal: i.e., \(f(x_1)=f(x_2)\text{.}\)
      üîó
      üîó
    • Graphically, this means that the points \(P=(x_1,f(x_1))\) and \(Q=(x_2,f(x_2))\) on the graph of \(f\) are distinct (they have different \(x\)-coordinates), but have the same \(y\)-coordinate \(y_0=f(x_1)=f(x_2)\text{.}\)
      üîó
      üîó
    • As a result, the horizontal line \(y=y_0\) intersects the graph of \(f\) in at least two distinct points.
      üîó
      üîó
    • Conversely, if some horizontal line \(y=y_0\) intersects the graph of \(f\) in at least two distinct points \(P=(x_1,f(x_1))\) and \(Q=(x_2,f(x_2))\text{,}\) then we must have \(f(x_1)=f(x_2)=y_0\text{,}\) showing that \(f\) is not one-to-one.
      üîó
      üîó
    üîó

    Theorem 2.10.5. Horizontal line test.

    Let \(\mathcal{C}\subseteq \R^2\) be the graph of a function \(f\text{.}\)
    üîó
    If for all \(c\in \R\text{,}\) the horizontal line \(y=c\) intersects \(\mathcal{C}\) in at most one point, then \(f\) is one-to-one.
    üîó
    Conversely, if there is horizontal line \(y=c\) that intersects \(\mathcal{C}\) at more than one point, then \(f\) is not one-to-one.
    üîó
    üîó

    Remark 2.10.6. Horizontal versus vertical line test.

    The horizontal line test probably reminds you of another line test introduced long ago in this course: namely, the vertical line test (Theorem¬†1.1.36). Make sure you do not conflate the two! Recall that the vertical line test is used to determine whether a given curve in the plane was in fact the graph of a function. By contrast, when using the horizontal line test, we know already that the curve \(\mathcal{C}\) is the graph of a function \(f\text{,}\) and wish to ascertain whether that function is one-to-one or not.
    üîó
    üîó

    Remark 2.10.7. Horizontal line test.

    Similarly to the vertical line test, it might be useful to envision the horizontal line test as a scanning procedure. Imagine scanning your graph \(\mathcal{C}\) with a device that produces a green horizontal line \(L\) that we move up and down across the entire plane.
    • If at some moment during the scan our horizontal line \(L\) hits two or more points, then our scanner emits a loud error beep, indicating that \(f\) has failed the horizontal line test, and is thus is not one-to-one.
      üîó
      üîó
    • If, on the other hand, we are able to scan the entire vertical range of \(\mathcal{C}\) without hearing the error beep, then \(f\) has passed the horizontal line test, and thus is one-to-one.
      üîó
      üîó
    üîó
    üîó
    Let‚Äôs see how this scanning analogy works when applying the horiztonal line test to \(f(x)=x^2\text{.}\) As illustrated in Figure¬†2.10.8, for lines \(y=c\) with \(c\leq 0\text{,}\) our scanner emits no error beep, since such lines intersect the graph of \(f\) in at most one point. However, as soon as our scanner line moves above the \(x\)-axis, it starts emitting error beeps, as each such line intersects the graph of \(f\) in two points. We conclude that \(f\) is not one-to-one, just as we did in Example¬†2.10.2.
    üîó
    Returning to our discussion about exponential functions at the beginning of this section, we observe that we were able to deduct the one-to-one property from another property of exponential functions: namely, their monotonicity. To repeat that argument, if \(f\) is increasing (or decreasing) on a subset \(I\) of its domain, then clearly it is one-to-one on that set: in the increasing case, values of \(f\) increase as the input values increase, thus distinct inputs have distinct outputs; similarly, in the decreasing case, values of \(f\) decrease as the input values increase, thus again distinct inputs have distinct outputs. We have thus established the following theorem.
    üîó

    Theorem 2.10.9. Monotonic functions are one-to-one.

    If \(f\) is monotonic on a subset \(I\) of its domain \(D\text{,}\) then \(f\) is one-to-one on \(I\text{.}\)
    üîó
    üîó

    Remark 2.10.10. One-to-one versus monotonic.

    It is important to observe that being one-to-one is not in general equivalent to being monotonic. However, in the special case where the function \(f\) is continuous and the relevant set \(I\) is an interval, then the two conditions are in fact equivalent. Since we deal almost exclusively with continuous functions, you will find that showing a function is monotonic is often the most convenient way to show it is one-to-one.
    üîó
    As an exercise, come up with a noncontinuous function that is one-to-one on its domain \(D\text{,}\) but not monotonic. Hint: as always, a piecewise defined function usually is a good place to start!
    üîó
    üîó

    Example 2.10.11. Monotonic functions are one-to-one.

    Show that the given function \(f\) is one-to-one on its domain \(D\) using a monotonicity argument.
    1. Linear function.
      \(f(x)=ax+b\text{,}\) \(a\ne 0\text{,}\) \(D=\R\)
      üîó
      üîó
    2. Integer power function.
      \(f(x)=x^{n}\text{,}\) \(n\) an odd positive integer, \(D=\R\text{.}\)
      üîó
      üîó
    3. \(n\)-th root function.
      \(f(x)=\sqrt[n]{x}=x^{1/n}\text{,}\) \(n\) an odd positive integer, \(D=\R\text{.}\)
      üîó
      üîó
    üîó
    Solution.
    1. As we have seen before, linear functions with nonzero slope are monotonic. Indeed, in the case given we have \(f'(x)=a\) which is either always positive (if \(a> 0\)) or always negative (if \(a< 0\)). We conclude that \(f\) is either increasing or decreasing on its entire domain, and thus one-to-one by Theorem¬†2.10.9.
      üîó
      üîó
    2. We might just as well use the vertical line test here, recalling what graphs of power functions \(f(x)=x^n\) look like when \(n\) is an odd positive integer. (See Figure¬†1.5.1.) However, we use a monotonicity argument instead by way of illustration. We have
      \begin{align*} f'(x) \amp =nx^{n-1}\text{.} \end{align*}
      Since \(n\) is odd, \(n-1\) is even, which means that \(f'(a)> 0\) for all \(a\ne 0\text{.}\) As a result \(f\) is increasing on the intervals \((-\infty, 0)\) and \((0,\infty)\text{.}\) Since furthermore \(f\) is strictly negative on \((-\infty,0)\text{,}\) is zero at \(x=0\text{,}\) and is strictly positive on \((0,\infty)\text{,}\) we conclude that \(f\) is increasing on \(\R\text{.}\)
      üîó
      üîó
    3. The argument here is similar to the one above. We have
      \begin{align*} f'(x) \amp =\frac{1}{n}\, x^{1/n-1}\\ \amp = \frac{1}{n}\, x^{1/n}x^{-1}\text{.} \end{align*}
      Since \(x^{1/n}\) and \(x^{-1}\) are both negative for \(x< 0\) and both positive for \(x> 0\text{,}\) we see that \(f'(x)> 0\) for all \(x\ne 0\text{.}\) (Note that \(f'\) is not defined at \(x=0\text{.}\)) As above we conclude that \(f\) is increasing on the intervals \((-\infty, 0)\) and \((0,\infty)\text{,}\) and since \(f\) is strictly negative on \((-\infty,0)\text{,}\) is zero at \(x=0\text{,}\) and is strictly positive on \((0,\infty)\text{,}\) we conclude that \(f\) is increasing on \(\R\text{.}\)
      üîó
      üîó
    üîó
    üîó
    üîó
    üîó

    Subsection Invertible functions

    As we will see, the one-to-one property is closely related to another important function property: invertibility.
    üîó

    Definition 2.10.12. Invertible function.

    A function \(f\colon A\rightarrow B\) is invertible if there is a function \(g\colon B\rightarrow A\) satisfying the following properties:
    \begin{align} g(f(a)) \amp = a \text{ for all } a\in A\tag{2.24}\\ f(g(b)) \amp = b \text{ for all } b\in B\text{.}\tag{2.25} \end{align}
    The function \(g\) in this case is called the inverse function of \(f\text{,}\) and we write \(g=f^{-1}\text{.}\)
    üîó
    üîó

    Warning 2.10.13. Inverse function notation.

    Unfortunately, we use the same superscript notation for an inverse of a function \(f^{-1}\) that we use for the reciprocal of a number \(a^{-1}\text{.}\) It is extremely important that you do not conflate these two operations! In particular, the inverse of a function \(f^{-1}\) is not the same thing as its reciprocal function: that is,
    \begin{align*} f^{-1}(x) \amp \ne \frac{1}{f(x)}\text{!!} \end{align*}
    As a simple counterexample, consider the function \(f(x)=2x\text{.}\) You can easily verify for yourself that the function \(f^{-1}(x)=\tfrac{1}{2}x\) satisfies
    \begin{align*} f^{-1}(f(x)) \amp = f(f^{-1}(x))=x\text{,} \end{align*}
    and is thus the inverse function of \(f\text{.}\) But clearly
    \begin{align*} f^{-1} \amp\ne \frac{1}{f} \end{align*}
    in this case since \(\tfrac{1}{2}x\ne \frac{1}{2x}\) for all \(x\ne -1,1\text{.}\)
    üîó
    üîó

    Remark 2.10.14. Inverse as a reverse function.

    Suppose \(f\colon A\rightarrow B\) is invertible with inverse \(f^{-1}\text{.}\) It is instructive to think of the defining equations (2.24) and (2.25) as saying that the function \(f^{-1}\) ‚Äúundoes‚Äù whatever \(f\) does to an input \(a\in A\text{,}\) and conversely that \(f\) ‚Äúundoes‚Äù whatever \(f^{-1}\) does to an input \(b\in B\text{.}\)
    üîó
    In more detail, think of \(f\) as an operation that takes inputs \(a\in A\) and transforms them into the element \(f(a)\in B\text{.}\) Similarly, think of \(f^{-1}\) as another process that takes inputs \(b\in B\) and transforms them into the element \(f^{-1}(b)\in A\text{.}\) The identity \(f^{-1}(f(a))=a\text{,}\) can then be understood as saying that if we first apply the process \(f\text{,}\) yielding \(f(a)\text{,}\) and then apply the process \(f^{-1}\) to this result, yielding \(f^{-1}(f(a))\text{,}\) we get back our original input \(a\text{.}\) In other words, the process \(f^{-1}\) reverses (or undoes) the process \(f\text{.}\) And conversely, the process \(f\) reverses (or undoes) the process \(f^{-1}\text{.}\) Figure¬†2.10.15 gives a schematic representation of this idea of a process and its reverse.
    üîó
    üîó
    According to Definition¬†2.10.12, to show that a given function \(f\colon A\rightarrow B\) is invertible, we must prove that an inverse function \(f^{-1}\colon B\rightarrow A\) exists. If the starting function \(f\) is simple enough, we are often able to find a formula for an inverse function \(f^{-1}\) via algebra, as the next example illustrates.
    üîó

    Example 2.10.16. Computing inverses.

    For the given function \(f\colon A\rightarrow B\text{,}\) show that \(f\) is invertible by providing an explicit inverse function \(f^{-1}\colon B\rightarrow A\text{.}\)
    1. \(f\colon \R\rightarrow \R\text{,}\) \(f(x)=(-2)x+5\)
      üîó
      üîó
    2. \(f\colon \R\rightarrow \R\text{,}\) \(f(x)=ax+b\text{,}\) \(a\ne 0\)
      üîó
      üîó
    3. \(f\colon \R\rightarrow \R\text{,}\) \(f(x)=(x+2)^3-7\)
      üîó
      üîó
    üîó
    Solution.
    1. For a function \(f^{-1}\) to satisfy the inverse equations (2.24) given any output \(y=f(x)=(-2)x+5\text{,}\) we need \(f^{-1}(y)=x\text{.}\) We can determine the formula for \(f^{-1}(y)\) by starting with the equation \(y=(-2)x+5\text{,}\) which expresses \(y\) in terms of \(x\text{,}\) and solving it for \(x\) in terms of \(y\text{:}\)
      \begin{align*} y \amp =(-2)x+5 \\ y-5 \amp =(-2)x\\ x \amp = \frac{y-5}{-2}=-\frac{1}{2}y+\frac{5}{2}\text{.} \end{align*}
      Thus our inverse function \(f^{-1}\) should be defined as
      \begin{align} f^{-1}(y) \amp = \frac{y-5}{-2}=-\frac{1}{2}y+\frac{5}{2} \tag{2.26} \end{align}
      for all \(y\in \R\text{.}\) Observe that \(y\) is just the name of a variable here, so we are free now to replace it with the more standard variable name \(x\text{,}\) and write \(f^{-1}(x)=-\tfrac{x}{2}+\tfrac{5}{2}\text{.}\)
      üîó
      Let‚Äôs verify that \(f^{-1}\) is indeed the inverse function of \(f\text{.}\) We have
      \begin{align*} f^{-1}(f(x)) \amp = f^{-1}((-2)x+5)\\ \amp = -\frac{1}{2}(-2x+5)+\frac{5}{2}\\ \amp = x-\frac{5}{2}+\frac{5}{2}\\ \amp = x \end{align*}
      and
      \begin{align*} f(f^{-1}(x)) \amp = f\left(-\frac{1}{2}x+\frac{5}{2}\right)\\ \amp = (-2)\left(-\frac{1}{2}x+\frac{5}{2}\right)+5\\ \amp = x-5+5\\ \amp = x\text{,} \end{align*}
      as desired.
      üîó
      üîó
    2. Reasoning exactly as above, doing the same algebra with the undetermined constants \(a\) and \(b\text{,}\) we find that
      \begin{align*} f^{-1}(x) \amp = \frac{x-b}{a}=\frac{1}{a}x-\frac{b}{a}\text{.} \end{align*}
      Interestingly, we have shown that (a) all non-constant linear functions are invertible, and (b) the inverse of a linear function is again a linear function!
      üîó
      üîó
    3. We use the same algebraic technique as in (a). That is, we start with the equation \(y=f(x)=(x+2)^3-7\) and try to solve for \(x\) in terms of \(y\text{.}\) The resulting expression in \(y\) yields a formula for \(f^{-1}(y)\text{.}\) Here goes:
      \begin{align*} y \amp =(x+2)^3-7\\ y+7 \amp =(x+2)^3\\ \sqrt[3]{y+7} \amp = x+2\\ x \amp = \sqrt[3]{y+7}-2\text{.} \end{align*}
      Thus we have \(f^{-1}(y)=\sqrt[3]{y+7}-2\text{.}\) If you prefer, we can replace the variable name \(y\) with \(x\) and write \(f^{-1}(x)=\sqrt[3]{x+7}-2\text{.}\) Again, for the sake of illustration, it is worthwhile that our function \(f^{-1}\) does indeed satisfy the inverse equations (2.24) and (2.25). We have
      \begin{align*} f^{-1}(f(x)) \amp = f^{-1}((x+2)^3-7)\\ \amp = \sqrt[3]{(x+2)^3-7+7}-2\\ \amp = \sqrt[3]{(x+2)^3}-2\\ \amp = x+2-2 \amp (\sqrt[3]{t^3}=t \text{ for all } t\in \R)\\ \amp = x \end{align*}
      and
      \begin{align*} f(f^{-1}(x)) \amp = f\left(\sqrt[3]{x+7}-2\right)\\ \amp = \left(\sqrt[3]{x+7}-2+2\right)^3-7\\ \amp = \left(\sqrt[3]{x+7}\right)^3-7\\ \amp = x+7-7 \amp (\sqrt[3]{t^3}=t \text{ for all } t\in \R)\\ \amp = x\text{,} \end{align*}
      as desired.
      üîó
      üîó
    üîó
    üîó
    üîó
    Let‚Äôs formalize the technique we employed in Example¬†2.10.16 for computing formulas for inverse functions (when they exist).
    üîó

    Procedure 2.10.17. Computing inverse function formula.

    Let \(f\colon A\rightarrow B\) be an invertible function. To algebraically derive a formula for \(f^{-1}\colon B\rightarrow A\) proceed as follows.
    1. Start with the equation
      \begin{align*} y \amp =f(x)\text{,} \end{align*}
      where the right side \(f(x)\) is the formula for \(f\text{.}\) The equation thus expresses an output \(y\) in terms of the input \(x\text{.}\)
      üîó
      üîó
    2. Algebraically solve the starting equation \(y=f(x)\) for \(x\) in terms of \(y\text{,}\) yielding an equation of the form
      \begin{align*} x \amp = g(y)\text{,} \end{align*}
      where \(g(y)\) is an mathematical expression solely in terms of the variable \(y\text{.}\)
      üîó
      üîó
    3. The inverse function \(f^{-1}\) then has formula \(f^{-1}(x)=g(x)\text{.}\)
      üîó
      üîó
    üîó
    üîó
    As hinted at above, when the starting invertible function \(f\) is complicated enough, the algebra that goes into Procedure¬†2.10.17 is often too complicated for us to successfully find an explicit formula for the inverse function \(f^{-1}\text{.}\) Accordingly, it pays to have some alternative methods for determining whether a given function is invertible. The next theorem does exactly this by linking invertibility to the one-to-one property.
    üîó

    Theorem 2.10.18. Invertibility equivalence.

    Let \(f\colon A\rightarrow B\) be a function.
    1. If \(f\) is invertible, then \(f\) is one-to-one.
      üîó
      üîó
    2. If \(f\) is one-to-one, then letting \(C=\range f\text{,}\) the function \(f\colon A\rightarrow C\) is invertible.
      üîó
      üîó
    3. \(f\) is invertible if and only if \(f\) is one-to-one and \(\range f=B\text{.}\)
      üîó
      üîó
    üîó
    üîó

    Proof.

    1. Assume \(f\) is invertible, and let \(f^{-1}\colon B\rightarrow A\) be its inverse. We have
      \begin{align*} f(x_1)=f(x_2) \amp \implies f^{-1}(f(x_1))=f^{-1}(f(x_2))\\ \amp \implies x_1=x_2 \amp (f^{-1}(f(a))=a \text{ for all } a\in A)\text{,} \end{align*}
      showing that \(f\) is one-to-one.
      üîó
      üîó
    2. Assume \(f\) is one-to-one. By definition of \(C=\range f\text{,}\) for every \(c\in C\) there is some \(a\in A\text{,}\) such that \(f(a)=c\text{.}\) Since furthermore \(f\) is one-to-one, there is in fact exactly one \(a\in A\) such that \(f(a)=c\text{.}\) Thus we can define a function \(f^{-1}\colon C\rightarrow A\) as follows: given \(c\in C\text{,}\) \(f^{-1}(c)=a\) where \(a\) is the unique element of \(A\) satisfying \(f(a)=c\text{.}\) It is now straightforward to verify that \(f^{-1}\) satisfies the inverse equations (2.24) and (2.25).
      üîó
      It is worthwhile observing where the one-to-one property comes into play in this argument. If we didn‚Äôt know that \(f\) was one-to-one, then there would be no guarantee that for each \(c\in C=\range f\) there was one and only one \(a\in A\) satisfying \(f(a)=c\text{:}\) there could be two such elements, or even more! In that case, our proposed definition of \(f^{-1}(c)\) as the element of \(a\) satisfying \(f(a)=c\) would not be well defined.
      üîó
      üîó
    3. This follows immediately from (1) and (2).
      üîó
      üîó
    üîó
    üîó
    Connecting invertibility with the one-to-one property allows us to show a given function is invertible without necessarily having to find an explicit inverse function. For example, each of the functions in Example¬†2.10.16 can be shown to be invertible simply by showing that they are monotonic, and hence one-to-one.
    üîó
    Of equal importance is the fact that Theorem¬†2.10.18 gives us alternative means for showing that a function \(f\) is not invertible. If we only had the definition of invertibility at your disposal, to do this we would have to prove somehow that no inverse function \(f^{-1}\) exists. That can be quite an onerous task, when you consider the fact that there are infinitely many functions out there that could serve as \(f^{-1}\text{.}\) By contrast, using xref="th_inv_equivalence"/>, we can show that \(f\) is not invertible simply by showing that (a) \(f\) is not one-to-one, or (b) \(\range f \ne B\text{.}\)
    üîó

    Remark 2.10.20. Invertibility equivalence.

    The function \(p(x)=2x^3+3x^2+6x+5\) is a good example of a function that we are able to show is invertible using Theorem¬†2.10.18 , but for which it is very difficult to find an explicit formula for the inverse function \(f^{-1}\text{.}\) Indeed, in order to solve the equation

    Subsection Relating functions and their inverses

    Not surprisingly, if \(f\) is an invertible function, then there are many close connections between properties of \(f\) and properties of its inverse function \(f^{-1}\text{,}\) starting with the fact that
    \begin{align*} f(a)=b \amp \iff f^{-1}(b)=a\text{.} \end{align*}
    The next theorem compiles a bunch of these similarities for your convenience. A guiding principle here is that we move back and forth between information about \(f\) and information about \(f^{-1}\) by reversing the roles of inputs and outputs. This is most transparently seen in statement (3) below. Statement (4) is seen to be a direct consequence of (3) once you know that reflecting a point \((x,y)\) through the line \(y=x\) yields the point \((y,x)\text{,}\) which is not entirely obvious.
    üîó

    Theorem 2.10.25. Inverse function compendium.

    Let \(f\) be an invertible function.
    1. We have
      \begin{align} f^{-1}(f(a))\amp =a \tag{2.27}\\ f(f^{-1}(b))\amp =b \tag{2.28} \end{align}
      and
      \begin{align} f(a)=b \amp \iff f^{-1}(b)=a \tag{2.29} \end{align}
      for all \(a\in \operatorname{domain} f\) and \(b\in \range f\text{.}\)
      üîó
      üîó
    2. We have
      \begin{align} \range f \amp =\operatorname{domain} f^{-1}\tag{2.30}\\ \operatorname{domain} f \amp = \range f^{-1}\text{.}\tag{2.31} \end{align}
      üîó
      üîó
    3. The point \(P=(x,y)\) is on the graph of \(f\) if and only if the point \(Q=(y,x)\) is on the graph of \(f^{-1}\text{.}\)
      üîó
      üîó
    4. The graph of \(f^{-1}\) is the reflection of the graph of \(f\) through the line \(y=x\text{.}\)
      üîó
      üîó
    üîó
    üîó
