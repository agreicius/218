    Subsection 6.1.2 Why Inverse Functions?

    In order to study several categories of transcendental functions, you first need to know about inverse functions. Intuitively, an inverse function is a function that "undoes" another function. In this section we will first review some relevant properties of functions, define what it means to "undo" or "invert" a function, determine conditions under which we can invert a function, and then summarize some useful properties of inverse functions.

    Subsection 6.1.3 Brief Review of Functions

    Before discussing inverse functions, letâ€™s recall the basic definitions and terminology for functions from the beginning of the year:
    ðŸ”—
    A function \(f\) from a set \(A\) to a set \(B\) is a rule \(f\) that associates to each element \(x\) of the domain set \(A\) exactly one element \(y=f(x)\) of the range set \(B\text{.}\) We will use the notation
    \begin{equation*} f: A\rightarrow B \end{equation*}
    to represent a function \(f\) together with its domain set \(A\) and range set \(B\text{.}\) This notation is pronounced "\(f\) from \(A\) to \(B\text{.}\)" If \(x\) and \(y\) are variables that represent elements of the sets \(A\) and \(B\text{,}\) respectively, then we say that \(y\) is a function of \(x\) and write \(y=f(x)\text{.}\) The variable \(x\) is called the independent variable and represents the "input" of the function. The function \(f\) sends each input \(x\) to one and only one "output," some value of the dependent variable \(y\text{.}\) Notice that \(y\) depends on \(x\text{,}\) according to the assignment defined by the function \(f\text{.}\)

    Subsection 6.1.4 Defining Inverse Functions

    Because every function is a process that converts a collection of inputs to a corresponding collection of outputs, a natural question is: for a particular function, can we change perspective and think of the original functionâ€™s outputs as the inputs for a reverse process? If we phrase this question algebraically, it is analogous to asking: given an equation that defines \(y\) as a function of \(x\text{,}\) is it possible to find a corresponding equation where \(x\) is a function of \(y\text{?}\)
    ðŸ”—

    Example 6.1.3.

    It is possible to solve the equation \(F=\frac{9}{5}C+32\) for \(C\text{.}\) In other words, we are able to isolate \(C\) through a series of algebraic steps. Doing so results in the equation which takes Fahrenheit temperature inputs \(F\) and produces the corresponding Celsius temperature outputs \(C\text{,}\) meaning a function \(C=h(F)\text{.}\)
    ðŸ”—
    We start with the equation
    \begin{equation*} F=\frac{9}{5}C+32 \end{equation*}
    and subtract \(32\) from both sides, to get
    \begin{equation*} F-32=\frac{9}{5}C\text{.} \end{equation*}
    ðŸ”—
    Then to get \(C\) by itself, we multiply both sides by \(\frac{5}{9}\) to get
    \begin{equation*} C=\frac{5}{9}(F-32)\text{.} \end{equation*}
    ðŸ”—
    Note that the equation \(C=\frac{5}{9}(F-32)\) expresses \(C\) as a function of \(F\text{.}\) Call this function \(h\) so that \(C=h(F)=\frac{5}{9}(F-32)\text{.}\)

    Inverses Reverse Each Other! In ExampleÂ 6.1.3, we found that for the function \(F=g(C)=\frac{9}{5}C+32\text{,}\) itâ€™s possible to solve for \(C\) in terms of \(F\) and write \(C=h(F) = \frac{5}{9}(C-32)\text{.}\) The first function, \(g\text{,}\) converts Celsius temperatures to Fahrenheit ones; the second function, \(h\text{,}\) converts Fahrenheit temperatures to Celsius ones. Thus, the process \(h\) reverses the process of \(g\text{,}\) and likewise the process of \(g\) reverses the process of \(h\text{.}\) This is also why it makes sense that \(j=h\circ g\) and \(k = g\circ h\) are so simple, namely that \(j(C) = h(g(C)) = C\) and \(k(F) = g(h(F)) = F\text{,}\) meaning both compositions end up back where they started. If, for instance, we take a Celsius temperature \(C\text{,}\) convert it to Fahrenheit, and convert the result back to Celsius, we arrive back at the Celsius temperature we started with: \(h(g(C)) = C\text{.}\)
    ðŸ”—
    Similar work is sometimes possible with other functions. When we can find a new function that reverses the process of the original function, we say that the original function "has an inverse function" and make the following formal definition.
    ðŸ”—

    Definition 6.1.4. The Inverse of a Function.

    Let \(f:A\to B\) be a function. If there exists a function \(g: B\to A\) such that
    \begin{equation*} g(f(a))=a \textrm{ and } f(g(b))=b \end{equation*}
    for each \(a\) in \(A\) and each \(b\) in \(B\text{,}\) then we say that \(f\) has an inverse function and that the function \(g\) is the inverse of \(f\).
    ðŸ”—
    A function is called invertible if it has in inverse function.
    ðŸ”—

    Example 6.1.5.

    Subsection 6.1.5 When is a Function Invertible?

    Not all functions have inverses.
    ðŸ”—

    Example 6.1.15.

    For example, consider the squaring function \(f(x) = x^2\) with the domain \(\mathbb{R}\text{.}\) How could we reverse the process defined by \(f\text{?}\)
    ðŸ”—
    Observe that \(f(2) = 4\) and \(f(-2)=4\text{.}\) An inverse of \(f\) would have to send the input 4 back to both \(2\) and \(-2\) in order to reverse \(f\text{.}\) This violates our rules for a function since an input must have one and only one output. Thus \(f(x) = x^2\) does not have an inverse.
    ðŸ”—
    In particular, \(\sqrt{x^2} = x\) is not always true, so that \(\sqrt{x}\) is not a true inverse to \(x^2\text{.}\) Why not? Find a value for \(x\) such that \(\sqrt{x^2}\neq x\text{.}\)
    ðŸ”—
    ðŸ”—
    Itâ€™s important to note in DefinitionÂ 6.1.4 that we say "If there existsâ€¦." That is, we are not guaranteed that an inverse function exists for a given function. Thus, we might ask: how can we determine whether or not a given function has a corresponding inverse function? As with many questions about functions, there are often three different possible ways to explore such a question: through a table, through a graph, or through an algebraic formula.

    Theorem 6.1.23. Horizontal Line Test.

    A function whose graph lies in the \(xy\)-plane has a corresponding inverse function if and only if every horizontal line intersects the graph at most once. When the graph passes this test, the horizontal coordinate of each point on the graph can be viewed as a function of the vertical coordinate of the point.
    ðŸ”—
    ðŸ”—

    Definition 6.1.24. One-to-One.

    A function \(f\) whose graph passes the horizontal line test is said to be one-to-one. This is because the inputs and outputs of such a function pair uniquely with each other, without any "doubling up" of multiple inputs to one output.
    ðŸ”—
    Another way of stating that a function is one-to-one is that, for any values \(a\) and \(b\) of the independent variable,
    ðŸ”—
    if \(a\neq b\text{,}\) then \(f(a)\neq f(b)\text{.}\)
    Equivalently, using the contrapositive, we can phrase this as
    ðŸ”—
    if \(f(a)=f(b)\text{,}\) then \(a=b\text{.}\)

    Subsection 6.1.6 Properties of Inverse Functions

    When a function has an inverse function, we have observed several important relationships that hold between the original function and the corresponding inverse function, which we summarize here. Note that many of these properties seem to be saying the same thing, only with a different emphasis.
    ðŸ”—

    Theorem 6.1.32. Properties of an inverse function.

    Let \(f:A\to B\) be a function whose domain is \(A\) and whose range is \(B\) be such that \(f\) has an inverse function \(f^{-1}\text{.}\) Then:
    1. The functions \(f\) and \(f^{-1}\) reverse one anotherâ€™s processes. Symbolically, \(f^{-1}(f(a))=a\) for every input \(a\) in the domain of \(f\text{,}\) and similarly, \(f(f^{-1}(b))=b\) for every input \(b\) in the domain of \(f^{-1}\text{.}\)
      ðŸ”—
      ðŸ”—
    2. If \(b=f(a)\text{,}\) then we can express the exact same relationship from a different perspective by writing \(a=f^{-1}(b)\text{.}\)
      ðŸ”—
      ðŸ”—
    3. The domain of \(f^{-1}\) is \(B\) and its range is \(A\text{.}\) Symbolically, \(f^{-1}:B\to A\text{,}\) meaning that \(f^{-1}\) is a rule that assigns to each element \(b\) of the set \(B\) and element \(a\) of the set \(A\) via \(a=f^{-1}(b)\text{.}\)
      ðŸ”—
      ðŸ”—
    4. If a point \((a,b)\) lies on the graph of \(f\text{,}\) then we write \(b=f(a)\text{.}\) From this, we can equivalently say that \(a=f^{-1}(b)\text{.}\) Hence, the point \((b,a)\) lies on the graph of \(a=f^{-1}(b)\text{.}\)
      ðŸ”—
      ðŸ”—
    5. The graph of \(y=f^{-1}(x)\) is the graph of \(y=f(x)\) reflected across the line \(y=x\text{.}\)
      ðŸ”—
      ðŸ”—
    ðŸ”—
    ðŸ”—
    Property (e) above is new but follows immediately from property (d): because inverse functions reverse inputs and outputs, inverting a function reverses the \(x\)- and \(y\)-values in the \(xy\)-plane. This means that the graph of the inverse of a function is a reflection of the graph of the original function across the line \(y=x\text{.}\)
