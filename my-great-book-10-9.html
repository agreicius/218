    Subsection 8.8.2 Derivatives of Inverse Trigonometric Functions

    In this section, we develop formulas for differentiating the inverse trigonometric functions, along with a formula for differentiating any inverse function. We begin by developing a formula for the arcsine function.
    ðŸ”—
    Recall that if \(y=\sin^{-1} x\text{,}\) then for \(-\frac{\pi}{2}\leq y\leq \frac{\pi}{2}\text{,}\) we can write \(\sin y = x\text{.}\) Then we take the derivative of both sides with respect to \(x\) and remember that \(y\) is a function of \(x\text{,}\) so we have to use implicit differentiation/the chain rule on the left-hand side:
    \begin{equation*} \begin{aligned}\frac{d}{dx}(\sin y) \amp = \frac{d}{dx}(x) \amp \leftarrow\text{ derivative with respect to } x\\ \cos(y)\cdot\frac{dy}{dx} \amp = 1 \amp \leftarrow\text{ chain rule on the left-hand side } \\ \frac{dy}{dx} \amp = \frac{1}{\cos y} \amp \leftarrow\text{ solving for } \frac{dy}{dx} \end{aligned} \end{equation*}
    ðŸ”—
    But remember that \(y\) is a function of \(x\text{,}\) and so we would like to be able to write its derivative \(\frac{dy}{dx}\) as a function of \(x\text{.}\) We substitute \(y=\sin^{-1}x\) (this is what we started with) and remember that we can simplify compositions of trigonometric and inverse trigonometric functions as in SectionÂ 8.5.3:
    \begin{equation*} \begin{aligned}\frac{dy}{dx} \amp = \frac{1}{\cos y} \amp \\ \amp = \frac{1}{\cos(\sin^{-1} x)} \amp \leftarrow\text{ substituting } y=\sin^{-1}x\\ \amp = \frac{1}{\sqrt{1-x^2}}\amp \leftarrow\text{ simplification from \href{my-great-book-10-6.html#sec_simplifyingcompositionsoftrigandinversetrig}{SectionÂ 8.5.3} } \end{aligned} \end{equation*}
    ðŸ”—
    Thus we have shown that
    \begin{equation*} \frac{d}{dx}(\sin^{-1} x) = \frac{1}{\sqrt{1-x^2}} \end{equation*}
    ðŸ”—
    An interesting fact about the derivative of the arcsine function is that its domain is slightly smaller than the domain of the original arcsine function. The graphs of these two functions are below. Note that the domain of \(f(x) = \arcsin x\) is \([-1,1]\) while the domain of \(f'(x) = \frac{1}{\sqrt{1-x^2}}\) is \((-1,1)\text{.}\)
    ðŸ”—
    If you look closely at the graph of \(f(x) = \arcsin x\text{,}\) you should notice that at the ends of the domain the tangent lines will be vertical (the tangent lines at the corresponding points on the graph of \(y=\sin x\) are horizontal). Since a vertical line has undefined slope, the derivative does not exist at these points. This means that the derivative of arcsine is not defined at \(x=1\) or \(x=-1\text{.}\) This is precisely what we see in the graph of \(f'(x) = \frac{1}{\sqrt{1-x^2}}\text{:}\) there is a vertical asymptote at \(x=1\) and \(x=-1\text{.}\)
    ðŸ”—
    If none of the above made any sense to you, fear not! You can still use the formula for the derivative without understanding it. The next few examples demonstrate how to take the derivative of functions involving arcsine.
    ðŸ”—

    Example 8.8.2.

    Calculate the derivative of \(f(x) = (\sin^{-1}x)^2\text{.}\)
    ðŸ”—
    Solution.
    The function \(f(x) = (\sin^{-1}x)^2\) is a composition of \((\cdot)^2\) with the inverse trigonometric function \(\sin^{-1}x\) on the inside. By the chain rule,
    \begin{equation*} \frac{d}{dx}[(\sin^{-1}x)^2] = 2(\sin^{-1}x)^1\left(\frac{1}{\sqrt{1-x^2}}\right) = \frac{2\sin^{-1}x}{\sqrt{1-x^2}} \end{equation*}

    ðŸ”—
    ðŸ”—

    Example 8.8.3.

    Calculate the derivative of \(g(x) = \sin^{-1}(x^2)\text{.}\)
    ðŸ”—
    Solution.
    The function \(g(x) = \sin^{-1}(x^2)\) is a composition of \(\sin^{-1}(\cdot)\) with \(x^2\) on the inside. By the chain rule,
    \begin{equation*} \frac{d}{dx}[(\sin^{-1}(x^2)] = \frac{1}{\sqrt{1-(x^2)^2}}\cdot 2x = \frac{2x}{\sqrt{1-x^4}} \end{equation*}

    Example 8.8.4.

    Calculate the derivative of \(h(x) = x\sin^{-1}(3x+1)\text{.}\)
    ðŸ”—
    Solution.
    The function \(h(x)=x\sin^{-1}(3x+1)\) is a product of two functions, \(x\) and \(\sin^{-1}(3x+1)\text{,}\) and thus we begin with the product rule. We will also need the chain rule to differentiate the composition \(\sin^{-1}(3x+1)\text{:}\)
    \begin{equation*} \begin{aligned}h'(x) \amp = (1)\cdot \sin^{-1}(3x+1) +x\cdot\frac{1}{\sqrt{1-(3x+1)^2}}\cdot(3)\\ \amp = \sin^{-1}(3x+1) +\frac{3x}{\sqrt{1-(3x+1)^2}}. \end{aligned} \end{equation*}

    Subsection 8.8.3 Derivatives of Arbitrary Inverse Functions

    We could develop formulas for the derivatives of the arccosine and arctangent functions the same way that we developed the above formula for the derivative of arcsine. On the other hand, it would be nice to develop one general formula for derivatives of any inverse function instead of developing them one by one, so weâ€™ll do that instead in this section.
    ðŸ”—
    Suppose now that \(f\) and \(g=f^{-1}\) are differentiable functions that are inverses of each other. Then by definition, \(y=f(x)\) if and only if \(g(y)=x\) and \(f(g(x)) = x\) for every \(x\) in the domain of \(g=f^{-1}\text{.}\) Taking the derivative of \(f(g(x)) = x\text{:}\)
    \begin{equation*} \begin{aligned}\frac{d}{dx}[f(g(x))] \amp = \frac{d}{dx}[x]\\ f'(g(x))g'(x) \amp = 1\amp \leftarrow\text{ using the chain rule } \\ g'(x) \amp = \frac{1}{f'(g(x))}\amp \leftarrow\text{ solving for } g'(x)\\ (f^{-1})'(x) \amp = \frac{1}{f'(f^{-1}(x))}\amp \leftarrow\text{ recall that \(g = f^{-1}\) and substitute} \end{aligned} \end{equation*}
    ðŸ”—
    Thus we have found that if we know \(f(x)\) has derivative \(f'(x)\) and inverse \(f^{-1}(x)\text{,}\) the derivative of the inverse function \(f^{-1}(x)\) is
    \begin{equation*} \left[f^{-1}(x)\right]^\prime =(f^{-1})'(x) = \frac{1}{f'\left(f^{-1}(x)\right)} \end{equation*}
    meaning the reciprocal of the composition \(f'\circ f^{-1}\text{.}\)
    ðŸ”—

    Example 8.8.6. Another Way to Deduce the Derivative of Arcsine.

    Note that this is consistent with our work finding a derivative for \(f^{-1}(x) = \sin^{-1}x\text{.}\) Because \(f(x) = \sin x\) has derivative \(f'(x) = \cos x\text{,}\) we found that the derivative of \(f^{-1}(x) = \sin^{-1}x\) was
    \begin{equation*} (f^{-1})'(x) = \frac{1}{f'\left(f^{-1}(x)\right)} = \frac{1}{\cos(\sin^{-1}x)} \end{equation*}
    before simplifying. We also know from SectionÂ 8.5.3 that \(\cos(\sin^{-1}x) = \sqrt{1-x^2}\text{,}\) and so
    \begin{equation*} (f^{-1})'(x) = \frac{1}{f'\left(f^{-1}(x)\right)} = \frac{1}{\cos(\sin^{-1}x)} = \frac{1}{\sqrt{1-x^2}} \end{equation*}
    ðŸ”—
    ðŸ”—
    Visual Interpretation: We can make sense of this formula for the derivative of the inverse visually, using what we know about graphs of inverse functions.     \begin{equation*} b=f(a) \Leftrightarrow f^{-1}(b) =a\text{.} \end{equation*}
    We can graph the tangent line to \(y=f(x)\) at \(x=a\) and when we reflect over the line \(y=x\text{,}\) we should get the tangent line to \(y=f^{-1}(x)\) at \(x=b\text{.}\) Take a careful look at the graph in FigureÂ 8.8.7 below and how it relates to what was just said:

    Subsection 8.8.4 Derivatives of Arccosine and Arctangent

    Using the formula for the derivative of the inverse of a function from the previous section, we can quickly find derivatives for \(y=\cos^{-1}x\) and \(y=\tan^{-1}x\text{.}\)
    ðŸ”—

    Example 8.8.14.

    If \(f(x) = \cos x\) and \(0\leq x\leq \pi\text{,}\) we know that \(f^{-1}(x) = \cos^{-1}x\) for \(-1\leq x\leq 1\text{.}\) Recall that \(f'(x) = -\sin x\text{.}\) Then
    \begin{equation*} \frac{d}{dx}[\cos^{-1}(x)] = \frac{1}{f'(f^{-1}(x))} = \frac{1}{-\sin(\cos^{-1}x)} = \frac{-1}{ \sqrt{1-x^2}} \end{equation*}
    since \(\sin(\cos^{-1}x) = \sqrt{1-x^2}\text{.}\) To see this, let \(x=\cos \theta\) be the adjacent side of a triangle with hypotenuse \(1\text{,}\) which then has opposite side length \(\sqrt{1-x^2}\) by the Pythagorean Theorem. (You have done several such simplifications already and should be able to do this yourself.)
    ðŸ”—
    ðŸ”—

    Example 8.8.15.

    If \(f(x) = \tan x\) and \(-\frac{\pi}{2}\lt x\lt \frac{\pi}{2}\text{,}\) we know that \(f^{-1}(x) = \tan^{-1}x\) for all real numbers \(x\text{.}\) Recall that \(f'(x) = \sec^2 x\text{.}\) Then
    \begin{equation*} \frac{d}{dx}[\tan^{-1}(x)] = \frac{1}{f'(f^{-1}(x))} = \frac{1}{\sec^2(\tan^{-1}x)} = \cos^2(\tan^{-1}x) \end{equation*}
    since \(\sec = \frac{1}{\cos}\text{.}\) In order to simplify \(\cos^2(\tan^{-1}x)\text{,}\) let \(x=\tan \theta\) so that we have a triangle with opposite side \(x\) and adjacent side \(1\text{:}\)
    Using the Pythagorean Theorem, the hypotenuse is \(\sqrt{1+x^2}\) and so
    \begin{equation*} \frac{d}{dx}[\tan^{-1}(x)] = \cos^2(\tan^{-1}x) = \left(\frac{1}{\sqrt{1+x^2}}\right)^2 = \frac{1}{1+x^2} \end{equation*}
    ðŸ”—
    ðŸ”—
    Note that the domain of \(\arctan x\) and the domain of its derivative \(\frac{1}{1+x^2}\) are both \((-\infty,\infty)\) since \(1+x^2\) is never equal to zero! The graphs of these two functions are below:

    We have now proven the following theorem, which gives us formulas for the derivatives of our three inverse trigonometric functions:
    ðŸ”—

    Theorem 8.8.17. Derivatives of Inverse Trigonometric Functions.

    For all values of \(x\) at which the following functions are defined,
    \begin{equation*} \begin{aligned}\frac{d}{dx}\left[\sin^{-1}x\right] \amp =\frac{1}{\sqrt{1-x^2}}\\ \\ \frac{d}{dx}\left[\cos^{-1}x\right]\amp =\frac{-1}{\sqrt{1-x^2}}\\ \\ \frac{d}{dx}\left[\tan^{-1}x\right]\amp =\frac{1}{1+x^2} \end{aligned} \end{equation*}

    The next three examples demonstrate how to take derivatives of functions involving the arctangent and arccosine functions.
    ðŸ”—

    Example 8.8.19.

    Calculate the derivative of \(f(x) = \tan^{-1}\sqrt{x}\text{.}\) What is the domain of \(f(x)\text{?}\) Of \(f'(x)\text{?}\)
    ðŸ”—
    Solution.
    The function \(f(x) = \tan^{-1}\sqrt{x}\) is a composition of the arctangent function with \(\sqrt{x}\) on the inside. By the chain rule,
    \begin{equation*} f'(x)= \frac{1}{1+(\sqrt{x})^2}\cdot \frac{1}{2}x^{-1/2} = \frac{1}{1+x}\cdot \frac{1}{2\sqrt{x}} = \frac{1}{2\sqrt{x}(1+x)} \end{equation*}
    ðŸ”—
    The domain of \(f(x) = \tan^{-1}\sqrt{x}\) is \(x\geq 0\) since arctangent can take any argument, but we need a nonnegative number inside the square root.
    ðŸ”—
    The domain of \(f'(x) = \frac{1}{2\sqrt{x}(1+x)}\) is \(x>0\) since we do not want to a denominator of 0.
    ðŸ”—
    It is also true that the derivative \(f'(x)\) is undefined when \(1+x = 0\text{,}\) so when \(x=-1\text{,}\) but this is not in the domain of \(f(x)\) so is not relevant. The derivative of a function can only be defined on the domain of the function.
    ðŸ”—
    ðŸ”—
    ðŸ”—

    Example 8.8.20.

    Calculate the derivative of \(g(x) = \dfrac{5}{\cos^{-1}x}\text{.}\)
    ðŸ”—
    Solution.
    Here, it is especially tempting to try to simplify using the \(-1\) as an exponent, but in this context, the \(-1\) superscript indicates the arccosine function, not the reciprocal of cosine. We can instead rewrite
    \begin{equation*} g(x) = 5(\cos^{-1}x)^{-1} \end{equation*}
    in order to take the derivative using the chain rule with inside function \(\cos^{-1}x\text{:}\)
    \begin{equation*} g'(x) = 5(-1)(\cos^{-1}x)^{-2}\cdot \frac{-1}{\sqrt{1-x^2}} = \frac{5}{(\cos^{-1}x)^2\sqrt{1-x^2}}\text{.} \end{equation*}
    You could also use the quotient rule, but the above computation is much quicker.

    Example 8.8.21.

    Calculate the derivative of \(h(x) = \sin^2(\tan^{-1}e^{3x})\text{.}\)
    ðŸ”—
    Solution.
    Weâ€™ll begin by rewriting the notation \(\sin^2 u\) as \((\sin u)^2\) so we can see the correct order for the differentiation rules. The function \(h(x)\) is actually a composition of five functions: \((\cdot)^2, \sin (\cdot), \tan^{-1}(\cdot), e^{(\cdot)}\text{,}\) and \(3x\text{.}\) Thus we will have to apply the chain rule several times and will proceed carefully, writing several intermediate steps:
    \begin{equation*} \begin{aligned}h'(x) \amp = \frac{d}{dx}\left(\sin^2(\tan^{-1}e^{3x})\right)\\ \amp = \frac{d}{dx}\left(\left(\sin(\tan^{-1}e^{3x})\right)^2\right)\amp \leftarrow\text{ rewrite function } \\ \amp = 2(\sin(\tan^{-1}e^{3x}))^1\frac{d}{dx}(\sin(\tan^{-1}e^{3x}))\amp \leftarrow\text{ chain rule with } (\cdot)^2\\ \amp =2\sin(\tan^{-1}e^{3x})\cos(\tan^{-1}e^{3x})\frac{d}{dx}(\tan^{-1}e^{3x})\amp \leftarrow\text{ chain rule with } \sin (\cdot)\\ \amp =2\sin(\tan^{-1}e^{3x})\cos(\tan^{-1}e^{3x})\frac{1}{1+(e^{3x})^2}\frac{d}{dx}(e^{3x})\amp \leftarrow\text{ chain rule with } \tan^{-1} (\cdot)\\ \amp =2\sin(\tan^{-1}e^{3x})\cos(\tan^{-1}e^{3x})\frac{1}{1+(e^{3x})^2}e^{3x}\frac{d}{dx}(3x)\amp \leftarrow\text{ chain rule with } e^{(\cdot)}\\ \amp =2\sin(\tan^{-1}e^{3x})\cos(\tan^{-1}e^{3x})\frac{1}{1+(e^{3x})^2}e^{3x}(3)\amp \leftarrow\text{ derivative of the innermost function, } 3x\\ \amp =\frac{6e^{3x}\sin(\tan^{-1}e^{3x})\cos(\tan^{-1}e^{3x})}{1+3^{6x}}\amp \leftarrow\text{ simplifying } \end{aligned} \end{equation*}
