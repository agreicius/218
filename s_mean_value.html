    Theorem 2.2.1. Rolle‚Äôs theorem.

    Assume \(f\) is continuous on \([a,b]\) and differentiable on \((a,b)\text{.}\) If \(f(a)=f(b)\text{,}\) then there is an input \(c\in [a,b]\) satisfying \(f'(c)=0\text{.}\)
    üîó
    üîó

    Proof.

    Let \(C=f(a)=f(b)\) be the common value of \(f\) at the endpoints. If \(f(x)=C\) for all inputs \(x\in [a,b]\) (i.e., \(f\) is a constant function), then \(f'(x)=0\) for all inputs \(x\in (a,b)\text{,}\) and our desired conclusion is trivially satisfied.
    üîó
    Assume now that \(f\) is not constant. Since \(f\) is continuous on \([a,b]\) there are points \(c\) and \(d\) in \([a,b]\) where \(f\) attains its absolute maximum and minimum values. Since \(f\) is not constant on \([a,b]\text{,}\) we must have \(f(c)\ne C\) or \(f(d)\ne C\text{.}\) Assume without loss of generality that \(f(c)\ne C\) is an absolute extreme value of \(f\) on \([a,b]\text{.}\) The input \(c\) must be an interior point of \([a,b]\text{,}\) since \(f(c)\ne f(a)\) and \(f(c)\ne f(b)\text{.}\) Since \(f\) is differentiable everywhere on \((a,b)\text{,}\) Theorem¬†2.1.11 implies \(f'(c)=0\text{,}\) as desired.
    üîó
    üîó

    Remark 2.2.2. Interpretations of Rolle‚Äôs theorem.

    Let \(f\) satisfy the assumptions of Rolle‚Äôs theorem on the interval \([a,b]\text{.}\)
    üîó
    Geometrically, we interpret Rolle‚Äôs theorem as follows: if the endpoints of the graph of \(f\) over \([a,b]\) lie on the same horizontal line, then there is an input \(c\in (a,b)\) for which the tangent line to the graph of \(f\) at \(c\) is horizontal.
    üîó
    In terms of rate of change, we interpret Rolle‚Äôs theorem as follows: if the average rate of change of \(f\) between \(a\) and \(b\) is equal to zero, then there is an input \(c\in (a,b)\) where the instantaneous rate of change \(f'(c)\) is equal to zero.
    üîó
    üîó
    Rolle‚Äôs theorem turns out to be a special case of the mean value theorem, where the function in question satisfies \(f(a)=f(b)\text{.}\) As a peculiar twist, however, the mean value theorem can also be seen as just a ‚Äútilted‚Äù version of Rolle‚Äôs theorem. In logical terms, the two theorems, though quite different in content, are logically equivalent!
    üîó

    Theorem 2.2.3. Mean value theorem (MVT).

    Assume \(f\) is continuous on \([a,b]\) and differentiable on \((a,b)\text{.}\) There is an input \(c\in (a,b)\) satisfying
    \begin{equation} f'(c)=\frac{f(b)-f(a)}{b-a}\text{.}\tag{2.2} \end{equation}
    üîó
    üîó

    Proof.

    We employ a little trick that reveals MVT to be a simple consequence of Rolle‚Äôs theorem. Define
    \begin{equation*} g(x)=f(x)-\frac{f(b)-f(a)}{(b-a)}(x-a)\text{.} \end{equation*}
    As is easily verified, \(g\) is continuous on \([a,b]\text{,}\) differentiable on \((a,b)\) and satisfies
    \begin{align} g(a) \amp =g(b)=f(a)\tag{2.3}\\ g'(x) \amp = f'(x)-\frac{f(b)-f(a)}{(b-a)}\text{.}\tag{2.4} \end{align}
    Applying Rolle‚Äôs theorem to \(g\) on \([a,b]\text{,}\) we conlude that
    \begin{equation*} g'(c)=0 \end{equation*}
    for some \(c\in (a,b)\text{.}\) We conclude, using (2.4) we conclude that
    \begin{equation*} 0=f'(c)-\frac{f(b)-f(a)}{b-a}\text{,} \end{equation*}
    or equivalently
    \begin{equation*} f'(c)=\frac{f(b)-f(a)}{b-a}\text{,} \end{equation*}
    as desired.
    üîó
    üîó

    Corollary 2.2.5. Constant function characterization.

    Assume \(f\) is continuous on an interval \(I\text{.}\) If \(f\) satisfies \(f'(x)=0\) for all interior points \(x\in I\text{,}\) then \(f\) is a constant function on \(I\text{.}\) As a consequence, a function is constant on an interval \(I\) if and only if its derivative is the zero function on \(I\text{.}\)
    üîó
    üîó

    Proof.

    Take any two distinct points \(c,d\in I\text{.}\) We may assume without loss of generality that \(c< d\text{.}\) The assumptions on \(f\) guarantee that it is both continuous (since differentiable) on \([c,d]\) and differentiable on the interior of \((c,d)\text{.}\) The mean value theorem then implies that we have
    \begin{equation*} f'(e)=\frac{f(d)-f(c)}{d-c} \end{equation*}
    for some \(e\in (c,d)\text{.}\) By assumption, however, we have \(f'(e)=0\text{.}\) But then
    \begin{align*} f'(e)=\frac{f(d)-f(c)}{d-c}\amp \implies 0=\frac{f(d)-f(c)}{d-c}\\ \amp\implies f(d)-f(c)=0\\ \amp\implies f(c)=f(d)\text{.} \end{align*}
    We have shown that for any \(c,d\in I\) we have \(f(c)=f(d)\text{.}\) In other words, \(f\) is a constant function on \(I\text{,}\) as desired.
    üîó
    üîó

    Corollary 2.2.6. Functions with identical derivatives.

    Assume \(f\) and \(g\) are continuous on an interval \(I\text{.}\) If \(f'(x)=g'(x)\) for all interior points \(x\in I\text{,}\) then there is a constant \(C\in \R\) satisfying
    \begin{equation*} f(x)=g(x)+C \end{equation*}
    for all \(x\in I\text{.}\) In plain English, if two functions have identical derivative functions over an interval \(I\text{,}\) then they differ by a constant.
    üîó
    üîó

    Proof.

    Let \(h(x)=f(x)-g(x)\text{.}\) Since
    \begin{equation*} h'(x)=f'(x)-g'(x)=0 \end{equation*}
    for all interior points \(x\in I\text{,}\) we conclude by Corollary¬†2.2.5 that \(h\) is a constant function on \(I\text{:}\) i.e., there is a constant \(C\) such that \(h(x)=C\) for all \(x\in I\text{.}\) But then
    \begin{align*} h(x) =C \text{ for all } x\in I\amp \implies f(x)-g(x)=C \text{ for all } x\in I\\ \amp \implies f(x)=g(x)+C \text{ for all } x\in I\text{,} \end{align*}
    as desired.
    üîó
    üîó

    Example 2.2.7. Number of roots.

    Show that the function \(f(x)=x^3-27x-7\) has exactly one root in the interval \(I=[-2,2]\text{.}\)
    üîó
    Solution.
    Since \(f\) is continuous everywhere, and satisfies \(f(-2)> 0\) and \(f(2)< 0\text{,}\) the intermediate value theorem implies there is an element \(c\in [-2,2]\) satisfying \(f(c)=0\text{:}\) i.e., \(f\) has at least one root \(c\in [-2,2]\text{.}\) It remains to show that \(c\) is the only root of \(f\) in \([-2,2]\text{.}\)
    üîó
    Suppose by contradiction that \(f\) had two distinct roots \(c,d\in [-2,2]\text{.}\) Without loss of generality, we can assume \(c< d\text{.}\) Since \(f\) is differentiable everywhere, the mean value theorem would imply that there exists \(e\in (c,d)\) satifying
    \begin{equation*} f'(e)=\frac{f(d)-f(c)}{d-c}=\frac{0-0}{d-c}=0\text{,} \end{equation*}
    where we have used the fact that \(f(c)=f(d)=0\) since \(c\) and \(d\) are roots of \(f\text{.}\) But this is impossible since, as the derivative \(f'(x)=3x^2-27\) satisfies \(f'(x)< 0\) for all \(x\in [-2,2]\text{.}\) (See diagram below.) Since we have reached a contradiction, we conclude that there is only one root of \(f\) lying in \([-2,2]\text{,}\) as desired.

    Example 2.2.9. Interesting inequality.

    Prove that \(\sqrt{1+x}< 1+\frac{1}{2}x\) for all \(x> 0\text{.}\)
    üîó
    Hint.
    Consider \(h(x)=\sqrt{1+x}-(1+\frac{1}{2}x)\text{.}\) We wish to show that \(h(x)< 0\) for all \(x> 0\text{.}\)
    üîó
    üîó
    Solution.
    Let \(h(x)=\sqrt{1+x}-(1+\frac{1}{2}x)\text{.}\) The desired inequality is equivalent to the inequality \(h(x)< 0\) for all \(x> 0\text{,}\) which we now endeavor to demonstrate. First observe that \(h'(x)< 0\) for all \(x> 0\text{.}\) Indeed, we have
    \begin{align*} h'(x) \amp = \frac{1}{2\sqrt{1+x}}-\frac{1}{2}\\ \amp < \frac{1}{2}-\frac{1}{2} \amp (x> 0\implies \sqrt{1+x}> 1)\\ \amp = 0\text{.} \end{align*}
    Next, since \(h\) is continuous on and differentiable on \([0,\infty)\text{,}\) given any \(x> 0\text{,}\) the mean value theorem implies that there exists a \(c\in (0,x)\) satisfying
    \begin{equation*} h'(c)=\frac{h(x)-h(0)}{x-0}=\frac{h(x)-0}{x}=\frac{h(x)}{x}\text{.} \end{equation*}
    This implies
    \begin{equation*} h(x)=xh'(c)< 0\text{,} \end{equation*}
    since \(x> 0\) and \(h'(c)< 0\text{.}\) We have shown that \(h(x)< 0\) for all \(x>0 \text{,}\) and hence that \(\sqrt{1+x}< 1+\frac{1}{2}x\) for all \(x> 0\text{.}\)
    üîó
    üîó
    üîó

    Corollary 2.2.10. Taylor‚Äôs theorem (\(k=1\)).

    Assume \(f\) is twice differentiable on an interval \(I\) containing \(a\) as in interior point. Let \(L(x)=f'(a)(x-a)+f(a)\) be the linearization of \(f\) centered at \(a\text{.}\) For all \(x\in I\) we have
    \begin{equation*} f(x)-L(x)=\frac{f''(c)}{2}(x-a)^2\text{.} \end{equation*}
    As a result, if \(\abs{f''(x)}\leq M\) for some constant \(M\in \R\) and all \(x\in I\text{,}\) then we have
    \begin{equation*} \abs{f(x)-L(x)}\leq \frac{M}{2}\abs{x-a}^2 \end{equation*}
    for all \(x\in I\text{.}\)
